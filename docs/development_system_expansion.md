# 发展系统扩展规划 (v1.2.0)

## 概述
本文档规划了据点发展系统的深度扩展，包括新的发展路径、建筑分支、长期加成机制。

## 新增发展路径

### 1. 军事路径 (Military Path)
**主题**：强化驻军与防御
- **Lv1 - 军营建设**：驻军+2，防御+5%
- **Lv2 - 城防加固**：城防+10，防御+10%
- **Lv3 - 骑兵营地**：可招募骑兵，移动速度+1
- **Lv4 - 要塞化**：城防+20，防御+20%，围攻回合+1
- **Lv5 - 兵工厂**：武器产出+30%，可制造特殊兵种

### 2. 商业路径 (Commerce Path)
**主题**：贸易与资源流通
- **Lv1 - 商业街**：金币产出+20%
- **Lv2 - 市集**：所有资源产出+10%
- **Lv3 - 贸易站**：可与相邻据点交易资源
- **Lv4 - 商业联盟**：金币产出+40%，商业路线+1
- **Lv5 - 金融中心**：金币产出+60%，利息收入+5金/回合

### 3. 文化路径 (Culture Path)
**主题**：文化与民心
- **Lv1 - 文化馆**：民心+10，威望+1/回合
- **Lv2 - 艺术学院**：民心+20，技能点+1/回合
- **Lv3 - 大剧院**：民心+30，相邻据点民心+5
- **Lv4 - 文化遗产**：民心+50，威望+3/回合
- **Lv5 - 文明灯塔**：民心+70，全国威望+2/回合

### 4. 科技路径 (Technology Path)
**主题**：研究与创新
- **Lv1 - 工坊**：建筑建造速度-1回合
- **Lv2 - 研究所**：新建筑解锁，研究速度+50%
- **Lv3 - 大学**：科技点+2/回合，可研究高级技术
- **Lv4 - 科学院**：科技点+4/回合，技术等级+1
- **Lv5 - 知识殿堂**：科技点+6/回合，所有技术效果+20%

### 5. 宗教路径 (Religion Path)
**主题**：信仰与秩序
- **Lv1 - 神殿**：秩序+5，民心+5
- **Lv2 - 修道院**：秩序+10，民心+10，腐败-5
- **Lv3 - 大教堂**：秩序+15，民心+15，相邻据点秩序+3
- **Lv4 - 圣地**：秩序+25，民心+25，腐败-10
- **Lv5 - 信仰中心**：秩序+40，民心+40，全国腐败-5

## 建筑分支系统

### 分支选择机制
玩家在升级到 Lv3 时可选择一个分支方向，之后的升级沿该分支进行。

```
Lv1: 基础建筑
  ↓
Lv2: 进阶建筑
  ↓
Lv3: 分支选择点 (选择 A、B、C 之一)
  ├─ 分支 A (攻势)
  ├─ 分支 B (防守)
  └─ 分支 C (平衡)
  ↓
Lv4-5: 分支特化
```

### 示例：农业建筑分支
- **分支 A - 集约农业**：粮食产出+50%，人口容纳-20%
- **分支 B - 生态农业**：粮食产出+30%，人口容纳+20%，腐败-10%
- **分支 C - 平衡农业**：粮食产出+40%，人口容纳+10%

## 长期加成机制

### 发展积累系统
每个据点有"发展点数"，通过建筑升级、政策激活、事件完成等方式积累。

```gdscript
# 发展点数来源
- 建筑升级：+10 发展点
- 政策激活：+5 发展点/回合
- 事件完成：+15-30 发展点
- 里程碑达成：+50 发展点

# 发展点数用途
- 解锁特殊建筑
- 加速建筑升级
- 获得永久加成
```

### 里程碑系统
据点达到特定条件时解锁里程碑，获得永久加成。

```
里程碑示例：
- "商业繁荣"：金币产出累计 10000 → 金币产出+10% 永久
- "文化中心"：民心平均 > 80 持续 5 回合 → 威望+1/回合 永久
- "科技进步"：研究完成 5 项技术 → 研究速度+20% 永久
- "军事强国"：赢得 10 场战斗 → 驻军防御+15% 永久
```

## 协同加成系统

### 路径协同
多条发展路径同时升级时获得额外加成。

```
协同表：
- 军事 + 商业：驻军维护费-10%
- 商业 + 文化：金币产出+15%
- 文化 + 宗教：民心+10%
- 宗教 + 军事：驻军士气+20%
- 科技 + 所有：所有效果+5%
```

### 邻接加成
相邻据点的发展路径互相加成。

```
邻接加成规则：
- 相邻据点有相同路径：该路径效果+10%
- 相邻据点有互补路径：双方都+5%
- 相邻据点形成"链"：链中每个据点+5%
```

## 平衡考量

- **进度速度**：避免过快达到 Lv5，保持长期目标感
- **路径选择**：每条路径都应有独特优势，避免最优解
- **协同强度**：协同加成不应超过 30%，避免破坏平衡
- **里程碑难度**：里程碑应具有挑战性但可达成

## 相关系统接口

- `ProductionCalculator`：计算发展路径的产出加成
- `GovernanceSystem`：政策与发展路径的互动
- `MoraleCorruptionSystem`：文化/宗教路径与民心的关系
- `BuildingRegistry`：新建筑的注册与效果定义
