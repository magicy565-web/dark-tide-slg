## Event 01 — 无声的第一刀「夜袭」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.14-17 |
| 情感基调 | 无声、精准、令人毛骨悚然的效率 |
| 战斗条件 | 在天亮前（8回合内）找到叶隐并将其逼退 |
| 特殊机制 | 叶隐每回合随机隐身消失，只能通过「落叶痕迹」判断方位；夜间全场视野-2；被叶隐袭击的单位不会死亡，仅昏迷 |
| 敌方配置 | 叶隐(Lv.17) 单独行动 |
| 奖励 | 300 EXP, 600G, 青色落叶×1, 解锁叶隐档案第一页 |

### 场景描写

月沉。

暗潮军团西翼营地陷入最深的黑暗——篝火在半个时辰前被细雨浇灭，值夜的哨兵裹紧油布披风，靠在木栅栏上与困意作斗争。林间的雾气像活物一样在帐篷之间匍匐游走，吞掉了一切声响。

没有风。树梢纹丝不动。

然后，一片叶子落在了哨兵的肩头。

青色的。不是这个季节该有的颜色。

哨兵低头去看——

世界倾斜。地面迎面扑来。他甚至没来得及发出声音就失去了意识。倒地前最后看到的，是一双布鞋无声踏过湿泥，没有留下脚印。

肩上的青叶被夜风卷走，飘入雾中。

### 剧情对话

#### 第一幕：无人发觉

[营地。玩家被副官叫醒。]

**副官：** 大人！出事了——西翼哨卡全部失联！

[玩家赶到西翼。六名哨兵倒在各自的岗位上，姿势各异——有的靠墙坐着，有的趴在桌上，乍看像是睡着了。每个人身上都放着一片青色的叶子。]

**副官：** 全部昏迷。没有外伤。脉搏平稳，像是……被人精准击晕了。

[副官蹲下检查。]

**副官：** 颈后。全部是颈后同一个位置。一击。

[他拿起一片青叶。]

**副官：** 银月议庭的……游侠？不，普通游侠做不到这个。六个哨位分布在三百步范围内，巡逻间隔最短的只有半刻钟。要在不惊动任何人的情况下逐一击倒——

*——她在看着我们。*

[玩家环顾四周。雾气翻涌。林缘的黑暗浓得像墨汁。]

**副官：** 大人，我建议立刻——

[一片青叶飘落，落在玩家脚前。]

[所有人抬头。树冠上空无一物。]

#### 第二幕：追踪

[玩家下令搜索。士兵们举着火把进入林缘。]

**斥候甲：** 没有脚印……这不可能。地面这么湿——

**斥候乙：** 这里！断枝！高度大约——

[断枝在两丈高的位置。]

**斥候乙：** ……她在树上移动。

[远处传来一声极轻的响动——像是布料擦过树皮。所有火把同时转向那个方向。]

[什么都没有。]

[然后，火把最密集处的反方向，又一名士兵无声倒下。肩上，一片青叶。]

**副官：** 她在引导我们的视线……！大人，收缩阵型！不要分散！

### 战斗阶段

[战斗开始。全场笼罩在夜雾之中，视野范围仅2格。叶隐不在场上任何可见位置。]

[第1回合——叶隐从暗处发动袭击。一道黑影掠过，最外围的一名士兵膝盖一软，倒地昏迷。他的身上飘落一片青叶。]

*——目标：外围。最慢的那个。*

[第2回合——玩家调整阵型。叶隐从完全相反的方向出现，击晕另一名落单的单位后瞬间消失。地面上出现一串微弱的「落叶痕迹」，持续1回合后消散。]

**副官：** 循着落叶！她的移动路线——

[第3回合——玩家根据落叶痕迹判断叶隐方位。若判断正确，叶隐短暂现形——一个裹着深灰色布衣的瘦小身影，面孔被兜帽遮得只露出下巴。她停了不到一秒便再次融入黑暗。]

[第4回合——叶隐改变策略，在三个方向同时制造声响（投石、摇枝、落叶），真身从第四个方向突入。]

*——噪音。三个假的，一个真的。他们能分辨吗。*

[第5-6回合——玩家逐渐缩小包围圈。叶隐的行动空间被压缩，但她开始利用玩家自己的士兵作掩护，在人与人之间的缝隙中穿行。]

[第7回合——若玩家成功将叶隐逼至林缘角落，她会短暂停下。月光从云层裂缝中泄下，照亮她的轮廓——比预想中矮小得多，肩膀窄而薄，像是一截会移动的枯枝。]

[第8回合——天际出现最初的灰白。叶隐抬头看了一眼东方。]

*——天亮了。撤。*

[她向后一跃，消失在晨雾中。地面上只留下最后一片青叶，和七名昏迷但毫发无伤的士兵。]

### 战后

[晨光穿透雾气。营地恢复秩序。七名被击晕的士兵陆续醒来，除了颈后有些酸痛外没有任何伤势。]

**副官：** ……一个人。一整夜。击倒了我们七个人。没有杀任何一个。

[他看着手里的青叶。]

**副官：** 大人，银月议庭的暗杀者不会留活口。这不是暗杀。这是……

[他顿了顿，找不到合适的词。]

**副官：** ……警告？

[玩家看向林缘。晨雾中，一棵老橡树的枝头挂着什么东西——走近才看清，是一枚银月议庭的任务令牌，被一片青叶压在树杈上。令牌上刻着一个目标名字。]

[那是玩家的名字。]

**副官：** 她的任务是刺杀您。但她只是……敲晕了几个哨兵然后走了。

*——林缘深处，一双眼睛在枝叶缝隙间注视着营地。注视了很久。然后无声地合上，消失在绿意之中。*

> **获得奖励：** 300 EXP / 600G / 青色落叶×1 / 解锁叶隐档案第一页
>
> *「青色落叶」——不属于任何已知树种的叶片。触感冰凉，脉络清晰如刀刻，经过数日仍不枯萎。据说银月议庭的游侠以此为信物，含义是："我来过。我可以杀你。但我选择不。"*

## Event 02 — 影中之影「三点骚扰」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.16-19 |
| 情感基调 | 焦躁、被玩弄于鼓掌之间 |
| 战斗条件 | 三处据点各坚守5回合不被突破（任一据点失守则失败） |
| 特殊机制 | 三线同时战斗，叶隐在三组之间轮换出现；叶隐专挑SPD最低的单位下手；被叶隐击晕的单位3回合后自动苏醒 |
| 敌方配置 | 叶隐(Lv.19) + 银月游侠×3(Lv.15) + 林地伏击兵×9(Lv.13)，分三组 |
| 奖励 | 450 EXP, 1000G, 夜行者的束带×1, 解锁叶隐档案第二页 |

### 场景描写

入夜。暗潮军团行军纵队穿越苍梧林道——这是通往银月议庭腹地唯一的通路。古木参天，树冠遮蔽了所有星光，火把的光芒被头顶密密层层的枝叶吞噬，只照亮脚下三步的范围。

纵队被迫拉成一条细长的线。前军、中军、后军之间的距离超过了喊话能传达的范围。

副官看着地图皱眉。

林道两侧的灌木丛沙沙作响。可能是风。也可能不是。

### 剧情对话

#### 第一幕：第一声倒地

[行军途中。前军传来骚动。]

**传令兵：** 报告！前军左翼哨兵被击晕——又是颈后同一位置！现场发现青色落叶！

**副官：** 又是她……大人，上次的手法。要加强警戒吗？

[话音未落。]

**传令兵（后方）：** 报告——！后军辎重队遭到袭击！三名护卫昏迷！粮草未失，但——所有绳结被割断，货物散落一地！

**副官：** 前后同时？！不可能——她一个人——

**传令兵（中军）：** 中军也出事了！两名斥候被绊索放倒，绑在树上——还活着，身上贴满了青叶。

[副官愣了三秒。]

**副官：** ……她不是一个人。

#### 第二幕：影子的规律

[玩家紧急部署。三处据点各安排守备力量。]

**副官：** 大人，根据前两次的规律——她每次都优先攻击速度最慢的单位。笨重的步兵、负重辎重兵……

[远处林中传来一声猫头鹰的叫声。然后是两声。然后是三声。]

**副官：** ……那不是猫头鹰。是信号。

[三个方向同时响起树枝断裂的声音。]

*——三点。同时。最慢的先倒。*

### 战斗阶段

[战斗开始。画面分为三个区域：林道前段、中段、后段。每个区域各有一组敌人从林中涌出。]

[前段——银月游侠×1 + 林地伏击兵×3。他们从树冠跃下，专攻重甲单位的关节缝隙。]

[中段——银月游侠×1 + 林地伏击兵×3。利用绊索和烟雾制造混乱。]

[后段——银月游侠×1 + 林地伏击兵×3。切割辎重队绳索，制造障碍物阻挡增援。]

[第1回合——叶隐出现在前段。她从一名重甲兵背后无声滑过，手刀精准落在颈后。重甲兵晃了晃，膝盖一弯，轰然倒地。她已经不在那里了。]

*——重甲。最慢。先清。*

[第2回合——叶隐出现在后段。从前段消失到后段出现，间隔不到半个回合。她从树冠穿行的速度超出了所有人的预判。]

[第3回合——叶隐回到中段。一名试图转身追踪她的士兵被从背后点了穴位，瘫软倒地。身上被放了三片青叶——像是某种黑色幽默。]

*——三片。你转了三次头。每一次都错。*

[第4回合——三处同时压力陡增。银月游侠们配合默契，似乎通过鸟鸣声传递信息。叶隐不再现身，但每个区域都能感觉到她的「存在感」——偶尔飘落的青叶，突然断裂的树枝，士兵们越来越紧绷的神经。]

[第5回合——最后一波攻势。叶隐同时在三处制造「落叶风暴」——大量青叶从树冠飘落，遮蔽视野。在混乱中，她最后击晕了一名试图保护同伴的年轻士兵。]

*——够了。消息已送到。*

[战斗结束。三组敌人如潮水般退入林中，干净利落，不留一人。]

### 战后

[清晨。玩家清点损失。]

**副官：** 十一人昏迷，全部无外伤，预计今日内可恢复。辎重损失为零——她们割断了绳子，但没有拿走任何东西。

[他翻了翻手中厚厚一叠报告。]

**副官：** 大人，这不是战斗。这是……演习。她在测试我们的反应速度和调度能力。

[副官将一片青叶放在桌上。]

**副官：** 而且她发现了我们的弱点——每一个被击晕的都是速度最慢的单位。她在告诉我们：你们的慢兵，在她面前就是靶子。

[桌上还有一样东西——一根细绳，系着三枚松果，编成某种简单的结。]

**副官：** 这挂在中军帐篷入口。不知道什么意思。

[玩家拿起松果绳结。三枚松果，三个方向，三次攻击。但绳结的编法很柔软，不像警告，更像是……]

*——某种她不知道怎么用语言表达的东西。*

> **获得奖励：** 450 EXP / 1000G / 夜行者的束带×1 / 解锁叶隐档案第二页
>
> *「夜行者的束带」——包裹足踝的深灰色布条，纤维中织入了消音的苔藓。穿戴者脚步声降低50%。使用痕迹表明它的主人体型很小，步幅极短，但频率极高。*

## Event 03 — 角色事件·壹「受伤的幼鹿」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情 |
| 推荐等级 | Lv.18-20 |
| 情感基调 | 杀意瓦解、不可遏制的温柔 |
| 战斗条件 | 无 |
| 特殊机制 | 本事件无战斗。全程叶隐独白以[动作描写]和*内心独白*为主，对话不超过5个字 |
| 敌方配置 | 无 |
| 奖励 | 200 EXP, 解锁叶隐档案第三页（含手绘幼鹿图案） |

### 场景描写

深夜。暗潮军团营地后方三里处的溪谷。

月光像碎银一样洒在溪水上，被乱石激成无数跳跃的光点。两岸是密密匝匝的蕨类植物，高及人腰，在夜风中缓缓起伏如同绿色的呼吸。

溪谷深处，一块平坦的青石上，躺着一只幼鹿。

它的后腿被捕兽夹咬住了。血已经干涸成暗红色的硬壳，但伤口仍在渗液。幼鹿的眼睛又大又亮，倒映着月光和漫天的星斗，没有挣扎——要么是放弃了，要么是痛到已经无力挣扎。

树冠上。叶隐趴在一根横枝上，俯视着这一切。

她来这里不是为了幼鹿。她来这里是为了等。溪谷是通往暗潮主帅帐篷的一条隐蔽路线，她已经在这根树枝上趴了四个时辰，等待最佳的突入时机。

但她的视线一直落在那只幼鹿身上。

四个时辰了。

### 剧情对话

#### 第一幕：任务

[叶隐趴在枝上。银月议庭的任务令牌挂在腰间，在月光下发出暗淡的冷光。]

*——目标：暗潮军团指挥官。*

*——路线：溪谷→北坡→营地后方排水沟→主帅帐篷。*

*——窗口：丑时三刻至寅时。守卫换岗间隙，47秒。*

*——足够了。*

[她的手指摸到袖中的短刃。冰冷的金属贴着皮肤，让她清醒。]

*——动手。*

[她没有动。]

*——动手。*

[幼鹿发出一声极低的呜咽。像是喉咙被什么东西堵住了。]

*——不关我的事。*

[月光移动了一寸。幼鹿的眼睛在光线中变换角度，从银白变成琥珀色。它看着树冠的方向——不是看叶隐，是看月亮。好像在等天亮。]

*——…………*

*——它在等天亮。*

*——但它等不到了。失血量。体温。再过两个时辰——*

*——不关我的事。*

#### 第二幕：47秒

[丑时三刻。远处营地传来换岗的脚步声和低语。]

*——窗口开了。47秒。*

[叶隐的身体微微绷紧，准备跃下。]

[幼鹿又叫了一声。比上一次更弱。]

*——35秒。*

[她的脚趾在树枝上扣紧又松开。]

*——28秒。*

[幼鹿的头缓缓垂下，贴在青石上。呼吸变浅了。]

*——19秒。*

*——…………*

*——12秒。*

*——…………不关我的事。*

*——7秒。*

[叶隐从树上跃下。]

[她没有跳向北坡。]

[她跳到了幼鹿身边。]

#### 第三幕：不需要理由

[叶隐蹲在幼鹿面前。幼鹿惊恐地挣动了一下，随即因疼痛而僵住。叶隐伸出手——那只手比幼鹿的蹄子大不了多少——轻轻按在它的脖颈上。]

*——别动。*

[她的手指找到了捕兽夹的机关。锈蚀的铁质，粗制滥造，是猎人随手丢弃的旧物。她用短刃撬开弹簧——金属摩擦的声音在寂静中格外刺耳，她下意识皱了皱眉。]

[捕兽夹弹开。幼鹿的后腿从铁齿中释放，鲜血重新涌出。]

[叶隐从怀中取出一卷绷带——那是为任务准备的急救物资——沉默地清洗伤口，上药，包扎。动作快而精准，像是做过无数次。]

[幼鹿大大的眼睛注视着她。不再恐惧。]

**叶隐：** ……别舔。

[幼鹿歪头。]

**叶隐：** 会感染。

[她站起身。抬头望向北坡方向。营地灯火如旧。]

*——窗口已经过了。*

*——任务失败。*

[她低头看了一眼幼鹿。它正试探性地活动被包扎的后腿，踉跄了一下，没有倒。]

*——…………*

*——算了。*

[她转身走入黑暗。走了三步，停下。回头看了一眼。幼鹿站在月光中，歪歪斜斜的，但站着。]

*——活过今晚就好。*

[她消失在蕨丛中。没有声音。没有痕迹。只有一片青叶留在捕兽夹旁边。]

### 战后

[次日。玩家的斥候在溪谷发现了被撬开的捕兽夹、染血的绷带碎片和一片青叶。]

**副官：** 她昨晚来过。从这条路可以直插我们营地后方——这是条暗杀路线。

[他蹲下检查。]

**副官：** 但她没有执行任务。她停在了这里——为了这个？

[他拿起染血绷带。旁边还有幼鹿蹄印，歪歪扭扭地延伸向林中。]

**副官：** 一只受伤的小鹿。她放弃了暗杀任务……为了救一只小鹿。

[他看向玩家。]

**副官：** 大人，这个人——她到底是什么样的刺客？

## Event 04 — 影之陷阱「森林迷途」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.19-22 |
| 情感基调 | 窒息般的压迫感、看不见的猎人 |
| 战斗条件 | 穿越叶隐的陷阱森林，到达对面出口（10回合限制） |
| 特殊机制 | 每格可能存在陷阱（绊索/落穴/迷烟/倒木）；叶隐不主动攻击，仅操控陷阱；触发陷阱的单位SPD-2持续2回合；找到3个「安全标记」可开辟安全路径 |
| 敌方配置 | 叶隐(Lv.22) 隐藏于场外操控陷阱 + 陷阱×15 |
| 奖励 | 500 EXP, 1200G, 蛛丝陷阱线轴×1, 解锁叶隐档案第四页 |

### 场景描写

苍梧深林。正午。

但在这片林子里，正午和午夜没有区别。树冠在十丈高处交织成密不透光的穹顶，只有偶尔的缝隙漏下一两道光柱，像是巨大暗室中偶然打开的窗。光柱中浮尘飞舞，除此之外，一切都沉在墨绿色的幽暗中。

地面是一层厚厚的落叶——不是青色的，是正常的枯褐色。腐殖质的气味浓郁得近乎甜腻。每一步踩下去都会陷入脚踝深度的松软层，发出湿润的「噗嗤」声。

前方的路看起来完全正常。

没有异样。

这正是最不正常的地方。

### 剧情对话

#### 第一幕：入口

[暗潮军团先遣队站在林缘。玩家审视着前方的密林。]

**斥候：** 大人，这片林子是通往银月议庭东侧据点的必经之路。绕行的话需要多走四天。

**副官：** 但根据情报，叶隐的活动范围就在这片林区。她如果设置了陷阱——

**斥候：** 属下先行探路——

[斥候迈出一步。脚下的落叶发出「咔哒」一声轻响——绊索。]

[一张由蛛丝般细线编织的网从两侧树干弹出，将斥候兜头罩住。网线极细但韧性惊人，斥候越挣扎缠得越紧。]

**斥候：** 唔——！

[副官拔刀割网。花了整整半刻钟才把人弄出来。]

**副官：** ……这只是第一步。

[林中深处，无声无息。但所有人都感觉到了——有人在看着他们。]

#### 第二幕：标记

[前进途中。一名细心的士兵注意到了什么。]

**士兵：** 大人！这棵树——树干上有刻痕。

[一棵老橡树的树皮上，用刀尖刻着一个极小的符号：一片叶子的轮廓。]

**副官：** 她的标记？

[那棵树周围半丈范围内的地面，落叶的纹理微妙地不同——更平整，没有被动过的痕迹。]

**副官：** ……这是安全区。她标记了安全路径。

[他皱眉。]

**副官：** 为什么？她设了满林子的陷阱，又留下安全标记？这是在考验我们，还是……在给我们留活路？

### 战斗阶段

[战斗开始。地图为密林，布满隐藏陷阱。玩家需要从南端到达北端出口。]

[第1回合——前进。第一名士兵踩入落叶堆，触发「落穴陷阱」——地面塌陷，人落入一个半丈深的坑。坑底铺满柔软的苔藓。]

*——不深。不会受伤。只是浪费时间。*

[第2回合——另一名士兵触发「迷烟陷阱」。一团灰绿色的烟雾从脚下升起，气味辛辣但无毒。吸入者视野模糊3回合。]

[第3回合——玩家发现第一个「安全标记」。以此为线索，可推断出一条安全路径的片段。]

[第4回合——有人触发「倒木陷阱」。一根大腿粗的枯木从头顶落下——但砸落的位置精确偏移了两尺，恰好落在人身侧。巨大的声响，零伤害。]

*——吓一跳就好。不需要真的砸到。*

[第5-6回合——陷阱越来越密集，但玩家逐渐掌握规律——所有陷阱都没有致命设计。落穴有缓冲，网罟不勒脖，迷烟无毒，倒木偏移。这是一片「不会杀人的陷阱森林」。]

[第7回合——第二个「安全标记」被发现。结合前两个，安全路径已清晰大半。]

[第8回合——第三个「安全标记」。完整的安全路径开启。]

[第9-10回合——沿安全路径前进。两侧仍有陷阱触发的声响——是给没走对路的人准备的——但安全路径上一片坦途。]

[到达出口时，最后一棵树上挂着一小束野花，用蛛丝陷阱线绑着。]

### 战后

[队伍走出密林。所有人都松了一口气。清点人数——无人受伤，只有几个人身上沾满了苔藓和迷烟的气味。]

**副官：** 十五个陷阱。零伤亡。

[他看着手中的野花束。]

**副官：** 大人，她的陷阱——没有一个是用来杀人的。全部都是拖延、迷惑、恐吓。而且她留了安全标记……

[他顿了顿。]

**副官：** 这到底是防御工事还是……迷宫游乐场？

[出口处的泥地上有一行浅浅的脚印——赤足，很小——延伸向林中深处。脚印旁边用树枝摆了一个箭头，指向正北方。]

**副官：** ……她还给我们指路了。

> **获得奖励：** 500 EXP / 1200G / 蛛丝陷阱线轴×1 / 解锁叶隐档案第四页
>
> *「蛛丝陷阱线轴」——指甲盖大小的木轴上缠着数十丈透明细线，强韧程度堪比钢丝却几乎不可见。线轴上有细密的使用磨痕，说明被反复使用过。仔细观察会发现线的一端系了一个极小的铃铛——不是用来报警的，是用来驱赶误入陷阱的小动物的。*

## Event 05 — 角色事件·贰「被放逐的影子」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情 |
| 推荐等级 | Lv.21-23 |
| 情感基调 | 背叛、被遗弃、无声的崩溃 |
| 战斗条件 | 无 |
| 特殊机制 | 剧情事件。叶隐第二次暗杀失败，被银月议庭除名。全程以叶隐视角叙述 |
| 敌方配置 | 无 |
| 奖励 | 250 EXP, 解锁叶隐档案第五页（含除名令副本） |

### 场景描写

银月议庭。月华殿。

殿内光线幽蓝——穹顶镶嵌的月光石在夜间自行发光，将所有事物都浸在一层冷冽的银辉中。十二根白桦木柱从地面直抵穹顶，柱身上的银箔铭文在光中缓缓流动，像是活着的文字。

议庭圆桌。十二把高背椅，坐了九人。三把空椅积满了灰尘——那三位长老已经在上一次暗潮入侵中阵亡。

桌子正中央是一枚被退回的任务令牌。令牌上刻着的目标名字尚未被血液浸润——这意味着任务未完成。

叶隐跪在圆桌前方三步远的地面上。她的头垂着，兜帽遮住了面容。这是她在议庭中一贯的姿势——跪着，低头，等待指令。像一件等待被使用的工具。

### 剧情对话

#### 第一幕：第二次失败

**长老·苍叶：** 这是第二次了。

[沉默。叶隐没有动。]

**长老·苍叶：** 第一次，你报告说「窗口期已过」。我勉强接受了。但这一次——

[他拿起桌上一份报告，念出。]

**长老·苍叶：** 「渗透成功。距目标帐篷十二步。目视确认目标。」——你自己写的。到了这一步，为什么没有动手？

[叶隐的肩膀微微动了一下。像是要开口。然后没有。]

**长老·银枝：** 回答。

**叶隐：** ……看到了。

**长老·苍叶：** 看到了什么？

**叶隐：** 俘虏。

[长老们交换视线。]

**叶隐：** ……他们在发粮。

*——暗潮的营地里，俘虏没有被关在笼子里。他们在排队。领粥。碗是干净的。有一个士兵把自己的面饼分给了一个小孩。那个小孩笑了。*

*——执行者不应该看这些。*

*——但我看到了。*

**长老·苍叶：** 你是说——因为暗潮军团善待了俘虏，你就无法下手？

[叶隐没有回答。]

**长老·苍叶：** 你是执行者。不是裁判。谁是敌人，由议庭决定。你的工作是——

**叶隐：** 杀。

**长老·苍叶：** ……对。杀。

[沉默拉长。月光石的辉光在叶隐的兜帽边缘勾出一条冰冷的白线。]

#### 第二幕：除名

**长老·银枝：** 表决吧。议题：游侠叶隐，两次任务失败，是否除名。

[议庭九人。逐一表态。]

**长老甲：** 除名。

**长老乙：** 除名。

**长老丙：** ……除名。

[一个接一个。声音在大殿中回响。每一个「除名」都像是一滴水落入深潭。]

**长老·银枝：** 七票赞成，两票弃权。通过。

[他看向叶隐。]

**长老·银枝：** 叶隐。从此刻起，你不再是银月议庭的游侠。交出令牌、武装和一切议庭配发物资。你的名字将从游侠录中消除。

[叶隐跪在地上。一动不动。]

**长老·银枝：** 此外——你不得进入银月议庭辖下任何领地。苍梧林、白桦谷、银月湖——这些地方不再是你的家。

*——家。*

*——那个字。*

*——它们说那个字的时候，声音和说「除名」一样平。*

[叶隐缓缓解下腰间的令牌，放在地上。然后是袖刃。然后是束带。然后是工具包。一件一件，摆成整齐的一排。]

[她站起身。没有抬头。]

**长老·苍叶：** 你没有什么要说的吗？

[叶隐走向大殿门口。]

[她停在门槛上。]

**叶隐：** ……照顾那只鹿。

[然后她走进了夜色里。]

#### 第三幕：玩家视角

[次日。暗潮军团斥候截获了一份银月议庭的内部通报。]

**副官：** 大人，银月议庭发布了除名公告——就是那个叶隐，被开除了。原因是「两次关键任务失败，丧失执行者资格」。

[他翻到第二页。]

**副官：** 有意思的是附注：「叶隐第一次任务失败的时间节点，与我们在溪谷发现被救幼鹿的时间完全吻合。」

[他放下通报。]

**副官：** 她第一次是为了救一只鹿放弃了刺杀您。第二次是因为看到我们善待俘虏而下不了手。然后她被自己人抛弃了。

[窗外，远方苍梧林的轮廓在晨光中显得格外安静。]

**副官：** 大人。她现在——无处可去了。

### 战后

[无战斗，无战后处理。]

[但当天傍晚，营地外围的哨兵报告：在营地以北半里的小丘上，有人看到一个灰色的身影坐在一块石头上，面朝苍梧林的方向。坐了很久。]

[等哨兵走近时，只剩下一片青色落叶。]

*——没有家了。*

*——没有任务了。*

*——那只鹿还好吗。*

> **获得奖励：** 250 EXP / 解锁叶隐档案第五页
>
> *叶隐档案第五页是银月议庭除名令的副本，措辞冰冷而公式化。但在纸张边缘，有人用指甲刮出了一行几乎看不见的痕迹。借光线斜照才能辨认。那是一个词，反复刻了三遍：*
>
> *「为什么。为什么。为什么。」*

## Event 06 — 遭遇战·转折「荒野独行」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.22-25 |
| 情感基调 | 孤独、无方向的暴力、自我放逐 |
| 战斗条件 | 击败所有敌人或坚守8回合直到叶隐主动撤退 |
| 特殊机制 | 叶隐作为第三方势力出现，同时对抗盗匪和玩家；叶隐HP低于30%时触发剧情——她不再躲闪，站在原地喘息；盗匪攻击叶隐时玩家可选择介入 |
| 敌方配置 | 叶隐(Lv.25，装备降级) + 荒野盗匪×6(Lv.20) + 黑爪狼×4(Lv.18) |
| 奖励 | 550 EXP, 1400G, 断裂的短刃×1, 解锁叶隐档案第六页 |

### 场景描写

荒原。银月议庭辖境以南十里，不属于任何势力的灰色地带。

这里没有名字。地图上只标注了一片空白和「未勘测」三个字。地面是龟裂的黄土，偶尔点缀着已经枯死的灌木骨架。天空低压着，铅灰色的云像是要塌下来。风裹着沙粒刮过平坦的大地，发出呜呜的悲鸣。

叶隐站在旷野中。

没有兜帽了——被风吹走了，或者她不在乎了。露出的面容出乎意料地年轻，看起来不到二十岁。短发被风吹得乱七八糟，贴在额头上。眼睛是深灰色的，像这片荒原的天空。

她的装备明显降级了——没有了议庭配发的束带和工具包，只剩下一柄有裂纹的短刃和身上那件磨出了破洞的灰色布衣。脚上没有鞋，赤足踩在龟裂的黄土上。

她面前站着一群盗匪和半驯化的黑爪狼。他们围成半圆，切断了叶隐的退路。

### 剧情对话

#### 第一幕：无处可去

[盗匪头子骑在一匹瘦马上，上下打量叶隐。]

**盗匪头子：** 嘿——又一个从银月那边跑出来的？看这身破布……被赶出来的吧？

[叶隐没有回答。]

**盗匪头子：** 跟我们走吧，小姑娘。荒原上一个人活不久。我们那儿至少有口热饭——当然，你得干活。

[他笑了。那笑容让身后几个匪徒也跟着笑。]

**叶隐：** ……不。

**盗匪头子：** 不？你不太懂现在的处境啊。

[他一挥手，黑爪狼们低吼着逼近。]

**盗匪头子：** 再问一次。跟我们走？

[叶隐握紧了断裂的短刃。]

*——无所谓了。*

*——来吧。*

#### 第二幕：遭遇

[此时——暗潮军团的巡逻队从南面山坡出现。双方对视。]

**副官：** 大人！前方有战斗——是盗匪，还有……

[他拿出望远镜。]

**副官：** 是叶隐！她被盗匪包围了。看起来状态很差——没有装备，孤身一人。

[叶隐也看到了暗潮的旗帜。她的表情没有变化。]

*——暗潮。曾经的目标。*

*——现在无所谓了。谁都无所谓了。*

### 战斗阶段

[战斗开始。三方混战：玩家、叶隐（独立第三方）、盗匪+黑爪狼。]

[第1回合——叶隐先动（SPD最高）。她冲向最近的盗匪，短刃划出一道弧线——但断裂的刀刃只在对方铠甲上留下一道浅痕。]

## Event 07 — 角色事件·叁「第二只小鹿」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情 |
| 推荐等级 | Lv.23-25 |
| 情感基调 | 微小的温暖、差一点的笑容 |
| 战斗条件 | 无 |
| 特殊机制 | 剧情事件。叶隐独自返回溪谷发现幼鹿已痊愈。全程几乎无对白 |
| 敌方配置 | 无 |
| 奖励 | 200 EXP, 解锁叶隐档案第七页（含第二幅手绘） |

### 场景描写

苍梧林外围。溪谷。

和那个夜晚一样的月光，一样的碎银洒在溪水上。蕨类植物比上次更高了，绿意更浓。空气中有初春的湿润气息——泥土在解冻，萌芽在拱出地面，一切都在悄然生长。

叶隐蹲在溪谷边缘的灌木丛后面。

她不该来这里。银月议庭的除名令禁止她进入辖区——苍梧林虽然不算核心领地，但这片溪谷在灰色边界上。如果被游侠巡逻队发现，等待她的将不仅是驱逐。

但她还是来了。

赤足踩在湿润的苔藓上，身上还是那件破了更多洞的灰色布衣。手臂上的伤口潦草地用树叶和藤蔓包扎着，看起来是自己处理的。

她看着那块青石。那块曾经放着一只受伤幼鹿的青石。

青石上什么都没有。

### 剧情对话

#### 第一幕：空石

[叶隐蹲在灌木丛后面，注视着空荡荡的青石。]

*——不在。*

*——死了？*

[她的手指微微收紧，扣住膝盖。]

*——失血过多。感染。野兽。*

*——都有可能。*

*——毕竟只是包扎了一下而已。*

*——那天晚上走的时候就知道。可能救不活。*

*——但还是——*

[溪水潺潺。月光平静地照着一切，不带任何感情。]

[叶隐站起身，准备离开。]

#### 第二幕：蹄声

[她转身的瞬间，身后的蕨丛沙沙作响。]

[叶隐的身体本能地压低，手摸向腰间——那里已经没有武器了。]

[蕨丛分开。]

[一只鹿走了出来。]

[不是幼鹿了。半年的时间让它长大了不少——肩高已经到叶隐的腰部，头上冒出了两个小小的角苞。毛色从幼年的斑点转变为均匀的褐色。]

[它的后腿上有一道浅浅的疤痕。捕兽夹留下的。]

[叶隐僵住了。]

*——活着。*

[鹿歪着头看她。然后，像是认出了什么，它迈着轻快的步子走到叶隐面前，用鼻子拱了拱她的手。湿湿的，温温的。]

*——你怎么——*

*——怎么知道是我。*

*——那天那么黑。你受着伤。应该记不住。*

[鹿又拱了拱她的手。然后低头去闻她的赤足。闻了闻，打了个响鼻。]

*——……嫌我脏？*

[叶隐蹲下来。鹿凑过来，把下巴搁在她的肩膀上。重量比预想中沉。温度比预想中暖。]

*——活着。*

*——真的活着。*

[月光照在叶隐的脸上。她的嘴角动了一下。非常轻微。不是笑——是笑的前兆。是某种肌肉记忆试图启动但已经生疏到几乎忘记的东西。]

[那个表情只持续了不到一秒就消失了。]

[但它确实存在过。]

#### 第三幕：旁观

[远处。暗潮军团的巡逻队恰好经过溪谷上方的山脊。]

**斥候：** 大人——下面！

[玩家循声望去。月光下的溪谷中，一个灰色的小小身影蹲着，一只半大的鹿把头搁在她肩上。画面安静得像一幅画。]

**副官：** 是叶隐。那只鹿——就是她当初救的那只。

[他们看了一会儿。]

**副官：** ……她蹲在那里多久了？

**斥候：** 属下观察了大约一刻钟。她一直那个姿势没动。

[叶隐似乎察觉了什么。她抬起头，望向山脊方向——但在月光和距离的双重遮掩下，她看不清上面的人。]

[鹿被她的动作惊了一下，往后退了两步。叶隐伸出手——]

**叶隐：** ……没事。

[鹿停了停。又凑了回来。]

[叶隐轻轻摸了摸鹿后腿上的疤痕。]

**叶隐：** 好了。

[两个字。是对鹿说的，也像是对自己说的。]

[她站起身。鹿蹭了蹭她的腿。她弯腰，在鹿的额头上极轻极快地碰了一下。像是不好意思做这种事。]

*——别跟来。*

[她转身走入夜色。鹿站在溪谷中，目送着她的背影消失。]

### 战后

[玩家从山脊下来，在青石旁发现了新鲜的痕迹——赤足脚印、鹿蹄印，交叠在一起。]

**副官：** 她没有进入苍梧林深处。只到溪谷就停了。

[青石上放着什么东西。走近看——是一小团鹿毛，被一片青叶包裹着。]

**副官：** 大人，我觉得……她是来看那只鹿的。就只是看看。

## Event 08 — 防御反击「一个人的战争」

### 场景描写

灰境。荒原边缘的无名小村。

村子没有围墙。三十几户低矮的泥砖房围着一口井排成不规则的环形，像是一群蜷缩取暖的小兽。炊烟从草屋顶的缝隙中升起，在灰色的天空下显得格外无力。

村子里住着不到百人——大多是流民、逃兵、不属于任何势力的边缘人。没有驻军，没有民兵，最强壮的青年也只拿得动锄头。

叶隐在村子外围的老树上住了三天了。没有人邀请她，没有人知道她在那里。她每天夜里从树上下来，用赤足在村子周围走一圈——检查脚印、兽迹、蹄印。然后爬回树上。

今天傍晚，她看到了地平线上的烟尘。

马蹄扬起的。很多匹。

*——盗匪。*

*——冲着村子来的。*

[她从树上跃下。拎着一柄不知从哪里捡来的锈迹斑斑的旧短刀，赤足走向村口。]

*——没人认识我。*

*——但他们会死。*

*——这个理由够了。*

#### 第一幕：无人请求

[傍晚。盗匪骑兵在村外半里处停下列阵。村民们惊恐地关紧了门窗——他们见过盗匪。知道接下来会发生什么。]

**村民甲：** 那是……谁？什么时候来的？

**村民乙：** 没见过。一个小姑娘——她疯了吧？拿把破刀就想挡盗匪？

**盗匪首领：** 嚯？有看门狗？

[叶隐不说话。]

**盗匪首领：** 让开，小丫头。我们只要粮食和钱。不伤人——大部分人。

[叶隐不说话。]

**盗匪首领：** 听不懂人话？

**叶隐：** ……走。

**盗匪首领：** 什么？

**叶隐：** 走。

[一个字。两次。她的声音很轻，但完全没有发抖。]

**盗匪首领：** 哈——！弟兄们听见了吗？她让我们走！

[盗匪们大笑。]

**盗匪首领：** 有意思。杀了她。

[与此同时——暗潮军团巡逻队在距村子一里处听到了马蹄声和喧嚣。]

**副官：** 大人，前方有村落遭遇袭击——是荒原盗匪。

[斥候用望远镜观察。]

**斥候：** 大人！村口有一个人独自面对盗匪——是叶隐！

**副官：** 叶隐？她怎么会在这里？这个村子和她没有任何关系——

[望远镜中。叶隐的身影在盗匪群中显得渺小到了荒谬的地步。]

**副官：** 大人，要支援吗？

[叶隐看了她一眼。]

**叶隐：** 关门。

[她转身回到村口。]

**盗匪首领：** 小丫头——！让开——！

*——气脉封穴。议庭的技术。至少这个还没被收走。*

**小女孩：** 姐姐！你受伤了！

[叶隐直起身。看了小女孩一眼。]

**叶隐：** ……没事。

[她转身准备离开。]

**小女孩：** 等一下！

**小女孩：** 给你的！妈妈说帮了忙的人要给吃的！

*——多久没吃过了。*

*——有人给的东西。*

[她接过面饼。]

**叶隐：** ……嗯。

[她把面饼塞进怀里，转身走了。没有回头。但她的手一直按在怀里面饼的位置，走出很远了还在按着，好像怕它消失一样。]

[玩家目送她离开。]

**副官：** 大人，她浑身是伤，饿了不知道多久，拿着一把破刀——为了一群不认识的人，一个人挡在了村口。

苍梧山。无名瀑布。

水声震耳。一道白练从三十丈高的悬崖顶端坠下，砸在底部的乱石上炸成无数飞沫，形成一层永恒的水雾。彩虹在雾气中若隐若现。

### 剧情对话

**斥候：** 洞口内壁刻着叶子的标记。和叶隐在陷阱森林里留下的一模一样。

**副官：** 伤亡？

### 场景描写

傍晚。焦土林地。天空是燃烧般的橘红色。

两棵白桦树苗在夕阳中投下细长的影子——它们又长高了。叶隐的那棵到了膝盖的高度，枝条上的叶片在风中翻出银白的底色，已经有了白桦树的雏形。玩家的那棵也不甘落后，虽然矮一点，但枝干粗壮结实。

焦土林缘。一群人影从南面逼近。火光在他们手中跳动——火把、火油罐、引火物。

盗匪残党。上次被叶隐和玩家联手击退后，他们并没有消失，而是带着更明确的目的回来了——烧毁苍梧林南缘残余林地，清出一片空地建立据点。

叶隐站在两棵树前。

她不知道什么时候来的。灰色披风被风吹得鼓起来。手里攥着一柄新短刀——不是银月议庭的制品，是某个铁匠铺里最便宜的那种。但磨得很锋利。

*——不能烧。*

*——这两棵不能烧。*

### 剧情对话

#### 第一幕：火光

**盗匪残党首领：** 又是你。

[他提着一罐火油，火把的光在他脸上投出摇晃的阴影。]

**盗匪残党首领：** 上次那顿打我记着呢。不过今天我带了二十个人——还有火。你知道火有多快吗？你再怎么打，一罐油泼出去，这片地方就什么都没了。

[叶隐不说话。她的视线在盗匪手中的火把和火油罐之间来回移动。]

**盗匪残党首领：** 让开。我们要的是地，不是你的命。你这种没家的野狗，去哪儿不好？非要守在这种烂地方？

[叶隐的手指收紧了短刀握柄。]

**叶隐：** ……有树。

**盗匪残党首领：** 什么？

**叶隐：** 这里有树。

**盗匪残党首领：** 两棵破树苗？那玩意儿——一把火的事！

[叶隐的眼神变了。之前是平静的灰色。现在——还是灰色，但是暴风雨前的灰色。]

*——不可以。*

#### 第二幕：增援

[暗潮军团巡逻队赶到。]

**副官：** 大人！焦土方向发现火光——盗匪要烧林！叶隐已经在那里了！

[玩家立刻率队赶往。]

[到达时，叶隐已经独自击倒了三名冲在最前面的盗匪——全部颈后一击，昏迷。但剩余的盗匪开始分散行动，企图绕过叶隐从侧翼投掷火油。]

[第1回合——叶隐站在两棵树正前方。四名盗匪冲锋，她在一个呼吸之间完成了一套流水般的动作——侧闪、膝顶、掌击、肘撞——四人全部倒地。昏迷。]

*——如果被点着——*

*——不会的。先灭了火源。*

*——不是我的。是他的。*

*——披风没了。*

*——…………*

*——树还在。*

**叶隐：** ……别动。

**叶隐：** 不杀你。别逼我改主意。

[第7-8回合——首领被制服。剩余盗匪见状溃散。玩家部队追击清剿。]

[战斗结束。]

[夕阳已落。焦土在暮色中变成一片深沉的灰。两棵树苗毫发无伤。]

[叶隐坐在两棵树之间。没有了披风，她抱着膝盖，看起来更小了。背上有几处火油灼伤——不严重，但在微微发红。]

[玩家走过来。]

[叶隐没有抬头。]

**副官：** 她刚才用身体挡住了纵火——为的是您种的那棵。

[他将自己的外套递给叶隐。叶隐抬头看了看外套，又看了看副官。]

[她犹豫了很久。然后接过来，披在肩上。没有说谢谢。但她把外套裹紧了。]

**副官：** 叶隐，你的背——需要处理。

[叶隐摇头。]

**叶隐：** ……树没事。

[她偏了偏头，看了看两棵完好的树苗。]

*——都在。*

*——两棵。都在。*

[她把脸埋进膝盖里。不知道是因为累了还是因为别的什么。]

[很久之后她站起来，把副官的外套叠好放在地上。]

**叶隐：** ……还你。

[她消失在夜色中。赤足踩过的焦土上没有留下任何脚印。]

[第二天，那件外套出现在暗潮军团营地门口，叠得整整齐齐。上面放着一小束刚采的野花。]

| 项目 | 数据 |

### 场景描写

清晨。暗潮军团行军路上。

队伍沿着苍梧林外缘的官道向北移动。晨雾还没散，林缘的轮廓在灰白色的光线中显得模糊而安静。士兵们的铠甲和武器发出有节奏的金属碰撞声，马蹄踩在泥地上的闷响像是大地的脉搏。

副官最先注意到的。

队伍最后面——不是最后一排士兵的位置，而是更后面，大约五十步开外——有一个灰色的小小身影。

她跟着队伍的方向走。速度和队伍完全一致。距离始终维持在五十步，不多不少。

### 剧情对话

#### 第一幕：发现

**副官：** 大人。

[他凑到玩家身边，压低声音。]

**副官：** 后面。五十步。别回头看——不然她可能会跑。

[玩家用余光向后瞥。晨雾中，一个裹着灰色布衣的身影（披风在上次战斗中报废了）赤足走在官道上。短发，瘦削，低着头，像是在数自己的脚步。]

**副官：** 从营地出发就跟着了。我确认了三次——她不是在去某个方向恰好同路。她在跟着我们。

[他想了想。]

**副官：** 准确地说，她在跟着您。

#### 第二幕：五十步

[队伍继续前进。叶隐继续跟着。]

[中途休息时，队伍停下。叶隐也停下。她在五十步外找了棵树，靠着树干坐下来。没有水，没有食物。只是坐着。]

**副官：** 要叫她过来吗？

[玩家思考。]

[一名后卫士兵试图向叶隐走去——叶隐立刻站起来，退了五步。士兵停下，叶隐也停下。士兵往回走，叶隐慢慢回到原来的位置。]

**副官：** ……像一只野猫。靠不近。但也不走。

[午后。队伍再次出发。叶隐再次跟上。]

[傍晚扎营时。叶隐在营地外围五十步处蹲着。]

**副官：** 大人，一整天了。她就这么跟着。不说话。不靠近。不离开。

[他递给后卫士兵一份口粮和一壶水。]

**副官：** 放在她五十步的距离上。我们别看着她。

[士兵照做。将口粮和水壶放在叶隐和营地之间的中点上，然后回来。]

[一刻钟后。副官假装巡查时瞟了一眼——口粮和水壶不在了。叶隐还在原处蹲着。]

#### 第三幕：三十步

[第二天。]

[叶隐和队伍的距离变成了四十步。]

[第三天。三十步。]

[第四天早上——]

**副官：** 大人。

[玩家回头。]

[叶隐站在队伍最后一排士兵身后十步的位置。她低着头，兜帽遮住面容。没有说话。]

[最后一排的士兵们面面相觑。]

**士兵甲：** 她……是加入我们了？

**士兵乙：** 没人通知我啊。

**士兵甲：** 要问问她吗？

**士兵乙：** 你问？我可不敢。上次她一个人打翻了二十个盗匪——

[叶隐在他们身后无声地走着。她好像听到了他们的对话，脚步微微停顿了一下——然后继续走。]

[玩家走到队伍后方。叶隐抬头。那双灰色的眼睛看着玩家——里面有警惕，有犹豫，有某种不知道怎么开口的东西。]

[玩家没有问"你要加入吗"。没有问"你为什么跟着我们"。]

[玩家只是微微点了点头。然后转身继续走。]

[叶隐站在原地。看着玩家的背影。]

*——他没有问。*

*——他没有赶我走。*

*——他只是——点了一下头。*

*——…………*

*——这样就可以了。*

[她迈开脚步。跟上了队伍。]

[距离不再是五十步、三十步或十步。]

[她走在了队伍的最后一排里。士兵们不自觉地给她让出了一个位置。没有人命令他们这么做。]

[叶隐走在那个位置上。赤足踩在别人踩过的路上。]

*——跟着。就好了。*

### 战后

[当天晚上。扎营。]

[叶隐没有帐篷。她在营地边缘找了一棵树，像往常一样准备爬上去。]

[副官走过来，在树下放了一条毯子和一份热汤。]

**副官：** 不用上树了。今晚不冷，但地上比树上暖和。

[叶隐看着毯子和热汤。]

[她没有上树。她把毯子铺在树根旁，坐在上面。端起热汤。]

[第一口。她停了一下。]

*——热的。*

*——有味道。*

*——上次喝到热的东西是什么时候。*

*——不记得了。*

[她低着头，很小口很小口地喝着。热汤的蒸气在她面前升起，模糊了她的轮廓。]

[整个营地都听不到她的声音。但她在那里。]

[这就够了。]

## Event 16 — 征服完成「两棵树」

| 项目 | 数据 |
|------|------|
| 类型 | 结算 |
| 推荐等级 | — |
| 情感基调 | 无言的圆满、不需要语言的存在 |

### 场景描写

一年后。

苍梧林南缘。曾经的焦土林地。

不再是焦土了。

一年的时间，雨水和风将灰烬冲进了泥土深处，变成了养分。野草率先收复了全部领地，将黑灰色的地面变成了深浅不一的绿。蕨类植物从边缘向中心蔓延，灌木丛一簇簇冒出来，偶尔有野花在绿意中探出彩色的头。

而焦土正中央——

两棵白桦树。

不再是树苗了。

叶隐的那棵高过了头顶，银白的树干在阳光下闪着柔和的光，枝条向四面八方伸展，叶片在风中翻转出银绿相间的波浪。它长得快而挺拔，像一个不说话但站得很直的人。

玩家的那棵矮半尺，但树干更粗，树冠更圆。它的枝条向叶隐那棵的方向倾斜——不是因为风，是因为两棵树的树冠已经开始交汇。最长的枝条在空中相触，叶片交叠在一起，分不清是哪棵树的。

两棵树下是一片荫凉。一年前这里什么都没有。现在有了影子。

### 剧情对话

#### 第一幕：归来

[叶隐站在两棵树前。]

[她不再赤足了——穿着一双简单的布鞋，是营地里某个好心的士兵给的。灰色的布衣换成了暗潮军团的便装，但她自己加了一些改动——袖口收紧了（方便行动），腰间多了一条自己编的束带。短发长了一些，但还是乱糟糟的。]

[她站在那里，仰头看着树冠。一年前种下的时候，她得蹲下来才能平视树苗。现在要仰头。]

*——长大了。*

[风吹过。叶片沙沙作响。两棵树的声音混在一起，分不出哪棵是哪棵。]

*——真的长大了。*

#### 第二幕：一起

[玩家从身后走来。]

[叶隐没有回头。但她的身体不再绷紧了——一年前，任何人从身后接近都会让她本能地进入战斗状态。现在，她知道是谁。从脚步声就知道了。]

[玩家站到她旁边。两个人并排看着两棵树。]

[很长的沉默。但不是尴尬的沉默。是两个人在同一个地方看同一样东西时，不需要用语言填补的那种沉默。]

**副官远远站着，没有走近。他对身旁的士兵轻声说：**

**副官：** 别过去。让他们待一会儿。

#### 第三幕：最长的句子

[叶隐看着两棵树交汇的树冠。阳光透过叶片在她脸上投下斑驳的光影。]

[她伸出手，触碰了一下最低的那根枝条——两棵树的枝条在此相交。她的手指从一棵树的叶片滑到另一棵树的叶片，没有间断。]

*——连在一起了。*

[她收回手。低下头。看着自己的脚——穿着布鞋的脚。站在两棵树投下的同一片影子里。]

[然后她开口了。]

[声音很轻。还是一贯的平淡语气。但这一次，她说了比平时多得多的话。]

**叶隐：** 树不需要说话。

[停顿。]

**叶隐：** 只需要有人在旁边。
