## Event 01 — 开战宣言「无声的开局」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.12-15 |
| 情感基调 | 冷静的观察，精密的计算，最后出手的人 |
| 战斗条件 | 击退红叶侧翼骑兵队 |
| 特殊机制 | 红叶骑兵SPD+2；红叶在HP降至60%时主动撤退；侧翼突袭每2回合触发一次 |
| 敌方配置 | 红叶(Lv.15) + 银甲骑兵×6(Lv.12) + 轻骑斥候×4(Lv.10) |
| 奖励 | 250 EXP, 600G, 银甲碎片×1, 解锁红叶档案第一页 |

### 场景描写

天城王朝南境，被称作"金穗原"的万顷粮仓，正值秋收尾声。

田野间金色的麦浪已割去大半，留下一截截齐整的麦茬，在午后的阳光下泛着枯淡的光泽。远处的谷仓群鳞次栉比，每一座都被漆成朱红色，屋顶悬挂着银甲公爵府的旗帜——一枚银色盾牌上嵌着天平的纹章。风从南方吹来，夹杂着成熟谷物与泥土的气息，还有一丝若有若无的铁锈味。

那铁锈味来自北方。

战争已经打了三天了。天城王朝北境的防线在暗潮军团的第一波攻势中摇摇欲坠，各路诸侯纷纷出兵响应王廷的勤王令——唯独南境，唯独金穗原，唯独银甲公爵府，迟迟没有动静。

公爵府书房的窗户大敞着，让秋风长驱直入。满墙的书架上不是兵书，而是账本——按年份、按地区、按品类分门别类，整整齐齐。一张红木长桌上铺着南境地图，但地图上标注的不是兵力部署，而是各条商道的运行状态、关税收入和物资库存。

红叶站在窗前，背对着书房里焦急等待的三名信使。

她穿着一件暗红色的织锦长裙，外罩银灰色的薄甲——那是一种仪式性多于实用性的轻甲，更像是贵妇的装饰而非战士的护具。长发用一根银簪束在脑后，几缕碎发被风吹到脸侧。她端着一盏茶，茶面上的热气在秋风中几乎看不见。

三名信使跪了已经一刻钟了。

### 剧情对话

[红叶慢慢转身，目光从窗外的麦田收回，落在三名信使身上。她的表情平静得像一面未起波澜的湖水。]

**信使甲：** 公爵夫人！北境急报——暗潮军团已突破磐石河防线，苍将军的先锋部队正在灰谷隘口苦战！王廷连下三道勤王令，各路诸侯已——

**红叶：** 我知道。

[她轻轻吹了吹茶面，啜了一口。动作从容，仿佛信使带来的不是战报而是天气预告。]

**红叶：** 磐石河三天前破的。灰谷隘口的战况，我昨天就收到了商队的鸽信。苍将军——哦，是凛将军的副官吧？打得不错，至少比我预期的多撑了十二个时辰。

**信使乙：** 那、那为何公爵府至今——

**红叶：** 至今没有出兵？

[她放下茶盏。瓷器与桌面接触的声音极轻，却让三名信使同时绷紧了身体。]

**红叶：** 你们替我算一笔账。

[她走到地图前，纤细的手指点在北境的位置上。]

**红叶：** 北境到这里，急行军五日。我的骑兵快一些，三日半。但粮草辎重至少要六日。也就是说，如果我现在出兵，前锋抵达时已是第四天，而辎重要到第七天才能跟上。

**信使甲：** 可是王廷的命令——

**红叶：** 王廷的命令是"勤王"。没有指定路线，没有指定时间，也没有指定——

[她微微一笑。]

**红叶：** 从哪个方向。

*——北境已经成了绞肉机。谁先冲进去谁先死。东境的平川公出兵最快，他的步兵现在大概已经在泥地里跋涉了。西境的落叶侯第二个响应，忠心可嘉，但他的补给线拉得太长。至于南境的我……*

*——我为什么要做第三个冲进绞肉机的人？*

[红叶转向信使们。]

**红叶：** 回去告诉王廷。银甲公爵府已接到勤王令，正在集结兵力。预计……

[她看了一眼窗外，像是在计算什么。]

**红叶：** 预计三日后出发。

**信使甲：** 三日——！

**红叶：** 粮草需要筹备。战马需要检视。银甲骑兵的装备需要保养。总不能让我的人穿着生锈的铠甲上战场吧？

[她的语气里带着一丝不容置疑的笑意。]

**红叶：** 我会亲自率军出征。但——以我认为最合适的方式。

[信使们退下后，红叶独自站在地图前。笑意从她脸上褪去，露出底下冷静而锋利的目光。]

*——三天。三天足够我看清局势了。谁在赢，谁在输，谁会是最后站着的那个人。到那时候……*

[她的手指从北境缓缓滑向地图的西侧——暗潮军团侧翼的空档处。]

*——我再决定站在哪一边。*

### 战斗阶段

[三日后。金穗原西北方的丘陵地带。暗潮军团的侧翼补给线正从一片低矮的枫树林间穿过。]

[红叶终于出兵了。但她没有去北境增援，而是绕了一个大弧线，直插暗潮军团侧翼——那里是补给线最薄弱的位置。]

**红叶：** 全军听令。

[她骑在一匹毛色如暗银的战马上，银甲在斑驳的树影间时隐时现。身后是三百银甲骑兵——数量不多，但装备精良，每一匹战马都膘肥体壮，每一副铠甲都打磨得能映出人影。]

**红叶：** 目标是侧翼辎重队，不是主力。记住我的规矩——

[她举起右手，竖起三根手指。]

**红叶：** 第一，不恋战。冲散敌阵后立刻脱离。第二，不追击。逃跑的敌人不值一枚铜币。第三——

[她放下手，目光冷冽。]

**红叶：** 不许死。死了的士兵不能替我打第二仗。

**银甲骑兵队长：** 遵命！

[战斗开始。银甲骑兵从枫树林中冲出，如同一柄银色的匕首刺入暗潮侧翼。]

*——战争和做生意一样。不是看你能赚多少，而是看你能亏多少。*

[第1回合：红叶骑兵SPD+2，从侧翼发动突袭，暗潮辎重队阵型大乱。]

**红叶：** 分三队，交替冲击。像割麦子一样——一排一排来。

[红叶本人在队列后方指挥，从不亲自冲锋。她的目光不断在战场上扫视，像一个商人在清点货架上的存货。]

[第2回合：暗潮护卫队开始组织反击，重甲步兵结阵。]

**红叶：** 不要硬碰硬。左翼绕过去，攻击他们的弓手。

[第3回合：暗潮侧翼增援赶到。战场上敌军数量开始增加。]

**银甲骑兵队长：** 夫人！敌军援兵来了，至少两个中队——

**红叶：** 我看到了。

[她低头看了一眼手腕上缠着的一串算珠——那是她独特的习惯，别人用念珠祈祷，她用算珠计算。]

*——我们已经摧毁了三辆粮车，击散了两队弓手。伤亡……四人轻伤，无人阵亡。够了。*

**红叶：** 收兵。

**银甲骑兵队长：** 夫人？我们正占优势——

**红叶：** 优势再过两分钟就没有了。他们的重甲已经完成结阵，弓手正在重新列队。我们继续冲下去，最好的结果是惨胜。

[她调转马头。]

**红叶：** 我不做惨胜的生意。

[银甲骑兵迅速脱离战场，消失在枫树林的深处。整个袭击从开始到结束，不到一刻钟。]

[第4回合：红叶主动撤退。暗潮追兵在枫林边缘停下——林中已布满了绊马索和陷阱。]

**暗潮队长：** 不追了！林子里有埋伏！

[远处，红叶回头望了一眼。]

*——没有埋伏。只有几根绳子和一些木桩。但他们不知道啊。*

*——战争中最廉价的武器，是恐惧。*

### 战后

[枫树林深处的临时营地。红叶下马，接过侍从递来的湿巾，仔细擦拭手上并不存在的血迹。]

**银甲骑兵队长：** 夫人，此战斩获敌军辎重车三辆，击溃护卫队约八十人，我方仅四人轻伤。大捷！

**红叶：** 不算大捷。

[她解下手腕上的算珠，一颗一颗地拨着。]

**红叶：** 三辆辎重车的物资，按市价折算约八百金。扣除我们的马匹饲料、装备损耗和行军口粮……净利润大约五百金。

**银甲骑兵队长：** ……夫人，这是战争，不是做买卖。

**红叶：** 有区别吗？

[她将算珠重新缠回手腕，目光投向北方。]

*——第一仗，试探完毕。暗潮的侧翼防御松散，指挥体系反应迟缓，但兵员数量确实庞大。正面硬碰硬是死路一条，但如果从缝隙中找利润……*

[她从怀中取出一本小巧的账本，在上面写了几行字。]

*——"暗潮军团·侧翼评估：防御力D，反应速度C，兵力A。结论：可骚扰，不可决战。"*

*——与此同时，那支暗潮先锋军的指挥官……*

[她的笔尖停顿了一下。]

*——倒是个有趣的对手。*

[红叶合上账本，抬头望向北方天际隐约可见的烽烟。]

**红叶：** 传令下去。明日转移营地，向西北方移动二十里。

**银甲骑兵队长：** 西北？那不是更靠近暗潮主力——

**红叶：** 靠近，但不接触。就像……

[她想了想，嘴角微微上扬。]

**红叶：** 就像在对方的钱袋旁边放一把刀。不一定要割开它，但要让他知道刀在那里。

[她翻身上马，暗银色的战马在夕阳下打了一个响鼻。]

**红叶：** 走了。天黑之前要扎好营。今晚的账还没算完呢。

[银甲骑兵无声地列队，消失在金穗原起伏的丘陵之间。战场上只留下几辆翻倒的粮车和一地散落的谷物。风吹过，麦穗沙沙作响，像是金币碰撞的声音。]

> **获得奖励：** 250 EXP / 600G / 银甲碎片×1 / 解锁红叶档案第一页
>
> *「银甲碎片」——从红叶骑兵队遗落的护甲上取下的一枚碎片。银质，打磨精细，内侧刻着一个极小的天平纹样。翻过来看，背面有用针尖划出的细小文字："收支平衡。"——据说这是银甲骑兵团的座右铭。*

## Event 02 — 前哨遭遇「银甲的邀约」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.15-18 |
| 情感基调 | 猫鼠游戏，优雅而危险的试探 |
| 战斗条件 | 坚持6回合，抵御红叶骑兵的游击骚扰 |
| 特殊机制 | 红叶骑兵每2回合从随机方向出现（东/西/南三个方向）；击退一波后2回合刷新；红叶本人不参战，仅在第5回合现身观战 |
| 敌方配置 | 银甲骑兵×4(Lv.15)每波 + 轻骑斥候×2(Lv.13)每波，共3波 |
| 奖励 | 350 EXP, 800G, 红叶的亲笔信×1, 解锁红叶档案第二页 |

### 场景描写

暗潮军团的前哨营地设在金穗原北缘一处名叫"枯枫岭"的低矮丘陵上。

这里曾是南境最大的枫树林之一，但三年前的一场山火烧去了大半。如今残存的枫树东一棵西一棵地立着，光秃秃的枝丫像伸向天空的焦黑手指。烧焦的树桩间长出了半人高的野草，在夜风中沙沙作响，让人分不清那是草叶摩擦还是马蹄踏在落叶上的声音。

月亮从云层后面探出半张脸。清冷的月光洒在枯枫岭上，把每一棵残树都变成了扭曲的影子。

暗潮前哨营地的篝火刚刚点燃。几名斥候正在巡逻，火光在他们的铠甲上跳动，映出疲惫而警觉的面孔。今天已是进入南境的第三天，除了第一天遭到那次短暂而凌厉的侧翼突袭外，再没有遇到任何抵抗。

太安静了。

### 剧情对话

[前哨营地中。暗潮指挥官正在检查地图，一名斥候匆匆跑来。]

**斥候：** 报告！西南方向发现骑兵活动痕迹，至少两个小队——但没有看到人影，只有马蹄印和……这个。

[斥候递上一样东西。是一枝刚折下不久的枫树枝，枝头系着一条银色丝带。]

**暗潮指挥官：** ……什么意思？

**斥候：** 不知道。每隔三里就有一枝，一路排列得整整齐齐，像是……像是路标。

*——路标？引向哪里？是陷阱，还是……邀请？*

[暗潮指挥官正犹豫间，营地外围突然传来马蹄声——不是奔袭的雷鸣，而是小跑的清脆节奏，像是有人在散步。]

**哨兵：** 来者何人！

[月光下，一匹纯白色的马从枯树间缓缓走出。马背上的骑手穿着轻甲，手持一面小旗——不是战旗，而是通商用的白旗，上面绣着银色的天平纹样。]

**信使：** 银甲公爵府使者，奉夫人之命递送书函一封！

[信使将一只密封的竹筒高高举起，然后放在地上，调转马头，从容离去。始终未进入弓箭射程之内。]

[暗潮指挥官打开竹筒，取出一张折叠整齐的信笺。纸张是南境特产的金穗笺，带着淡淡的墨香和一丝茶的气息。字迹娟秀而有力。]

**信中内容：**

*"致暗潮军前锋指挥官阁下：*

*连日行军辛苦。南境秋夜寒凉，不知贵军将士可有足够的冬衣和热食？我听说北方人不习惯南方的湿冷——关节疼是小事，士气低落才是大事。*

*金穗原的枫叶虽已烧去大半，但南山坡还有一片幸存的老林，这个季节正是最好看的时候。月下赏枫，配一壶热茶，是南境人秋天最大的享受。*

*如果贵军指挥官有兴趣，三日后月圆之夜，旧枫林中的废弃茶寮恭候大驾。只需带一名随从。我也一样。*

*这不是陷阱。如果是，我不会提前三天告诉你。*

*当然，如果你们更喜欢在这片焦土上反复兜圈子，我也不介意。我的骑兵正好需要练习夜间行军。*

*——红叶"*

[暗潮指挥官读完，沉默良久。]

**暗潮副官：** 这是什么意思？劝降？还是诱敌？

**暗潮指挥官：** ……都不像。倒更像是——

[话未说完，营地西侧突然响起急促的号角声。]

**哨兵：** 敌袭——！西侧发现骑兵！

### 战斗阶段

[银甲骑兵从西侧枯树林中冲出，如同银色的狼群。月光下，他们的铠甲反射着冷冽的光芒。]

**暗潮指挥官：** 列阵！弓手就位！

[第1回合：第一波银甲骑兵×4从西侧突袭，冲击前哨营地的防御工事。轻骑斥候×2在外围游弋，切断通讯线路。]

*——来了。不是正面强攻，是骚扰。和上次一样的套路——不，方向变了。*

[银甲骑兵的战术极为灵活。他们不与防御阵线硬碰，而是沿着营地外围高速移动，用弓骑射击后迅速脱离。]

**暗潮副官：** 他们不接近！一直在外面绕圈——

**暗潮指挥官：** 稳住。他们想拖乱我们的阵型。所有人坚守位置，不许追击！

[第2回合：银甲骑兵撤退，消失在夜色中。短暂的安静。]

**暗潮副官：** 走了？

**暗潮指挥官：** 没有。听——

[第3回合：第二波银甲骑兵从东侧杀出！完全相反的方向。]

**哨兵：** 东侧！东侧也有——！

[营地陷入混乱。士兵们刚把盾墙转向西面，东面又遭到冲击。]

**暗潮指挥官：** 分兵防守！东西两面各留一队——

*——不对。如果我分兵，她就等着各个击破。如果我不分兵，她就永远从我的侧面咬。这是——*

*——这是一道选择题。而两个选项都是错的。*

[第4回合：东侧骑兵撤退。再次沉寂。但这次沉寂的时间更短——]

[第5回合：第三波从南侧出现。这一次，队列后方多了一个身影。]

[月光下，一匹暗银色的战马悠然立在百步之外。马背上的女人没有穿重甲，而是披着一件银灰色的斗篷，长发在夜风中微微飘动。她手里端着——]

**暗潮副官：** ……她在喝茶？！

[红叶确实端着一盏茶。在战场上。在月光下。在两军对峙之间。]

**红叶：** ——

[她没有说话。只是远远地举起茶盏，朝着暗潮营地的方向微微致意，然后轻轻啜了一口。]

[她的身后，银甲骑兵继续进行最后一轮骚扰射击。]

*——这个女人……*

[第6回合：银甲骑兵全数撤退。红叶最后看了暗潮营地一眼，调转马头，消失在月色中。]

[战场上恢复了安静。只有几支射在地上的箭矢还在微微颤动。]

### 战后

[天亮后，暗潮士兵在清理战场时发现了一样东西。]

[在营地南侧，银甲骑兵最后出现的位置，地上插着一支箭。箭头没有开刃——这不是用来杀人的箭。箭杆上系着一张小纸条。]

**纸条内容：**

*"六个回合。你们的指挥官判断力不错，至少没有追出来。三日后的邀请依然有效。*

*另外，你们东侧哨位的换岗时间太规律了，每两个时辰一次。建议打乱一下。免费的建议，不收钱。*

*——红叶*

*追伸：那个信使说你们的指挥官读信时的表情很有趣。我更期待喝茶时的表情了。"*

[暗潮指挥官读完纸条，久久没有说话。]

**暗潮副官：** ……这到底是在打仗，还是在——

**暗潮指挥官：** 她在告诉我，她随时能打进来，但她选择不打。

*——这比直接攻击更让人不安。*

[暗潮指挥官将纸条折好，放入怀中。他抬头望向南方——金穗原在晨曦中铺展开来，金色的麦茬在朝阳下闪烁。某个看不见的地方，那个喝着茶的女人正在看着他。]

**暗潮指挥官：** ……加强巡逻。所有换岗时间随机化。

[他停顿了一下。]

**暗潮指挥官：** 另外，三日后……我亲自去。

**暗潮副官：** 大人！那很可能是陷阱——

**暗潮指挥官：** 也许吧。但一个会在战场上喝茶的敌人，值得我走一趟。

[他最后看了一眼那张纸条上的字迹，目光在"免费的建议"四个字上停留了一瞬。]

*——免费。她特意强调了"免费"。也就是说，在她眼里，一切都有标价。包括这场战争。*

*——有趣。*

## Event 03 — 角色事件·壹「月下茶会」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情 |
| 触发条件 | 威望>=15 |
| 情感基调 | 商人的逻辑，利益至上的世界观，冰面下的暗流 |
| 玩家抉择 | 接受贸易协定（获取资源但泄露情报） vs 拒绝（获得谈判筹码） |
| 奖励 | 200 EXP, CG「月下茶会」, 解锁红叶档案第三页 |

### 场景描写

南山坡。幸存的枫树林。

月圆之夜，整片枫林被银色的月光浸透，像是沉在一缸清水底部。深秋的枫叶已变成暗红色，在月光下呈现出一种近乎黑色的浓烈——只有风吹过时，叶片翻转，露出背面银灰色的绒毛，才闪过一线亮光，像是树上挂满了无数只在黑暗中眨眼的瞳孔。

林间有一座废弃的茶寮。木质结构已经朽了大半，屋顶塌了一角，藤蔓从破洞中垂下来，在月光里摇曳。但地面被人仔细打扫过，铺着干净的竹席。一张矮桌摆在茶寮正中——不，不是普通的矮桌——那是一张红木嵌螺钿的茶案，做工极其精美，和这座破败的茶寮形成荒诞的对比。

茶案上摆着两套茶具。白瓷，薄如蝉翼。旁边一只红泥小炉上，铁壶正在咕嘟冒着热气。

红叶已经在了。

她换下了银甲，穿着一件暗红色的绸质长衫，领口和袖口绣着极细的银线暗纹。长发散下来，只在耳后别了一朵干燥的白梅花。手腕上那串算珠还在，但在月光下更像是一件装饰品。她盘腿坐在竹席上，正用竹夹拨弄炉中的炭火。动作不急不缓，像是已经等了很久，又像是根本不在意对方来不来。

### 剧情对话

[脚步声从林间传来。枯叶在脚下发出细碎的声响。红叶没有抬头。]

**红叶：** 你来了。我还以为你会在林子外面犹豫至少半个时辰。

[她终于抬起目光。月光落在她的脸上，照出一双极为明亮的眼睛——不是那种热烈的光，而是冷的，像是两枚刚铸好的银币。]

**红叶：** 请坐。水刚好开。

[她开始泡茶。动作行云流水——提壶、润杯、注水、刮沫、分茶——每一步都精准得像是在执行一套演练过千百遍的仪式。]

**红叶：** 铁观音。南境特产，金穗原南山坡自家庄园出品。和北方那些浓烈的砖茶不一样，我们喜欢香气清远、回甘绵长的。

[她将一盏茶推到对面。]

**红叶：** 尝尝？

[等对方入座，品了一口茶后。]

**红叶：** 味道如何？

[不等回答，她自顾自地继续。]

**红叶：** 这茶有个有趣的地方。第一泡是香的，第二泡是甜的，第三泡是苦的。大多数人喝到第二泡就停了——甜的谁不喜欢？但真正懂茶的人知道，第三泡的苦味才是这款茶真正的价值。因为苦过之后，第四泡、第五泡……会有一种从舌根蔓延开的回甘，那是前面任何一泡都比不了的。

[她端起自己的茶盏，浅浅啜了一口。]

**红叶：** 你觉得我为什么要说这些？

*——因为我接下来要跟你谈的事，就像这壶茶。第一口是香的，第二口是甜的——但你得有耐心喝到第三口。*

[她放下茶盏，目光变得认真起来。笑意还挂在嘴角，但眼底的冷光更清晰了。]

**红叶：** 好了，闲话说完了。我想你也不是来喝茶的。

[她从袖中取出一卷绢帛，展开在茶案上。那是一张手绘的商路图——标注极其详尽，每一条路线、每一个驿站、每一处关卡的通行费用都写得清清楚楚。]

**红叶：** 这是天城王朝南境到东境的全部商路。我经营了十二年，每一条路上的石头我都认得出来。暗潮军团现在最大的问题是什么？不是兵力，不是士气——是补给线太长。你们的粮草从北方运来，过磐石河、穿灰谷、绕桐岭……到前线已经损耗了三成。

[她用手指在地图上画了一条线。]

**红叶：** 但如果走南境的商路……损耗不到一成。而且沿途有现成的仓储设施——我的仓储设施。

[她抬起眼。]

**红叶：** 我不关心正义和邪恶。我不关心谁坐在王座上。我关心的是——如果天城王朝倒了，谁来保证南境商路的安全？那些商路是我半辈子的心血。

**红叶：** 所以，我有一个提议。

[她从绢帛下面又抽出一张纸，上面密密麻麻写满了条款。]

**红叶：** 贸易协定。暗潮军团承认南境商路的独立运营权，不征收额外战争税，不征用民间商船。作为交换，我向你们提供南境的粮草补给和部分情报——仅限战术层面的情报，不涉及我自己的兵力部署。

[她将纸推到对面。]

**红叶：** 你可以理解为……我在买保险。不管最后谁赢，我的商路要活下来。

[长久的沉默。风穿过枫林，带落几片暗红色的叶子，旋转着落在茶案上。]

**红叶：** 你在犹豫。

[她端起茶壶，给双方都续了一杯——第二泡。]

**红叶：** 让我猜猜你在想什么。你在想："接受了，是不是就被她牵着走了？她给的情报是真是假？补给里会不会下毒？"

[她笑了。]

**红叶：** 很好。会这样想说明你不笨。

**红叶：** 但你忽略了一件事。我给你情报，同时也在向你暴露我自己。你现在知道了南境商路的走向、我的仓储位置、我的物资储备——这些信息的价值远高于我能从你那里得到的任何东西。

[她倾身向前，压低声音。]

**红叶：** 我在赌。赌你值得投资。

*——这才是真话。不是买保险，是风险投资。她在所有势力之间布网，谁最后赢了，她就和谁做生意。我只是她的选项之一。*

*——但知道自己是"选项之一"……反而没那么难接受了。至少她是诚实的。*

#### 玩家抉择

**【选项A】接受贸易协定**

**红叶：** ……成交。

[她从袖中取出一枚小巧的银色印章——天平纹样——在协定上盖了一下。]

**红叶：** 这份协定只有你我知道。如果泄露出去，对你我都没好处。

[她将协定折好，一人一份。]

**红叶：** 第一批物资三日后到。会伪装成普通的行商车队。接头暗号是——

[她想了想。]

**红叶：** "今年的枫叶红得早。"回复："是因为雨水多。"

[她站起身，拍了拍裙上的灰。月光下，她的身影修长而优雅。]

**红叶：** 合作愉快。希望这是我们之间许多笔交易中的第一笔。

*（获得：南境粮草补给×3，南境商路情报地图×1。代价：暗潮部分行军路线信息暴露给红叶。）*

---

**【选项B】拒绝协定**

[对方摇头。红叶的动作停顿了一瞬——极短暂的，几乎看不出来。但她的手指在茶盏上多停留了一秒。]

**红叶：** ……哦？

[她将协定收回，重新卷好，塞回袖中。动作依然从容，但速度比刚才快了一拍。]

**红叶：** 理由？

[听完对方的回答后。]

**红叶：** 有意思。

[她沉默了一会儿，开始倒第三泡茶。苦的那泡。]

**红叶：** 你知道吗，拒绝我的人不多。不是因为我的条件多好——而是因为大多数人没有底气拒绝"免费"的好处。

[她端起茶盏，一饮而尽。]

**红叶：** 但你拒绝了。这意味着两件事。第一，你不缺物资，至少短期内不缺。第二……你认为和我做交易的代价比它表面上看起来的要高。

[她站起身。]

**红叶：** 两个判断都是对的。

[她微微一笑——这次的笑和之前不同，带着一丝真实的兴味。]

**红叶：** 你比我以为的更难对付。很好。那下次我就得开出更好的条件了。

*（获得：红叶好感度+5。未来谈判中红叶会给出更优厚的条件。）*

---

### 尾声

[无论选择哪个选项。红叶走到茶寮门口，停下脚步，回头。]

**红叶：** 最后说一件不算秘密的秘密。

[月光从破损的屋顶倾泻而下，照亮她的半张脸。]

**红叶：** 天城王朝的宰相大人……最近对南境越来越不满了。说我"拥兵自重"、"首鼠两端"。呵。

[她的笑容带了一丝寒意。]

**红叶：** 我经营南境十二年，缴纳的税金占王廷岁入的四成。没有我的商路，王廷连士兵的军饷都发不出来。但他们只记得我"不够忠诚"。

**红叶：** 忠诚。

[她念这个词的方式，像是在品尝一颗酸涩的果实。]

**红叶：** 在王廷眼里，忠诚的定义是"不思考，服从命令"。抱歉，我做不到。我的每一个决定都要经过计算。包括——

[她的目光落在对方身上。]

**红叶：** 包括今晚来赴这个约。

[她转身走入枫林。暗红色的衣衫在银白色的月光中渐渐远去，像一片逆风飘飞的落叶。]

**红叶：** （声音从远处传来）茶具你带走吧。算是见面礼。白瓷的，南境官窑出品。市价……大概够你半个月的军粮。

[脚步声消失在林间。竹席上只留下两盏未凉的茶、一只红泥小炉，和满地旋落的暗红色枫叶。]

*——她走了。但那双像银币一样冷的眼睛里，最后一瞬好像闪过了什么别的东西。*

*——不确定。也许只是月光的错觉。*

> **获得奖励：** 200 EXP / CG「月下茶会」/ 解锁红叶档案第三页
>
> *CG描述「月下茶会」——废弃茶寮内，月光从破损的屋顶倾泻而下。红叶一袭暗红长衫端坐于红木茶案前，手指轻扣瓷盏边缘，薄唇微启似在说些什么。茶烟袅袅上升，与月光交融，模糊了她脸上那副永远恰到好处的微笑。身后枫林如墨，唯有几片叶子被风吹入画面，在银色光柱中旋转坠落。*

## Event 04 — 防御反击·一「金币铸就的防线」

| 项目 | 数据 |

### 场景描写

金穗原南端，澄河渡口。

这里是南境通往外界最重要的水运枢纽——三条河流在此汇聚，形成一片开阔的冲积平原。渡口码头上林立着大大小小的货栈、船坞和仓库，桅杆如林，曾经日夜不停地装卸着丝绸、茶叶、瓷器和粮食。

但现在，渡口变成了一座军事要塞。

红叶用金币——确切地说，用大量的金币——在短短七天内将这里改造成了一道看上去固若金汤的防线。木质栅栏被替换成石墙，码头上的吊臂被改装成投石机，仓库变成了兵营。三道防线层层叠叠地横亘在渡口前方，每一道都驻扎着不同旗号的佣兵团。

第一道防线：铁犁团。穿着杂色铠甲的近战佣兵，大多是北境流落的老兵，体格壮硕但眼神涣散。他们的盾牌上漆着铁犁的纹样，但油漆是新的——显然刚刚换了雇主。

第二道防线：穿云社。弓手为主的佣兵团，据说来自东境山区。他们蹲在石墙后面，弓弦上搭着箭，面无表情。

第三道防线：杂牌混合部队。看得出是临时拼凑的——步兵、弓手、甚至还有几个骑马的斥候，旗号各异，纪律松散。

三道防线的最后方，一座临时搭建的木质高台上，红叶端坐于太师椅中。她换回了银灰色的薄甲，但依然像参加宴会多过上战场。一名侍从替她撑着伞遮阳，另一名侍从端着茶盘。她的膝上摊开着一本厚厚的账本。

她在算账。

### 剧情对话

**暗潮副官：** 她在七天里把渡口变成了这个样子？

**暗潮指挥官：** 不是她变的。是钱变的。

[远处，红叶的高台上似乎有什么动静。一名骑手从防线后方奔出，径直冲向暗潮阵线，在弓箭射程外勒马。]

**骑手：** 银甲公爵夫人有话转达！

**暗潮指挥官：** 说。

[骑手调转马头，留下一句。]

**骑手：** 夫人说，茶叶是准备好了的。

[暗潮军团发起进攻。]

**侍从：** 是，夫人。

**侍从：** 夫人，要不要再加钱——

*——八万金买来的不是防御，是时间。*

**斥候：** 报告！在指挥高台上发现一件东西——

## Event 05 — 角色事件·贰「清洗之夜」

| 项目 | 数据 |

### 场景描写

夜。天城王朝南境首府，银甲城。

大雨倾盆。

银甲城是南境最繁华的城市——或者说，曾经是。它因红叶而繁华，因商路而富庶。城中最高的建筑不是官衙也不是神殿，而是银甲商会总部：一座五层的石制楼阁，每一层的窗户都镶着彩色玻璃，屋顶的风向标是一座纯银打造的天平。

此刻，那座天平正在大雨中剧烈摇晃。

城中四处燃起了火光——不是因为雷电，而是因为人。数百名穿着王廷禁军黑甲的士兵正在城中各处同时行动：封锁商会总部、查抄沿街的仓库、逮捕商队的掌柜和管事。他们的动作整齐划一，显然经过了周密的计划。

商会总部的大门被撞开。禁军涌入，将里面的人一个个押了出来——账房先生、船运管事、仓储总管、各条商路的负责人——他们被推搡着跪在雨中，有的在求饶，有的在哭喊，有的已经吓得说不出话。

一名身穿墨绿官袍的中年男人站在商会门前的台阶上，手持一卷盖着鲜红大印的诏书。他的声音透过雨幕，冰冷而清晰。

### 剧情对话

**宰相府密使：** 奉宰相大人令——银甲公爵夫人红叶，里通外敌，私通暗潮叛军，证据确凿！即日起，查封银甲商会一切产业，冻结南境所有商路经营权，逮捕红叶及其一切同党！

[跪在雨中的人群炸开了锅。]

**账房先生：** 大人饶命啊！小人只是管账的——

**仓储总管：** 冤枉！夫人何时通过敌——

**禁军队长：** 闭嘴！

[一脚将仓储总管踹倒在泥水里。]

**禁军队长：** 搜！每一间屋子，每一个柜子，每一块地砖都给我翻开来！特别是地下密室——情报说商会底下有三层暗库！

[镜头切换。]

[银甲城西北角，一条隐蔽的水道入口。]

[暴雨将水道的水位抬高了近两尺，浑浊的水流几乎没过膝盖。一个身影在齐腰深的水中艰难前行——红叶。]

[她已经不是那个在高台上悠然喝茶的公爵夫人了。暗红色的衣衫被雨水和泥浆浸透，贴在身上。长发散落，湿漉漉地垂在脸侧。银灰色薄甲的肩部被什么东西划破了一道口子，露出里面渗血的皮肤。她的右手死死攥着那本随身账本——唯一带出来的东西。]

[她身后跟着两个人：一个是她的贴身侍女阿桐，另一个是商会的老门房周叔。三个人。从整个银甲商会的数百号人里，只逃出来三个人。]

**阿桐：** 夫人，水太深了——您的伤——

**红叶：** 别停。

[她的声音沙哑，但语气依然平稳。]

**红叶：** 水道出口在前方三百步。出去之后往南走，穿过柳林镇，到码头找老陈头——他的船在第七号泊位。

**周叔：** 夫人……那些被抓的人——

**红叶：** 我知道。

[她没有回头。]

*——黄掌柜，跟了我十一年。船运的老郑，八年。仓储的小方，五年，是我亲手从一个街边小贩培养出来的。还有账房的刘先生，他的算盘打得比我还快……*

*——他们都被抓了。因为我。*

[水道尽头，铁栅栏已经被人提前撬开了——红叶在修建商会时就预留了这条逃生路线。但当时她以为，需要用到这条路的那天永远不会来。]

[三人从水道出口钻出来。外面依然是倾盆大雨。远处银甲城的方向，火光映红了半边天空——他们在烧仓库。]

**阿桐：** （忍不住哭出声）夫人……我们什么都没有了……

**红叶：** ——

[红叶站在雨里，水从她的发梢、指尖、衣摆不停地流淌。她望着远处的火光，表情在雨幕中看不分明。]

[良久。]

**红叶：** 不对。

[她低头看了看手里攥着的账本。浸水了，但还能读。]

**红叶：** 我还有这个。

**阿桐：** ……一本账本有什么用？商会都没了……

**红叶：** 账本里记着每一条商路的走向、每一个客户的喜好、每一处隐蔽仓库的位置——这些不在银甲城里，在我脑子里。他们可以烧掉我的仓库，抢走我的金币，抓走我的人——但他们抢不走我的脑子。

[她用力擦了一把脸上的雨水——或者泪水，分不清。]

**红叶：** 而且——

[她拍了拍周叔的肩膀。]

**红叶：** 我还有你们两个。

**周叔：** 夫人……

**红叶：** 走吧。站在雨里感慨不能当饭吃。

[三人踉踉跄跄地走进黑暗中。身后，银甲城的火光越烧越旺，映在雨帘上像一面破碎的铜镜。]

[走了约莫半个时辰。红叶的脚步突然停了。]

[她站在一棵孤零零的枯树下——树冠挡不了多少雨，但至少能遮住头顶。她靠着树干，缓缓滑坐下来。]

**阿桐：** 夫人！

**红叶：** 没事。就……坐一会儿。

[她将账本抱在怀里，像抱着一个孩子。雨水打在她的脸上，她闭上了眼睛。]

**红叶：** ……阿桐。

**阿桐：** 在。

**红叶：** 你还记得我刚嫁到银甲公爵府的时候吗？

**阿桐：** 记得。夫人那时候才十六岁。

**红叶：** 十六岁。什么都不懂。公爵大人比我大二十三岁，娶我只是因为我父亲的商路值钱。我嫁过去的第一天，他对我说——

[她睁开眼睛，雨水顺着睫毛滑下来。]

**红叶：** 他说："在这个家里，你只需要做两件事——生孩子，和管账。"

*——于是我学会了管账。管得比他更好。管到整个南境的商路都姓了红叶而不是银甲。*

*——公爵大人死后，王廷容忍了我五年。因为他们需要南境的税金。但现在战争来了，他们不再需要一个"不够忠诚"的公爵夫人来替他们赚钱了——他们需要的是一个听话的、可以随意支配的南境。*

*——所以他们清洗了我。*

[她低下头，看着怀中的账本。]

**红叶：** 忠诚。

[她念出这个词。声音在雨里显得格外空旷。]

**红叶：** 忠诚真的没有保质期……

[阿桐和周叔等着她说下去。]

**红叶：** 因为它从来就不是真的。

### 场景描写

清洗之后的第三天。南境边缘，柳溪谷。

雨还没停。

三天的连绵阴雨将柳溪谷变成了一片泽国。溪水暴涨，漫过两岸的卵石滩，将低洼处的柳树根部浸在浑浊的水中。原本清澈见底的柳溪变成了一条黄褐色的浊流，夹裹着断枝和泥沙，发出沉闷的轰鸣。

谷中有一座废弃的石桥——桥面已经断裂，只剩两个桥墩和半截残拱，像一个张开的嘴巴。桥下的水流湍急，卷着白沫冲刷桥墩，发出令人牙酸的摩擦声。

红叶就在断桥的这一侧。

她的状况很糟糕。三天的逃亡已经耗尽了她最后的体力。衣衫上全是泥浆和荆棘划出的口子，左臂用一截布条潦草地包扎着——那是在穿过柳林镇时被禁军的箭擦伤的。她的脸色苍白，嘴唇干裂，眼窝深陷，但那双眼睛依然亮着——像快要燃尽的灯芯，最后一点火焰反而烧得格外清晰。

她身边只剩五个人：侍女阿桐、老门房周叔，还有三个不知从哪里跟上来的残兵——他们不是银甲骑兵团的正规军，而是商队的护卫，本可以在清洗之夜逃走，却鬼使神差地一路跟到了这里。

六个人，三匹瘦马，一把半锈的剑，几根削尖的木棍。

这就是银甲公爵夫人红叶现在的全部家当。

南境边陲，一座无名小镇的废弃客栈。

雨终于停了，但天空依然阴沉沉的，像一块洗过太多次的灰布。客栈已经荒废多年——屋顶的瓦片缺了一大半，裸露的椽木上长满了青苔。二楼的地板朽了，踩上去吱呀作响，每一步都像是在和整栋建筑商量"你到底还塌不塌"。

阿桐在角落里用破布给周叔换药。三个残兵挤在门口，轮流放哨。客栈外的世界安静得让人不安——没有追兵，没有马蹄声，没有任何声音。只有屋檐上残存的雨水一滴一滴地落在台阶上。

滴答。滴答。滴答。

像钟表在走。也像在倒计时。

[她的声音略微颤了一下。]

**红叶：** 已焚毁。

[翻页。翻页。翻页。一条一条，一项一项，像是在念一份冗长的讣告。每一行字后面都跟着同样的结尾——查封、冻结、焚毁、没收。]

**红叶：** ……银甲城西区三号仓库，存粮八百石——没收。码头第七号泊位，商船四艘——扣押。柳林镇中转站，骡马二十匹——征用。

[她翻到了最后几页。这些页面受损最严重，墨迹几乎完全被雨水冲走了。]

**红叶：** 私人账目……已经看不清了。

[她将账本翻回第一页，重新开始。但这一次，她不是在念数字。她在算。]

**红叶：** 目前剩余资产——

[她在一张空白页上写了起来。蜡烛的光太暗，她不得不凑近，差点被蜡烛燎到头发。]

**红叶：** 现金：约三十七枚金币——阿桐缝在内衣里的私房钱。

**阿桐：** （在角落里小声说）夫人，那是我攒了三年的……

**红叶：** 我知道。记账了。欠阿桐三十七金，利息按年化一成二计。

**阿桐：** ……夫人，我没有要利息——

**红叶：** 我要给。这是规矩。

[她继续写。]

**红叶：** 马匹：三匹，状况不佳。折合约十五金。武器装备：一把半锈的剑、三根削尖的木棍、周叔的一把小刀。折合——

[她想了想。]

**红叶：** 两金。还是算三金吧，周叔那把刀是好钢。

**周叔：** ……夫人。

**红叶：** 人员：六人。分别是——

[她停了。]

[笔尖悬在纸面上方，墨滴在犹豫中膨胀，最终落下，在纸上晕开一个小小的墨点。]

**红叶：** ……人员不算资产。

*——什么时候开始的？什么时候开始，我把人也当成了资产负债表上的一行数字？*

[她放下笔。用双手揉了揉太阳穴。蜡烛的光在她的手指间摇曳，在柜台上投下忽明忽暗的影子。]

**红叶：** 总资产：约五十五金。

[她盯着这个数字看了很久。]

**红叶：** 五十五金。

[又看了一会儿。]

**红叶：** 我十六岁嫁到公爵府的时候，嫁妆里有一百金。

[她发出一声极轻的笑——不是苦笑，而是一种更奇怪的声音，像是嘲笑和叹息的混合物。]

**红叶：** 做了十二年生意，到头来还不如一个十六岁的新嫁娘。

[她将账本往后翻——翻过了所有的数字和条目，翻到了最后几页。那些页面本应是空白的备用页，但——]

[她的手停住了。]

[最后一页里夹着一样东西。]

[一朵干燥的花。]

[是桃花。粉白色的花瓣已经褪成了淡黄，脆得像蝉翼，边缘微微卷曲。但花芯还保留着一丝若有若无的颜色，在蜡烛的光下显出极淡的粉红。]

[红叶的手指悬在花朵上方，像是怕碰碎它。]

**阿桐：** （走过来，看到那朵花，愣住了）那是……

**红叶：** ——

[她没有回答。]

[客栈里沉默了很久。蜡烛的火苗跳了两下，像是什么人的呼吸。]

**红叶：** ……小枫。

[她说出这个名字的方式和说任何一个数字都不一样。不是清晰的、精确的。而是模糊的、犹豫的——像是这个名字在嘴里含了太久，已经快要化掉了。]

**红叶：** 小枫是——

[她闭了闭眼睛。再睁开时，蜡烛的光在她的瞳孔里碎成了两点细小的火星。]

**红叶：** 是我的女儿。

[阿桐沉默了。周叔也停下了换药的动作。三个在门口放哨的残兵虽然听不太清，但本能地屏住了呼吸。]

**红叶：** 她三岁的时候——

*——三岁。还不会写自己的名字。但她会笑。笑起来的时候有两个酒窝，左边的比右边的深。她喜欢在花园里追蝴蝶，喜欢把桃花瓣一片一片摘下来，说是在"做花雨"——*

**红叶：** 那年冬天南境发了大疫。银甲城死了很多人。我……我有钱。我请了最好的郎中，买了最贵的药材，整整七天七夜没有离开她的床前——

[她的手指终于轻轻触到了那朵桃花。]

**红叶：** 七天。我守了七天。但最好的郎中说不行。最贵的药材没有用。最多的金币——

[她的声音碎裂了一个极短暂的瞬间，像瓷器上出现了一条发丝般的裂纹。但她立刻将它修补回去。]

**红叶：** ——买不回来。

[沉默。]

[蜡烛的火苗在沉默中燃烧，蜡油沿着烛身缓缓流淌，凝固成透明的泪珠形状。]

**红叶：** 小枫走之前，把一朵桃花放在我的手心里。她说——

[红叶的嘴唇动了动，但没有发出声音。]

[又试了一次。]

**红叶：** 她说："妈妈不要哭，小枫给你做花雨。"

[声音极轻。轻到几乎被屋檐上滴落的雨水盖过去。]

**红叶：** 从那以后，我把这朵花夹在账本里。不是因为它值钱——它不值一文钱。但它是……

[她将花朵小心翼翼地放回账本最后一页。动作的轻柔程度，和她翻阅前面那些数以万计的金额数字时完全不同——像是在安放一件无价之宝。]

**红叶：** 它是账本外面的东西。

[她合上账本。将它重新贴着胸口放好。]

*——十二年了。十二年来我把一切都变成了数字。商路是数字，人脉是数字，甚至人——我差点把人也变成了数字。*

*——但这朵花，我一直没有给它标价。*

*——因为有些东西……*

[她站起身。拿起蜡烛，走向门口。门外的天空还是灰的，但东边的云层似乎薄了一些，有一丝白光正在渗透进来。]

**红叶：** 阿桐。

**阿桐：** 在。

**红叶：** ……以后算账的时候，人员那一栏，不写数字了。写名字。

**阿桐：** ……是。

[红叶走到门口，吹熄了蜡烛。天快亮了，不需要蜡烛了。]

**红叶：** 天亮了就走。我们还有路要赶。

[她最后回头看了一眼柜台上留下的那几滴蜡泪。它们在晨光中像几颗凝固的琥珀。]

*——五十五金。*

*——和一朵花。*

*——够了。从这里重新开始。*

## Event 08 — 防御反击·二「佣兵的忠诚」

### 场景描写

南境最南端，青石关。

这里是天城王朝与南方诸国之间的最后一道关口——也是红叶逃亡旅程的终点。过了青石关就是自由地带，王廷的追捕令到了这里便如废纸一张。

但追兵比预想中来得更快。

青石关建在两座低矮石山之间的隘口处，关墙早已年久失修，长满了藤蔓和苔藓。关门只剩一扇，另一扇不知被谁拆去当了柴烧。但地形依然有利于防守——隘口宽度只能容纳六骑并行，两侧石壁上全是凸出的岩架和天然的射击位。

红叶在三个时辰前到达了这里。

她已经精疲力竭了——左臂的伤口再次裂开，血从布条缝隙间渗出来，把半截袖子染成了暗红色。她的脸色苍白得几乎透明，嘴唇上干裂的皮一碰就渗血。但她依然站着——靠在关墙的石壁上，摊开那本账本，在最后一页空白处写着什么。

周叔在隘口前方用石头和断木搭了一道简易路障。三个残兵在路障后面检查武器——他们的武器依然寒酸：一把半锈的剑现在完全生锈了，木棍换成了从破关门上拆下来的铁钉木板。阿桐在后方撕布条准备绷带，动作熟练得让人心疼——她这几天已经包扎了太多伤口。

远处，官道上扬起了一片灰尘。

马蹄声如闷雷滚来。

### 剧情对话

**残兵甲：** ……来了。

[所有人都听到了。红叶合上账本，抬起头。]

**红叶：** 多少人？

**残兵甲：** （爬到关墙上看了一眼，脸色发白）至少……二十骑。后面还有步兵。

**红叶：** 禁军？

**残兵甲：** 黑甲。骑兵在前，弓手在后。领头的是个校尉——

**红叶：** 骑兵校尉。

[她闭了闭眼。]

*——追击队。不是普通巡逻兵，是专门来抓我的精锐。宰相大人真看得起我——派了一个骑兵校尉加二十多骑来追一个逃亡的商人。*

*——或许他觉得三千金的赏金不够，要确保用兵力拿到结果。*

[她睁开眼，环顾周围。五个人——周叔、阿桐、三个残兵。加上她自己，六个人。对面至少三十人。]

**红叶：** 青石关另一侧有出路吗？

**周叔：** 关南有一条山间小路，通往鹤鸣谷。但路口被乱石堵了——大概要一个时辰才能清理出来。

**红叶：** 一个时辰。

[她低头看了看自己的手。手指在微微发抖——不是因为恐惧，而是因为失血和疲劳。]

**红叶：** 周叔。

**周叔：** 在。

**红叶：** 你为什么还在这里？

[周叔愣了一下。]

**红叶：** 我认真问的。银甲商会没了。你的薪水已经三个月没发了。你今年六十二岁，在商会干了二十八年，按照合同条款，你的遣散金是四百八十金。但这笔钱——

[她苦笑了一下。]

**红叶：** 你大概永远拿不到了。

**周叔：** 夫人——

**红叶：** 如果你现在走，从东面的山坡翻过去，禁军不会追你。他们要的是我，不是一个老门房。你还有儿子在东境，对吧？

[周叔沉默了一会儿。然后他做了一件让红叶意外的事——他笑了。一个六十二岁的老人，在二十多个追兵即将到来的时刻，站在一道破败的关墙后面，笑了。]

**周叔：** 夫人，您算账确实算得好。全南境没人比您更会算账。

**红叶：** ……

**周叔：** 但有些账，您一直算错了。

[他从怀里掏出一样东西——一块旧铁牌，上面刻着模糊的编号和一个士兵的名字。]

**周叔：** 三十年前，我在北境当兵。冬天行军，断了粮，整个连只剩十七个人。连长把最后一块干粮分给了我们，自己嚼树皮。我问他为什么，他说——

[他把铁牌挂回脖子里。]

**周叔：** 他说："因为你们还年轻，还有路要走。"

**红叶：** ……

**周叔：** 连长后来没活过那个冬天。但他那十七个兵，活了十五个。

[他握紧手里的铁钉木板——那根本不能算武器，但他握得像是握着一杆长矛。]

**周叔：** 夫人。二十八年前，我从北境回来，找不到活干，是您——那时候您还是刚过门的小夫人——给了我一份门房的工作。您说："周叔，你在这里好好干，我不会亏待你的。"

**红叶：** ……我记得。

**周叔：** 二十八年了。您确实没亏待过我。逢年过节的红封从来没少过，我儿子去东境念书的路费是您出的，我老伴走的时候丧葬费也是您给的。这些——

[他拍了拍胸口。]

**周叔：** 这些不在合同里。

**红叶：** ——

**周叔：** 所以，夫人。您问我为什么还在这里？

[他转过身，面朝隘口外面越来越近的马蹄声。]

**周叔：** 因为有些东西不是钱能买的。也不是没了钱就会消失的。

[三个残兵互相看了看。其中一个——最年轻的那个，大概也就二十出头——清了清嗓子。]

**残兵甲：** 我、我也是。夫人，去年我妹妹生病，是商队的大夫免费给治的。那个大夫说是夫人吩咐的——商队沿途遇到百姓求医，能治就治，药费从商会公账出。

**残兵乙：** 我家在柳林镇。三年前发大水，是银甲商会的船来接的人。我爹我娘都是夫人的船救出来的。

**残兵丙：** ……我没什么大恩。但夫人每个月给商队护卫的工钱比别家多两成。这两成……够我弟弟读书了。

[红叶听着这些话，表情慢慢地发生着变化。]

*——我……这些事我都记得。但我记的方式是——"商队医疗支出，年均三百金，可列入公共关系经费"。"水灾救援，动用船只四艘，损失八百金，但获得了柳林镇商铺的优先租约"。"护卫薪资上浮两成，员工流失率降低四成，综合效益为正"。*

*——我把每一件事都变成了数字。*

*——但对他们来说，那不是数字。是生命。是家人。是未来。*

*——我一直以为我买的是忠诚。但他们留下来，不是因为我付了钱——*

*——是因为……*

[她说不下去了。不是嗓子哑了，而是有些领悟太大，语言装不下。]

**红叶：** ……好。

[她深吸一口气。然后做了一件她从来没做过的事——她向五个人微微鞠了一躬。银甲公爵夫人，向一个老门房、一个侍女和三个连姓名都没上过任何档案的商队护卫鞠了一躬。]

**红叶：** 拜托了。

[周叔的眼眶红了，但他用力眨了眨，忍回去了。]

**周叔：** 夫人放心。青石关虽破，但窄。他们进不来那么多人。

**红叶：** 我来指挥。

[她走到关墙上方——那里有一个天然的指挥位置，可以俯瞰整个隘口。她摊开账本，但这次写的不是数字——而是战场标注。]

**红叶：** 周叔守中路。三位——

[她看向三个残兵。]

**红叶：** ……你们叫什么名字？

**残兵甲：** 林大壮。

**残兵乙：** 方小河。

**残兵丙：** 陈七。

**红叶：** 大壮守左，小河守右，七弟在周叔后面补位。阿桐——

**阿桐：** 在！

**红叶：** 你到关南去清理路口的乱石。尽快。

**阿桐：** 可是夫人——

**红叶：** 清理出路来是我们活下去的唯一办法。去吧。

[阿桐咬着嘴唇跑向关南。]

[红叶转身面向隘口外。追兵已经到了。]

### 战斗阶段

[禁军骑兵在隘口前列阵。领头的骑兵校尉是个三十多岁的精悍男人，面容冷硬，眼神像刀。]

**骑兵校尉：** 银甲公爵夫人红叶！奉宰相令，缉拿叛国逆犯！束手就擒者可免一死——反抗者，格杀勿论！

[红叶站在关墙上，居高临下。风吹起她散乱的长发。]

**红叶：** 格杀勿论？校尉大人，你确定宰相要的是我的尸体而不是活人？三千金的赏金可写着"活捉"两个字呢。

**骑兵校尉：** 少废话！

**红叶：** 我从不废话。每一句都有报价的。

[她微微一笑。那个笑容——在疲惫、苍白、伤痕累累的脸上——带着一种奇异的锋利感。]

**红叶：** 来吧。

[第1回合：禁军骑兵冲入隘口。但隘口狭窄，只能并行六骑。周叔和三个残兵在路障后迎击。]

**周叔：** 来啊！老头子还没老到拎不动家伙的地步！

[周叔虽然六十二岁了，但当年北境当兵的底子还在。他用铁钉木板格开了第一柄长剑，然后一记横扫将一名骑兵从马上掀了下来。]

**红叶：** （在关墙上喊）左边两个是佯攻！右边那个拿长矛的才是主力——大壮，注意右翼！

[第2回合：禁军加大攻击力度。弓手从远处抛射箭雨。]

**红叶：** 贴墙！箭雨角度是四十五度——靠在石壁下方有死角，射不到！

[周叔和残兵们贴着关墙躲避箭雨。箭矢叮叮当当地射在石壁上，溅出火星。]

**陈七：** 周叔，他们人太多了——

**周叔：** 人多怎么了？门就这么宽，来一百个也得一个一个过！

[第3回合：禁军骑兵试图翻越关墙两侧的石壁。但石壁光滑且长满苔藓，几个攀爬的士兵接连滑落。]

**红叶：** 石壁上有苔藓就不用管——天然的防御。把注意力集中在关门！

[第4回合：禁军增援抵达——步兵×6从后方赶上来，加入攻击。压力骤增。]

**方小河：** 又来了！后面还有步兵——

**周叔：** 稳住！

[周叔的木板被一柄长刀劈断了。他丢掉断板，从地上捡起一面倒下的禁军丢弃的盾牌，继续战斗。]

**红叶：** 周叔，退后三步！让他们进来两个——关门内侧有个坑洼，骑兵进来会被绊住！

[周叔后退。两名禁军骑兵冲入关门，果然被地面的坑洼绊了一下——战马踉跄的瞬间，林大壮和方小河从两侧夹击，将骑兵拽下马。]

**红叶：** 好！就是这样——不要硬扛，用地形！

[第5回合：战斗进入白热化。周叔肩膀上挨了一箭，但咬牙拔出来继续战斗。林大壮的脸上被剑锋划了一道口子。方小河的左手已经不能动了。]

[红叶看着下方的战况，手指在账本边缘用力捏着——指甲陷入了纸页。]

*——他们在为我流血。为一个付不起他们薪水的人流血。*

*——这不在合同里。*

*——这不在任何一本账本里。*

[第6回合：红叶做出了一个决定。]

**红叶：** ……大壮！

**林大壮：** 夫人！

**红叶：** 关南五百步有一支行商队伍——我来的时候看到了他们的营火。去找他们的护卫头领，给他这个。

[她从怀中取出一个小布袋——那是她最后的全部现金。三十七枚金币。阿桐的私房钱。]

**红叶：** 告诉他——银甲公爵夫人红叶雇他打一仗。报酬就是这些。如果他嫌少——

[她将手腕上仅剩的半串算珠也解下来，放入布袋。]

**红叶：** 告诉他这串算珠的银珠是纯银的。值十金。凑个整数。

*——五十五金。我把最后的五十五金全押上了。*

*——但如果不押这一注，就没有明天了。*

[林大壮接过布袋，翻墙向关南跑去。]

[第6回合后半段：四名行商护卫从关南赶来增援！他们是跑商路的老手，战斗经验丰富——虽然算不上精兵，但在这种近身混战中绝对够用。]

**行商护卫头领：** 银甲公爵夫人？就是那个做南境茶路的？我走过她的商路——关税公道，驿站干净。冲这个，这仗我打了！

[友军增援加入战场。战况逐渐稳定。]

[第7回合：禁军的攻势被遏制。隘口处尸体堆积，禁军无法有效推进。]

**骑兵校尉：** ……撤弓手！换长矛队上——

**红叶：** 他要换长矛了！长矛在窄巷里施展不开——贴近了打，不给他距离！

[周叔带头冲了上去。他的动作已经没有刚开始那么灵活了——年纪和伤势都在拖他的后腿。但他的眼神依然坚定。]

**周叔：** 来！

[第8回合：关南方向传来阿桐的喊声。]

**阿桐：** 夫人——！路通了！路通了！

**红叶：** 撤退！所有人从关南撤——

[禁军发现目标要逃，发起最后的猛攻。]

**骑兵校尉：** 不许让她跑！冲——

[周叔挡在最后面。]

**周叔：** 你们先走！老头子断后！

**红叶：** 周叔——！

**周叔：** 夫人！快走！

[红叶咬紧了牙。她想说"不行"，想说"一起走"——但她看着周叔的眼睛，看到了三十年前那个连长的影子。]

*——"因为你们还年轻，还有路要走。"*

**红叶：** ……你给我活着出来。这是命令。

**周叔：** 哈！夫人，老周接的最后一道命令。

[红叶被残兵们架着从关南撤出。最后回头看了一眼——周叔一个人站在关门前，六十二岁的老人像一面破旧但不倒的墙。]

### 战后

[鹤鸣谷入口。追兵的声音渐渐远去——他们被阻在了青石关。]

[红叶靠在一棵松树上，大口喘气。阿桐在给她重新包扎手臂。方小河和陈七瘫坐在地上，浑身是血。行商护卫们已经散去了——他们的合同在战斗结束的瞬间就到期了。]

**方小河：** 周叔……周叔他——

[等了很久。关南方向传来脚步声——不是马蹄声，是脚步声。一瘸一拐的。]

[周叔从小路的尽头走出来。他浑身是血，左腿拖在身后，右手按着肋下的伤口。但他在笑。]

**周叔：** 夫人……命令执行完毕。老周……活着出来了。

[他走了两步，然后膝盖一软，倒了下去。]

**红叶：** 周叔！

[她冲上去，扶住了他。周叔的血染在她的手上、衣上。]

**周叔：** 没事……没事。皮肉伤。禁军校尉……让我挡了两剑，就不追了。大概是觉得……一个老头子不值得浪费功夫。

[他咧嘴笑了。缺了两颗门牙。]

**周叔：** 夫人……您以前总说"不做亏本的买卖"。

**红叶：** ……嗯。

**周叔：** 今天这一仗……您觉得亏不亏？

[红叶低下头。她的眼泪终于掉了下来——不多，就两滴。落在周叔沾满泥土和血迹的手背上。]

**红叶：** ……赚了。

[她用力擦掉眼泪——像是在擦掉一个不该出现在资产负债表上的数字。]

**红叶：** 赚大了。

[她将那块从周叔怀里滑出来的旧军牌捡起来，替他重新挂好。]

**红叶：** 走吧。鹤鸣谷里有药铺。你的伤——

[她站起来，伸手去拉周叔。但周叔太重了，她拉不动——几天的逃亡已经耗尽了她的力气。]

[方小河和陈七过来帮忙。三个人把周叔搀了起来。]

[六个人——缺了林大壮，他在行动中和行商护卫一起撤向了别的方向——一瘸一拐地走进了鹤鸣谷。身后是青石关沉默的轮廓。关墙上的藤蔓在傍晚的风中微微摇摆。]

[红叶走了几步，突然停下来。]

**红叶：** 我现在……身无分文了。

[她摸了摸空荡荡的袖子——算珠没了，金币没了。]

**红叶：** 连一杯茶都买不起了。

[然后她做了一件奇怪的事——她笑了。不是苦笑，不是商人的礼貌微笑，而是一个真正的、从胸腔深处涌上来的笑。]

**红叶：** 但是——

[她回头看了看身后的人。周叔被搀着，还在笑。方小河和陈七虽然满身是伤，但眼神是亮的。远处阿桐跑过来给每个人分发布条。]

**红叶：** 但是，好像也没那么糟。

*——身无分文。但身边有人。*

*——这笔账……以前怎么就没算过呢？*

## Event 09 — 角色事件·肆「最后的筹码」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情 |
| 情感基调 | 赤裸的谈判——不再从强者位置出发，而是用最后的一切换取最重要的东西 |
| 特殊事件 | 红叶以全部情报换取追随者的安全 |
| 奖励 | 300 EXP, CG「最后的筹码」 |

### 场景描写

暗潮军团前线营地，黄昏。

夕阳将天边烧成一片熔铜色，光线从帐篷的缝隙间斜斜刺入，在地面上拉出一道道长长的金色条纹。营地里炊烟升起，饭菜的香气和马粪的气味混合在一起，弥漫在微凉的秋风中。

营地东侧的哨卡前，一个身影出现了。

哨兵差点没认出来。那个曾经在月下茶会上优雅斟茶的女人，那个在高台上端坐着用金币指挥佣兵的公爵夫人——此刻站在夕阳里，像一棵被暴风折断又勉强站直的枯树。

红叶穿着一件不知从哪里找来的灰色粗布外袍，太大了，肩膀处垮下来，袖子长出一截，被她胡乱卷到手肘。里面那件暗红色的绸衣已经看不出原来的颜色了——泥渍、血渍、汗渍层层叠叠，像一幅失败的水墨画。头发用一根草绳胡乱扎在脑后，几缕碎发粘在额头上，被汗水和尘土糊成了暗灰色。

她的脸瘦了一圈。颧骨的线条从皮肤下面凸出来，显得那双眼睛格外大——但不再是银币般冰冷的光了。那双眼睛里有疲惫，有计算，有一种被打到最底层之后反而清澈起来的东西。

她身后站着四个人：周叔拄着一根树枝当拐杖，肋下的伤还没好利索，走几步就要喘一阵。阿桐背着一个布包袱，里面装着他们仅有的家当——几块干粮、一卷绷带、一把小刀。方小河的左手吊在脖子上，陈七的脸上那道剑伤还在结痂，狰狞地横过半张脸。

五个人。身无分文。满身是伤。

这就是银甲公爵夫人红叶全部的"谈判代表团"。

### 剧情对话

[哨兵拦住了他们。]

**哨兵：** 站住！什么人？

**红叶：** 银甲公爵夫人红叶。我要见你们的指挥官。

[哨兵上下打量了她一眼，目光里带着明显的怀疑。]

**哨兵：** ……你说你是银甲公爵夫人？

**红叶：** 前公爵夫人。现在的头衔大概是"通缉犯"。

[她的嘴角弯了一下——不算笑，更像是嘴唇抽搐了一下，碰巧弯对了方向。]

**红叶：** 如果你们的指挥官收到过一封写在金穗笺上的信和一张绑在箭杆上的纸条，他会愿意见我的。

[哨兵犹豫了一会儿，派人去通报。]

[等待的时间里，红叶站在营地门口，没有坐下。周叔想让她靠一靠，她摇了摇头。]

**红叶：** （低声，对周叔）谈判不能坐着。坐着的人是在求人。站着的人——至少看起来还有选择。

**周叔：** ……夫人，您都快站不住了。

**红叶：** 站不住也得站着。

[一刻钟后，她被带到了指挥帐内。]

[帐篷内的陈设简朴——一张行军桌，几把折叠椅，墙上挂着地图。桌上有一壶茶，还冒着热气。]

[红叶看到那壶茶，目光停顿了一瞬。她已经很多天没喝过热茶了。但她没有去拿。]

[她站在行军桌的对面，从怀中取出那本皱巴巴的账本，翻到中间某一页，平摊在桌上。]

**红叶：** 我只说一遍。

[她的声音沙哑但稳定。每一个字都像是从嗓子里一颗一颗掏出来的石子。]

**红叶：** 天城王朝国库有三处秘密金库，分别在北境磐石河下游的地下洞窟、东境桐岭关后方的废弃矿井、以及王都皇城西苑假山下方。总计储金约四十万两。

[她翻了一页。]

**红叶：** 宰相大人在南境到东境之间有一条私人走私线路，用官船走漕运水道，贩运铁矿和盐。每年利润约八万金——这笔钱不经国库，直接进入宰相的私人账户。路线图在这里。

[又翻一页。]

**红叶：** 天城王朝现存十七位封疆大臣中，有六位和宰相有私下的利益往来，四位和王室暗中不和，三位正在观望暗潮的动向。名单和详细关系网在这里。

[又翻一页。]

**红叶：** 宰相本人——他有一个弱点。他的嫡长子在三年前的科举中舞弊，被一个御史参过一本。宰相花了大力气把这件事压下去了，但证据还在。在南境第四号密库的铁箱里——密库的钥匙我有。

[她将账本合上，双手平放在上面。]

**红叶：** 以上。天城王朝的金脉、暗脉、人脉、软肋——我知道的全部。

[长久的沉默。帐篷外面传来远处的马嘶声和士兵操练的号令。]

**红叶：** 这些情报足够你们打赢这场战争。或者至少，足够你们把天城王朝的内部搅成一锅粥。

[她抬起头，目光直视对面。]

**红叶：** 我的条件——

[她停了一下。喉头动了一下，像是在咽什么东西。]

*——以前我谈判，开口就是数字。多少金，多少地，多少股份。条款一二三四，违约责任甲乙丙丁。每一次开口都胸有成竹，因为我手里永远握着最大的筹码。*

*——但这一次不一样。*

*——这一次我手里什么都没有了。我唯一的筹码就是脑子里的东西。而我要用它换的——不是金子，不是地盘，不是权力。*

**红叶：** 我的条件只有一个。

[她的声音突然变得很轻。不是策略性的轻——不是谈判中用来制造悬念的停顿——而是一种从胸腔深处发出的、真正的轻。像是说出来就会碎掉。]

**红叶：** 保护他们。

[她微微偏头，朝帐篷外面示意了一下——周叔、阿桐、方小河、陈七。四个站在暮色中等着她的人。]

**红叶：** 周叔六十二岁了，肋骨断了两根，需要郎中。阿桐跟了我十二年，什么苦都吃过了，给她一个安全的地方住就好。方小河的左手要是再不治就废了。陈七——陈七十九岁，还是个孩子。

[她的目光从帐篷的缝隙间穿过，落在外面那四个人身上。夕阳正好照在他们脸上。]

**红叶：** 他们不是我的筹码。他们是——

[她张了张嘴，似乎在找一个合适的词。找了很久。]

**红叶：** 他们是我欠着的人。

[沉默。]

[帐内茶壶的盖子被蒸汽顶得轻轻响了一声。]

**红叶：** 整个天城王朝的情报，换四个人的安全。

[她将双手从账本上移开，垂在身体两侧。]

**红叶：** 你可以觉得我在贱卖。以前的我也会觉得这笔买卖亏得离谱——四十万两金库的情报，换四个人的三餐和一张床。从投入产出比来看，简直荒谬。

[她的嘴角又弯了一下。这次倒确实有点像笑。]

**红叶：** 但我最近学会了一件事。

*——账本外面的东西。*

**红叶：** 有些东西的价值，不能用金子算。

[又一阵沉默。]

[她补充了最后一句话。声音已经很淡了，像说给自己听。]

**红叶：** ……这是我第一次做不以盈利为目的的交易。感觉很奇怪。但——

[她深吸了一口气。]

**红叶：** 但好像也没那么难。

### 尾声

[谈判结束后。]

[红叶走出帐篷。天已经全黑了，营地里点起了篝火。周叔几个人还在门口等着——准确地说，是周叔坐在地上等着，另外三个人围着他，阿桐正在给他换绷带。]

**周叔：** 夫人，怎么样？

**红叶：** ……成了。

[她的语气平淡得像是在说"今天的天气不错"。但她走到周叔身边的时候，腿突然软了一下——这一下软得很彻底，整个人直接跪坐在了地上。]

**阿桐：** 夫人！

**红叶：** 没事。就是——

[她低着头，用手撑着地面。肩膀在微微发抖。]

**红叶：** 就是突然……松下来了。

[她抬起头。火光映在她的脸上，照出嘴角一个极小的弧度——是真正的笑。疲惫的、释然的、有点不习惯但确实是真的笑。]

**红叶：** 他答应了。给你们安排住处、治伤、吃饭。管饱。

**方小河：** ……夫人，那您呢？

**红叶：** 我？

[她想了想。]

**红叶：** 我大概还有用。脑子里的东西不是一本账本就能写完的——后面还有很多情报需要详细解读。所以他暂时不会赶我走。

**陈七：** 那不就是……夫人也能留下？

**红叶：** ……也算是吧。

[她沉默了一下。]

**红叶：** 虽然理由不太体面——"还有利用价值所以暂时留着"。

**周叔：** 夫人。

**红叶：** 嗯？

**周叔：** 您刚才进去谈了多久？

**红叶：** 半个时辰。

**周叔：** 半个时辰。您谈的第一件事是什么？

[红叶愣了一下。]

**红叶：** ……你们。

**周叔：** 不是情报的价钱，不是自己的安排。是我们。

[他笑了。牙齿缺了两颗的笑。]

**周叔：** 夫人，您账算得越来越好了。

[红叶坐在地上，靠着一个木桩。有人递来一碗热粥——军营里最普通的粗粮粥，米粒都没煮开，糊成了一团。她接过来，捧在手心。碗壁的温度透过手掌传到全身。]

[她喝了一口。]

*——很难喝。但是热的。*

[她将碗沿抵在嘴唇上，闭上了眼睛。]

*——很久很久没有喝到热的东西了。*

*——南境最好的铁观音、金穗笺上的茶香、白瓷盖碗里升起的袅袅茶烟——*

*——都不如这一碗糊了的粗粮粥。*

[她将碗放下来。碗底还剩一点粥，映着篝火的光。]

**红叶：** 阿桐。

**阿桐：** 在。

**红叶：** 明天开始，帮我记一笔新账。

**阿桐：** 记什么？

**红叶：** ……还不知道。但应该和以前的不太一样。

[篝火噼啪作响。五个人围坐在火边，影子在地面上叠成一团暖色的形状。营地里有人在远处唱歌，走调的军歌，混着笑声和咳嗽声。]

[红叶将空碗放在膝盖上，仰头看了看天。]

[星星出来了。在逃亡的日子里，她一直没注意过头顶有星星。]

## Event 10 — 据点攻略「空城计」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.24-28 |
| 情感基调 | 虚实之间——一个空了的城，一个不空的心 |
| 战斗条件 | 攻破红叶旧据点，识破空城计 |
| 特殊机制 | 前3回合"欺诈阶段"：假人偶×12、陷阱×6、烟火幻象×4制造满编假象；每击败一个假目标，揭示一条线索（旗帜太新、脚印太少等）；第3回合结束自动揭穿，剩余真实敌人出现；红叶和少量忠仆从内部现身 |
| 敌方配置 | 欺诈阶段：稻草假兵×12(不可攻击，3回合后消失) + 机关陷阱×6(Lv.22，触发式) + 烟火幻象×4(触发后降低我方命中2回合)；真实阶段：红叶(Lv.25) + 忠仆护卫×4(Lv.22) + 老管家(Lv.20) |

### 场景描写

南境腹地，枫鸣堡。

这里是银甲公爵府在南境的辅城——不像银甲城那样繁华，但胜在隐蔽。枫鸣堡建在一座矮山的半腰处，四周被茂密的枫树林环抱。城墙是就地取材的青灰色石头砌成的，不高但厚实，依山势蜿蜒，远看像一条盘在山腰的灰色蛇。

从山脚仰望，枫鸣堡此刻看起来固若金汤。

城头上旌旗密布——银甲公爵府标志性的天平纹旗帜至少插了三十面，在秋风中猎猎作响。城墙垛口后面隐约可见一排排整齐的人影，戴着头盔，持着长矛，肃然而立。炊烟从城内多处升起，厚实而稠密，像是有数百人在同时生火造饭。城门紧闭，铁皮包裹的门板上还新刷了一层黑漆，闪着阴沉的光泽。

暗潮军的侦察骑兵在山脚下转了三圈，回去报告：

"枫鸣堡守军约五百人，粮草充足，士气旺盛。强攻需至少三倍兵力。"

但有一件事不太对。

侦察兵中有个老斥候——跟着暗潮军团打了二十多年仗，经验丰富。他蹲在山脚的一棵枫树下，掰着手指头算了半天，然后去找了指挥官。

他说：

"不对。旗帜是新的。这个季节南境多雨，旗帜露天挂三天以上就会褪色——但城头那些旗帜的颜色太鲜了，像是今天早上才挂上去的。而且城下的道路上——没有新鲜的马蹄印。五百人的驻军，一天下来城门口至少该有几十道进出的痕迹。但我看了三圈，只看到五六个人的脚印。"

"这座城，是空的。"

### 剧情对话

[暗潮军前锋队伍在枫鸣堡山脚集结。指挥官面前摊着侦察报告。]

**暗潮副官：** 老赵说城里是空的？可城头上明明有人影——

**暗潮老斥候赵：** 人影不动。我盯了半个时辰，城头那些"守军"一个都没换过位置。活人站半个时辰不动？不可能。除非——

**暗潮指挥官：** 除非那不是人。

**暗潮老斥候赵：** 稻草人。绑在木架上，套上头盔和铠甲，远看跟真人一样。北境的匈人三十年前就用过这招——不过他们做得粗糙。这位公爵夫人做得精细多了。

**暗潮副官：** 那炊烟呢？

**暗潮老斥候赵：** 湿柴堆在灶上就能冒烟。不需要人看着。

[暗潮指挥官看向枫鸣堡。夕阳将城墙染成金红色，旌旗在风中舞动，城头的"守军"沉默地伫立着——确实，细看之下，那份沉默太完美了。活人做不到的那种完美。]

**暗潮指挥官：** 空城计。

*——她知道自己守不住。所以不打算守——而是让我们以为她能守住，争取时间。*

*——争取时间做什么？*

**暗潮指挥官：** ……进攻。但小心陷阱。这种人不会只准备稻草人。

### 战斗阶段

[暗潮先锋队从三个方向逼近枫鸣堡。]

#### 欺诈阶段（第1-3回合）

[第1回合：先锋队靠近城墙。]

[正门方向，第一队士兵刚踏上通往城门的石阶——脚下的石板突然塌陷！]

**暗潮士兵：** 陷阱——！

[机关陷阱触发。塌陷的石板下是一个一丈深的坑，坑底插着削尖的竹签。两名士兵摔了进去，幸而被同伴拽住了一个。]

**暗潮指挥官：** 绕开石阶！从侧面爬上去！

[侧面的登山小路上，一条几乎看不见的细线被触发。树林间突然腾起一团巨大的白烟，伴随着噼里啪啦的爆裂声——烟火幻象。白烟中隐约可见人影晃动、刀光闪烁。]

**暗潮副官：** 有埋伏——！

**暗潮老斥候赵：** 别慌！是烟火弹！南境商队防山匪用的货色——看着吓人，不伤人！但小心烟雾——进去了眼睛看不清！

[烟雾弥漫，暗潮军命中率下降。士兵们在白烟中胡乱挥砍，打中的全是空气。]

[第2回合：先锋队推进到城墙下方。一名士兵搭起云梯，爬到垛口——面对面遇到了一个"守军"。]

[他举刀劈下去。刀砍在——稻草上。]

**暗潮士兵：** ……草人？！

[城头上的"守军"全是稻草人偶。穿着旧铠甲，戴着生锈的头盔，被绑在木架上。有的手里还"拿着"长矛——那些长矛是木棍削的，涂了一层银漆。从远处看以假乱真，近看破绽百出。]

**暗潮副官：** 全是假的！

[但城墙内侧还有机关。第二名翻墙的士兵踩中了一块松动的城砖——一桶沙石从城墙上方倾泻而下。]

[第3回合：更多的陷阱被触发。城门内侧有一条绊马索，连接着三个装满碎瓷的吊桶——马冲过去就会被砸得一脸碎片。内城甬道里用破家具堆了三道路障，每道路障后面都点着几根冒烟的湿柴，让人以为有人在防守。]

*——她一个人把整座城变成了一个巨大的陷阱盒。每一步都有东西等着你——不是要杀你，而是要吓你、拖你、让你犹豫。*

*——犹豫一分钟就多一分钟。对她来说，时间才是最贵的货币。*

[第3回合结束。所有假象耗尽——稻草人被推倒，陷阱被踩完，烟火弹烧尽。枫鸣堡的"满编驻防"像一层画皮被撕开，露出下面空荡荡的真相。]

#### 真实阶段（第4回合起）

[城堡内院。一座三层的石砌主楼前。]

[大门敞开。红叶站在门廊下。]

[她已经不是几天前在暗潮营地里那副狼狈模样了——不，她依然很狼狈，但狼狈的方式变了。她穿着一件洗过的暗红色长衫（虽然还有几处洗不掉的污渍），头发重新梳理过了，用一根木簪别着。脸上的灰尘擦掉了，但消瘦和疲惫擦不掉。]

[她身边站着四个人——不是周叔他们。是另外四个人。穿着陈旧但整齐的银甲骑兵制服，腰间佩着真正的剑。他们的眼神沉稳而坚定，是老兵的眼神。]

[他们身后还有一个老人——头发全白了，穿着一件褪色的灰袍，驼着背，但手里稳稳当当地端着一把铁壶。]

**红叶：** ……你们来得比我预计的快了半天。

[她的声音平静，甚至带着一丝赞许。]

**红叶：** 是那个老斥候看出来的吧？脚印那个破绽我其实发现了，但来不及处理——铺假脚印需要在城门口来回走三十趟，我的人不够。

[她从门廊下走出来，站在台阶上。]

**红叶：** 枫鸣堡守军——全部在这里了。四名银甲骑兵旧部。他们在清洗之夜从南境各处自行赶来的——我没叫他们。还有许叔——

[她偏头看了一眼那个端铁壶的老人。]

**红叶：** 枫鸣堡的老管家。堡里最后一个人。他帮我做了那些稻草人。

**许叔：** （声音苍老而平淡）老朽绑稻草人的手艺是夫人教的。夫人十七岁那年来枫鸣堡避暑，说庄稼地里的稻草人做得太丑，亲手重新做了一批。没想到今天派上了这个用场。

**红叶：** ……那时候是为了好看。今天是为了活命。

[她回过头，面对暗潮军。]

**红叶：** 你们已经攻进来了。我也没有别的陷阱了——用完了。

[她将双手抬起，手心朝外。]

**红叶：** 所以——这场仗到此为止吧。

[第4回合：红叶和她的忠仆们放下武器。但四名银甲骑兵没有立刻服从。]

**银甲骑兵甲：** 夫人！我们还能打——

**红叶：** 不打了。

**银甲骑兵甲：** 可是——

**红叶：** 我说不打了。

[她的声音忽然变得锋利——像以前在商会里裁决争议时的那种语气。但下一秒就软了下来。]

**红叶：** 你们不远千里赶来，不是为了死在一座空城里的。

[银甲骑兵们互相看了看。最后——一个接一个地——将剑放在了地上。]

[战斗结束。]

### 战后

[暗潮军进入枫鸣堡。整座城堡果然几乎是空的——府库被搬空了，粮仓见底，兵器架上只剩几把生锈的旧剑。但所有的建筑都保存完好，没有被破坏的痕迹。城内的水井、仓库、马厩甚至花园都维护得干干净净。]

**暗潮副官：** 她……在投降之前还把城打扫了一遍？

[指挥官在主楼的书房里发现了一样东西——桌上放着一张纸，用一只茶杯压着。]

**纸上内容：**

*"枫鸣堡是我十二年前亲手设计的。每一块石头的位置我都记得。*

*堡内有暗道三条——北墙第七块石头是机关门，通向山后；东厢房地窖有地道连接枫树林；主楼二层书架后面是密室，里面有南境的全套水文地图。*

*我把这些告诉你，是因为你们现在是这座堡的主人了。我不希望它被糟蹋。*

*另外，许叔年纪大了，走不了远路。如果可以的话，让他继续留在堡里做管家。他在这里住了四十年，枫鸣堡每一棵树开什么花他都知道。*

*——红叶*

*追伸：书房柜子里有一罐铁观音。不是顶好的，但比军营里的茶叶强。我留给你的——就当是房东给新住户的见面礼。"*

[指挥官打开柜子。果然有一罐茶。罐子是普通的陶罐，但被擦得一尘不染。]

*——一座空城。几十个稻草人，十几个陷阱，四个不肯走的老兵，一个四十年没离开过的老管家，和一罐铁观音。*

*——这就是银甲公爵夫人最后的全部家当。*

*——但她交出来的时候，把每一样东西都安排得妥妥帖帖——仿佛不是在投降，而是在做一笔清清爽爽的交割。*

>
> *「空城棋谱」——一本薄薄的手册，封面写着"枫鸣堡防务手册·特殊方案丙"。内页详细记录了如何用最少的人力制造最大的防御假象：稻草人的制作规格、陷阱的布置图、烟火弹的配方、甚至假脚印的踩踏频率都有精确计算。最后一页用红笔写了一行备注："此方案仅在万不得已时使用。因为它只能骗敌人一次——第二次就不管用了。就像信用一样。"*

## Event 11 — 角色事件·伍「交易的艺术」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情 |
| 情感基调 | 最精彩的交易——失去一切的人做出的最完美的一笔买卖 |
| 特殊事件 | 红叶正式提出投效条件，展现她不可替代的价值 |
| 奖励 | 350 EXP, 天城情报全图×1 |

### 场景描写

暗潮军团中军大营，主帅帐。清晨。

帐外的营地刚刚起床，号角声在薄雾中回荡。几个伙夫正在大锅前熬粥，蒸汽从锅盖缝隙间窜出来，在冷空气中凝成白色的柱子。远处有马在嘶鸣，铁匠铺里传来叮叮当当的锤击声——有人在修整武器。

帐内，行军桌上铺着一张巨大的地图——天城王朝全境图，标注着已知的兵力部署和粮道走向。地图边缘用铜镇纸压着，被早晨的穿堂风吹得微微鼓起。桌上还有昨夜批阅到一半的军报、一盏已经凉透的茶、和一只缺了角的砚台。

帐门被人从外面掀开了。

红叶走了进来。

她和几天前在营地门口那副凄惶模样判若两人。头发洗过了，乌黑的长发用一根银簪整齐地束在脑后——那根簪子是阿桐从行囊最底层翻出来的，是红叶唯一留下的首饰。脸也洗过了，虽然消瘦没有恢复，但气色好了不少——至少嘴唇有了血色。

她穿着一件素白色的长衫。不是逃亡时的粗布，也不是公爵夫人时期的锦缎——是一件普通但干净的棉布衫，袖口和领口的针脚很细，看得出是阿桐连夜赶制的。外面罩了一件浅灰色的对襟短褂。

整个人看起来——怎么说呢——像一个准备赴约的商人。不是奢华的那种，而是沉稳、干练、胸有成竹的那种。

她右手拿着一卷文书。左手端着——一只茶盏。自己带来的。里面是热茶，茶烟袅袅上升。

### 剧情对话

[红叶在行军桌对面站定。没有坐——没人请她坐，而且她也不打算坐。站着的人掌握节奏。这是谈判的基本功。]

**红叶：** 早上好。

[她先把茶盏放在桌角——一个不占地方但能看到的位置。然后将手中的文书展开，平铺在地图旁边。]

[那卷文书有三页。每一页都写满了密密麻麻的字，字迹整齐，条理分明。一看就不是随手写的，而是花了相当的时间和心思整理的。]

**红叶：** 这是我这几天整理的。

[她用手指点在第一页上。]

**红叶：** 第一部分——天城王朝全境资产清单。

[指尖滑过一列列的数字和地名。]

**红叶：** 国库三处秘密金库的详细位置和进入方式，我上次说过了，这里有补充。除此之外——各州郡的实际税收数据，不是报给王廷的虚数，是真实的数字。这些数据是我经营商路时从各地税吏手里陆续收集的。王廷以为南境年税收是十二万金，实际上是二十一万金。中间的差额——

[她微微一笑。]

**红叶：** 一部分被地方官中饱私囊了。一部分被宰相的人截留了。还有一小部分……是我的运营成本。

[她翻到第二页。]

**红叶：** 第二部分——粮道地图。天城全境十一条主要粮道，我走过九条。每条粮道的运力、损耗率、沿途驿站补给能力、季节性变化——全部在这里。另外标注了三条隐蔽的山间小路，官方地图上没有，但商人们世代相传。这三条路可以绕过天城王朝现有的所有关卡。

[她的手指在地图上画出三条弯弯曲曲的虚线。]

**红叶：** 用好了，你们的补给线可以缩短四成。

[翻到第三页。]

**红叶：** 第三部分——人。

[她的声音在这个字上稍稍加了一点重量。]

**红叶：** 宰相的幕僚团有十二个人。七个是他自己培养的，三个是买来的，两个是被抓了把柄不得不跟从的。被买来的三个人里——有一个叫孙应年的，掌管军需调拨。他欠了赌坊四千金，宰相替他还了，所以他听话。但如果有人出更高的价——或者把他欠赌债的事捅出去——他就不那么听话了。

**红叶：** 还有。王廷禁军统领赵永和，表面上对宰相唯命是从，但他的女儿嫁给了东境平川公的次子。平川公和宰相面和心不和——所以赵永和的忠诚是有保质期的。保质期大概到——

[她闭上眼睛算了一下。]

**红叶：** 到宰相开始动平川公的那天为止。根据我的判断，大约三个月。

[她将三页文书都摊开，一字排在地图旁边。白纸黑字，清清楚楚。]

**红叶：** 金脉、粮脉、人脉。天城王朝的三条命脉——全部在这三张纸上。

[长久的沉默。]

[红叶端起桌角的茶盏，啜了一口。动作从容——和很久以前在月下茶会上一模一样。但这次茶盏里的不是南境铁观音，大概只是军营里最普通的粗茶。]

**红叶：** 现在谈条件。

[她放下茶盏。]

**红叶：** 第一，商务管理权。你们打下的地盘迟早要恢复经济运转。粮食要流通，商税要征收，货币要稳定。这些事情——恕我直言——你们不擅长。我擅长。给我管理权，六个月内，我能让你们控制区的税收翻倍。

**红叶：** 第二，一个头衔。不需要多大——商务参事、通商使、随便叫什么都行。我需要一个名义上的身份来和各地商人打交道。没有头衔，他们不会信任一个通缉犯。

[她停了一下。]

**红叶：** 第三——

[她的声音忽然轻了下来。锋利的谈判口吻消失了，露出底下一层柔软得不太自然的东西。]

**红叶：** 一杯好茶。

[她低头看了看手中的茶盏。]

**红叶：** 这几天只喝过雨水和军营的粗茶。我想喝一杯好茶。

[她抬起头，嘴角弯出一个弧度。那个弧度里有疲惫，有自嘲，有一丝几乎看不见的——恳求。]

**红叶：** 你看——这大概是有史以来最不对等的交易了。一整个王朝的情报——换一份工作、一个虚名、和一杯茶。

[她将茶盏放回桌角。盏底发出一声极轻的响。]

*——这确实是一笔不对等的交易。但不对等的方向——和她说的相反。*

*——她给的太多了。不是因为她不知道自己手里东西的价值——她比任何人都清楚。而是因为她现在需要的东西，本来就不需要那么高的代价。*

*——一份工作。一个名分。一杯茶。*

*——她已经不再计算利润了。她只是在找一个可以重新开始的地方。*

**红叶：** 当然，如果你觉得这些条件太高——

[她故意停了一拍。商人的本能还在，哪怕在这种时候也会预留谈判空间。]

**红叶：** 茶叶我可以自己想办法。

### 尾声

[谈判结束后。]

[红叶走到帐篷门口，将文书卷好留在了桌上。她站在门帘前，背对着帐内，似乎在犹豫要不要回头说点什么。]

[最后她没有回头。只是——]

**红叶：** ……谢谢。

[又是这两个字。但这次说得比上一次——在柳溪谷断桥下那次——流畅了一点点。只有一点点。]

[她掀开门帘走了出去。晨雾还没散尽，阳光从东边的山头后面透出来，在薄雾中折射出一片柔和的金色。]

[帐外，阿桐在等她。]

**阿桐：** 夫人？

**红叶：** ……成了。

**阿桐：** 都答应了？

**红叶：** 嗯。包括茶叶。

[阿桐愣了一下，然后笑了。]

**阿桐：** 夫人还是那个夫人。谈什么都不忘加一条茶叶。

**红叶：** 这是原则问题。人可以没有钱，不能没有好茶喝。

[她说这话的时候，语气里有一种久违的——轻松。不是装出来的那种。是真正的、从肩膀上卸下了什么重物之后的轻松。]

[两人并肩走在营地中。晨雾渐散，炊烟升起，士兵们的谈笑声在远处此起彼伏。红叶走了几步，忽然停下来。]

**红叶：** 阿桐。

**阿桐：** 嗯？

**红叶：** 你还记得我以前做大买卖的时候是什么感觉吗？签下一笔几万金的合同——心跳加速，手心出汗，脑子里全是数字在飞。

**阿桐：** 记得。夫人每次签完大单都要连喝三壶茶。

**红叶：** 嗯。

[她望着东方渐渐亮起来的天空。]

**红叶：** 今天这笔——

[她想了想。]

**红叶：** 今天这笔没有心跳加速。也没有数字在飞。只有——

[她将双手拢在袖子里。晨风有一点凉。]

**红叶：** 只有一种很安静的感觉。像是——终于把一本乱了的账整理好了。

[她重新迈开脚步。]

**红叶：** 走吧。去看看周叔。然后——去找一壶好茶。我有预感，他们的后勤仓库里一定藏着好东西。

**阿桐：** 夫人，您现在还没正式上任呢——

**红叶：** 所以要趁着还没上任先去翻仓库。上任了就不好意思了。

[阿桐无奈地笑了。两个人的身影消失在渐渐热闹起来的营地深处。]

## Event 12 — 防守击退「商队的防线」

| 项目 | 数据 |

### 场景描写

南境通商官道，枫鸣堡以南三十里处的"锦带坡"。

这段官道是南境最重要的商路之一——从枫鸣堡出发，沿山脚蜿蜒南下，穿过一片半丘陵半平原的地带，最终通向南方诸国的边贸市场。官道两侧曾经种着整排的银杏树，秋天落叶时像是铺了一条金色的地毯。但战乱以来疏于维护，银杏树倒了不少，路面也坑坑洼洼，积水和泥浆让马车走得颤颤巍巍。

红叶正式接管暗潮控制区的商务工作后做的第一件事，是重新开放这条商路。她亲自写了三十封信——是的，三十封，用她那手漂亮的行书——分别寄给散落在南境各处的商人们，告知他们："商路安全有保障，过路费减半，损失由我个人担保赔偿。"

信的落款是"暗潮军团商务参事·红叶"。新头衔后面还加了一行小字："如对商路安全有疑虑，可致信本人。我的信用虽然不值钱了，但我的账本还在。"

第一支敢走这条路的商队在今天出发了。

六辆满载货物的马车排成长列，缓缓行进在锦带坡的官道上。车上装的是南方诸国的香料、药材和布匹——这些物资在战区价值翻倍，对暗潮控制区的经济恢复至关重要。商队的掌柜是一个四十多岁的矮胖男人，姓钱，人称"钱半城"，是南境商界的老人了——也是第一个回应红叶那封信的人。

红叶骑着一匹借来的栗色马走在商队旁边。她没有穿铠甲——她本来就不是武将——而是穿着那件素白长衫，外罩浅灰短褂，腰间别着一卷地图和那本永远不离身的账本。她的身后跟着六名商队自带的护卫，以及暗潮军派来的先锋小队。

**钱半城：** 红叶姑奶奶——不，红叶大人——这条路真的安全？上个月流岩寨的人还在这一带劫了两支商队呢。

**红叶：** 流岩寨在柳溪谷被打散了一次，伤了元气。但没有被彻底剿灭——他们的据点在北面的山沟里，派兵清剿不划算。

**钱半城：** 那……

**红叶：** 所以我来了。

[她拍了拍腰间的账本。]

**红叶：** 我答应了三十个商人，商路安全由我担保。第一支商队如果被劫了——我的信用就完了。一个商人的信用完了，比没了命还严重。

**钱半城：** ……可您现在手上也没多少兵啊。万一——

**红叶：** 万一的事我已经算过了。

[她朝前方的丘陵扫了一眼。目光在几个制高点上停留了一瞬——那些位置已经提前布置好了暗潮军的哨兵。]

**红叶：** 走吧，钱掌柜。到了南渡口，我请你喝茶。

### 战斗阶段

[商队行至锦带坡中段，官道在这里拐了一个弯，两侧是矮丘陵和倒伏的银杏树。视野受限。]

[红叶的眉头皱了一下。]

**红叶：** ……停。

**钱半城：** 怎么了？

**红叶：** 安静。

[她侧耳倾听。风声、马蹄声、车轮碾过碎石的声音——然后，一个不该出现的声音。]

**红叶：** 鸟。

**钱半城：** 啊？

**红叶：** 刚才路过那片银杏林的时候有鸟叫。现在没有了。鸟不叫了——说明有人把它们惊走了。

[她拉住马缰。]

**红叶：** 全队停车！护卫列阵！前方有——

[话没说完，两侧丘陵上同时响起了尖锐的哨声。]

[山匪从灌木丛中冲出来。]

[第1回合：山匪从东西两侧夹击。骑兵直扑商队前端，试图冲散车队；弓手从丘陵高处抛射箭雨，覆盖中段；步兵从后方包抄，切断退路。]

**山匪首领：** 弟兄们！六辆大车！发财了——！

**红叶：** 不要慌！

[她的声音穿透了混乱——清晰、稳定、不容置疑。]

**红叶：** 车队分三组！前两辆靠右，中间两辆原地不动，后两辆靠左——拉开间距！不要让他们一锅端！

[商队护卫迅速按指示调整阵型。六辆马车分散到三条路线上，增大了山匪同时攻击的难度。]

**红叶：** 护卫分两队——甲队守前，乙队守后。中间的车交给暗潮的人。

[她调转马头，退到队伍中段的一辆马车旁边。从车上抽出一面旗帜——银甲商会的天平旗——将旗杆插在车辕上。]

**红叶：** 让他们看到旗。

**商队护卫甲：** 看旗有什么用——

**红叶：** 流岩寨的人知道这面旗。他们的前任寨主和我做过三年生意——我供他们盐和铁，他们替我看路。后来换了新寨主才断了往来。但老人还认这面旗。看到旗，至少有人会犹豫——犹豫一秒就少死一个人。

[第2回合：天平旗升起。果然，冲在最前面的几个山匪看到旗帜后明显减速了。有人在喊："是银甲的旗——那不是红叶的人吗？"]

**山匪首领：** 管她是谁！红叶都落魄成这样了——还能翻什么浪？给我冲！

[犹豫只持续了几秒。但几秒已经足够——商队护卫利用这几秒的空隙完成了防御部署。]

[「商人的信用」技能激活。红叶在后方指挥——每击败一名山匪，全体友军获得短暂的ATK+1提升，同时累积金币奖励。]

**红叶：** （高声）护卫听令——每打倒一个，记五十金！战后现结！

**商队护卫乙：** 当真？！

**红叶：** 红叶的账本上写的数字，什么时候赖过？

[护卫们的士气肉眼可见地涨了一截。挡在马车前面的两个护卫吼了一声，将一个试图爬上车辕的山匪踹了下去。]

[第3回合：山匪弓手从高处集中射击中段车辆。一支箭射穿了第三辆车的帆布顶棚，钱半城抱着头钻到了车底。]

**钱半城：** 我的货——！我的香料——！

**红叶：** 钱掌柜！人比货重要！蹲好别动！

**钱半城：** 那一车香料值八百金——！

**红叶：** ……你的命值多少？

[钱半城噎住了。]

**红叶：** （对暗潮军先锋）弓手在东侧丘陵第二层台地上！绕过去从侧面打——他们只看着路上，不防侧后！

[第4回合：山匪增援抵达。四名山匪精锐从北面山沟中杀出——他们是流岩寨最后的底牌，装备比普通山匪好得多。]

**山匪首领：** 老子的精锐来了！看你怎么挡——

[红叶做出了一个所有人都没预料到的决定——她从后方指挥位置走了出来。]

[不是冲锋。她不会冲锋。她骑着那匹栗色马，从中段缓缓走到前线和后方之间的位置——整个战场的正中央。]

**商队护卫甲：** 大人！太危险了——

**红叶：** 我需要看到全局。后方看不到西侧的情况——那里有死角。

[她站在战场中央，目光快速扫过三个方向。手里翻开账本——不是在记账，而是在用账本背面的空白页画战场草图。]

**红叶：** 听我指挥！

[全队INT+3战术支援激活。]

**红叶：** 西侧那四个精锐不要硬碰——引到银杏树桩那里，树桩之间骑兵施展不开！前方的步兵已经散了一半，集中力量先清后方——后方一通，退路就有了！弓手已经被你们的人逼下了台地，不用管他了！

[她的指令如同流水——每一条都精准地指向战场上最关键的节点。不是将军式的热血指挥，而是商人式的冷静调度——把每一份战力像金币一样精确地分配到最需要的地方。]

[第5回合：战局逆转。山匪精锐被引入银杏树桩区域后行动受限，被暗潮先锋队围歼。后方的包抄被击退，退路打通。]

**山匪首领：** ……撤！撤！不打了——

[第6回合：山匪全面溃退。流岩寨余党消失在北面的山沟中。]

[战斗结束。六辆马车——完好四辆，轻微受损两辆（帆布被射穿、车轴歪了，但货物无大碍）。人员：护卫轻伤三人，无人阵亡。]

### 战后

[锦带坡。战场上散落着山匪丢弃的武器和几顶破毡帽。商队的马在路边打着响鼻，护卫们坐在地上喘气。]

[钱半城从车底下钻出来，满头灰尘，但眼睛瞪得溜圆。]

**钱半城：** ……没死？都没死？

**红叶：** 都没死。

[她下了马。腿在微微发抖——她不是武将，在战场中央站了那么久，肾上腺素退去后身体开始算总账。]

**钱半城：** 货呢？货怎么样？

**红叶：** 四辆完好，两辆小伤。你那车值八百金的香料——

[她走到第三辆车旁边，掀开被箭射穿的帆布。里面的香料包完好无损——箭射穿帆布后插在了一个空木箱上。]

**红叶：** 一根箭毛都没碰到。运气好。

**钱半城：** （长出一口气）菩萨保佑……红叶大人保佑……

[红叶没有理会钱半城的念叨。她走到护卫队中间，翻开账本。]

**红叶：** 清点战果。

[护卫们报了数。]

**红叶：** ……击败二十二人。按每人五十金计——

[她在账本上写了一笔。]

**红叶：** 一千一百金。现在没有现钱，记在我的账上。到南渡口后从商税中支取。

**商队护卫乙：** 大人，这钱——您从哪儿出？您不是说自己身无分文——

**红叶：** 身无分文是以前的事了。我现在有头衔、有薪俸——虽然还没发第一个月的——更重要的是，我有信用。

[她合上账本。]

**红叶：** 钱掌柜。

**钱半城：** 在在在——

**红叶：** 回去之后帮我传个话。就说——第一支商队安全到达了。路上遇到了匪，但货和人都没事。

**钱半城：** 这个当然——

**红叶：** 然后加一句：红叶说了，以后走这条路，她负责。

[钱半城看着她。战后的红叶——衣衫上沾了尘土和马汗，头发被风吹散了一半，嘴唇干裂，但眼神里有一种钱半城在很多年前——在银甲商会鼎盛时期——见过的东西。]

**钱半城：** ……红叶大人。我做了三十年生意。见过的商人比我吃过的盐还多。

**红叶：** 嗯？

**钱半城：** 您是第一个亲自上战场保自己商队的。

[红叶愣了一下。然后她低头看了看自己——满身尘土、手指发抖、站都站不稳——确实不像一个商人该有的样子。]

**红叶：** ……以前不用。以前有钱，可以雇人。现在没钱了——只好自己来。

[她停了一下。]

**红叶：** 不过——

[她望向那六辆马车。车上的货物在阳光下闪着暖色的光——香料的气味在风中弥散，混着尘土和汗水的味道。]

**红叶：** 自己来的感觉……也不坏。

*——以前保护商队是为了利润。今天保护商队是因为——钱掌柜信了我的话。那三十个商人信了我的话。*

*——他们信的不是银甲公爵夫人的权势，也不是四十万金的身家。他们信的是——*

*——一封信。一个承诺。一个已经一无所有的人说出的"我负责"三个字。*

*——这种信任比金子重。也比金子脆。所以更要用命去保。*

## Event 13 — 角色事件·陆「新的账本」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情 |
| 情感基调 | 自动加入触发——不再是交易，而是归属 |
| 特殊事件 | 红叶自动加入队伍，成为永久角色（外交型加入，跳过事件14-15） |
| 奖励 | 400 EXP, 新账本×1, CG「新的账本」 |

### 场景描写

暗潮军团营地，红叶的临时居所。深夜。

说是"居所"，其实就是一顶普通的军帐——和士兵们住的一样大，一样简陋。帐内一张行军床、一张折叠桌、一把椅子。墙角堆着几卷地图和一摞文书。桌上有一盏油灯、一方砚台、几支毛笔，和一只半满的茶壶。

帐篷顶上有一个小洞——不知是什么时候磨穿的——正好够一根手指粗细。月光从那个洞里漏进来，在地面上投下一个亮晶晶的小圆点，像一枚掉在地上的银币。

红叶坐在桌前。

油灯的火苗被穿帐而过的夜风吹得东倒西歪，她的影子在帐壁上忽大忽小，像一个不安分的幽灵。她穿着一件半旧的寝衣——阿桐用旧布缝的，接了两块不同颜色的补丁，但洗得很干净。长发披散着，垂在肩头和椅背上。

她面前摊着两本账本。

一本是旧的——那本从银甲城大火中抢出来的、被雨水泡过的、皱巴巴的、缺了角的老账本。纸页发黄翘曲，墨迹大半模糊，但最后一页里夹着的那朵桃花还在——褪了色的，几乎透明的，脆弱得像一声叹息。

另一本是新的。

新账本的来历有点寒酸——是红叶用军营里废弃的公文纸自己裁的，阿桐帮她用针线缝好了书脊，封面是一块裁剪整齐的灰布。纸张粗糙，边缘参差不齐，有些页面上还能隐约看到被裁掉的半截公文字迹。

但红叶已经在第一页上端端正正地写了四个字：

**新账·卷一**

新账本目前是空的。她一直没有动笔——从拿到这本账本到现在已经三天了，她还没写下第一笔。

今夜她要写了。

### 剧情对话

[红叶将旧账本翻到最后一页。那朵桃花在油灯的光下显出一种几乎透明的淡金色。她看了它很久。]

**红叶：** ……小枫。

[声音极轻。像是怕吵醒什么人。]

**红叶：** 妈妈要开一本新账了。

[她将桃花从旧账本里小心翼翼地取出来，放在桌面上。然后拿起新账本，翻开第一页。]

[提笔。蘸墨。]

[笔尖悬在纸面上方。]

[停了很久。]

*——以前开新账本从来不犹豫。第一行永远是日期和开户金额。"某年某月某日，初始资金若干金。"然后是资产清单、预期收益、风险评估——一切都是数字，清晰、冷静、可计算。*

*——但这本新账……我不知道第一行该写什么。*

*——写什么呢？"初始资金：零金"？"资产清单：无"？*

*——不。这次不是那种账本了。*

[她深吸一口气。落笔。]

[写下的不是数字。是一个名字。]

**"周叔。"**

[她看着这两个字。笔尖上的墨还没干，在灯光下微微发亮。]

[然后她继续写。]

**"阿桐。"**

**"方小河。"**

**"陈七。"**

[四个名字。一行一个。字迹工整——和她写任何一个金额数字时一样工整。但笔画之间有一种不同的力度。写数字的时候，笔是快的，精准的，像算盘珠子噼啪作响。写名字的时候，笔是慢的，沉的，每一划都像是在石头上刻字。]

[她停了一下。]

[又写了几个名字。]

**"许叔。"**

——枫鸣堡的老管家。她走了之后他留在了堡里，暗潮军同意让他继续做管家。

**"林大壮。"**

——那个在青石关替她去找行商护卫的残兵。后来听说他平安了，跟着那支行商队伍去了南方。

**"钱半城。"**

——第一个相信她的商人。

[名字一个一个地写下去。纸页上慢慢排满了人名——不是按照资产价值排列的，也不是按照职位高低排列的。是按照她想起来的顺序。]

[写着写着，她停了。]

[笔尖在下一行的开头悬停。]

*——还有一个人。*

*——那个人的名字写不写？*

*——按照商业逻辑，不应该写。因为那个人不是我的员工，不是我的客户，不是我的合伙人。从任何一个账本的分类来看，那个人都不属于我的资产负债表。*

*——但是——*

*——在柳溪谷，他救了我。在营地门口，他收留了我。在帐篷里，他听完了我的条件，然后答应了——不是因为我给的情报值多少钱，而是……*

*——我不知道是为什么。我算不出来。我试过了。算不出来。*

*——也许这就是应该写的原因。*

[她落笔。]

[在所有名字的最下方，空了一行，写了最后一个名字。]

[然后在名字的后面——用极小的字——加了两个字。]

**"——账户持有人。"**

[她看着这一页。满满一页的名字。]

*——这是我的新账本。*

*——资产清单：无。初始资金：零。*

*——但是——*

[她将那朵桃花从桌面上拿起来，夹进新账本的最后一页。和旧账本里的位置一样——最后一页。]

*——但有这些人。*

*——他们不是数字。他们不是资产。他们是——*

[她合上账本。双手按在封面上。灰布封面下面是粗糙的公文纸，针线缝的书脊，和一整页的人名。]

**红叶：** ……名字。

[她自言自语。]

**红叶：** 原来账本里最值钱的东西，是名字。

[帐篷外面传来巡逻士兵的脚步声，踩在碎石上沙沙作响。远处有人在轻声哼歌。夜风穿过帐篷顶上那个小洞，把油灯的火苗吹得一歪——月光的那枚"银币"在地面上微微晃动。]

[红叶将账本抱在怀里——和她在暴雨中抱着旧账本逃出银甲城时一模一样的姿势。但这一次，她不是在逃。她坐在一张虽然简陋但安全的椅子上，头顶有帐篷遮着风雨，桌上有还没凉的茶。]

[她靠着椅背，闭上了眼睛。]

**红叶：** ……就这样吧。

[油灯的火苗终于安静了下来——风停了。帐篷内恢复了温暖的、橘黄色的光。]

[红叶抱着新账本，在椅子上睡着了。]

[桌上的旧账本摊开着，最后一页空空荡荡——桃花已经搬家了。]

### 加入触发

[第二天早晨。]

[红叶走进主帅帐。她手里拿着新账本，翻到最后一页——写着名字的那一页。]

[她将账本翻转过来，正对着帐内的人，让他看到最下方那行字。]

**红叶：** 我想了很久，不知道该用什么方式正式加入。写一份投效书？太正式了。送一份见面礼？我没钱。说一番慷慨激昂的话？我不擅长。

[她将账本合上。]

**红叶：** 所以我只做了一件我擅长的事——开了一本新账。

[她将账本放在行军桌上。]

**红叶：** 第一页写的是所有留在我身边的人。最后一行是你的名字。

[她直视对方。那双眼睛不再是银币的冷，也不是琥珀色的脆弱——而是一种更深的、更稳定的颜色。像是深秋的枫叶在落地之前最后呈现的那种红——暗的，沉的，不再飘摇。]

**红叶：** "账户持有人"——这是商人之间最高规格的信任。意思是，我把我所有的一切——所有的能力、所有的情报、所有的未来——存在你这里。不设取款密码，不设到期日。

[她顿了一下。]

**红叶：** 当然，也不设利息。

[嘴角弯了一下——终于有了一点红叶式的幽默。]

**红叶：** 所以——从今天起，我的账本就是你的账本。我的计算就是你的计算。

[她伸出手。]

**红叶：** 成交？

[一个握手。一笔没有金额的交易。一本写满名字的新账本。]

*——红叶正式加入队伍。*

*——不是因为被打败了。不是因为被收买了。不是因为走投无路别无选择。*

*——而是因为她在账本的最后一行写下了一个名字，然后发现——自己是心甘情愿的。*

### 尾声

[红叶走出主帅帐。阳光正好。]

[周叔拄着拐杖等在外面。]

**周叔：** 夫人，成了？

**红叶：** ……别叫我夫人了。公爵府都没了。

**周叔：** 那叫什么？

**红叶：** ……红叶就好。

[周叔笑了。]

**周叔：** 红叶。好听。比"公爵夫人"好听。

[阿桐从旁边冒出来。]

**阿桐：** 夫——红叶姐，我昨天在后勤仓库里找到了一包铁观音。不是顶好的，但——

**红叶：** 泡上。

**阿桐：** 给谁泡？

**红叶：** 给所有人泡。

[她望向营地。晨光下，士兵们在操练，马匹在饮水，炊烟从各处升起。这个营地不是银甲城，不是枫鸣堡，不是任何一个她曾经拥有过的地方。但此刻——]

**红叶：** 给所有人泡。这是我请的。

[她拍了拍怀里的新账本。]

**红叶：** 记在新账上——"支出：铁观音一包。用途：庆祝开户。"

## Event 16 — 征服完成「第一笔新账」

| 项目 | 数据 |

| 类型 | 结算 |
| 情感基调 | 不止是交易——有些东西的价值无法计算 |

### 场景描写

天城王朝南境，重建中的银甲城。一个月后。

银甲城正在从废墟中醒过来。

城墙还没有完全修复——北段坍塌的部分用木栅栏临时围了起来，上面挂着"施工中"的牌子。但南段的城墙已经重新刷了白灰，在午后的阳光下亮得晃眼。城门口的石狮子被熏黑了一半，有个年轻石匠正蹲在旁边，一锤一凿地修补狮子缺掉的耳朵。

街道上恢复了人气——虽然还不到战前十分之一，但已经有商铺开始重新营业了。卖早点的摊贩支起了油布棚，蒸笼里白气腾腾，包子的香味飘了半条街。有个小女孩蹲在路边用树枝在地上画画，画的是一匹马和一面旗——她大概觉得画的是天平旗，但实际上更像一只蝴蝶。

银甲商会总部的原址上只剩一片焦黑的地基。红叶没有在原址重建——她在城东靠近码头的位置租了一间旧仓库，挂上了一块新招牌。招牌是她自己写的字，黑底银字：

**「新银甲商务局」**

旧仓库不大，两层，木质结构，楼梯踩上去吱呀响。一楼被改成了办公区，几张拼起来的长桌上堆满了文书、地图和算盘。墙上贴着南境商路重建规划图，用红蓝两色的线标注已恢复和待恢复的路线——红色远多于蓝色，但蓝色的线正在一天天减少。二楼是红叶的私人办公室兼住处，窗户正对码头，可以看到河上来来往往的船只。

此刻是下午。红叶坐在二楼办公桌前，面前摊着一份厚厚的报告。

### 剧情对话

[阿桐端着一壶茶走上来。楼梯吱呀了三声。]

**阿桐：** 红叶姐，茶。铁观音，新到的——钱掌柜送来的，说是南山坡今年的头茬。

**红叶：** 放这里。

[她没有抬头。手里的毛笔在报告上勾勾画画，时不时在旁边的纸上写几个数字。]

**阿桐：** 还在写审计报告？

**红叶：** 快写完了。最后一部分。

**阿桐：** 您已经写了三天了……

**红叶：** 南境经济烂了十二年——不对，烂了不止十二年，我在的时候表面光鲜，底子一样烂。只不过我用商路的利润把窟窿糊住了。现在窟窿全露出来了——糊是糊不住了，得一个一个填。

[她翻了一页。]

**红叶：** 银甲城的商税体系要重建。原来的税制有二十七项附加税——二十七项！卖一斤米要交五种税，其中两种是宰相私设的。我全砍了，只留了四项基础税。

**阿桐：** 那税收会不会不够？

**红叶：** 短期会。但税低了，商人才愿意回来。商人回来了，交易量上去了，总税收反而会超过原来。这笔账——

[她在纸上写了几行算式。]

**红叶：** 六个月。六个月后税收会持平。十个月后翻倍。我算过了。

**阿桐：** ……红叶姐对数字的自信，十二年没变过。

**红叶：** 数字不会骗人。骗人的是用数字的人。

[她放下笔，拿起茶壶给自己倒了一杯。茶汤清澈，带着淡淡的兰花香。她端起来，先闻了闻——闭上眼睛，深吸一口气——然后才啜了一小口。]

**红叶：** ……好茶。钱半城这回没糊弄我。

[她将茶盏放下，继续写报告。]

[写了约莫半个时辰。窗外的阳光从正西移到了偏西，码头上的影子拉长了。远处有船在鸣笛。]

[红叶终于写完了最后一行。]

[她将毛笔搁在笔架上，拿起报告，从头翻到尾，逐页检查。]

**红叶：** 第一章，南境经济现状评估。第二章，商税改革方案。第三章，商路重建时间表。第四章，人力资源配置。第五章，粮储调配计划。第六章，风险预警和应急预案。第七章——

[她翻到最后一章。]

[顿了一下。]

**红叶：** 第七章。审计总结。

[她将报告最后一页摊开在桌面上。那一页上写满了数据——南境各项经济指标的汇总、预期收益曲线、投入产出比分析——一切都精确到个位数，一切都条理分明。]

[但最后一行——]

[最后一行不是数字。]

[红叶拿起报告，读出了最后一行。声音不高，像是说给自己听，也像是说给窗外的河水和远处的船笛听。]

**红叶：** "净利润——"

[她停了一拍。]

**红叶：** "无法计算。"

[阿桐在旁边整理文书，听到这句话手里的动作停了。]

**阿桐：** 红叶姐……您从来不写"无法计算"的。

**红叶：** 嗯。第一次。

[她将报告放在桌上，靠回椅背。目光从窗户穿出去，落在码头上——几个脚夫正在卸货，吆喝声此起彼伏。一个卖茶水的老太太坐在码头边上，面前的铜壶冒着热气。有个穿着旧军装的老人拄着拐杖慢慢走过——那是周叔，他每天下午都去码头散步，说是"替夫人看看船来了没有"。]

**红叶：** 阿桐。

**阿桐：** 嗯？

**红叶：** 你觉得——

[她端起茶盏。茶还是温的。]

**红叶：** 一杯茶值多少钱？

**阿桐：** 这杯？钱掌柜说是一两银子一斤的——

**红叶：** 不是问市价。

[她望着茶盏里的茶汤。清澈的液面映着窗外的天光，也映着她自己的脸——比一个月前多了些肉，气色也好了不少。]

**红叶：** 一杯茶——如果是在逃亡路上，三天没喝过热水的时候喝到的那一杯——值多少？

**阿桐：** ……

**红叶：** 如果是在青石关，周叔差点没命的那天之后，终于安定下来喝到的那一杯——值多少？

**阿桐：** 红叶姐——

**红叶：** 如果是有人——

[她的声音变得很轻。]

**红叶：** 如果是有人，什么都不求、什么都不问，只是把一杯热茶推到你面前——值多少？

[窗外的河面上有一阵风吹过，波光粼粼。码头上周叔的身影走到了卖茶水老太太面前，停下来，大概是在买茶。]

**红叶：** 我算了十二年的账。算过数十万金的买卖，算过王朝的税收，算过商路的利润率。但这一杯茶的价格——

[她将茶盏举到嘴边，浅浅啜了一口。]

**红叶：** 算不出来。

[她将茶盏放回桌面。拿起笔，在报告最后一行"无法计算"的后面又添了一句话。]

[阿桐凑过来看。]

**报告最后一行（完整版）：**

*"净利润：无法计算。品目：茶一杯，与不曾索要回报之人共饮。备注：此为本账本迄今为止最大的一笔收入。"*

[阿桐看了很久。然后她红了眼眶——但没有哭出来。她只是笑了一下。]

**阿桐：** ……红叶姐，您变了。

**红叶：** 变了吗？

**阿桐：** 以前的红叶姐不会写"无法计算"。

**红叶：** 以前的红叶姐也不会在审计报告里写茶。

[她将报告合好，压上镇纸。]

**红叶：** 这份报告今晚送过去。明天开始执行。南境的烂摊子——

[她站起身，走到窗前。]

[码头上，周叔买了两杯茶，一杯自己端着喝，另一杯放在码头的石墩上——他总是多买一杯，说是"留给夫人的"，虽然红叶几乎不去码头。但那杯茶每天都在那里等着。]

**红叶：** 南境的烂摊子会好的。给我时间。

[她回过头，拿起桌上那本新账本——写满名字的那一本。翻到最后一页，在桃花旁边的空白处添了今天的日期，然后写了一行字。]

**"第一笔新账。收入：一杯茶。支出：十二年。结余：足够。"**

[她合上账本。]

**红叶：** 阿桐，走。去码头。

**阿桐：** 做什么？

**红叶：** 周叔每天给我留一杯茶，我一次都没去喝过。今天去喝。

[她将账本塞入怀中，推开门走了出去。楼梯又吱呀了三声。]

[仓库门外的街道上，阳光正从屋檐间倾泻下来。卖早点的摊贩在收摊，那个画画的小女孩已经不在了，但她用树枝画的蝴蝶——或者天平旗——还留在地面上。]

[红叶踩过那幅画，朝码头走去。]

[风从河面上吹来，带着水汽和鱼腥味——还有一丝若有若无的茶香。]

[码头上，周叔远远看到了她，咧嘴笑了。缺了两颗门牙的笑。]

[他拿起石墩上那杯已经有点凉了的茶，朝她举了举。]

[红叶加快了脚步。]

*——有些东西的价值，不能用金子算。*

*——她花了十二年才学会这句话。*

*——但学会了。*
