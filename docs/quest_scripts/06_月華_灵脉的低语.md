## Event 01 — 开战宣言「灵脉的颤抖」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.14-17 |
| 情感基调 | 半闭的眼睛——超然而悲悯，仿佛在倾听远方的声音 |
| 战斗条件 | 击败月華灵脉守卫阵列（需在8回合内突破灵脉结界） |
| 特殊机制 | 月華「灵脉感应」：预知攻击方向，前2回合所有友方单位DEF+2；「灵脉之壁」每回合恢复全体5%HP |
| 敌方配置 | 月華(Lv.17) + 灵脉祭司×3(Lv.15) + 银月卫兵×6(Lv.13) + 灵树卫士×2(Lv.14) |
| 奖励 | 400 EXP, 900G, 灵脉碎晶×1, 解锁月華档案第一页 |

### 场景描写

银月议庭领地的最外缘——翡翠回廊。

这是一片古老到近乎不真实的森林。千年巨树的树冠彼此交织，形成一道又一道天然的穹顶，将日光切割成无数细碎的光斑洒落在苔藓覆盖的地面上。空气中浮动着淡淡的磷光——那是灵脉的气息，从地底深处沿着古树的根系渗透而出，令整座森林仿佛浸泡在一层稀薄的月光里。

每隔数十步便有一块灵脉标石。表面刻满精灵文字的花岗岩柱，顶端嵌着指甲盖大小的月白色宝石，明灭闪烁，如同森林的脉搏。若将耳朵贴近地面，隐约能听到一种极低极沉的嗡鸣——像是大地在呼吸。

一棵最古老的巨树根部，被掏空的树洞形成了一个天然的祭坛。月華坐在祭坛前的石台上。

她的姿态极其特殊：双腿盘起，双手覆在膝盖上，银色的长发垂落两肩如同流淌的水银。最引人注意的是她的眼睛——半闭着。不是困倦，不是冥想，而是一种介于"看"与"听"之间的状态。她的嘴唇极其微弱地翕动，仿佛在与某种不可见的存在对话。

灵脉标石上的月白宝石，随着她嘴唇的翕动一明一暗，节奏完全一致。

远处，暗潮军团的斥候已经触碰了翡翠回廊的外围结界。

### 剧情对话

[月華的眼皮微微一颤。她身旁站着的灵脉侍女——芒，立刻弯下腰。]

**芒：** 大人？

**月華：** ……七。

**芒：** 什么？

**月華：** 外围结界被触碰了七次。西北方向，间隔不规则。不是野兽，野兽会绕开灵脉标石。

[她的眼睛仍然半闭着。声音很轻，如同从水底传来。]

**月華：** 是人类。大约……二十到三十人。其中两个踩踏的方式比较重，可能穿着铁甲。其余的脚步很轻——斥候或轻步兵。

**芒：** 大人，您的意思是——

**月華：** 暗潮来了。

[她说这句话的语气极其平淡，仿佛在陈述"今天有露水"。]

[一名银月卫兵急匆匆地从林道跑来，单膝跪下。]

**银月卫兵：** 禀报灵脉祭司大人！西北外围发现暗潮军团斥候！数量不明，但后方尘烟浓重，主力部队应在半日行程之内！

**月華：** 二十三人。其中五人携带弓弩，两人重甲，其余轻装。主力……

[她歪了歪头，像是在倾听什么。灵脉标石的光芒急促闪动了几下。]

**月華：** 主力约两千。骑兵不多，大部分是步兵。他们的军靴在践踏灵脉，大地告诉我……它们很痛。

[卫兵愣住了。芒在旁边轻声解释。]

**芒：** 月華大人与灵脉相连。整片翡翠回廊的一草一木、一石一泉，都是她感知的延伸。

**月華：** 芒。

**芒：** 在。

**月華：** 通知议庭。翡翠回廊需要防御。

[她终于睁开眼睛。是纯粹的金色，瞳孔在磷光中收缩成极细的竖线——精灵的眼睛，在灵脉浸润千年后已经不像凡人的眼睛了。]

**月華：** 然后……

[她站起来。动作很慢，仿佛从一个极深的梦境中一层层剥离出来。银色长发从石台上滑落，尾端拂过苔藓地面。她比想象中要高，也比想象中要瘦——灵脉祭司的身体更像是灵脉本身的容器，而非战士的武器。]

**月華：** 告诉来犯者。这片森林不是战场。

*——大地又在说话了。它说："有人来了。"它说得很害怕。没关系，我在听。我一直在听。*

[银月卫兵领命离开。月華缓步走向翡翠回廊的主干道。每走一步，脚下的苔藓便轻轻发光，如同大地在回应她的脚步。灵脉标石依次亮起，从她身后向远方延伸，像一条被点燃的引线。]

[暗潮先锋部队已经推进到回廊的第一道灵脉标石线。领队是一名面容冷硬的暗潮百夫长。]

**暗潮百夫长：** 什么鬼地方？石头会发光，树根像蛇一样乱动……

**暗潮士兵甲：** 百夫长，前方有人！

[月華出现在林道尽头。磷光环绕在她周围，银发在暗绿色的森林中亮得刺眼。]

**月華：** 你们踩到灵脉标石了。

[语气依然平淡得不像是在面对入侵者。]

**暗潮百夫长：** ……什么？

**月華：** 你脚下。那块刻着文字的石头。你踩到它了。它已经在那里一千四百年。请你挪开。

[百夫长低头看了一眼，嗤笑一声。]

**暗潮百夫长：** 听好了，精灵。暗潮军团奉命征服此地。识相的话，交出翡翠回廊的地图和灵脉矿脉位置——

**月華：** 灵脉不是矿。

**暗潮百夫长：** ——否则我们的大军会在天黑前——

**月華：** 灵脉不是矿。

[她第二次说这句话时，声音没有提高，但脚下的苔藓忽然扩散出一圈刺目的白光。灵脉标石齐齐发出嗡鸣，如同一架巨大风琴的最低音被猛然按下。暗潮士兵们不由自主地后退了一步。]

**月華：** 灵脉是大地的血管。你们想挖开大地的血管，取走它的血液。我不允许。

[她抬起右手。手心朝上，掌中凭空凝结出一枚拳头大小的月白色光球。光球表面流转着精灵文字——那是灵脉的语言。]

**月華：** 我再说一次。请你……挪开脚。

[百夫长咬了咬牙，拔刀。]

**暗潮百夫长：** 全军列阵！攻击那个精灵！

*——他没有听。人类总是不听。灵脉说过，人类的耳朵太小，听不到大地的声音。可是……我必须试着让他们听到。哪怕只是痛。*

[月華闭上眼睛。不，不是完全闭上——还是那种半闭的状态。她的嘴唇翕动，灵脉标石的光芒从脚下向四面八方扩散。地面开始震动。]

**月華：** 灵脉——回应我。

[翡翠回廊的巨树根系从地底涌出，如同活物般在暗潮士兵面前形成一道又一道盘根错节的壁障。磷光编织成半透明的护盾，笼罩在银月守军身上。]

### 战斗阶段

[战斗开始。月華站在后方的灵脉祭坛附近，双手持续维持灵脉连接。「灵脉感应」发动——所有友方单位获得DEF+2，因为月華能够预知敌人的攻击方向并通过灵脉网络传递警示。]

*——第一波。从西北。三个人，两把刀一支弓。弓箭手会先射……不，他在犹豫。他看到了灵脉的光。他害怕了。可是另外两个不怕。他们冲上来了。*

[月華的声音通过灵脉传递到每一个银月卫兵的脚底，如同微弱的震动。]

**月華：** 左翼。两人。持刀。

[银月卫兵立刻调整阵型。暗潮刀兵冲到壁障前，被根系缠绕，攻势受阻。]

**灵脉祭司甲：** 大人的灵脉感应……真的能看到敌人的动向？

**芒：** 不是"看到"。是"感受到"。他们踩在大地上。大地把振动传给灵脉，灵脉传给月華大人。整片翡翠回廊就是她的眼睛。

[第3回合，暗潮投入重甲步兵试图强行突破根系壁障。月華皱了皱眉。]

**月華：** ……重的。他们的靴子钉了铁钉。灵脉标石被压碎了一块。

[她的语气中第一次出现了一丝不易察觉的波动——不是愤怒，更像是一种无法言说的心痛。]

**月華：** 灵脉之壁。

[地面涌出更密集的根系，编织成坚固的木质壁障。同时，灵脉的磷光流入每一位银月守军的身体，恢复5%HP。]

[遭受重击时——]

**月華：** ……

[她的身体晃了一下。芒立刻扶住她。]

**芒：** 大人！

**月華：** 灵脉……被切断了一小段。就像……手指被针扎了一下。没事。

*——痛。不是身体的痛。是大地的痛传过来了。每一个灵脉标石被破坏，都像是我自己的骨头被折断了一根。但我不能断开连接。如果我断开，翡翠回廊就只是一片普通的森林。*

[第5回合，暗潮百夫长亲自带队冲锋。]

**暗潮百夫长：** 砍那些发光的石头！那是她力量的来源！

**月華：** ……他很聪明。

[她睁开金色的眼睛，目光直视前方。]

**月華：** 但灵脉不是石头。灵脉在地下。在地下几百尺深的地方，像河流一样流动。你可以砍断树枝，但你砍不断树根。

[她双手合十，然后猛然向两侧展开。灵脉标石的光芒暴涨——整个翡翠回廊在一瞬间亮如白昼。]

**月華：** 灵脉·溢流。

[灵脉能量从地底喷涌而出，化作无数道月白色的光流在暗潮士兵之间穿梭。不是致命的攻击，而是一种强烈的压制——令他们双腿发软、头脑昏沉、战意消退。]

### 战斗结算

[暗潮先锋部队溃退。百夫长被灵脉光流弹飞，重重撞在一棵巨树上，爬起来时满脸惊恐。]

**暗潮百夫长：** 撤……撤退！这个精灵不是人——！

[月華缓缓收回双手。灵脉标石的光芒回落到正常的微弱闪烁。她站在原地，银发因灵脉能量的残余而轻轻飘浮。]

**月華：** 他们会回来。

**芒：** 是。下一次会带更多人。

**月華：** 嗯。

[她重新走向祭坛，坐下，恢复那个半闭眼睛的姿态。]

**月華：** 灵脉在说话。

**芒：** 它说什么？

**月華：** 它说……"还有更多脚步。从远处来。很多、很多。"

[她停顿了一下。]

**月華：** 它还说……"不只是人类。"

*——大地的低语从未停止。千年来我一直在听。听到季节的更替，听到河流改道，听到种子在土壤里翻身。可今天的声音不一样。今天的低语里有恐惧。千年来第一次……灵脉在害怕。*

[灵脉标石的光芒在最后一个音节后微微颤抖了一下，然后归于沉默。]

## Event 02 — 前哨遭遇「低语者」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.16-19 |
| 情感基调 | 预知的战场——用灵脉预见敌人的行动，却无法预见自己的未来 |
| 战斗条件 | 在月華的灵脉预知引导下击退暗潮前哨部队（保护3个灵脉标石不被破坏） |
| 特殊机制 | 「灵脉预知」：每回合开始时揭示敌方1个单位的行动意图；灵脉标石被破坏则月華INT-1（永久） |
| 敌方配置 | 暗潮斥候长(Lv.19) + 暗潮破法师×2(Lv.17) + 暗潮轻步兵×8(Lv.15) + 爆破工兵×3(Lv.16) |
| 奖励 | 450 EXP, 1000G, 灵脉预知之书×1, 解锁月華档案第二页 |

### 场景描写

翡翠回廊深处，灵脉标石密集分布的核心区——月牙泉。

地下灵脉在此处上涌，形成一汪浅浅的弧形泉池。泉水是半透明的月白色，夜间自行发光，远看如同一弯落入森林的新月。泉水表面不断有微小的气泡涌起，每一个气泡破裂时都会发出极轻极细的声响——像是有人在水底低声说话。

月華坐在月牙泉的边缘，赤足浸入泉水中。银色长发的末端也泡在水里，随着灵脉的脉动微微摆动。她的眼睛依然半闭着，但这一次，她的表情中多了一层此前没有的东西——专注之中夹杂着困惑。

三块核心灵脉标石以三角形排列在月牙泉周围。它们比外围的标石高出一倍，顶端嵌着的月白宝石有拳头大小，光芒强烈而稳定，如同三颗心脏在同步跳动。

远处传来树枝折断的声音。很轻，但在灵脉网络的放大下，对月華而言如同惊雷。

### 剧情对话

[月華的赤足在泉水中微微蜷缩了一下。]

**月華：** ……他们换了路线。

[芒正在给灵脉标石补充祈祷铭文，闻言抬头。]

**芒：** 从哪个方向？

**月華：** 东南。上一次是西北。他们在寻找灵脉网络的薄弱点。

[她从泉水中抽出双脚，站起身来。赤足踩在湿润的苔藓上，脚底的灵脉纹路微微发光。]

**月華：** 而且这一次……带了不同的人。

**芒：** 不同？

**月華：** 灵脉告诉我，有三个人的脚步声和其他人不一样。他们走路的时候，灵脉会……回避。

[她皱了皱眉，选择了一个更准确的词。]

**月華：** 灵脉在躲他们。就像血管躲开毒针一样。他们身上有某种……对灵脉有害的东西。

**芒：** 破法师？

**月華：** 比破法师更专门。暗潮军团研究过灵脉了。他们知道标石是节点。

*——大地在告诉我它害怕。不是害怕普通的脚步。是害怕那种能让它感到灼烧的脚步。暗潮的人类很快。上一次的失败让他们学到了东西。他们在适应……而灵脉不会适应。灵脉只会说话。只会请求。只会害怕。*

[一名银月卫兵从树冠上滑下，落地无声。]

**银月卫兵乙：** 灵脉祭司大人！东南方向确认暗潮部队，约三十余人！其中有三人穿戴不明装置，发出暗红色微光——我们的斥候靠近时，手中的灵脉导引石突然碎裂了！

**月華：** ……灵脉导引石碎了？

[她的金色眼睛完全睁开了。这是罕见的——月華几乎从不完全睁眼，因为灵脉网络的信息量过于庞大，全开的感知会让她头痛。但此刻，她需要看清。]

[只有月華看到了。她什么都没说。]

深夜。月牙泉。

月華独自坐在月牙泉中央的一块天然石台上。泉水刚好没过她的腰际。她没有穿祭司的正式法袍，只着一件素白的内衣，银色长发散在水面上如同一张银色的网。她的姿态和往常不同——不是端坐冥想，而是整个人后仰，双手撑在身后的石台上，面朝天空。

[芒抱着一件披风走到泉边。]

**芒：** 大人，夜深了。水凉——

**月華：** 芒。过来。把脚放进水里。

**月華：** 不。更深处。

**月華：** 那就是它。

**芒：** 大人的身体——

**月華：** 不重要。

**月華：** 芒。如果有一天灵脉的声音停了……帮我照顾翡翠回廊。

[没等芒回答，她继续走了。]

月華站在灵脉之泉旁。她的身体微微前倾，一只手按在胸口——灵脉标石的碎裂让她感受到了实实在在的物理疼痛。

她的脸上有泪痕。

### 剧情对话

[银月卫兵们匆忙在灵脉之泉周围布防。但士气低迷——外圈标石的碎裂让所有精灵都感受到了灵脉的衰弱，如同置身于一个正在停跳的心脏旁边。]

**芒：** 大人——您在哭？

[月華抬手摸了摸自己的脸颊。她看了看指尖的湿润，似乎自己也很惊讶。]

**月華：** ……是吗。

**芒：** 一千四百年来我从未见大人哭过。

**月華：** 因为一千四百年来灵脉从未受过这样的伤。

[她将手从胸口移开。手心上有一个淡淡的红色印记——灵脉灼伤的痕迹，从体内透出来的。]

**月華：** 外圈八块标石被摧毁了。每一块碎裂的时候，我都感觉到了。像是有人用锤子敲碎了我的一根肋骨。

**芒：** 大人，也许应该暂时断开一些灵脉连接，减轻——

**月華：** 不行。断开连接就无法感知腐蚀塔的频率。我需要完整的连接才能找到它们的弱点。

[远处，暗潮攻城将正在指挥部队推进。两座腐蚀塔的嗡鸣声在树林中回荡，令鸟兽远避。]

**暗潮攻城将：** 加大腐蚀塔的功率！让那些发光的石头全部碎掉！精灵的力量来自石头——石头碎了，精灵就是软蛋！

[腐蚀塔的暗红色光波加强。灵脉之泉的水柱又矮了一截。]

**月華：** ……

[她跪在灵脉之泉旁边，双手按在地面上。苔藓在她掌下枯萎了——不是因为她的触碰，而是因为灵脉能量正在从这片土地中流失。]

**月華：** 大地在哭。

[她的声音几乎被腐蚀塔的嗡鸣淹没。但银月卫兵们都听到了，因为她的声音通过灵脉网络传递到了他们的脚底。]

**月華：** 你们听到了吗？大地在哭。它说它很痛。它说标石碎了就像它的骨头折了。它问我为什么不帮它。它问我为什么让那些东西伤害它。

[她的泪水落在枯萎的苔藓上。落下的地方，苔藓微弱地亮了一下——灵脉对灵脉祭司泪水的回应。]

*——一千四百年。我听了一千四百年的低语。那些声音教会了我季节的规律、河流的走向、种子发芽的时机。灵脉从不抱怨。从不叫喊。它只是低低地说话，像一个耐心的老者。但今天，它的声音变了。不再是低语。是悲鸣。一千四百年来第一次，灵脉在哭。而我……只能跟着它一起哭。*

[月華站起身。擦掉眼泪。她的金色眼睛中银色的纹路更密了——灵脉正在过度向她倾注信息，如同一个将死的人拼命抓住最后的救命稻草。]

**月華：** 芒。让所有人听我说。

[她的声音通过灵脉网络传递到战场上每一个银月卫兵的耳中。]

**月華：** 灵脉之泉不能失守。它是翡翠回廊的心脏。如果心脏停了，整片森林——三千年的古树、所有的灵脉标石、月牙泉、灵脉之心——全部会在七天内枯死。

[她深吸一口气。]

**月華：** 目标是两座腐蚀塔。摧毁它们，灵脉的自愈能力就能恢复。注意——我说过，不要用灵脉魔法攻击腐蚀塔。用物理手段。刀砍，枪刺，把它们推倒都行。

**银月卫兵戊：** 大人，腐蚀塔周围有大量守卫——

**月華：** 我会为你们开路。

[她抬起双手。灵脉之泉的水柱——那道已经矮到只剩两丈的灰白色水柱——忽然开始旋转。水柱的颜色从灰白变成了刺目的纯白。]

**月華：** 灵脉，我知道你在痛。但请再给我一次力量。最后一次——不，倒数第二次。

### 战斗阶段

[战斗开始。灵脉之泉进入不稳定状态——水柱的高度和颜色不断变化，映射着灵脉网络当前的状态。]

[第1回合。暗潮攻城兵冲向中圈标石。月華通过灵脉网络预知他们的路线，引导卫兵进行防守。]

**月華：** 北面三人。南面五人。攻城兵优先攻击标石——拦在标石前面。用你们的身体挡。

[一名银月卫兵被暗潮攻城兵击中，飞退撞在标石上。标石没碎，但卫兵的铠甲凹陷了。]

**月華：** ……对不起。

[第3回合。腐蚀塔功率再次提升。一块中圈标石在腐蚀波的持续照射下开始龟裂。]

**月華：** ！

[碎裂。标石炸开。月白色的碎片飞散。月華全身剧烈一颤——又一根"肋骨"断了。]

[灵脉悲鸣发动。碎裂的标石释放出无差别的灵脉冲击波，对全场所有单位造成伤害。暗潮士兵和银月卫兵同时被击退。]

**暗潮攻城兵乙：** 那些石头会爆炸？！

**月華：** 不……那不是爆炸。是灵脉在……挣扎。

[她咳了一口血。]

**月華：** 一个受伤的生物在被继续伤害时，会无差别地挣扎。灵脉也一样。它已经分不清敌友了。它只是在痛。

[第5回合。暗潮破法师精锐组成阵型，同时释放腐蚀波瞄准灵脉之泉本身。]

**暗潮攻城将：** 直接打泉眼！让它彻底干涸！

**月華：** 不许碰它。

[她的声音忽然冰冷。金色的眼睛完全变成了银色——灵脉能量溢出了虹膜。那一瞬间，她不像是一个精灵。她像是灵脉本身。]

**月華：** 灵脉——共鸣·过载。

[所有尚存的标石同时被激活到极限。月白色的光芒冲天而起，将灵脉之泉的水柱重新推高到五丈以上。光芒形成一面巨大的护盾罩在泉眼上方。]

*——痛。从头到脚都在痛。灵脉的悲鸣直接灌进了我的脑子。二十多个标石同时过载——这不是人能承受的。但我不是人。我是精灵。我是灵脉祭司。我的存在意义就是——替大地承受它无法独自承受的痛苦。*

[第7回合。银月卫兵突击组冲到南侧腐蚀塔下方。塔基由黑色金属构成，极其坚硬。]

**银月卫兵己：** 砍不动！这金属太硬了！

**月華：** 塔基第三节和第四节之间——有焊接缝隙。灵脉能感觉到那里的金属密度不均。用枪尖——捅进去——然后撬。

[卫兵依言。枪尖刺入缝隙，发出刺耳的金属摩擦声。腐蚀塔的结构开始松动。]

**月華：** 再用力！

[腐蚀塔倾斜。倒塌。暗红色的光芒熄灭。灵脉之泉的水柱瞬间亮了一些——一半的压迫消失了。]

[第9回合。第二座腐蚀塔也被摧毁。灵脉之泉完全恢复月白色的光芒。水柱高度回升。]

### 战斗结算

[暗潮攻城部队在失去腐蚀塔支援后迅速溃退。灵脉之泉保住了。但代价惨重——中圈标石被摧毁四块，外圈仅剩三块完好。多名银月卫兵重伤。]

[月華跪倒在灵脉之泉边。水柱的飞沫落在她身上，银色长发湿透了，贴在苍白的面颊上。她的双手按在地面，指尖在泥土中抠出了浅浅的痕迹。]

**芒：** 大人！大人，您还好吗？！

**月華：** 灵脉在安静下来。悲鸣停了。它在……自愈。

[她慢慢抬起头。她的眼睛恢复了金色，但银色纹路比以前更深了。]

**月華：** 芒。我做了一个决定。

**芒：** ……

**月華：** 第三个未来。我选第三个。

[芒没有说话。她已经知道了。从月華看到三个未来的那天起，她就知道月華会选这个。]

**月華：** 但不是现在。倒计时还没有到。而且……我需要找到翠玲。在变成凡人之前，我需要见她一面。

*——大地的悲鸣停了。但我知道，如果不做点什么，它还会再哭。会哭得更响。直到哭声变成死寂。我不能让那一天到来。哪怕代价是——不再听到低语。永远的沉默。*

[她站起身。接过芒递来的披风。回头看了灵脉之泉最后一眼。泉水中映出她的倒影——银发金眸的精灵祭司。她知道，下一次看到自己的倒影时，那个人可能已经完全不同了。]

**月華：** 走吧。还有仗要打。

## Event 07 — 角色事件·叁「永生的代价」

| 项目 | 数据 |

| 类型 | 剧情 |
| 推荐等级 | — |

### 场景描写

翡翠回廊东缘——古老的精灵墓园。

这里葬着的不是精灵的尸体——精灵几乎不会自然死亡，死于意外或战斗的精灵会被灵脉吸收，归于大地。这里埋葬的是精灵们"放弃"的东西：旧名字、旧身份、一段不再需要的记忆。每一座小小的石冢下面，是一片写满精灵文字的树叶——经过灵脉浸润后永不腐朽。

月華坐在一座无名石冢旁。这座石冢是她自己的——八百年前，她把自己少女时期的名字埋在了这里。那时候她刚成为灵脉祭司，被告知要舍弃世俗的一切。她照做了。把名字、家族、朋友的面容，都埋在了这片树叶上。

八百年后的今天，她把那片树叶挖了出来。

### 剧情对话

[月華将那片永不腐朽的树叶捧在手心。精灵文字在月光下微微闪烁。她的嘴唇动了动，念出了上面的字——一个名字。然后她笑了。那种笑容极其罕见地出现在她脸上——不是灵脉祭司的淡然微笑，而是一个普通精灵女孩的、带着些许苦涩的笑。]

**月華：** ……好久不见了。

[芒站在几步之外，不敢靠近。这种私密的时刻，侍女不该在场。但月華叫住了她。]

**月華：** 芒，过来。我想和你说说话。

**芒：** 大人……那片叶子——

**月華：** 八百年前的旧东西了。上面写着我做灵脉祭司之前的名字。那时候我还有家族姓氏。有父母。有一个很喜欢追着我跑的弟弟。

[她把树叶翻过来。背面还有更多文字——密密麻麻的小字，像是日记。]

**月華：** 成为灵脉祭司的第一天，长老告诉我："从今天起你不再是某某家的女儿。你是灵脉的容器。灵脉不需要名字，不需要家族，不需要情感。你要做的只有一件事——倾听。"

**芒：** ……

**月華：** 我照做了。我把名字埋在这里。把家人的面容从记忆中慢慢模糊掉。花了大约两百年。两百年后，我再也记不清弟弟的脸了。只记得他跑起来的样子——很快，总是在笑。

*——永生。精灵以永生为荣。我们有无限的时间去学习、去成长、去完善自己。但没有人提到永生的另一面——永生意味着你要看着所有短命的存在一个一个离开。树木还好，千年古树能陪你很久。但动物不行。朋友不行。你养的那只猫头鹰——三十年后它就死了。你在学院认识的那个人类法师——还没来得及记住他的魔法就已经老死了。*

**月華：** 芒。你多大了？

**芒：** 三百二十岁，大人。

**月華：** 三百二十。对精灵来说还很年轻。你还记得你少女时候喜欢什么吗？

**芒：** 我……喜欢在溪边编花环。还有追月光蝶。

**月華：** 你看。三百二十年了，你还记得。

[她低头看着手中的树叶。]

**月華：** 我一千四百岁了。我不记得少女时候喜欢什么。我不记得父母的面容。我甚至不确定我的弟弟是真的还是我编造的记忆。一千四百年的灵脉低语，把那些东西全部冲刷掉了。灵脉太吵了。太持续了。它永远不停。白天说话，晚上说话，睡觉的时候也在说话。慢慢地……灵脉的声音就成了我脑子里唯一的声音。

**芒：** 大人……

**月華：** 这就是永生的代价，芒。不是肉体的代价。是记忆的。情感的。自我的。你活得越久，灵脉的声音就越大，而你自己的声音就越小。到最后——你还是你吗？还是你已经变成了灵脉的一部分，一个会说话的标石？

[她把树叶小心地折起来，放进衣袋里。]

**月華：** 我经常想这个问题。尤其是在安静的夜晚。灵脉的声音稍微小一点的时候，我偶尔能听到自己的声音。它在问我："月華，你最后一次因为自己的事情而不是灵脉的事情感到开心，是什么时候？"

[沉默。]

**月華：** 我想不起来。

**芒：** ……大人。那翠玲呢？

[月華的表情变了。极其微妙——如同水面被一粒沙子击中，产生了一圈不易察觉的涟漪。]

**月華：** 翠玲……

**芒：** 您提到翠玲的时候，声音和平时不一样。灵脉的声音会变小吗？

**月華：** ……

[她沉默了很久。]

**月華：** 翠玲第一次来翡翠回廊的时候，拿了一把花插在灵脉标石上。她说："你们的石头太冷了，需要点颜色。"

[她嘴角微微上扬。]

**月華：** 灵脉标石被插花以后发出了一种……很奇怪的频率。不是不高兴。是——困惑。灵脉不理解"装饰"这个概念。但翠玲不管。她就是那种人。不管你有什么规矩、什么传统、什么一千四百年的历史——她觉得需要花就插花。

**芒：** 翠玲姑娘确实……很有个性。

**月華：** 她和灵脉完全相反。灵脉低语，她大声说话。灵脉永恒，她活在当下。灵脉感知一切，她只在乎眼前的人。

[月華站起身。银色长发在夜风中飘动。]

**月華：** 芒。你问我翠玲的时候灵脉的声音会不会变小。答案是——不会。灵脉的声音永远在。但翠玲在的时候，我能听到另一个声音。比灵脉更小，但更真实。是我自己的声音。在说——

[她停顿了一下。金色的眼睛在月光下显得格外明亮。]

**月華：** 在说："我很开心。"

*——永生的代价是失去自己。但翠玲让我想起了"自己"是什么。她不关心灵脉的低语。不关心第三种力量。不关心精灵的使命。她只关心：你今天有没有吃饭。你的黑眼圈是不是比昨天深了。你已经多久没有笑了。*

*——如果切断灵脉意味着失去永生，那我也将失去这一千四百年形成的一切——灵脉的声音、感知、力量。但也许……我也会重新获得一些东西。灵脉声音消失之后的那片沉默里，也许我能重新听到自己的声音。重新记起弟弟的脸。重新学会因为自己的事而开心。*

**月華：** 芒。帮我写一封信。寄给翠玲。

**芒：** 写什么？

**月華：** 就写……"花还插在标石上。没枯。灵脉好像学会了怎么养花。"

[芒笑了。这是整个任务链中第一次出现真正温暖的笑声。]

**芒：** 就写这些吗？

**月華：** 就这些。

[她最后看了一眼精灵墓园。那些无名石冢在月光下排列成行，像是一排沉默的问号。每一座下面都埋着一个精灵放弃的东西。]

[月華想：也许很快，这里会多一座新的石冢。不埋名字，不埋记忆。埋的是——永生。]

[她转身离开。脚步比来时轻快了一些。]

灵脉重启后的森林比以前更加生机勃勃——古树的叶冠更加繁茂，灵脉标石的光芒更加稳定明亮，月牙泉的泉水清澈到能看见泉底的每一粒沙砾。整座森林散发着一种充沛而安宁的力量，如同一个大病初愈的人终于恢复了健康。

然而站在这座焕然一新的森林中的月華，看起来像是一个不属于这里的异乡人。

她穿着一件从银月卫兵那里借来的轻甲——灵脉祭司不穿甲，但凡人需要。灰白色的头发被简单地束在脑后。手中拿着一根普通的木杖——不是灵脉祭司的法杖，是从厨房借来的擀面杖用灵木改造的。

她的棕色眼睛看着翡翠回廊，表情很复杂。她能看到灵脉标石在闪烁，但她听不到它们的声音了。就像一个失聪的人看着乐队在演奏——知道音乐在响，却什么都听不到。

**月華：** ……会渴。原来渴是这种感觉。嘴巴发干，喉咙紧。然后喝了水就舒服了。

**芒：** 大人一直不觉得渴吗？

**月華：** 灵脉会调节。以前不太能感受到身体的需求。现在全来了。渴、饿、冷、热、困——

[她打了个哈欠。]

**月華：** 困。原来困也是这种感觉。眼皮很重。

**芒：** 大人昨晚没睡好？

**月華：** 太安静了。习惯了灵脉的低语，忽然什么声音都没有……反而睡不着。就像住在瀑布旁边一千四百年，忽然瀑布停了。

**芒：** 暗潮……还在进攻？他们不知道灵脉已经重启了？

**银月卫兵辛：** 应该是前线溃散的残部。通讯中断，可能不了解最新情况。他们还带着腐蚀器。

**月華：** 腐蚀器对重启后的灵脉无效。灵脉重启时改变了共振频率，旧的腐蚀器对不上了。他们带着一堆废铁。

[她说这些的时候，声音很平静。但芒注意到——她的判断不再基于灵脉感知。这些是她在灵脉之心仪式之前就已经计算好的结论。她现在只能依靠知识和推理，而不是感知。]

**月華：** 我去。

**芒：** 大人？！您现在的状态——

**月華：** 我知道我的状态。INT降了很多，灵脉系的技能全废了，体力也不如以前。但我还会基础的祭司术。而且——

[她拿起那根用擀面杖改造的木杖，在手中转了一圈。动作不太协调——失去灵脉连接后她的精细动作退化了不少。]

**月華：** 我需要确认一件事。

**芒：** 什么？

**月華：** 没有灵脉的我，还能不能战斗。我不是要证明什么。是因为——去南方救翠玲的路上不可能一帆风顺。如果没有灵脉的我连一场小规模战斗都撑不过，那我需要重新制定计划。

[战斗开始。翡翠回廊的重启灵脉自动运行，提供全场DEF+1。月華不再需要维持——灵脉网络现在是自主运行的。]

**月華：** 需要一根好一点的杖。

[翠玲站起来。她认出了那个身影——虽然头发颜色不对，身形也比记忆中瘦了一些。]

**月華：** 灰白。不是全白。

**翠玲：** 你这个白痴。一千四百岁了还会摔跤。

**月華：** 以前灵脉会提前告诉我哪里路滑。现在不会了。

[两个人沉默了一会儿。风从烧焦的原野上吹过来，带着硝烟的苦味。]

**翠玲：** ……你现在能活多久？

**月華：** 和你一样。几十年。

**翠玲：** 几十年。

**月華：** 嗯。

[翠玲伸出手。这次是她主动的。她握住了月華的手——那只没有灵脉纹路的、普通的、温暖的手。]

**翠玲：** 那别浪费。

[月華的棕色眼睛微微湿润了。她点了点头。]

**月華：** 不浪费。

[翠玲拉着月華的手，转向玩家的队伍。]

**翠玲：** 喂！多一个人。她跟我们走。

[玩家队伍。月華被拉着走进来。她看着周围的人——她从未以凡人的视角看过这么多人类。以前透过灵脉，她看到的是脚步的频率、心跳的节奏、生命力的流动。现在她看到的是——脸。表情。眼睛里的光。]

**月華：** ……你们好。我叫月華。以前是银月议庭的灵脉祭司。现在——

[她低头看了看自己：灰白头发、棕色眼睛、普通木杖、破旧斗篷、膝盖上结了痂。]

**月華：** 现在只是一个凡人。但我还会一些基础的祭司术。治疗、束缚、简单的防护。如果不嫌弃的话——

**翠玲：** 不嫌弃。走了。

[翠玲没有松开月華的手。]

**【系统提示：月華（Gekka）加入了队伍。】**

*——翠玲的手很暖。比灵脉温暖。比大地温暖。这是凡人的体温。三十六度左右。几十年后会变凉。但在那之前——在那之前的每一天——我都不想松手。*

[月華跟着队伍走向前方。她的脚步比翡翠回廊的时候快了一些——也许是因为有人牵着。身后，烧焦的原野上，一块不知道从哪里滚来的石头表面——微微闪烁着月白色的光。]

[灵脉记得她。即使她已经听不到了。]

### 场景描写

某日清晨。队伍宿营地。

这是一个普通得不能再普通的清晨。天际线刚刚泛出鱼肚白，几片薄云被染成淡淡的粉橙色。远处的山脊线将天空和大地分割成两半。近处是一片长满野草的小丘，队伍的帐篷散落其间。篝火已经熄灭了，只剩几缕青烟从灰烬中袅袅升起。

月華站在小丘的最高处。面朝东方。

她昨晚没有睡好——作为凡人的第十四个夜晚，她仍然不习惯脑子里的安静。但今天醒得很早。不是灵脉叫醒了她——再也不会了——而是一种更原始的力量：光。破晓的第一缕光线穿过帐篷的缝隙照在她的眼皮上。她的凡人眼睛对光线变得敏感了——以前金色的精灵瞳孔可以自动过滤强光，现在棕色的凡人瞳孔不行了。

她被阳光晃醒了。然后她发现——她从来没有认真看过日出。

一千四百年。每一个清晨她都在倾听灵脉的汇报——夜间的地壳微动、水位变化、植物的生长数据。太阳升起只是灵脉信息流中一个不起眼的数据点。她从来没有——真正地——看过太阳是怎么从地平线上一点一点地升起来的。

现在她看了。

### 剧情对话

[翠玲从帐篷里钻出来。头发比月華的还乱。她打着哈欠走到月華身边，顺着她的视线看向东方。]

**翠玲：** ……你在看什么？

**月華：** 日出。

**翠玲：** 日出？天天都有。

**月華：** 嗯。天天都有。但我以前没看过。

[翠玲揉了揉眼睛。她还不太清醒。]

**翠玲：** 一千四百年没看过日出？

**月華：** 看过。但不是"看"。是灵脉告诉我"太阳升起了"，然后我记录一下数据。从来没有站在这里，用自己的眼睛，看它是怎么一点一点亮起来的。

[太阳从山脊线上露出了一小片弧形。浅金色的光从地平线向天空蔓延，将薄云依次点燃。]

**月華：** ……很慢。

**翠玲：** 什么？

**月華：** 日出很慢。灵脉的数据里它只是一个时间点——"日出：卯时三刻"。但实际上它不是一个点。它是一个过程。从第一缕光到整个太阳升出来，大概要——

[她看了看天。]

**月華：** 一刻钟？

**翠玲：** 差不多吧。

**月華：** 一刻钟。

[她沉默了一会儿。太阳的边缘从山脊后露出更多。]

**月華：** 以前我有无限的一刻钟。但没有一个用来看日出。现在我的一刻钟有限了。但这一个——

[她的嘴角微微上扬。棕色的眼睛映着初生的阳光。]

**月華：** 这一个很好。

[翠玲斜眼看着她。然后伸手拨了拨月華的灰白色头发——被清晨的露水打湿了一些，贴在额头上。]

**翠玲：** 你的头发。

**月華：** 怎么了？

**翠玲：** 以前是银色的，会发光。像月光。现在灰白色的，不发光了。

**月華：** 嗯。

**翠玲：** ……还挺好看的。

[月華看了翠玲一眼。翠玲的表情很认真——不是安慰，是真的觉得好看。]

**月華：** 谢谢。

**翠玲：** 不过看起来老了一点。

**月華：** 我确实变老了。以前灵脉维持着精灵的青春。现在——正常速度地变老了。

**翠玲：** 你现在看起来多大？

**月華：** 大概……三十岁左右的人类？

**翠玲：** 三十。行。

[翠玲从口袋里掏出一根皮筋——不知道从哪儿弄的——走到月華身后，把她散乱的灰白色长发简单地扎成了一个低马尾。]

**翠玲：** 头发太长了，打仗的时候碍事。要不要剪短？

**月華：** ……以前银色的时候我不舍得剪。灵脉的力量有一部分通过头发传导。但现在没有灵脉了。

**翠玲：** 那？

**月華：** 随便你。

**翠玲：** 那改天帮你剪。留到肩膀差不多。

[翠玲扎完头发，走回月華身旁。两个人并肩看着日出。太阳已经完全升出了山脊线——一个完整的、金色的圆盘悬在天际，将整片原野染成了蜜糖般的暖色。]

**翠玲：** 月華。

**月華：** 嗯。

**翠玲：** 你现在能活多久？

**月華：** 你问过了。几十年。和你一样。

**翠玲：** 几十年。如果从三十岁算起……六七十？八十？

**月華：** 不确定。灵脉断绝后的精灵没有先例。也许六十年。也许四十年。也许更少。我不知道。

[翠玲沉默了一会儿。]

**翠玲：** 那别浪费。

**月華：** 你也说过了。

**翠玲：** 值得说两遍。

[月華笑了。不是灵脉祭司那种超然的淡笑。是一个凡人的笑。带着眼角的细纹，带着微微发颤的嘴角，带着棕色眼睛里的泪光。]

**月華：** 不浪费。

*——第一次凡人的日出。阳光照在皮肤上，有温度。不是灵脉传来的温度数据，是真的——热。微微地热。毛孔在舒张。眼睛要眯起来。鼻子闻到了露水蒸发的味道——青草和泥土混合的，湿润的，活着的味道。*

*——一千四百年的灵脉低语。数不清的信息。季节的更替、河流的走向、种子的发芽、大地的心跳。但从来没有——从来没有一个数据告诉过我，凡人的早晨是这样的。*

*——暖。安静。短暂。珍贵。*

[太阳完全升起。新的一天开始了。]

[翠玲拍了拍月華的肩膀。]

**翠玲：** 走了。早饭该做了。你能生火吗？

**月華：** 基础祭司术里有点火的。

**翠玲：** 那去点。

[月華拿起她那根磨平了底端的硬木杖，跟着翠玲走向营地。灰白色的低马尾在晨风中轻轻摆动。]

[走了几步，她忽然回头看了一眼东方。太阳已经升高了一些，光芒更加耀眼。她用凡人的棕色眼睛直视了一秒——立刻被晃得眯起了眼。]

**月華：** ……太亮了。

**翠玲：** 你不是精灵吗？精灵不怕光？

**月華：** 以前不怕。精灵的瞳孔会自动调节。现在——

[她揉了揉眼睛。]

**月華：** 现在跟你们一样了。太阳太亮会刺眼。

**翠玲：** 适应一下就好了。人类从小就适应了。

**月華：** 嗯。我有几十年的时间来适应。

[翠玲回头看了她一眼。月華正揉着眼睛、拄着木杖、灰白色头发被风吹到脸上——看起来完全不像一个曾经拥有一千四百年寿命和整个灵脉网络的精灵祭司。她看起来像一个普通的、略显狼狈的、正在学习怎么做凡人的——人。]

[翠玲笑了。]

**翠玲：** 快走。粥要凉了。

**月華：** 来了。

[她们走向营地。身后，太阳继续升高。灰白色的长发和晨光融在一起，分不清哪里是头发，哪里是阳光。]

[远方。极远的远方。翡翠回廊的灵脉标石在这一刻同时闪烁了一下——时间极短，不到一秒。如果有人在场，可能会觉得那是灵脉网络的一次正常脉动。]

[但芒知道那不是。]

[在翡翠回廊的月牙泉旁，芒看着标石闪烁了一下，然后微微一笑。]

**芒：** 灵脉说——"早安。"
