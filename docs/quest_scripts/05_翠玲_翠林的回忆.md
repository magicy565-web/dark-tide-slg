## Event 01 — 开战宣言「千箭之林」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.18-22 |
| 情感基调 | 精灵射手长的绝对骄傲，猎手俯视猎物 |
| 战斗条件 | 突破翠玲箭阵封锁线（存活3单位以上抵达前线） |
| 特殊机制 | 翠玲「千箭齐鸣」每回合对随机3单位造成ATK×0.6远程伤害；翠林地形：树冠遮蔽使远程命中率-15%，但翠玲无视此效果 |
| 敌方配置 | 翠玲(Lv.22) + 精灵射手×6(Lv.19) + 精灵斥候×4(Lv.17) + 树灵哨兵×2(Lv.18) |
| 奖励 | 400 EXP, 900G, 翠色箭羽×1, 解锁翠玲档案第一页 |

### 场景描写

翠林——银月议庭领地最西端的千年古森林。

阳光几乎无法穿透层叠的树冠。三百尺高的常青巨木如同大地的肋骨，密密匝匝地撑起一片永恒的绿色穹顶。地面覆盖着厚厚的苔藓，踩上去没有一丝声响。空气中弥漫着树脂与露水混合的清冽气息，偶尔有一缕金色的光柱从树冠缝隙间刺下来，照亮飘浮的孢子，像是森林在呼吸时吐出的微尘。

但那些光柱正在被切断。

树冠之间，无数银色的丝线在逆光中闪烁——那是弓弦。每一棵巨木的枝杈间都藏着精灵射手，他们的翠绿斗篷与树叶融为一体，只有在拉弓的瞬间，弦上反射的光芒才会泄露位置。而那些光芒……多得像星空倒映在了森林里。

最高的那棵巨木顶端，一个身影站在仅容双足的枝头，逆光而立。银白色的长弓斜挎在背上，几乎与她等高。风吹动她及腰的翠绿长发，发梢缠绕着几片活的叶子——那是翠林认主的标记。

翠玲·希瓦纳尔。精灵射手长。一万零三箭，箭箭命中。

她低头俯瞰着林间小径上缓缓推进的暗潮军队，嘴角微微上扬。那不是笑——那是猎手看到猎物自投罗网时的本能反应。

### 剧情对话

[翠玲单手搭在树干上，另一只手垂在身侧。她甚至没有去拿弓。身后的精灵射手们已经全部就位，箭矢上弦，等待命令。]

**翠玲：** ……真的来了啊。

[她歪了歪头，翠绿的瞳孔中映出林间小径上蠕动的队列。]

**翠玲：** 铁甲、长矛、盾牌阵……哦，还有攻城弩。

[她轻轻吹了声口哨。]

**翠玲：** 真是盛大的葬礼队伍。只不过——你们不知道棺材是给自己准备的。

[身旁的精灵副官银叶快步跃上枝头，单膝跪地。]

**银叶：** 射手长，暗潮先锋已进入外围射程。约六百人，重步兵为主，后方有弩车三辆。

**翠玲：** 六百人？

[她终于转过身来，面朝银叶。逆光中看不清她的表情，但声音里带着一丝真诚的困惑。]

**翠玲：** 只有六百人？他们是来攻打翠林的，还是来野餐的？

**银叶：** ……射手长，根据斥候回报，这只是先锋。主力——

**翠玲：** 我知道后面还有。但那不重要。

[她从背上取下银弓。弓身是用千年银杉的芯木削成，通体泛着淡淡的月光色泽，弓臂上缠绕着活的藤蔓——那是翠林之心赐予的祝福。]

**翠玲：** 银叶，你知道这片森林有多少棵树吗？

**银叶：** ……属下不知。

**翠玲：** 我也不知道。但我知道每一棵的名字。

[她轻轻抚过弓弦，弦音清越如泉。]

**翠玲：** 踏入翠林的那一刻，他们就已经输了。这里的每一片叶子都是我的眼睛，每一根枝条都是我的箭台。

*——一万零三支箭。从第一支练习箭射穿树叶开始，到昨天射落入侵的灰鹫为止。没有一支落空。翠玲·希瓦纳尔，从不失手。*

[她张弓搭箭，瞄准林间小径上走在最前方的暗潮旗手。距离——四百步。在任何弓手的有效射程之外。]

**翠玲：** 告诉所有人。等我的第一箭。

[银叶的眉心微蹙。]

**银叶：** 射手长，四百步……即使是您——

**翠玲：** 银叶。

[声音平静得像湖面。]

**翠玲：** 我什么时候说过"试试看"这种话？

[银叶沉默片刻，随即行礼退下。]

[翠玲拉满弓弦。她的呼吸极其缓慢——吸气，屏息，瞳孔收缩。风从东南方吹来，树冠沙沙作响。她在计算风速、湿度、箭矢重量、目标移动速度。这些数据在她脑中如同水流般自然汇聚，不需要思考，只需要——]

**翠玲：** ……去吧。

[箭离弦。]

[四百步外，暗潮旗手手中的军旗旗杆被拦腰射断。旗面在风中翻滚落地。旗手本人毫发无伤——她故意的。]

*——杀人太简单了。让他们知道自己随时会死，才是真正的恐惧。*

**翠玲：** 翠林射手听令——

[她举起银弓，弓臂上的藤蔓发出绿色的荧光。整片森林仿佛在呼应她的号令，树叶沙沙作响如千军万马。]

**翠玲：** 千箭齐鸣。

[一瞬间，从四面八方的树冠中射出无数箭矢。箭雨从天而降，密集到遮蔽了树冠间仅存的光柱。那些箭矢精准地落在暗潮队列的间隙中——没有射人，全部钉入泥土，在队列周围形成了一个密不透风的箭之牢笼。]

[暗潮士兵们僵在原地。被箭包围，却无一人受伤。恐惧比伤痛更具杀伤力。]

**翠玲：** （对着林下高喊）听好了，踩在翠林泥土上的人——

[她的声音在树冠间回荡，被森林放大了数倍。]

**翠玲：** 你们有一次机会转身离开。我数到十。数完之后，下一轮箭……不会再这么客气了。

[沉默。然后，暗潮队列中传来一声粗犷的怒吼——是先锋队长在下令继续推进。]

**翠玲：** ……

[她歪了歪头，像是在观察一只不知死活的虫子。]

**翠玲：** 好吧。那就别怪我了。

### 战斗阶段

[战斗开始。翠玲站在最高的树冠上，居高临下。「千箭齐鸣」发动——每回合随机3个敌方单位被箭雨覆盖。]

*——翠林是活的。她的根须延伸到每一寸土地下面，她的枝叶覆盖每一个角落。在这里，我不需要眼睛。森林就是我的眼睛。*

[第1回合，暗潮重步兵举盾推进。箭矢如雨点般落在盾面上，发出密集的叮当声。但精灵射手们不急——他们等待盾与盾之间露出的缝隙。]

**翠玲：** 别浪费箭。等缝隙。呼吸之间的缝隙。

[第2回合，暗潮弩车开火，轰碎了一棵巨木的主干。树木轰然倒塌，扬起漫天碎叶。]

**翠玲：** ……！

[她的目光骤然锐利。那棵树——她叫它"歌者"，因为风穿过它枝叶时会发出像歌谣一样的声音。]

**翠玲：** 你们……砍了我的树。

[声音冷得像刀锋。]

**翠玲：** 银叶。弩车。全部。

**银叶：** 是！

[精灵射手集中火力，三轮齐射摧毁全部弩车。翠玲本人亲手射出三箭——每一箭精确命中弩车的绞盘轴承。]

[受到重击时——]

**翠玲：** 嗤……能碰到我？

[她从枝头跃起，在空中翻转，箭矢擦过她的发丝。落在另一棵树的枝杈上时，她已经回射了一箭。]

[HP降至50%以下时——]

**翠玲：** 有点意思。真的有那么一点点意思。

*——很久没有被逼到需要移动位置了。也许……不，不可能。他们只是虫子。运气好的虫子。*

### 战斗结束·过场

[暗潮先锋部队溃退。林间小径上散落着断裂的旗帜和被丢弃的盾牌。翠玲站在倒下的"歌者"树干上，低头看着脚下的年轮。]

**翠玲：** ……三百二十七圈。你活了三百二十七年。

[她蹲下身，手掌贴上断面，指尖微微颤抖——这是她在整场战斗中唯一一次失去平静。]

**翠玲：** 我会让他们付出代价。每一棵树，每一片叶子，都会得到偿还。

[她站起身，背对着暗潮撤退的方向。]

**翠玲：** 银叶，清点损失。

**银叶：** 是。射手长……您没事吧？

**翠玲：** 我什么时候有过事？

[她跃上树冠，消失在绿色的海洋中。]

*——一万零四。"歌者"的仇。*

## Event 02 — 前哨遭遇「傲慢的距离」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.20-24 |
| 情感基调 | 傲慢依旧，但开始认真审视对手 |
| 战斗条件 | 在翠玲的远程压制下突破三道防线 |
| 特殊机制 | 翠玲「傲慢之距」：射程+2格，超远距离攻击附带减速效果（SPD-2，持续1回合）；树冠走廊：精灵射手可在树冠间快速转移 |
| 敌方配置 | 翠玲(Lv.24) + 精灵射手×8(Lv.20) + 精灵游侠×3(Lv.21) + 树灵卫士×3(Lv.19) |
| 奖励 | 450 EXP, 1000G, 银杉树脂×2, 解锁翠玲档案第二页 |

### 场景描写

翠林外围——被精灵们称为"低语走廊"的狭长林带。

这里的树木比翠林深处矮上一截，但依然高耸入云。树与树之间由天然的藤桥相连，形成了一张悬在半空的立体交通网。精灵射手们可以在这张网上无声地奔跑、跳跃、变换射击位置——而地面上的入侵者只能仰头挨打。

清晨的雾气尚未散尽。乳白色的雾从地面升起，刚好淹没到人腰的高度，使得地面部队的脚步被完全遮蔽。但对于在树冠上俯瞰的精灵来说，雾气中移动的身影反而更加醒目——如同白纸上的墨点。

翠玲今天换了位置。她不在最高的树上——她站在"低语走廊"中央那棵被称为"守门人"的古橡树上，银弓横在膝头，双腿悬空晃荡，看起来悠闲得像在郊游。

### 剧情对话

[翠玲坐在"守门人"的横枝上，嘴里叼着一片嫩叶。银叶站在相邻的树枝上，表情严肃。]

**银叶：** 射手长，斥候回报。暗潮军团调整了战术——前排全换成了塔盾兵，后方跟着法师组。他们要用魔法护盾抵消我们的箭雨。

**翠玲：** 嗯。

[她把嘴里的叶子吐掉，看着它旋转着落入雾中。]

**翠玲：** 所以他们学聪明了一点点。从虫子变成了——会躲的虫子。

**银叶：** ……射手长，我认为不应该轻视他们。上次的指挥官——

**翠玲：** 上次的指挥官是个蠢货。但这次的……

[她眯起眼睛，望向雾气深处。一支队伍正在缓慢而有序地推进，塔盾兵组成移动的铁墙，缝隙极小，阵型稳固。]

**翠玲：** 阵型不错。有人教过他们怎么在森林里行军。

*——有意思。通常人类进了翠林就像无头苍蝇一样乱撞。这支部队……有人做了功课。*

**翠玲：** 银叶，把射手分成三组。第一组留在原位，第二组向东翼迂回，第三组……跟我来。

**银叶：** 射手长要亲自——

**翠玲：** 我要测试一下他们的指挥官。

[她站起身，银弓在手，翠绿的斗篷在晨风中猎猎。]

**翠玲：** 如果他只是个照本宣科的书呆子，三箭就够了。如果他是个真正的战术家……

[嘴角勾起一丝兴味。]

**翠玲：** 那我可能需要五箭。

[她从枝头跃下，在藤桥网络中无声穿行，如同一只翠色的飞鸟。]

### 战斗阶段

[战斗开始。翠玲位于地图后方高台，射程覆盖全场。「傲慢之距」生效——她的攻击范围比普通弓手多出整整两格。]

*——距离是弓手最好的朋友。只要我和他们之间永远隔着这段距离，他们就永远只能被动挨打。这就是弓手的法则——绝对不要让猎物靠近。*

[第1回合，暗潮塔盾兵组成铁壁阵推进。翠玲射出第一箭——箭矢精准命中盾与盾之间仅有一指宽的缝隙，穿透击中后方的法师。]

**翠玲：** 你们的盾举得不错。但缝隙——

[她又射一箭，从另一个角度穿入缝隙。]

**翠玲：** ——在我眼里和城门一样宽。

[第3回合，暗潮阵型突然变化——塔盾兵分成两列，中间释放出三名突击骑兵，直奔翠玲所在的树木！]

**翠玲：** 哦？

[她挑了挑眉。骑兵们速度极快，利用雾气掩护，转眼已到树下。]

**翠玲：** 用骑兵冲森林？这主意要么蠢透了，要么——

[骑兵们没有试图爬树。他们拔出绳索，套住树干基部，三匹马同时发力——]

**翠玲：** ——要么聪明得过分。

[她脚下的树开始剧烈摇晃。翠玲不得不跳到相邻的树上。]

*——他们不是来射我的。他们是来拆我脚下的平台的。有意思……真的有意思。*

**翠玲：** 银叶！处理掉那些马！我不想再损失第二棵树！

[受到攻击时——]

**翠玲：** 嗯？你居然看到我了？

[她侧身避开一支反击的弩箭，弩箭擦过她的面颊，留下一道浅浅的血痕。]

[翠玲抬手摸了摸面颊上的血，看了看指尖。]

**翠玲：** ……

*——三千六百一十二天。距离上一次被碰到，隔了整整三千六百一十二天。*

[HP降至50%以下时——]

**翠玲：** 好吧，我承认……你比虫子要大一号。也许是……老鼠？

[笑意从眼角消失了。取而代之的是专注——真正的、弓手对猎物的专注。]

**翠玲：** 认真模式。

[她拉弓的速度骤然提升，连珠三箭，每一箭之间的间隔不到半息。]

### 战斗结束·过场

### 场景描写

翠林东区——燃烧中的废墟。

火焰从南面蔓延而来。暗潮军团不再试图正面突破翠林——他们选择了最残酷的方式：烧。

特制的火油被投掷到树干根部，火焰沿着树皮向上攀爬，速度快得像活物。枯萎的树叶成了最好的燃料，一棵树从起火到化为焦黑的骨架，只需要短短一刻钟。浓烟遮蔽了所有光源，空气变得灼热而窒息。灰烬如同黑色的雪花从天空飘落。

翠玲站在火线前方。她的斗篷边缘被火星烧出了几个洞，面颊上沾满了烟灰。银弓在手，但她的箭——该射向谁？

南面是暗潮的纵火兵，正在系统性地点燃每一棵树。

北面是暗潮的突击部队，趁着混乱试图穿过翠林。

三支力量，在燃烧的森林中交汇。

**翠玲：** ……灭了这边，那边又烧起来。

**翠玲：** ……

### 场景描写

翠林核心区。围攻战第三日。

千年之母周围已经被翠玲和残余的精灵射手们改造成了最后的防线。枯死的树干被砍倒、堆叠成简陋的路障。地面的枯根被削尖，做成了拒马。没有时间修建像样的工事——一切都是就地取材，用死去的翠林来保护还活着的翠林。

翠玲蹲在一段枯木后面，正在做箭。

不是用精灵的银杉木做的正规箭矢——箭筒里的好箭在Event 06就几乎耗尽了。她现在用的是枯枝。削尖一端，尾端绑上几片干树叶代替箭羽。粗糙、不平衡、精度极差。但这是她目前唯一能做的。

银弓的弦也换了。不是千年之母的藤丝——那已经断了。她用枯藤搓了一根替代品。能拉，能射，但力量和弹性远不如从前。

### 剧情对话

[翠玲用牙齿咬住枯枝的末端，一手削皮，一手固定。旁边的地上放着十几支完成的枯枝箭，歪歪扭扭，像是小孩子的手工作品。]

**银叶：** 射手长……这些箭……

**翠玲：** 我知道很丑。

[她把削好的箭杆放在眼前瞄了瞄，歪的。叹了口气，继续削。]

**翠玲：** 但好看的箭和难看的箭射出去都是一样的——只要够尖，能扎进肉里就行。

**银叶：** 可是精度——

**翠玲：** 精度的问题我用经验补。射了一万零三支箭的手不会因为箭杆歪了就打不中。

*——骗人的。枯枝做的箭，精度最多只有正规箭矢的六成。有些射出去直接在空中碎了。但我不能说出来。银叶已经够害怕了。*

[远处传来号角声。暗潮第三波攻势开始了。]

**银叶：** 来了！

**翠玲：** 多少人？

**银叶：** 比前两次加起来还多。而且……有灰精灵带头。

[翠玲停下手中的削刀。抬头望向防线外。浓烟和灰尘中，影影绰绰的人影正在逼近。走在最前面的三个身影——尖耳，银发，精灵的面孔。但他们的眼睛是灰色的，没有翠林精灵的翠绿。]

**翠玲：** ……叛徒。

[声音里的愤怒比面对暗潮人类时更深沉。]

**翠玲：** 所有人，听好了。

[她站起身，把枯枝箭筒背上肩。银弓上的枯藤弦在风中嗡嗡作响——声音沙哑，不像从前那样清越。]

**翠玲：** 今天我们用的不是银杉箭，是枯枝。弦不是千年之母的藤丝，是枯藤。但我们脚下踩的——

**翠玲：** 还是翠林的土。

**翠玲：** ……！

**翠玲：** 十支里能用七支。够了。

**灰精灵队长：** 翠玲·希瓦纳尔。两百年前我们一起在箭的摇篮练箭的时候，你可没有用这种垃圾射过我。

**翠玲：** 灰渊。两百年前你还配叫翠林的精灵。现在你只配吃我的枯枝箭。

**翠玲：** 嘁……

**灰渊：** 看到了吗？你的箭——碎了。就像翠林一样碎了。放弃吧，翠玲。翠林已经死了。

**翠玲：** 翠林没死。

**翠玲：** 千年之母还活着。只要她还有一口气——翠林就没死。

**翠玲：** 哈……哈……

**翠玲：** 还有三支箭……

[她看了看箭筒。只剩三支枯枝箭。]

**翠玲：** 三支。选好目标。

**翠玲：** 千年之母……再借我一点力。最后一点就好。

**翠玲：** ……撑住。

[枯藤弦——断了。第二次断弦。]

[暗潮部队在玩家的侧翼攻击和翠玲的拼死抵抗下终于退却。千年之母保住了——但防线几乎全毁，枯木路障被踏成碎片。]

[翠玲坐在千年之母的根须上，把断弦的银弓放在身旁。她的右手三根手指上缠着从斗篷上撕下的布条，血已经渗透了布料，把绿色的布染成了暗红色。]

**银叶：** 射手长，您的手——

**翠玲：** 没事。皮肉伤。

[她活动了一下手指，疼得微微抽气，但没有出声。]

**翠玲：** 弦又断了。

**银叶：** 我去找材料——

**翠玲：** 找什么材料？翠林里的藤都枯了。

**翠玲：** 银叶……你说，弓手没有弓弦，还算弓手吗？

**银叶：** ……射手长。

**翠玲：** 随便问问。别当真。

[她靠回千年之母的树干，闭上眼睛。一片枯黄的叶子从头顶飘落，落在她的膝上。她没有睁眼，只是用手指轻轻碰了碰那片叶子。]

**翠玲：** ……又一片。

### 场景描写

千年之母的根须间。清晨。

翠林几乎已经没有绿色了。枯黄和灰白取代了一切。千年之母的树干上只剩最后一片叶子——孤零零地挂在最高处的枝条上，在晨风中摇摇欲坠。

翠玲坐在千年之母的根须上，银弓放在身旁。发丝弓弦还在——染血的、汗渍斑驳的、但还完整。她的面前铺着一块精灵丝帛，上面整齐排列着所有她还拥有的东西：

[她拿起一支银杉箭，在指尖转动。箭矢在晨光中闪烁。]

**翠玲：** 但现在我在想……这一万零三支箭，保护了什么？

**翠玲：** 你来得正好。我有话跟你说。

**银叶：** 射手长——您——

**翠玲：** 我要放下弓。

**银叶：** 不——您不能！您是翠林三百年来最强的射手！您是"不落之箭"！您——

**翠玲：** 银叶。

**翠玲：** "不落之箭"射了一万零三支箭，没有保住翠林。也许……该试试不射箭的办法了。

**翠玲：** 你说你能救翠林。灵脉修复的方法你在找。但修复需要时间，而暗潮不会给时间。

**翠玲：** 所以我不再射你。也不再射任何人。从今天起——我的箭留给翠林。不是用来杀人，是用来守护。

[她蹲下身，手掌贴在千年之母的根须上。根须微微震颤——微弱，但还在。]

**翠玲：** 千年之母还有一口气。最后一片叶子还在。只要她活着，翠林就有希望。我要守住这最后一口气。

[她站起来，看着玩家。翠绿的眼瞳中没有了骄傲，没有了傲慢——也没有了绝望。取而代之的是某种新的东西——平静的、深沉的决意。]

**翠玲：** 不用弓也能守护。用手，用身体，用两百四十年积累的对翠林的了解。

**翠玲：** ……好轻。

**翠玲：** 原来不拿弓的时候，手……这么轻。

*——两百年了。第一次觉得手是空的。但空的手——才能抓住新的东西。*

### 场景描写

一年后。

翠林——或者说，曾经的翠林。

灵脉修复工程持续了整整一年。玩家调遣的灵脉修复师团队拆除了暗潮的要塞，逐段疏通被堵塞的灵脉主干道。过程漫长、艰难，期间还经历了三次暗潮的破坏性反扑。但最终——灵脉重新流动了。

最先感受到变化的是千年之母的残骸。那棵枯死的巨木不会再活过来了——一千年的生命已经燃尽。但她的根须在地下依然延伸着，成为了灵脉的管道。灵力通过她的根须流向翠林的每一个角落，就像母亲的血液通过脐带流向孩子。

嫩芽——长大了。

不再是指甲大小的两片叶子。一年的时间，它长成了一棵一人高的小树。银白色的树干——和千年之母一模一样。翠绿的叶子茂密而鲜亮，在阳光下闪着柔和的光。它扎根在千年之母的残骸上，从古树的养分中汲取力量，向上，向上。

而在它旁边——三步之遥——另一棵树也破土而出。

没有人种下它。没有人浇灌它。它自己从灰烬中长出来的。枝叶比第一棵矮一些，但同样翠绿，同样生机勃勃。两棵小树的枝条在风中相互交缠，像是两只手在握着。

翠林的其他地方也在缓慢恢复。溪流重新流淌——虽然只是细流。月光花的种子在潮湿的溪边萌发——虽然还只是幼苗。几棵新生的小树在灰烬中冒头——虽然只有手臂粗细。

远远不是从前的翠林。但——是活着的翠林。

### 剧情对话

[翠玲站在两棵小树之间。她的翠绿长发重新长回了原来的长度——比剪掉搓弓弦之前还长了一些。银弓挂在背上，弓弦是新的——玩家用修复灵脉时回收的千年之母藤丝重新搓制。]

[她看起来和一年前完全不同了。不是外表的变化——是气质。曾经高高在上的骄傲消失了，取而代之的是一种沉静的温和。她的站姿不再是弓手的紧绷，而是松弛的、安定的。]

[银叶站在远处，手臂的伤早已痊愈。她的眼眶有些红——但嘴角是笑的。]

[玩家走来。]

**翠玲：** 你来了。

[笑。不是傲慢的笑，不是苦涩的笑。只是——笑。]

**翠玲：** 看。

[她指向两棵小树。]

**翠玲：** 左边那棵是千年之母留下的种子长成的。我们知道。但右边那棵——

[她歪头看着右边那棵自行生长的小树，表情有些困惑，也有些温柔。]

**翠玲：** 没有人种。没有人浇。它自己从灰烬里长出来的。精灵长老院查了所有记录，说这从来没有发生过。

[她蹲下身，手指轻轻触碰右边小树的树干。]

**翠玲：** 你知道精灵怎么说吗？他们说这棵树是"灰烬之子"——从毁灭中自行诞生的生命。不属于千年之母的血脉，不属于翠林任何一棵古树的后代。它是……全新的。

[她站起身，目光在两棵小树之间移动。左边的银白色树干，右边的嫩绿色树干。枝条相互交缠。]

**翠玲：** 一棵是继承。一棵是新生。

[她转向玩家。]

**翠玲：** 就像我们。

[沉默了一小会儿。然后她从背上取下银弓。]

**翠玲：** 一年了。灵脉修好了。翠林在恢复。千年之母……虽然不在了，但她的孩子在长大。

[她把银弓举起来，对着两棵小树之间的天空。那里，从树冠的缺口中，一束阳光正好落下来——金色的光柱打在灰烬中萌发的新草上。]

**翠玲：** 我说过。第一万零四支箭——射向明天。

[她从箭筒中取出一支箭。不是银杉箭——那三支已经在捕获决战中用完了。也不是枯枝箭——翠林已经没有枯枝了。]

[这是一支新箭。箭杆是用两棵小树自然落下的嫩枝制作的——一半银白，一半嫩绿。箭羽是新生月光花的花瓣。箭镞是银弓之塔回炉的精灵银。]

[新翠林的第一支箭。]

**翠玲：** 师父教我射箭的时候说：弓手的第一箭，决定了你射箭的理由。

[她搭箭拉弦。新弦的声音——既有千年之母的沧桑，又有新生藤丝的清脆。两种声音交织成一种从未有过的旋律。]

**翠玲：** 我的第一箭，射穿了一片树叶。理由是——师父让我射的。

[拉弓至半满。]

**翠玲：** 之后的一万零二支箭，理由各不相同。为了骄傲，为了翠林，为了银月议庭，为了杀人，为了救树。

[拉弓至全满。弓弦颤抖。]

**翠玲：** 第一万零三支，理由是——悲伤。

[她调整了瞄准角度。不是对着前方，而是——向上。对着树冠缺口中那束阳光。]

**翠玲：** 第一万零四支——

[深呼吸。]

**翠玲：** 理由是明天。

[放弦。]

[箭矢飞上天空。穿过树冠的缺口，穿过阳光的金色光柱，笔直飞入蓝天深处。越飞越高，越飞越小，最终——消失在了光芒中。]

[看不到它落在哪里。也许它永远不会落下来。]

[翠玲放下弓，微微笑着。]

**翠玲：** 这支箭不需要命中任何目标。

[她把银弓重新挂回背上。转身面向玩家。]

**翠玲：** 因为——明天不是靶心。明天是方向。

[她走到玩家面前，伸出右手——那只拉了一万零四支箭的手。]

**翠玲：** 翠玲·希瓦纳尔。翠林射手长——不对，前射手长。

[笑了笑。]

**翠玲：** 现在只是……一个为翠林种树的精灵。以后请多指教了。

[她的右手被玩家握住。手掌上满是两百年拉弓留下的茧——但掌心是温暖的。]

[银叶在远处擦了擦眼泪，然后快步跑过来。]

**银叶：** 射手长——！不对——翠玲——！不对——呃——

**翠玲：** 叫什么都行。只要别叫"不落之箭"了。

[她仰头望向天空——箭矢消失的方向。]

**翠玲：** 那支箭还在飞呢。让它一直飞下去吧。

*——第一万零四支箭。射向明天。*

*——它会飞很久很久。*

*——因为明天——永远不会到达。所以箭——永远不会落下。*

*——"不落之箭"。原来这个名字——应该是给这支箭的。*
