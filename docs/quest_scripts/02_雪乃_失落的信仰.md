## Event 01 — 开战宣言「神殿的钟声」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.12-15 |
| 情感基调 | 虔诚的后方，温柔与恐惧并存 |
| 战斗条件 | 5回合内击溃敌方先锋（雪乃不可攻击） |
| 特殊机制 | 敌方「祝福」每回合+1兵力；雪乃在后方医疗帐篷，不可操作不可被攻击 |
| 敌方配置 | 暗潮先锋兵×6(Lv.10) + 暗潮祭司×2(Lv.12)（祝福源） |
| 奖励 | 250 EXP, 600G, 祈祷珠×1 |

### 场景描写

王都神殿区的青铜钟声敲响了第七遍——那是战争钟，每一声都比上一声更沉。

神殿广场已经被临时改建为前线中转站。白色医疗帐篷一字排开，帆布在晚风中微微鼓胀，像是无数沉默的肺在呼吸。空气中交织着草药的苦涩、止血药粉的辛辣和铁锈般的血腥气。帐篷里的灯火透过帆布渗出来，远远望去像是一排昏黄的灯笼，暖色的光与广场上冰冷的月色泾渭分明。

雪乃跪在第三顶帐篷里。她的白色祭司袍已经被血渍浸透了下摆，膝盖处的布料因为反复跪下而磨得发亮。她的双手悬在一名伤兵的腹部上方，掌心有淡淡的银白色光芒流淌而出——那是治愈术，像是一层极薄的月光被编织成了纱布，缓缓覆盖在翻开的伤口上。

她的额头沁着细密的汗珠。黑色长发用一根简单的白色缎带束在脑后，但几缕碎发仍然垂落在脸颊两侧，随着她每一次呼吸而轻轻摇曳。她的嘴唇微微张开，无声地念着某种祈祷文——嘴唇的动作极轻，像是怕惊扰了什么。

### 剧情对话

[帐篷内。伤兵痛苦地呻吟着，身体不自觉地挣动。雪乃轻轻按住他的肩膀，银白色的治愈光芒在她掌下扩散。]

**雪乃：** 不要动，很快就好。……伤口比看起来浅。你很幸运。

**伤兵甲：** 嘶……痛……

**雪乃：** 我知道。再忍一下。数到十就不痛了。来，跟我一起——一、二……

[伤兵咬着牙，跟着她的节奏数着。数到七的时候，伤口表面已经被一层银色的光膜覆盖，血止住了。伤兵的呼吸逐渐平缓。]

**雪乃：** 好了。剩下的让身体自己慢慢来。喝一点水，不要急着起身。

[她站起来，膝盖发出一声细微的咯吱响——跪太久了。助手明里端着一盆热水挤进帐篷。]

**明里：** 雪乃大人，外面又送来十二个重伤的。第五帐篷已经满了，第六帐篷的草药快用完了……

**雪乃：** 十二个。……重伤的先分到第三帐篷来，我亲自处理。轻伤的用草药包止血，让见习祭司去第六帐篷补位。

**明里：** 可您已经连续治愈了四个时辰……

**雪乃：** 明里。

[她的声音很轻，但有一种不容置疑的平静。]

**雪乃：** 治愈的优先顺序是什么？

**明里：** ……失血过多者优先。内脏受损者优先。断肢者——

**雪乃：** 对。然后是普通刀伤、箭伤、钝击伤。按顺序来，不要乱。乱了就会有人等不到。等不到的人——

[她停顿了一瞬。]

**雪乃：** ——就再也等不到了。

[明里低下头，快步出了帐篷。雪乃转身查看下一个伤兵——一个年轻士兵，左臂被从肩膀到手肘整个劈开，骨头外露，白森森的。他已经昏迷了。]

**雪乃：** ……很深。

[她深吸一口气，将双手覆上伤口。银白色光芒涌出，比方才更亮也更密集。她的眉心微微蹙起——这种程度的治愈需要大量集中精力。]

**伤兵乙：** （醒转，尖叫）啊——！

**雪乃：** 别动！你动了我就没法对齐骨头！

[她一手按住伤兵的肩膀，一手维持治愈术。她的力气不大，但按得极准——正好压在神经节上，伤兵的挣扎停了下来。]

**伤兵乙：** 疼……疼啊……求你……

**雪乃：** 我知道疼。骨头接回去一定会疼。但疼说明神经还活着，还能接。你该庆幸。

[她的声音温柔却绝不动摇，像是一条溪流——柔软但始终朝着同一个方向。]

**雪乃：** 咬住这个。

[她用膝盖夹着一根木棒递过去。伤兵咬住了。木棒上已经有无数个牙印——不知道之前有多少人咬过。]

[治愈完成。伤兵的手臂包扎好了，骨头接上了，但需要至少三周才能恢复。雪乃抹了一把额头的汗，站起身。]

**老祭司：** 雪乃大人。您该休息了。再这样下去您自己会倒的。

**雪乃：** 外面还有多少人在等？

**老祭司：** ……还有二十多个。

**雪乃：** 那就还不能休息。

**老祭司：** 可是——

**雪乃：** 害怕也会害怕。累了也会累。

[她回头看了老祭司一眼。她的眼睛很亮，但眼底有一层薄薄的红血丝。]

**雪乃：** 但如果我停下手，就有人会死。

[老祭司叹了口气，不再劝了。]

[帐篷帘掀开，一名传令兵冲进来。]

**传令兵：** 雪乃大人！前线急报——第二波攻势已经打响，预计半个时辰后会有新一批伤员送到！

**雪乃：** 人数？

**传令兵：** 预估……三十到五十。

[雪乃沉默了两秒。然后微微点头。]

**雪乃：** 明里，把第七帐篷也开出来。备品不够的话去神殿储藏室取。跟守卫说是我的口谕。

**明里：** 是！

[明里跑出帐篷。雪乃正要转向下一个伤兵时，传令兵又补了一句。]

**传令兵：** 对了……凛团长的骑士团在前线担任先锋。情况很激烈。

[雪乃的手停在半空中。]

[只有一瞬。不到一秒。]

[然后她的手落回伤兵身上，继续治愈。]

**明里：** （小声，在帐篷门口回头看了一眼）……雪乃大人？

**雪乃：** 嗯？

**明里：** 凛团长……会没事的吧？

[雪乃没有回答。她低着头，银白色的治愈光芒在她掌下稳定地流淌。她的表情没有任何变化。]

[但明里注意到——雪乃按在伤兵肩膀上的那只手，指尖微微收紧了。]

**雪乃：** ……专心做你的事，明里。

[夜深了。帐篷外传来远处的喊杀声和金属碰撞声，像是暴风雨在很远很远的地方翻滚。]

[又一个伤兵被抬进来。这次是一个很年轻的女兵，脸色白得像纸，腹部被长枪贯穿。雪乃一看就知道——这个伤太重了。普通治愈术不够。]

**雪乃：** ……要用高阶治愈了。

**老祭司：** 高阶？！雪乃大人，您现在的灵力——

**雪乃：** 够一次。

**老祭司：** 如果用完了，后面的人——

**雪乃：** 后面的人，我休息一刻钟就能继续。她不行。她等不了一刻钟。

[她将双手悬在女兵腹部上方。闭上眼。低声念诵祈祷文——这次出声了，声音轻柔如风中的铃铛。]

**雪乃：** 万物之光，听闻祈祷。伤痛者求安息，流血者求止息。请将您的慈悲——

[银白色光芒从她掌心爆发，比之前所有治愈都更亮更浓烈。女兵腹部的伤口以肉眼可见的速度愈合——皮肉重新生长，血止住了。]

[雪乃睁开眼。她晃了一下。老祭司赶紧扶住她。]

**雪乃：** ……没事。站得住。

[她确实站住了。但她的脸色比方才苍白了一个色度。]

### 战斗阶段

[战斗开始。玩家操控天城前线部队迎击暗潮先锋。雪乃在后方医疗帐篷区域，不可操作，不可被攻击。]

[敌方暗潮祭司每回合释放「祝福」，使一名存活的暗潮士兵兵力+1。]

*——钟声又响了。第八遍。每一声都比上一声更沉。*

*——前线的人在拼命。我能做的，只有让他们拼命的时候不用担心身后。*

[第1回合。暗潮先锋冲击天城防线。画面角落可以看到医疗帐篷中雪乃的剪影——她跪在伤兵身边，治愈的光芒未曾间断。]

[第2回合。暗潮祭司「祝福」发动，一名暗潮士兵兵力+1。]

[第3回合。新一批伤兵被送入帐篷区。画面闪过雪乃忙碌的身影。]

**雪乃：** （远处传来的声音）这边，把他放平！——不要拔箭！拔箭会大出血！

[第4回合。「祝福」再次发动。暗潮兵力持续增加。]

[第5回合。最后一轮攻势。]

### 战后

[战斗结束。天城军惨胜。前线的喊杀声渐渐平息，取而代之的是伤兵的呻吟和收拾战场的嘈杂声。]

[医疗帐篷里。雪乃终于停了下来。她坐在帐篷角落的一只木箱上，双手放在膝盖上，手指微微颤抖——不是因为害怕，是因为连续五个多时辰的治愈术消耗了她几乎全部的灵力。]

[她低头看着自己的手。手掌心有一层淡淡的灼烧痕迹——过度使用治愈术的副作用。]

**明里：** 雪乃大人……结束了。今天一共治疗了七十三人。其中重伤二十一人，全部脱离危险。没有人死在帐篷里。

**雪乃：** ……七十三。

[她轻轻点头。]

**雪乃：** 那就好。

[她站起身，掀开帐篷门帘走了出去。]

[外面是神殿广场。月亮已经升到了天幕正中央，冷白色的光铺了一地。远处是王都的城墙轮廓，更远处——前线的方向——还有零星的火光在闪烁。]

[雪乃站在帐篷前，望着前线的方向。]

[很久。]

[晚风吹过来，掀起她被血浸透的袍角。她将双手合十，置于胸前。闭上眼。嘴唇无声地翕动——在祈祷。]

**明里：** （走出帐篷，站在她身后，轻声）……雪乃大人。在为前线的人祈祷吗？

[雪乃没有回答。她的眼睛仍然闭着，双手合十的指尖微微用力。]

[明里没有再问。她安静地站在雪乃身后，等着。]

[很久之后，雪乃睁开眼。她放下双手。望着远方的火光，轻声说了一句话。声音很小，小到明里几乎没有听清。]

**雪乃：** ……请平安回来。

[她说的是"请平安回来"。不是"请你们平安回来"。]

[是"请平安回来"。]

[明里听出了区别。但她什么也没说。]

## Event 02 — 前哨遭遇「战地天使」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.15-18 |
| 情感基调 | 不分敌我的慈悲，信念的初次展露 |
| 战斗条件 | 击退双方混战部队或坚守6回合 |
| 特殊机制 | 雪乃在战场中央搭白色帐篷，帐篷为不可攻击区域；帐篷周围2格内所有单位（含敌方）每回合恢复5%HP |
| 敌方配置 | 暗潮游击兵×5(Lv.13) + 暗潮斥候×3(Lv.14) + 天城溃兵×4(Lv.11)（混战状态） |
| 奖励 | 350 EXP, 800G |

### 场景描写

前线以西十二里。一片被战火蹂躏过的荒原。

这里昨天刚经历了一场遭遇战——天城的一支侦察队与暗潮的游击队在此不期而遇，双方缠斗了半日后各自撤退，留下了满地的尸体和还在呻吟的伤兵。没有人来收拢他们。两边都没有。

荒原中央，一顶白色帐篷突兀地立在那里。

帐篷很小，只够容纳四五个人。帆布是纯白的——在这片焦土、血迹、断刃遍布的荒原上，那抹白色刺目得近乎荒谬。帐篷顶部插着一根细长的竹竿，竿头系着一条白色缎带，在风中慢慢飘动。那是天城神官的治愈标识——意味着"此处有救助，不分身份"。

雪乃蹲在帐篷前。她的祭司袍的下摆扎进了腰带里，方便行动。袖子卷到肘部以上，露出纤细但布满细小烫伤痕迹的前臂——那是常年调配药剂的痕迹。她正在为一个躺在担架上的人处理伤口。

那个人穿着暗潮的黑色军服。

### 剧情对话

[荒原上。雪乃将一条浸了草药汁的绷带缠上暗潮伤兵的小腿。伤兵意识模糊，偶尔发出微弱的呻吟。]

**雪乃：** 骨头没断。但肌腱受损了，需要固定至少十天。……你醒着吗？能听到我说话就眨眨眼。

[暗潮伤兵勉强眨了一下眼。]

**雪乃：** 好。不要动。我给你缝合侧面那道伤口。会疼，但你忍着。

[她低头仔细缝合伤口。针线穿过皮肉时伤兵痛得全身痉挛，但她的手极稳，每一针的间距几乎完全相同。]

[帐篷后方，一名天城溃兵靠在帆布上，右臂吊着绷带。他看着雪乃治疗暗潮士兵，脸上露出难以置信的表情。]

**天城溃兵甲：** 雪乃大人……您在做什么？

**雪乃：** 缝合伤口。你看不出来吗。

**天城溃兵甲：** 我是说——他是暗潮的人！昨天就是他们杀了我们连队的弟兄——

**雪乃：** 我知道。

[她的手没有停。]

**天城溃兵甲：** 那您为什么还要——

**雪乃：** 他也是某人的儿子。

[天城溃兵张了张嘴，没有说出话来。]

**雪乃：** 也许是某人的丈夫。也许是某人的父亲。他的家人现在也许正在某个地方等他回去。……你的家人不也在等你吗？

[溃兵低下头。]

**雪乃：** 伤口不分阵营。痛苦也不分阵营。

[她缝完最后一针，用牙齿咬断线头。然后给伤口覆上一层药膏，再用干净的布包扎好。]

[远处传来脚步声。三名暗潮士兵持武器从荒原东侧接近——他们是来搜寻昨天走散的同伴的。]

**暗潮斥候：** 站住！你——

[他看到了白色帐篷和帐篷前的雪乃。然后他看到了躺在地上被治疗的暗潮同伴。]

**暗潮斥候：** ……你在治疗我们的人？

**雪乃：** （站起身，平静地面对他们）是的。

[暗潮斥候们互相对视。他们的手握着武器，但没有举起来。]

**暗潮士兵甲：** 你是天城的祭司。

**雪乃：** 是。

**暗潮士兵甲：** 天城的祭司，治疗暗潮的伤兵。为什么？

**雪乃：** 因为他受了伤。需要有人治。

**暗潮士兵甲：** ……你不怕我们杀了你？

[雪乃看着他。她的眼神清澈而安静，像是深冬里被冻住的湖面——冰下有东西在流动，但表面毫无波澜。]

**雪乃：** 怕。

[暗潮士兵明显愣了一下。]

**雪乃：** 你们有武器，我没有。你们有三个人，我只有一个。我当然怕。

[她向前迈了一步。]

**雪乃：** 但如果我因为害怕就不救人——那我跪在神像前念的那些祈祷词，算什么呢？

[暗潮斥候盯着她看了很久。然后他慢慢把刀收回鞘里。]

**暗潮斥候：** ……你是个奇怪的人。

**雪乃：** 也许吧。请不要攻击帐篷里的人。不管他们穿什么颜色的军服。

[暗潮斥候看了看帐篷里——帐篷里不只有暗潮伤兵，还有两名天城溃兵。所有人都受了伤，并排躺着，被同样的白色绷带包扎。]

**暗潮斥候：** ……天城的人也在？

**雪乃：** 在。他们也受了伤。我不能治了这个不治那个。你能接受吗？

[暗潮斥候沉默了片刻。然后他点了点头。]

**暗潮斥候：** 我去把伤兵带回去。至于你的帐篷——我们不动。

**雪乃：** 谢谢。

[暗潮斥候带着两名士兵走向帐篷。他弯腰查看暗潮伤兵的伤势时，不经意间和旁边的天城溃兵对上了视线。两人几乎同时别开头。]

[但帐篷太小了。他们的肩膀几乎挨在一起。]

**天城溃兵乙：** （低声嘟囔）……别碰我。

**暗潮伤兵：** （虚弱地）……谁想碰你。

[尴尬的沉默。]

[雪乃端着一碗水走进帐篷。她先递给天城溃兵，然后又倒了一碗递给暗潮伤兵。两碗水从同一个水壶倒出来。]

**雪乃：** 都喝。脱水比流血更危险。

[两边的伤兵各自喝水。谁也没有说话。但那种剑拔弩张的敌意——淡了一点。只淡了一点。]

[过了一会儿。天城溃兵乙低头看着自己空了的碗。然后他看了看暗潮伤兵——对方还没喝完，手在发抖，水洒出来一半。]

[天城溃兵犹豫了很久。然后他伸出没受伤的那只手，帮暗潮伤兵稳住了碗。]

[两人对视了一眼。什么也没说。]

[雪乃在帐篷外看到了这一幕。她微微垂下视线，嘴角出现了一个极浅极浅的弧度。]

**明里：** （跑来）雪乃大人！西面又发现了一队暗潮游击兵在靠近！

**雪乃：** ……人数？

**明里：** 大概五六个！带着武器！

[雪乃的表情恢复平静。她看向西面——尘土在地平线上扬起。]

**雪乃：** 把白色旗帜举高。让他们看清楚。

**明里：** 可是如果他们不管旗帜——

**雪乃：** 那就祈祷他们还有一点良心。

[她的声音没有颤抖。但她将双手合十，在胸前轻轻按了一下——那是她每次害怕时的习惯动作。]

## Event 03 — 角色事件·壹「通敌的慈悲」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情（无战斗） |
| 情感基调 | 第一次质疑，信仰与现实的碰撞 |
| 奖励 | 200 EXP, 银色祈祷链×1, CG「沉默的神像」 |

### 场景描写

王都。军务司。

这栋灰色石头建筑坐落在神殿区的东侧，与神殿隔着一条窄窄的石板巷。建筑外墙上攀着枯死的常春藤，像是一张干瘪的渔网贴在石壁上。窗户又窄又高，阳光只能以极吝啬的角度切入室内，在地板上投下刀刃般锋利的光条。

空气中有灰尘和旧纸的气味。角落的铁火盆里烧着廉价的木炭，烟气呛人，却驱不散石壁渗出的阴冷。

### 剧情对话

#### 第一幕：战场边缘

[三天前。前线与后方之间的灰色地带。一片被遗弃的农田，田埂上散落着生锈的农具和被遗弃的行囊。]

[雪乃跪在田埂旁。她面前躺着一个暗潮士兵——重伤。腹部被天城长枪刺穿，虽然枪被拔出了，但伤口仍在汩汩地涌血。他的脸色灰白，眼球开始上翻。]

[雪乃的治愈术笼罩在他的腹部。银白色光芒在血泊中流淌，试图封闭那个致命的伤口。]

**明里：** 雪乃大人！那是暗潮的人！

**雪乃：** 我有眼睛，明里。我看到了他的军服颜色。

**明里：** 可是——

**雪乃：** 也看到了他的伤口。你觉得哪个更重要？

[明里张口结舌。]

[远处，几名天城巡逻兵发现了这一幕。其中一人快步走来——是同为神官系的明里的师姐，名叫葵。她穿着和雪乃同样的白色祭司袍，但胸前多了一枚军务司的铜质纹章。]

**葵：** 雪乃！你在做什么？！

**雪乃：** 治愈。

**葵：** 我看到了你在治愈！我问的是你为什么在治愈一个暗潮的兵！

[雪乃没有抬头。她的手稳稳地按在暗潮伤兵的腹部，治愈光芒一秒都没有中断。]

**雪乃：** 因为他快死了。

**葵：** 他是敌人！

**雪乃：** 他是伤者。

**葵：** 这是通敌！军务司会追究——

**雪乃：** 那就让他们追究。

[她终于抬起头，看着葵。她的表情很平静，平静到葵不由自主地后退了半步。]

**雪乃：** 神的光芒不分敌我。这是入门第一天教的。你忘了吗，葵？

**葵：** 那是祈祷词！不是让你在战场上治疗敌方士兵的！

**雪乃：** 那你告诉我——什么时候祈祷词变成了只在嘴上念念的东西？什么时候"神的光芒不分敌我"变成了"神的光芒只照自己人"？

[葵说不出话了。她的脸涨得通红，在愤怒和羞愧之间摇摆。]

**雪乃：** 如果你想阻止我，可以。用力推开我的手就行。

[她重新低头，继续治愈。]

**雪乃：** 但你推开我的手的时候，这个人就会死。你准备好了吗？

[葵站在原地。手抬起来了。又放下了。最终她转身离开，嘴里低声骂了一句什么。]

[暗潮伤兵的伤口终于止住了血。他的呼吸微弱但稳定。雪乃收回双手，手掌心被灼烧的痕迹又深了一层。]

**雪乃：** （对昏迷的暗潮伤兵，轻声）……活下去。不管你是谁的敌人，总有人在等你回去。

#### 第二幕：审讯室

[军务司审讯室。一间狭长的房间，只有一面窄窗。窗外的光像一把刀切进来，正好劈在房间中央的石质桌台上。]

[雪乃坐在桌台一侧的木椅上。她的祭司袍已经换过了，干净整洁，但袖口处有一小块血渍——也许是自己的手伤渗出来的。]

[对面坐着军法官——一个面容严肃的中年男人，面前摊着一份卷宗。葵站在他身后，不敢看雪乃。]

**军法官：** 雪乃。王宫神官。现任前线医疗帐篷总管。

**雪乃：** 是。

**军法官：** 三日前，你在灰色地带救治了一名暗潮重伤士兵。有同僚目击并举报。你承认吗？

**雪乃：** 承认。

[军法官似乎没料到她回答得这么干脆。他翻了翻卷宗。]

**军法官：** 根据天城战时条例第204条，在未经军务司批准的情况下，向敌方人员提供医疗救助，等同于资敌行为。你有什么要辩解的？

**雪乃：** 不是辩解。是提问。

**军法官：** ……什么？

**雪乃：** 神赐予我治愈的力量，难道是为了让我选择性地使用吗？

[军法官皱起眉头。]

**军法官：** 雪乃，这不是神学讨论的场合。

**雪乃：** 不是神学讨论。是事实。

[她的声音平稳如水。]

**雪乃：** 我的治愈术来自信仰。信仰的核心是什么？是"慈悲不分敌我"。如果我因为对方的军服颜色而拒绝治疗——我违背的不是军务司的条例，而是赐予我力量的根源。

**军法官：** 那你打算怎么办？每次遇到敌方伤兵都去治疗？

**雪乃：** 如果他会死，如果我能救——是的。

**军法官：** 你知道这意味着什么。

**雪乃：** 我知道。

[沉默。军法官盯着她看了很久。然后他合上卷宗。]

**军法官：** 判决如下。雪乃，鉴于你的行为虽出于善意但严重违反战时条例，且在同僚劝阻后仍然执意为之——剥夺你的前线医疗权限。即日起调离前线，转至后方物资管理部门。

[雪乃的眼睫毛颤了一下。]

**雪乃：** ……后方？

**军法官：** 是的。你不再被允许接触前线伤员。

**雪乃：** 前线每天有几十个人在流血。现在祭司人手已经——

**军法官：** 这不是你需要操心的事。

[雪乃闭上嘴。她的双手放在膝盖上，指尖微微蜷缩。]

**军法官：** 你可以离开了。

[雪乃站起身。她走到门口时，停了一步。]

**雪乃：** 军法官大人。

**军法官：** 还有什么事？

**雪乃：** ……没有。

[她走了出去。门在她身后关上的一瞬间，她听到了葵的声音——小到几乎是气音。]

**葵：** ……对不起。

[雪乃没有停。]

#### 第三幕：祈祷室

[深夜。王都神殿。祈祷室。]

[这是一间圆形的小房间。穹顶上镶嵌着彩色玻璃，但夜晚没有阳光穿过，只有几根白色蜡烛在石台上燃烧，烛光将四壁的阴影推成摇曳的巨人。正中央立着一尊石质神像——面容模糊，性别不明，双手摊开向下，像是在倾洒什么。]

[雪乃独自跪在神像前。]

[祈祷室里只有蜡烛偶尔爆出灯花的细微声响。她的影子被烛光投射在身后的墙壁上，显得比她本人高大得多、也孤独得多。]

**雪乃：** ……

[她双手合十。闭着眼。嘴唇在动，但发不出声音。]

[很久之后，她开口了。声音很低，像是在对自己说话，又像是在对神像说话。]

**雪乃：** 我做错了吗？

[沉默。神像不会回答。]

**雪乃：** 他们说我"通敌"。因为我治了一个快要死掉的人。他穿着黑色军服，所以我不应该救他。如果他穿白色军服，我就应该救他。

[她睁开眼，抬头看着神像模糊的面容。]

**雪乃：** 但您教给我的第一句话是什么？"痛苦者呼唤你时，不要问他的名字。"不要问他的名字。不是——不要问他的军服颜色。不是——不要问他站在哪一边。

[她的声音微微发颤。]

**雪乃：** 如果治愈痛苦是罪过，那您赐予我这力量——到底是为了什么？

[蜡烛的火焰跳了一下。像是风吹进来了。但祈祷室的门窗紧闭。]

**雪乃：** 我不认罪。我做不到认罪。

[她低下头，额头触到了合十的指尖。]

**雪乃：** 可是他们把我调走了。后方。物资管理。

[她的嘴角出现了一个极苦涩的弧度。]

**雪乃：** 物资管理。……我是祭司。我的手是用来治愈的。现在让我去数箱子。

[沉默。]

**雪乃：** ……我不怕受罚。不怕被骂。不怕被人说"通敌"。

[她停顿了。很长的停顿。长到蜡烛又爆了一次灯花。]

**雪乃：** 我怕的是——

[她抬起头。烛光在她的瞳孔中跳动。]

**雪乃：** 凛在前线。

[这四个字从她的嘴里落出来，轻得像羽毛，重得像铁。]

**雪乃：** 她每天都在最前面。她的铠甲每天都多一道裂痕。她受伤的时候——

[她的声音终于碎了。只碎了一瞬。]

**雪乃：** 她受伤了，谁来治她？

[祈祷室里安静得像是被抽走了所有空气。只有蜡烛的光在墙壁上无声地摇摆。]

[雪乃跪了很久。很久很久。直到蜡烛烧短了一截，烛泪在石台上凝成白色的小山丘。]

[最后，她站起来。膝盖已经跪得发麻，她踉跄了一步才站稳。她最后看了神像一眼。]

**雪乃：** ……如果您不回答我，那我就自己找答案。

[她转身走向门口。推开门。走廊上空无一人，只有冷风从窗户灌进来。]

[她迈出祈祷室的那一刻，低声说了最后一句话。]

**雪乃：** 凛……在前线。她受伤了，谁来治她？

[她的脚步声在走廊中渐渐远去。祈祷室里，蜡烛的火焰终于被穿堂风吹灭了。神像在黑暗中沉默地立着。双手仍然摊开向下。什么也没有倾洒出来。]

> **获得奖励：** 200 EXP / 银色祈祷链×1 / CG「沉默的神像」
>
> *「银色祈祷链」——一条银质的链子，末端坠着一枚小小的月形吊坠。链身因为长期贴身佩戴而变得温热。吊坠背面刻着两个字——已经被磨得很浅了，要对着光才能勉强辨认："守护"。这和凛的护臂上刻的字一样。也许是巧合。*
>
> *CG「沉默的神像」——圆形祈祷室中，蜡烛将尽。雪乃独跪在石质神像前，双手合十贴在额头上，长发散落在背后。神像面容模糊，双手向下摊开，空无一物。画面色调冷蓝，只有蜡烛周围有一圈暖黄色的光。雪乃的影子被拉得很长，投射在神像的脚下。*

## Event 04 — 防御反击·一「静默的壁垒」

| 项目 | 数据 |

### 场景描写

雪乃被调离前线后的第五天。

王都后方物资仓库。一排排木架上堆满了麻布袋和木箱——粮食、绷带、箭矢、铁锭。空气中弥漫着干草和霉味的混合气息。仓库的窗户又小又少，白天也显得昏暗。

雪乃坐在仓库角落的一张矮桌前。面前摊着一本厚厚的物资登记簿，旁边堆着今天新到的清单。她穿着普通的灰色工作服——不是祭司袍。军务司的人收走了她的祭司袍，说"调离期间不得穿着神官服饰"。

她在一丝不苟地登记物资。每一笔字都写得工工整整，和她治愈时缝合伤口的手法一样——精准、稳定、没有一丝多余。

但她的目光偶尔会飘向仓库深处堆放草药原料的区域。

### 剧情对话

#### 第一幕：灰色工作服

[白天。仓库。雪乃正在清点一批新到的箭矢。明里偷偷跑来看她。]

**明里：** 雪乃大人……

**雪乃：** 别叫"大人"了。我现在只是个仓库管理员。

**明里：** 才不是！您是……您永远是雪乃大人。

[雪乃微微笑了一下。那个笑容很淡，像是冬天的阳光——有温度，但暖不了什么。]

**雪乃：** 前线怎么样了？

**明里：** ……不太好。您走之后，重伤存活率下降了两成。见习祭司们经验不够，高阶治愈一个人都不会……

[雪乃登记物资的手停顿了一瞬。]

**雪乃：** 伤亡名单里……有没有骑士团的人？

**明里：** 有几个。不过凛团长没有受重伤——上次报告里说她只是轻伤。

[雪乃点了点头。她的表情几乎没变。但明里注意到——她握笔的手指松了一点。]

**雪乃：** ……那就好。

**明里：** 雪乃大人。您不该在这里。您应该在前线。

**雪乃：** 军务司不这么认为。

**明里：** 军务司的人根本不懂！前线的人在死——

**雪乃：** 明里。

[她的声音依然温柔，但多了一丝不容辩驳的坚定。]

**雪乃：** 不要因为我而给自己惹麻烦。回去吧。你的病人在等你。

[明里红着眼睛离开了。雪乃目送她走远。然后她放下笔，走向仓库深处堆放草药原料的区域。]

[她蹲下身，开始翻检草药。她的动作极快——辨认、分拣、挑选，每一株草药在她手中只停留一两秒就被归入不同的堆。这不是仓库管理员的手法。这是一个经验丰富的药剂师的手法。]

#### 第二幕：深夜的仓库

[深夜。仓库。所有人都已经离开。]

[雪乃坐在那张矮桌前。但桌上不再是物资登记簿——而是一排排小巧的布包、一个研钵、几瓶液体和散落的草药碎屑。]

[她在配药。]

[烛光下，她的手指在草药之间飞速穿梭。研磨、混合、称量、包裹。每一个动作都精确到了近乎强迫症的程度。她的嘴唇微微抿着，眉心有一道浅浅的竖纹——那是极度专注时才有的表情。]

[她已经连续配药三个时辰了。手指上的皮肤因为反复接触草药汁液而裂开了——细小的口子密密麻麻地布满指尖和指节。有几道裂口在渗血，血珠滴在草药上，被她仔细擦掉。]

**雪乃：** （自语，极轻）……这种配比的止血效果最好。再加半分白芷，可以抗感染。

[她打开一个小布包，将配好的药粉仔细装入。然后她拿起一张小小的纸条，用工整的字迹写下用法说明：]

*"止血散。外伤出血时直接撒于伤口。深伤口需先清洗再用。用量以覆盖伤口为准，不可过多。——Y"*

[她在每一份药包里都放了一张这样的手写说明。字迹清晰端正，即使在战场上光线不好的情况下也能看清。]

[桌上已经堆了二十多份药包。她还在继续。]

**雪乃：** （低声，对着空气）……我不能站前线了。但至少可以做这些。

[她的手没有停。即使手指裂开了，即使血在渗——她的手没有停。]

[门被轻轻敲了三下。雪乃浑身一僵。然后门推开了——是明里。]

**明里：** （低声）雪乃大人。药包我来拿了。

**雪乃：** ……小心点。监察队最近盯得紧。

**明里：** 我知道。我走后门。

[雪乃将包好的药包装进一个普通的麻布袋里——外面看起来和普通物资毫无区别。]

**雪乃：** 这一批有二十四份。止血散十二份，退烧药粉八份，高级伤药四份。每份里面都有用法说明，让他们一定要看。

**明里：** 嗯。

**雪乃：** 高级伤药不要随便用。只在重伤且没有祭司在场的时候才用。这东西原料太稀少了，我没法经常配。

**明里：** 我记住了。

[明里接过麻布袋，转身要走。然后她又回头。]

**明里：** 雪乃大人。前线的士兵都在传——"又有药了"。他们不知道是谁送的，但是……

**雪乃：** 不知道就好。

**明里：** 可是——

**雪乃：** 明里。如果被军务司发现，不只是我，你也会受牵连。所以——不知道就好。

[明里抿着嘴，点了点头。她把麻布袋藏在斗篷里，消失在夜色中。]

[雪乃回到桌前。她低头看着自己的手——十根手指没有一根完好的。裂口，血痕，草药汁染成的黑绿色渍痕。这双手昨天还在登记箭矢数量。现在——]

**雪乃：** （苦笑）……这才是这双手该做的事。

[她打了一盆冷水来洗手。水一接触裂开的皮肤就钻心地疼。她倒吸一口气，但没有把手缩回来。一点一点地把草药残渍洗干净。]

[然后她从怀里取出一小罐药膏——这是她给自己配的，最基础的皮肤愈合药膏。她仔细地涂在每一道裂口上，动作像是在对待别人的伤口一样轻柔。]

[涂完之后，她坐在矮桌前发了一会儿呆。]

[仓库的窗户很小，但够她看到外面的天空。月亮被薄云遮住了一半，星星稀稀落落的。]

**雪乃：** （望着窗外，极轻极轻地）凛……请平安。

[那两个字在空旷的仓库里回荡了一下，就消散了。像是从未被说出口一样。]

#### 第三幕：前线的议论

[前线营地。夜晚。几名天城士兵围坐在帐篷里检查装备。]

**足轻甲：** 喂，你看到了吗？今天又有药包送来了。

**足轻乙：** 嘘——小声点。监察队的人耳朵尖。

**足轻甲：** （压低声音）这已经是第三批了。止血散、退烧药粉……都是好东西。上次第四连的阿久受了重伤，就是用那个高级伤药撑到祭司赶来的。

**弓兵甲：** 谁送的？

**足轻甲：** 不知道。每次都是半夜出现在物资帐篷里。外面套着普通麻布袋，不仔细看根本分不出来。

**弓兵乙：** 但是里面每一份都有手写的用法说明。字写得特别好看，特别清楚。连"深伤口需先清洗再用"都写上了。

**祭司甲：** ……那个字迹。

[祭司甲沉默了一下。其他人看向他。]

**祭司甲：** 什么也没有。大概是我看错了。

[他没有看错。他认得那个字迹。在前线医疗帐篷的病历本上，他见过无数次那个工整到近乎虔诚的字迹。但他什么也没说。]

**足轻乙：** 不管是谁，这些药救了不少人的命。

**足轻甲：** 对。……真希望能当面谢谢那个人。

[帐篷外面，风吹过营地。远处有哨兵换岗的脚步声。月亮从云后面探出来，照亮了帐篷旁边堆放的物资——其中有几个普通的麻布袋。看起来和其他物资完全一样。]

[但如果打开其中一个，就会看到里面整齐码放的药包。每一份都用白布仔细包裹，每一份里面都有一张手写的用法说明。说明的末尾，都签着同一个字母——]

*Y。*

### 战斗阶段

[天城前线阵地。深夜。暗潮发动突然夜袭。]

[战斗开始。天城军全体因为"神秘草药包"的效果，初始HP+10%。]

*——她不在前线。但她的手，从未离开过这里。*

[第1回合。暗潮夜袭部队从三个方向同时突入天城营地。]

[天城足轻和弓兵仓促列阵迎敌。祭司在后方展开治愈阵。]

[第2回合。暗潮夜袭队长亲自冲锋，攻破天城左翼防线。]

[一名天城足轻被重击，HP骤降。跌至50%时——「草药包」发动。他从怀里掏出一个布包，按照说明撒在伤口上。HP恢复至70%。]

**足轻甲：** ——还能打！这药管用！

[第3回合。天城弓兵集中火力压制暗潮弓兵。祭司治愈前排伤兵。]

[第4回合。暗潮投入预备队试图扩大突破口。天城防线承压但未崩溃。]

[第5回合。天城祭司发动集体治愈。前线伤兵得到恢复后发起反击。]

[第6回合。暗潮夜袭部队在天城的顽强防御下开始后撤。天城军追击——战斗结束。]

### 战后

[战斗结束。天城营地灯火通明，士兵们在救治伤员、修补防御工事。]

[镜头切换——王都后方。仓库。]

[雪乃靠在窗边。她没有睡。她当然不知道前线此刻发生了什么——但她知道前线每天晚上都可能有战斗。每天晚上，都有人在用她配的药。]

[她低头看着自己的双手。手指上的裂口今天又深了一些。她重新涂上药膏，用干净的白布一根一根地包扎好。]

**雪乃：** （自语）……明天还要配一批退烧药粉。最近天冷，前线的人容易发烧。

[她拿起笔，在一张纸条上写下明天需要的草药清单。字迹依然工整。手在微微发抖——但字迹依然工整。]

[写完之后，她将纸条放在枕头旁边。然后走到窗前。]

[窗外是王都的夜空。看不到前线。看不到火光。只有黑沉沉的天幕和几颗疏落的星。]

[她将手贴在冰冷的窗玻璃上。玻璃上凝着一层薄薄的水雾，她的掌心留下一个温热的印记。]

**雪乃：** （望着前线的方向，声音轻得几乎不存在）凛……请平安。

[这是她每天晚上都会说的话。每天晚上，同一个方向，同一句话。]

[窗玻璃上的掌印慢慢被冷气重新覆盖，消失不见。]

[但明天晚上，同一个位置，会再次出现一个温热的掌印。]

## Event 05 — 角色事件·贰「圣光的真相」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情（无战斗） |
| 情感基调 | 信仰粉碎，真相比背叛更痛 |
| 奖励 | 300 EXP, 古代文献碎片×1, CG「破碎的光环」 |

### 场景描写

深夜。王都后方物资仓库。

仓库最深处有一面墙壁长年渗水，墙根的石砖上长满了暗绿色的苔藓。因为这片区域过于潮湿，所有正经物资都堆放在前半部分，后方只剩下几个被遗忘的旧木箱——箱体已经发霉，铁皮锁扣锈成了暗红色。雪乃每天登记物资时都要经过这里，却从未打开过它们。直到今夜，她在为前线配药寻找一种罕见草药时，撬开了其中一个箱子的盖板。

箱子里没有草药。

是书。几十卷泛黄的羊皮纸卷轴，墨迹褪色严重，卷轴两端的木轴被虫蛀得千疮百孔。卷轴上有封印用的蜡封——蜡封上压着神殿的纹章，但纹章的样式极古老，和如今通用的神殿纹章至少差了三代。这些文献，被封存在这里至少已有百年。

### 剧情对话

#### 第一幕：仓库的发现

[深夜。仓库角落。雪乃蹲在撬开的旧木箱前，手里捧着一卷羊皮纸。她身边只有一盏豆大的油灯，灯光在发霉的墙壁上投下摇曳的橘色光斑。]

[她的眼睛飞速扫过卷轴上的古文字。最初是好奇——这些是被遗忘的古籍，也许记载了失传的药方。但随着她往下读，她的表情开始变化。]

**雪乃：** ……"光之术"？

[她将卷轴凑近油灯。古文字写得密密麻麻，有些地方的墨迹几乎消失了，但关键段落仍可辨认。]

**雪乃：** （低声朗读）"初代祭司长记——第七纪。'圣光'之名并非自始有之。在神殿创立之初，此力量被称为'反转之力'——即将暗之能量反转为治愈能量的技法。力量的本源……"

[她的手指在某一行字上停住了。]

**雪乃：** "力量的本源，与暗之眷属所用之术……同根同源。"

[仓库里安静得能听见木箱里虫蛀的细微声响。]

**雪乃：** ……同根同源？

[她放下这卷，又拿起另一卷。展开。继续读。]

**雪乃：** "……为避免信众动摇，经初代祭司长与神殿议会决议，将'反转之力'更名为'圣光'，并在所有教典中删除有关力量起源的记述。此后一切教义将以'神赐'为统一口径……"

[她的手开始发抖。不是因为冷。]

**雪乃：** 删除……？统一口径……？

[她将卷轴展开到末尾。那里有一段被特意用不同颜色墨水追记的文字，字迹潦草而急促——像是后来的某个人偷偷补上去的。]

**雪乃：** "第三十七代文献官·遗笔——此文献本应在大净化中焚毁。我擅自藏了一份。也许永远不会有人读到。但如果有人读到了——请相信自己的眼睛。神殿建立在谎言之上。圣光是谎言。身份是谎言。唯一不是谎言的，是你掌心的温度。"

[雪乃的双手捧着卷轴，指尖已经完全白了。]

*——圣光是谎言。*

*——我从七岁起跪在神殿里学习的一切……都建立在谎言之上？*

**雪乃：** （极轻地，像是怕把什么东西震碎）……不。也许是伪造的。也许是有人故意——

[她低头再看了一遍封印的蜡封。那上面的古代神殿纹章不可能伪造——每一代纹章的微雕工艺都有独特的暗记，这是她在见习祭司时期就学过的鉴别法。]

**雪乃：** ……是真的。

[她将卷轴小心地卷好，藏在衣服里。然后她站起来，拿着油灯走向仓库出口。走了三步，她停住了。]

**雪乃：** （回头看了一眼那堆旧箱子）……还有多少？

[她回去又翻了翻。至少还有二三十卷。每一卷的封印上都压着不同时代的古代神殿纹章。]

*——整整百年的真相。被锁在发霉的箱子里。和蛀虫住在一起。*

#### 第二幕：神殿禁区

[两天后。深夜。王都神殿。]

[神殿的正殿在夜间关闭，但侧翼的文献馆仍有值夜的灯火。文献馆的最深处有一扇铁门——门上挂着"禁入"的铜牌，门锁是特制的神殿锁，需要长老级别以上的神力印记才能打开。]

[雪乃站在铁门前。]

[她穿着灰色工作服，头发用深色布巾包住，在昏暗的走廊里几乎与墙壁的阴影融为一体。她手里攥着从仓库找到的一枚古代祭司长令牌——那也是从旧箱子里翻出来的，令牌上残留的神力印记虽然微弱，但和铁门上的锁属于同一时代。]

*——也许还能用。也许已经失效了。*

[她深吸一口气，将令牌按在门锁上。]

[一秒。两秒。三秒。]

[锁内传出一声极低的咔嗒声。铁门无声地打开了一条缝。门缝里涌出一股封存了不知多少年的干燥气息——像是一座坟墓的呼吸。]

[雪乃侧身挤了进去。]

[禁区内是一间圆形的地下室。石壁上嵌着已经熄灭的灯座——她用随身带的火折子点亮了最近的一盏。微弱的火光照亮了房间的一角：石质书架从地面延伸到穹顶，上面密密麻麻地排列着更多的卷轴和石板。]

[但房间正中央的东西，让她彻底停住了脚步。]

[一个石台。石台上放着一个水晶球体——球体约拳头大小，内部有暗紫色的气流在缓缓旋转。那种颜色、那种流动方式——她在战场上见过。暗潮的术士施法时，手中凝聚的就是这种暗紫色的能量。]

**雪乃：** ……怎么会？

[她走近石台。石台底座上刻着古文字——和仓库卷轴上同一种文字。]

**雪乃：** （蹲下辨认）"原初之核。光与暗之共同起源。一切治愈与一切毁灭，皆从此出。神殿以此为'圣光'之种，暗之眷属以此为'深渊'之种。种子相同，开出的花不同。但根——永远是同一条根。"

[她伸出手，指尖在距离水晶球体几寸的地方停住了。球体内的暗紫色气流突然加速旋转，向她的方向涌来——像是感应到了什么。然后球体表面泛起一层银白色的光。]

[那是圣光的颜色。]

[暗紫与银白，在水晶球体内交织、缠绕、不分你我。]

**雪乃：** （退后一步，声音发颤）……一样的。真的是一样的。

[她环顾四周。石壁上还有更多刻字。她挨个辨认——全是历代祭司长的记录。每一代都知道。每一代都选择了沉默。]

*——他们全都知道。*

*——从第一代到现在。每一个站在神坛上宣讲"圣光是神的恩赐"的人——都知道这是谎话。*

[她的膝盖突然发软。她没有倒下，但不得不伸手扶住石台才能站稳。]

**雪乃：** 那我……

[她低头看着自己的手掌。掌心依然留着长年使用治愈术的灼烧痕迹。]

**雪乃：** 我以为我用的是"圣光"。我以为这份力量来自信仰。我以为——只要我足够虔诚，就对得起这份力量。

[她的声音在空荡的地下室里回荡，和石壁上的古文字交叠在一起。]

**雪乃：** 但它从来就不是"圣光"。……它和暗潮的魔法——同一个东西。

[水晶球体中的暗紫与银白仍然无声地旋转着，仿佛在嘲笑她。]

[她松开扶着石台的手。直起身。深深吸了一口气。然后她用之前来时的路原路返回，将铁门重新关上。走廊上依然空无一人。]

#### 第三幕：崩塌

[神殿后方的小花园。夜风很冷。]

[雪乃坐在花园角落的一棵老榕树下。树根虬结，从土里拱起了一段，形成了一个天然的座位。她就坐在那上面，双脚踩着落叶，双手垂在膝盖两侧。]

[月光透过树冠的缝隙落下来，在她身上投下斑驳的光影。她的脸色很白——比月光还白。]

[她就这样坐了很久。]

**雪乃：** （自语，声音平静得近乎冰冷）……七岁。我七岁进的神殿。

[她抬头看着树冠间的月亮。]

**雪乃：** 七岁的时候，师父让我第一次触摸圣光。我把手伸出去，掌心亮了。银白色的。那么温暖。师父说——"这是神回应了你的祈祷。"

[她慢慢地将右手摊开。掌心的灼烧痕迹在月光下清晰可见。]

**雪乃：** 我信了。从那天起一直信到现在。十四年。每天早课祈祷，每天晚课诵经。每一次治愈伤者的时候，我都会先在心里说一句——"感谢您的恩赐。"

[她攥紧了拳头。]

**雪乃：** 但没有"恩赐"。没有"神回应了你的祈祷"。有的只是……一种被改了名字的力量。一种和敌人手里一模一样的力量。

[停顿。夜风吹过，落叶在她脚边打了个旋。]

**雪乃：** 他们说暗潮的魔法是"邪恶的"。"不洁的"。"被诅咒的"。我也这么教过见习祭司。我对他们说——"我们的圣光是与暗潮魔法截然对立的力量。"

[她发出一声极低极短的笑——不是笑，是某种更接近呜咽的声音。]

**雪乃：** 截然对立？……同根同源。

[沉默。]

[很长的沉默。长到月亮从一片云后面完全走了出来，冷白色的光铺满了整个花园。]

**雪乃：** ……那我救过的那些人呢？

[她的声音终于有了裂痕。]

**雪乃：** 七十三个。第一夜救的七十三个人。战地帐篷里救的几百个人。他们伤口上覆盖的银白色光芒——那到底是什么？是"圣光"？是"暗之力"？还是——

[她双手捂住了脸。]

**雪乃：** ……还是只是一个骗局的副产品？

[她的肩膀在发抖。但她没有哭出声。她把所有声音都按在了掌心里。]

[过了很久，她放下手。眼眶是红的，但脸上没有泪痕。也许是风干的。也许是根本没有流出来。]

**雪乃：** ……我救治暗潮伤兵的时候，他们说我"通敌"。说我用"圣洁的力量"去救"邪恶的敌人"。

[她低声重复。]

**雪乃：** 圣洁的力量。邪恶的敌人。

[又一声短促的、接近碎裂的笑。]

**雪乃：** 原来全是一样的。力量是一样的。人也是一样的。从一开始——就没有什么圣洁和邪恶之分。

[她站起来。双腿因为坐太久而有些僵硬。她扶着树干站了一会儿，等血液重新流通。]

[然后她低头看着自己的手掌——那双布满灼烧痕迹和草药裂口的手。]

**雪乃：** 但是……伤口愈合了。那些人活下来了。不管这力量叫什么名字——它确实救了人。

[她握了握拳。又松开。]

**雪乃：** ……名字是假的。效果是真的。

[她慢慢走出花园。经过神殿正殿前的广场时，她停下脚步，回头看了一眼神殿的轮廓——月光下的白色穹顶庄严而空洞，像一顶华丽的空壳。]

**雪乃：** 我不需要你了。

[她说这句话时的语气，不像是在对建筑说话。像是在对某个她曾经深信不疑的存在告别。]

[然后她转回头，继续向仓库的方向走去。]

*——信仰碎了。但手还在。手还在，就能救人。*

*——不需要"圣光"这个名字。不需要"神官"这个身份。*

*——只需要这双手。*

## Event 06 — 遭遇战·转折「瘟疫村庄」

| 项目 | 数据 |

### 场景描写

天城王朝西南边境。柿子村。

这个村庄坐落在一条干涸河床的转弯处，村子周围是大片荒芜的农田——稻穗还没来得及收割就枯死了，黑褐色的稻秆歪歪斜斜地立在田里，像是一排排插在泥土中的枯骨。村口的石碑上刻着"柿子村"三个字，石碑下堆着几束已经枯萎的野花和一只碎了的陶碗——那是村民祈福用的。

瘟疫已经蔓延了十天。

村子里弥漫着一种甜腐的气味——那是高烧和坏疽混合后的特有臭味。大部分村民躺在自家的草席上，面色灰黄，身上长满了暗红色的疹子。有几户人家的门板被钉死了——里面的人已经死了，钉门板是为了防止尸毒扩散。苍蝇在钉死的门缝里进进出出，嗡嗡声在空旷的村道上回荡。

村子正中央的打谷场上，一顶白色帐篷被搭了起来。帐篷比雪乃之前在战地用的那顶更大，帆布上用木炭画了一个粗糙的十字——不是神殿的标志，只是一个最原始的"这里有救助"的记号。帐篷前挂着一面用白布和竹竿扎成的旗子，上面什么纹章也没有。

雪乃就在帐篷里。她已经在这里待了三天了。

### 剧情对话

#### 第一幕：违抗命令

[三天前。王都。物资仓库。]

[一名传令兵跑进来，手里拿着前线送来的急报。]

**传令兵：** 管理员！西南柿子村爆发瘟疫，已有三十余人感染——

**雪乃：** （放下登记簿）死了多少？

**传令兵：** ……七人。还在增加。村民向王都请求支援，但军务司认为该区域不具战略价值，已经驳回了。

[雪乃沉默了两秒。]

**雪乃：** 不具战略价值。

**传令兵：** 是的。那个村子不在任何补给线或防御线上。军务司说……现在前线的医疗资源本来就紧张，没有余力——

**雪乃：** 七个人死了。三十多个人在等死。因为他们的村子"不具战略价值"。

[传令兵被她的语气吓了一跳。雪乃的声音并不大，甚至可以说很平静——但那种平静下面有一层极薄的冰，踩上去随时会碎。]

**雪乃：** 药方我有。草药我能配。路程多远？

**传令兵：** 快马一天半。但是管理员——您没有离开王都的许可——

**雪乃：** 我知道。

[她已经站了起来。她走到仓库后方的草药区域，开始飞速地分拣药材。]

**传令兵：** 管理员？！您在做什么——

**雪乃：** 我在做我该做的事。

[她头也不回，双手在草药之间穿梭。分拣、称量、装袋。]

**雪乃：** 你可以去报告军务司。告诉他们仓库管理员雪乃擅自离岗了。顺便告诉他们——我带走了一批草药，数量和种类我会留一份清单。记在我账上。

[传令兵张口结舌地看着她将大包小包的药材装进背囊。]

**传令兵：** 这是违反军令——

**雪乃：** 你说得对。这是违反军令。

[她背起背囊，走到仓库门口。在门框处停了一步，回头看了传令兵一眼。]

**雪乃：** 但那些人正在死。

[她的眼神清澈而坚定。没有犹豫。没有恐惧。只有一种纯粹到近乎刺眼的东西——像是被打碎之后重新拼起来的信念，比原来更锋利。]

**雪乃：** 我不需要谁的许可去救一个正在死的人。

[她推开仓库的门。外面是王都灰蒙蒙的天空。她迈出去的那一步，踩碎了门槛上积的一层薄灰。]

#### 第二幕：瘟疫中的帐篷

[柿子村。第三天。]

[帐篷内。雪乃跪在一张草席前，面前躺着一个约莫十岁的小女孩。女孩浑身滚烫，暗红色疹子从脖子一直蔓延到手臂。她的意识时清时浊，偶尔发出猫一样的呜咽。]

[雪乃将一碗深褐色的药汤端到女孩嘴边。药汤冒着热气，苦味刺鼻。]

**雪乃：** 来，小口喝。很苦，但喝下去就不会那么烫了。

**小女孩：** （虚弱地）呜……不要……苦……

**雪乃：** 我知道苦。但你是勇敢的孩子对不对？喝完三口就不用喝了。来，第一口——

[小女孩勉强喝了一口，立刻皱起整张脸。]

**雪乃：** 好。真厉害。第二口——

[又一口。女孩的眼泪流了下来。]

**雪乃：** 最后一口。喝完我给你唱歌好不好？

**小女孩：** ……真的？

**雪乃：** 真的。

[女孩闭着眼，咽下了最后一口药。]

[雪乃把药碗放在一旁。她伸出手，覆在女孩的额头上。掌心有微弱的光芒涌出——不是银白色的。是一种更温暖的、偏向淡金色的光。]

*——这不是"圣光"。这只是……我的力量。不需要名字。不需要来源。它能退烧。这就够了。*

**雪乃：** （轻声哼唱）……风过山岗，水过溪旁，花落了还会再开。你闭上眼，我守在这，醒来的时候天就亮了……

[那是一首很古老的天城摇篮曲。雪乃的歌声很轻，轻到只有她和女孩之间的那一点距离才能听到。]

[女孩的呼吸逐渐平稳了。烧退了一些。她睡着了，眉头不再皱得那么紧。]

[雪乃收回手。她站起来走到帐篷门口，深深呼了一口气——帐篷外的空气虽然也弥漫着病腐的气味，但至少流通。帐篷里的气味已经浓到让人反胃。]

[一名老妇人拄着拐杖走过来。她的腿上也长了疹子，但症状较轻——已经吃过药了。]

**村长婆婆：** 祭司大人……

**雪乃：** 我不是祭司了。叫我雪乃就好。

**村长婆婆：** 雪乃大人。你已经三天没合眼了。这样下去你自己会倒的。

**雪乃：** （微笑）有人也对我说过一样的话。我当时的回答是——还有人在等，就不能停。

**村长婆婆：** 可你是一个人来的啊。连个帮手都没有。王都难道就派了你一个人吗？

[雪乃沉默了一拍。]

**雪乃：** ……王都没有派我来。我自己来的。

[村长婆婆愣住了。]

**村长婆婆：** 自己……？

**雪乃：** 嗯。军务司说柿子村"不具战略价值"。没有批准救援。

[老妇人的脸上露出一种复杂的表情——愤怒、苦涩、但又夹杂着一丝被人在意的感动。]

**村长婆婆：** 那你为什么来？你一个人过来，万一自己也染上了……

**雪乃：** 我配了预防的药。喝了。应该没事。

[她停顿了一下。]

**雪乃：** 至于为什么来——因为你们在这里。因为有人在生病。因为我能做些什么。

[她低头看着自己的手。]

**雪乃：** 不需要更多的理由了。

[远处突然传来哨声。帐篷外一个负责瞭望的村民少年跑过来，脸色发白。]

**瞭望少年：** 不好了！东面山路上来了一队人——穿黑色铠甲——是暗潮的兵！

[雪乃的眼神一凝。]

**雪乃：** 多少人？

**瞭望少年：** 十几个！拿着武器！好像是冲着村子来的！

[雪乃看了看帐篷里那些还在发烧的村民。然后她看向东面——尘土在山路上扬起。]

**雪乃：** ……这里有病人。搬不了。走不了。

[她转向身边仅有的几个还能站着的村民。]

**雪乃：** 把帐篷周围能搬的东西都搬过来垒成掩体。所有还能走动的人退到帐篷后面。不要主动攻击——但如果他们靠近帐篷，用一切能用的东西挡住他们。

**村长婆婆：** 大人——我们只是农民，打不过那些兵——

**雪乃：** 不需要打过。只需要拖住。

[她走回帐篷。把正在熬煮的大锅药汤搬到帐篷最里面。检查每一个病人的状况。然后她走回帐篷门口，拿起那根挂白旗的竹竿，将白旗举得更高。]

**雪乃：** （对自己说）……这里是医疗站。不分阵营。如果他们有一点人性——就不会攻击病人。

*——但如果他们没有呢？*

*——那就用我的身体挡在帐篷前面。*

### 战斗阶段

[战斗开始。暗潮游击队从东面进入村庄。玩家部队赶来增援。目标：保护雪乃的治疗帐篷6回合。]

[帐篷位于地图中央。HP 800。雪乃在帐篷内治愈村民，每回合瘟疫治愈进度+17%。进度条到100%时全村民得救。帐篷被摧毁则任务失败。]

*——她不会跑。帐篷里有三十个还在发烧的人。她跑了，他们就死了。*

[第1回合。暗潮破坏兵分两路接近帐篷。弓兵在后方掩护射击。玩家需要在帐篷外围建立防线。]

[帐篷内传出雪乃的声音。]

**雪乃：** ——别怕。外面的声音不要管。专心呼吸。药已经在起作用了。

[治愈进度：17%]

[第2回合。暗潮破坏兵突破左侧，试图接近帐篷。]

**暗潮破坏兵：** 队长！那不像是军用帐篷——里面好像是平民——

**雪乃：** （睁开眼，露出一个极淡的微笑）……醒了？感觉怎么样？

**雪乃：** 不了。我得回去了。我已经擅自离岗三天了。回去之后大概会有很多麻烦在等着我。

**雪乃：** 对。我答应了。

*——但这三十个人活了。*

*——这就够了。*

## Event 07 — 角色事件·叁「被逐出的祈祷者」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情（无战斗） |
| 情感基调 | 身份的丧失，以及丧失之后的发现 |
| 奖励 | 250 EXP, 被收走的祈祷珠(记忆)×1, CG「空手的祈祷」 |

### 场景描写

王都神殿。大殿。

神殿大殿是一座穹顶建筑，穹顶内侧绘着天城开国神话的壁画——诸神降临、赐予光明、驱逐黑暗。壁画的颜色经过几百年的烟熏和氧化，已经从鲜艳变成了一种暧昧的灰褐色，只有某些角落的金箔还在穹顶的高处反射着微弱的光芒。

大殿正中央是一座三阶石台，石台最高处放置着一尊与祈祷室中同样面容模糊的神像——但这尊更大，足有两人高。神像双手向下摊开，手指微微弯曲，像是在倾洒什么又像是在挽留什么。石台两侧各立着六根石柱，柱面雕刻着藤蔓和星辰的纹样。

今天的大殿不是平日的模样。石台前方被清空了，地面上铺着一条黑色长毡——那是天城神官系中专用于"剥夺仪式"的布。这条毡布一年用不了一次，但每个见习祭司在入门时都被告知过它的存在：如果你做了不可饶恕的事，你将跪在这条黑毡上。

### 剧情对话

#### 第一幕：审判席

[大殿内。三名神殿长老并排坐在石台的第二阶上。他们穿着暗金色的祭司袍，胸前佩戴着高阶神官的月形徽章。最中间的是首席长老——一个头发花白、面容严厉的老人，他的眉心有一道深深的竖纹，像是常年皱眉刻出来的。]

[大殿两侧站着十几名神官和见习祭司。他们被要求来观礼——"以儆效尤"。葵也在其中。她站在最角落，低着头不敢看前方。]

[大殿正门打开。雪乃走了进来。]

[她穿着灰色工作服——从被调到仓库之后就没穿过祭司袍。她的头发没有束起来，散落在肩膀两侧，几缕碎发贴在脸颊上。她的脸色仍然苍白——从柿子村回来后只过了一天，还没有完全恢复。]

[但她的脊背是直的。她的步伐平稳。她的视线平视前方，越过黑色长毡，越过三位长老，落在他们身后的那尊神像上。]

[她走到黑毡前方。站定。]

**首席长老：** 雪乃。

**雪乃：** 是。

**首席长老：** 你知道今天为什么被叫到这里。

**雪乃：** 知道。

[首席长老展开面前的一份文书。他的声音在穹顶下回荡，低沉而冰冷。]

**首席长老：** 王宫神官雪乃。罪状如下——第一，在灰色地带擅自治疗暗潮敌方伤兵，已受前线医疗权限剥夺之处分。第二，在调离期间，擅自利用仓库资源配制药物并秘密向前线运送，违反物资管理条例。第三——

[他抬起头，目光像两把铁钉钉在雪乃脸上。]

**首席长老：** 未经任何许可，擅自离开王都，前往非战略区域柿子村进行未授权的医疗活动。途中消耗军方草药资源，且在该区域引发暗潮游击队注意，间接导致交战事件。

[大殿里安静得能听到穹顶壁画上的金箔碎屑偶尔掉落的声音。]

**首席长老：** 对以上罪状，你有异议吗？

**雪乃：** 事实部分没有异议。

**首席长老：** "事实部分"？

**雪乃：** 是。我确实做了这些事。但我不认为"救治病人"是罪。

[首席长老的眉心竖纹更深了。]

**首席长老：** 雪乃，你是受过完整训练的神官。你应该比任何人都清楚——神殿的秩序是治愈力量的基石。没有秩序，就没有力量。你一个人的"慈悲"，动摇的是整个体系。

**雪乃：** ……秩序。

[她微微低下头。然后重新抬起来。]

**雪乃：** 长老大人。我有一个问题。

**首席长老：** 现在不是你提问的时候。

**雪乃：** "圣光"的力量来源——您知道吗？

[大殿里的空气突然凝固了。]

[三名长老的表情在同一个瞬间发生了微妙的变化——首席长老的瞳孔收缩了一瞬，左侧长老的手指在膝盖上攥紧了，右侧长老不自然地移开了视线。]

[这些反应只有一瞬。但足够了。]

*——他们知道。他们全都知道。*

**首席长老：** （语气骤然冷硬）这与今天的审判无关。

**雪乃：** 无关吗？

[她的声音很轻，但在穹顶下的回响让它变得清晰无比。]

**雪乃：** 你们教我——"圣光来自信仰，来自虔诚，来自对神的奉献"。你们教每一个见习祭司这些话。但你们心里——

[她停了一下。像是在选择用词，又像是在给对方最后一次机会。]

**雪乃：** 你们心里知道那不是真的。

[大殿两侧的神官和见习祭司面面相觑。有人露出困惑的表情，有人隐约感觉到了什么。]

**首席长老：** （猛地站起身）够了！

[他的声音像一记重锤砸在穹顶上。]

**首席长老：** 雪乃！你不仅违反了军令和神殿法规，现在还妄图动摇神殿教义！你——你是要叛教吗？！

**雪乃：** 不是我叛教。是教义叛了真相。

[这句话落下去，大殿里响起了一片极低的惊呼。]

[首席长老的脸色变得铁青。他深吸一口气，重新坐下，将文书重重拍在膝盖上。]

**首席长老：** 判决如下。

#### 第二幕：剥夺仪式

[首席长老的声音在穹顶下回荡，像是从很远很远的地方传来。]

**首席长老：** 鉴于王宫神官雪乃屡次违反军令与神殿法规，且在审判中对教义发表亵渎言论——经三长老一致决议，即日起，永久剥夺其神官身份。

[停顿。那两个字在穹顶下回荡了很久。]

**首席长老：** 收回祭司袍。收回祈祷珠。收回神官令牌。从今日起，雪乃不再是天城王朝的神官。不得进入神殿。不得使用神官名义。不得以神殿之名行使任何治愈行为。

[他向旁边的执事点了点头。]

[执事走到雪乃面前。他手里端着一个黑色漆盘——那是用来盛放被收回之物的。]

**执事：** （低声，几乎带着歉意）……雪乃大人。请交出您的祈祷珠。

[雪乃低下头。她的右手缓缓伸进衣领——那里挂着一条细绳，绳子末端系着那颗白色祈祷珠。那颗她从入门那天起就没有离过身的珠子。那颗被她每天握在掌心祈祷、表面因为反复摩挲而变得温润光滑的珠子。]

[她将绳子从脖子上取下来。祈祷珠在她的掌心里——珠面上有一条细小的裂纹。那是她曾经握得太紧时捏出来的。]

[她看了它很久。]

*——七岁那年，师父把它挂在我脖子上的时候说："这是你和神之间的纽带。握着它祈祷，神就能听到你。"*

*——我握了十四年。掌心的温度已经渗进了珠面里。*

*——现在——*

[她将祈祷珠放在黑色漆盘上。珠子在漆盘上轻轻转了半圈，发出一声极细的"叮"，然后安静地躺在那里。]

**执事：** ……神官令牌。

[雪乃从腰间解下令牌。铜质的令牌已经被她的体温暖热了。她放在漆盘上。又一声轻响。]

**执事：** 祭司袍已在此前调离时收回。确认无误。剥夺仪式完成。

[首席长老站起来。他看着雪乃，像是在看一件令人遗憾的废品。]

**首席长老：** 你从今天起，不再是我们中的一员。希望你好自为之。

[雪乃没有回答。她只是站在那里，双手空空地垂在身侧。]

[大殿两侧的神官和见习祭司中，有人在低声议论，有人在摇头叹息。明里站在人群后方，双手捂着嘴，眼泪无声地流。]

[葵站在角落。她的拳头攥得关节发白。嘴唇在发抖——像是想说什么，又不敢说。]

[雪乃转身。她走向大殿的门口。脚下的黑毡在她的步伐下发出细微的沙沙声。]

[走到门口时，她停了一步。没有回头。]

**雪乃：** 长老大人。最后说一句。

[首席长老没有回应。但他也没有阻止。]

**雪乃：** 柿子村的三十二个人都活了。

[她推开大殿的门。阳光涌进来，将她的剪影投射在黑毡上——一个孤单的、空手的影子。]

[门在她身后关上了。]

#### 第三幕：力量仍在

[神殿外。石板路。]

[雪乃走出神殿的大门，站在石阶上。风吹过来，掀起她散落的长发。她抬头看了看天空——晴天。很蓝。蓝得有些刺眼。]

[她低头看着自己的双手。空空的。没有祈祷珠。没有令牌。身上穿的是灰色工作服，不是祭司袍。]

[她就这样站了一会儿。]

[然后她注意到——石阶下面蹲着一只猫。一只灰色的瘦猫。它的后腿在发抖，走路一瘸一拐的——后腿受了伤，可能被什么东西砸到了。]

[雪乃蹲下来。]

**雪乃：** ……你也受伤了啊。

[猫警惕地看着她，发出低低的呜声。]

[雪乃伸出右手。掌心朝下，悬在猫的后腿上方。]

[她闭上眼。]

*——没有祈祷珠了。没有祭司袍了。没有"神官"这个身份了。*

*——如果力量真的来自"身份"和"信仰"——那现在应该什么都不会发生。*

[她的掌心开始发热。]

[然后——光出现了。]

[不是银白色。是淡金色的，温暖的光。从她的掌心涌出来，落在猫的后腿上。光芒比以前微弱一些，但它确确实实地在流动。猫的后腿上的淤肿以肉眼可见的速度消退。猫停止了呜声，抬头看着雪乃。]

[雪乃睁开眼。她看着自己掌心的光——看了很久。]

**雪乃：** ……还在。

[她的声音带着一种奇异的平静。不是如释重负，不是狂喜。而是一种更深的、像是地壳运动之后尘埃落定的平静。]

**雪乃：** 力量还在。不是因为我是"神官"。不是因为祈祷珠。不是因为祭司袍。

[猫的后腿已经不疼了。它站起来，试探性地走了两步。然后它回头蹭了蹭雪乃的手。]

[雪乃轻轻摸了摸猫的头。]

**雪乃：** 这力量……是我自己的。

[她站起来。阳光正好从云后面完全走出来，铺满了整个石阶。她站在光里，影子被拉得很长，延伸到身后紧闭的神殿大门上。]

*——他们收走了珠子。收走了袍子。收走了我的名字。*

*——但他们收不走这双手。*

[她将双手合十，放在胸前。没有祈祷珠的手指交叉在一起——掌心对掌心，空空如也，但温热。]

**雪乃：** （闭眼，极轻地）……不向任何神祈祷。只向自己的手承诺。

[她睁开眼。放下手。向石阶下面走去。]

[走了几步，她回头看了神殿的穹顶最后一眼。白色的穹顶在蓝天下闪着光——庄严的、空洞的光。]

[她转回头。不再回望。]

[那只灰猫跟在她脚边走了一段路。在巷口的转角处，猫停下了。雪乃也停下了。]

**雪乃：** （对猫）你要去哪里就去哪里。伤好了，就自由了。

[猫歪了歪头。然后它转身，轻巧地跳上了围墙，消失在另一侧。]

[雪乃看着猫消失的方向。]

**雪乃：** ……自由了。

[她继续向前走。走进王都的街道里。走进人群里。]

[没有祭司袍。没有祈祷珠。没有令牌。没有任何标志表明她是谁。她只是一个穿灰色工作服的年轻女人，走在人群中，和所有人一样渺小，和所有人一样普通。]

[但她的手是温热的。那双手里的力量，从未离开过。]

> **获得奖励：** 250 EXP / 被收走的祈祷珠(记忆)×1 / CG「空手的祈祷」
>

### 场景描写

前线与后方之间的缓冲地带。一片被称作"三不管地"的荒芜丘陵。

这片丘陵地带北面是天城的防线，南面是暗潮的推进线，中间是一条约三里宽的无主之地。地面上满是弹坑和壕沟的残迹，枯草在碎石间顽强地生长，呈现出一种灰黄色的、死而不僵的颜色。偶尔能看到生锈的断剑或半埋在泥土里的头盔——这里曾经是主战场，现在成了两军都不愿踏足的鬼域。

在这片无主之地的正中央，丘陵的最高点上，矗立着一组帐篷。

不是一顶——是六顶。排成两列三排，帐篷之间用绳索和木桩连接，形成了一个简陋但完整的医疗营地。帐篷全部是白色的。每一顶帐篷前面都挂着一面白旗——不是天城的白旗，不是任何阵营的旗帜。只是纯白的布，什么纹章也没有。

营地入口立着一块木板，上面用黑炭写着几行字：

*"此处收治一切伤者。不问阵营。不问身份。不问来路。"*

*"放下武器方可进入。"*

*"——雪乃"*

营地里，左侧帐篷里躺着天城的伤兵，右侧帐篷里躺着暗潮的伤兵。中间的帐篷是药房和手术帐。所有伤兵都使用同样的白色绷带、同样的药物、同样的草席。

雪乃站在中央帐篷前。她不再穿灰色工作服了——她穿着一件自己缝制的白色长袍。不是祭司袍，没有任何神殿的纹章或标记。只是白色的。干干净净的白。

帐篷已经修补过了——被箭射穿的洞和魔法灼穿的缺口上缝着不同颜色的布块，有的是天城士兵从军服上撕下来的灰白色布条，有的是暗潮伤兵出院时留下的一片黑色衣角。拼在一起，像是一面无声的百衲旗。

营地里的气氛比前些日子安静了许多。双线防御战之后，天城监察队和暗潮散兵短期内都不再来犯——也许是因为吃了亏，也许是因为消息传开了：那个"疯祭司"的营地有人守着，打不划算。不管原因是什么，医疗站获得了一段难得的喘息期。

雪乃在药房帐篷里清点药材存量。她的白色长袍袖口卷到肘部，露出前臂上细密的药剂灼伤痕迹。手指上的裂口终于愈合了大半，但新的薄茧又覆了上来——那是每天研磨草药磨出来的。

空气中弥漫着晾晒草药的苦涩香气，混着帐篷帆布被太阳晒热后的干燥气味。远处有鸟叫声——这片"三不管地"因为长期无人耕种，反而长出了一些野草和灌木，鸟儿不知道什么是战线，它们只关心虫子。

### 剧情对话

#### 第一幕：伤兵的口述

[黄昏。医疗站入口。明里扶着一个几乎站不住的人走进来——一个天城士兵，铠甲碎裂了大半，左眼用血污的布条缠着，右腿拖在地上，每走一步都在身后留下一道血迹。]

**明里：** 雪乃大人——不，雪乃姐——这个人从北面爬过来的。他说他一定要见你。

[雪乃放下手中的药杵，快步走出帐篷。一看到伤兵的伤势，她的瞳孔微微收缩——这不是普通的刀伤。铠甲上的碎裂方式、伤口的灼烧痕迹——是被高浓度魔法直接命中的。]

**雪乃：** 明里，热水、绷带、高级伤药。快。

[她蹲下来，双手覆上伤兵的胸口。淡金色的治愈光芒涌出，稳住了他最危急的内伤。伤兵的呼吸从濒死的急促变为虚弱但稳定的节奏。]

**伤兵：** （抓住雪乃的袖子，声音嘶哑）……你就是……雪乃？

**雪乃：** 是。先不要说话。你的肺被灼伤了，每说一个字都在加重——

**伤兵：** 不——必须说。必须……现在就说。

[他的眼睛——唯一没有被缠住的那只——死死盯着雪乃。那只眼睛里有一种被某种执念支撑着的光，像是快要燃尽的烛火在最后一刻突然亮了。]

**伤兵：** 凛团长……

[雪乃的手停了一瞬。不到半秒。然后治愈继续。]

**雪乃：** ……凛怎么了。

**伤兵：** 骑士团被解散了。三天前……王庭下了令。说凛团长"抗命不遵"。把骑士团拆了，人分到各个前线部队去。

[雪乃的表情没有变化。但她按在伤兵胸口的手指微微收紧了——治愈的光芒波动了一下。]

**雪乃：** ……继续。

**伤兵：** 苍——就是凛团长的副手——为了保护团长跟监察队动了手。被打成重伤。现在……不知道在哪。活不活着都不知道。

[明里端着热水和绷带跑回来，看到雪乃的侧脸——白得像纸。]

**明里：** 雪乃姐……？

**雪乃：** （声音极平稳）放下。帮我把他的腿伤处理一下。先止血。

[明里蹲下来处理腿伤。伤兵还在说——像是把这些话烂在肚子里太久了，一旦开始就停不住。]

**伤兵：** 凛团长自己……她没有回王都。她带着剩下的几个人直接去了前线最深处。最前面。最危险的地方。

**雪乃：** ……一个人？

**伤兵：** 几乎是一个人。剩下跟她走的不到五个。其他人不敢。前线最深处——那里的暗潮主力部队——

[他咳了起来。咳出了血。雪乃加大了治愈的力度。]

**雪乃：** 不要再说了。

**伤兵：** 最后……最后一句。

[他用力攥着雪乃的袖子。布料在他手里被拧出了褶皱。]

**伤兵：** 我是从第七防线撤下来的。撤退的时候，远远地看到过她。凛团长。一个人站在残垣上面。铠甲全碎了。身上全是血。但她还站着。

[他闭上眼。]

**伤兵：** 她还站着。但是……那个样子——

[他说不下去了。不是因为伤。是因为一个士兵目睹自己的指挥官走向死亡时的那种窒息。]

[雪乃沉默了很久。治愈的光芒在她掌下稳定地流淌，没有一丝波动。她的脸上什么表情都没有。什么都没有。]

**雪乃：** ……我知道了。谢谢你。休息吧。

[伤兵终于松开了她的袖子。他的手落下去，整个人陷入了昏迷。]

[雪乃站起来。她走到药房帐篷后面——那个谁也看不到她的角落。]

[她靠在帐篷的木柱上。双手垂在身侧。]

[很久。]

[然后她慢慢地、慢慢地将双手举到面前。看着自己的手掌。掌心的灼烧痕迹在黄昏的光里清晰可见。]

*——骑士团被解散了。苍受了重伤。*

*——凛一个人在最前线。铠甲碎了。身上全是血。*

*——还站着。*

*——但那个样子——*

[她的手指蜷缩起来。攥成拳。攥得指节发白。]

*——那个样子，是不打算活着回来的样子。*

**雪乃：** （极轻地，像是牙缝里挤出来的）……你这个人。

[她的声音碎了。只碎了一瞬。然后她深吸一口气，将碎掉的东西重新咽了回去。]

#### 第二幕：回忆与独白

[深夜。医疗站。所有伤兵都已入睡。]

[雪乃坐在营地外围的一块岩石上。脚下是干涸的壕沟，远处是模糊的地平线。天上没有月亮，只有星星——密密麻麻的星，像是谁在黑色的幕布上戳了无数针眼，光从针眼里漏下来。]

[她抱着膝盖。怀里贴着那个伤兵口述的内容——明里帮她记录了下来，写在一张粗糙的纸上。纸已经被她翻来覆去地看了很多遍，边角都起了毛。]

**雪乃：** （对着星空，声音低得像是自言自语）……第一次见你，是什么时候来着。

[风吹过荒丘。枯草在她脚边沙沙作响。]

**雪乃：** 是在神殿的回廊里。你穿着骑士见习生的铠甲，太大了，肩甲歪到一边去了。你经过我身边的时候绊了一跤——被自己的披风绊的。

[她的嘴角出现了一个极浅的弧度。不是笑。是某种比笑更深也更疼的东西。]

**雪乃：** 你摔在地上也不说"痛"。自己爬起来，把肩甲推正，继续走。我在你后面看着，想——这个人真倔。

[停顿。]

**雪乃：** 后来你受了伤来医疗帐篷。手臂上被砍了一道。很深。我给你缝合的时候，你一声也不出。我说"你可以叫疼的"。你说——

[她停了一下。像是在回忆那个声音的质感。]

**雪乃：** 你说"叫了也不会不疼"。

[又是那个极浅的弧度。]

**雪乃：** 你就是这样的人。什么都扛着。什么都不说。受了伤也不让人看到。……你以为我不知道吗。每次你来帐篷，我都看得到。看得到你又多了几道伤。看得到你越来越瘦。看得到你的眼睛越来越深——像是在往下陷。

[她低下头，将脸埋进膝盖里。]

**雪乃：** 我在后方数箱子的时候，你在前线挡刀。我在仓库配药的时候，你在阵地上流血。我在这里建医疗站的时候——你的骑士团被拆了。苍倒了。你一个人……

[她的肩膀微微发抖。但没有声音。她把所有声音都压进了膝盖和手臂围成的黑暗里。]

[很久之后，她抬起头。眼眶是红的。但没有泪痕。]

**雪乃：** ……你不是不打算活着回来。你是不知道可以回哪里。

[她展开那张记录着伤兵口述的纸。在星光下再看了一遍。然后她将纸叠好，贴在心口。]

**雪乃：** 你可以回这里。

[这句话说得很轻。像是怕被星星听到。又像是想被某个远在前线的人听到。]

**雪乃：** 你可以回到我这里来。

[风又吹过来了。把她的黑发吹向了前方——前线的方向。]

#### 第三幕：告别明里

[翌日。清晨。雪乃已经收拾好了一个背囊——药材、绷带、干粮、水壶。不多，但每一样都是精挑细选过的，恰好够一个人在野外行走七到十天。]

[明里站在帐篷门口。她已经知道了。]

**明里：** ……你要去找她。

**雪乃：** 嗯。

**明里：** 前线最深处。暗潮主力部队的推进区域。

**雪乃：** 嗯。

**明里：** 你没有武器。没有护卫。没有战斗能力。你的治愈术用来保护自己都不够——

**雪乃：** 明里。

**明里：** 你会死的！

[她的声音尖锐地破了音。帐篷里的伤兵们被惊醒了几个，迷迷糊糊地抬头看了看，又躺了回去。]

[雪乃走到明里面前。她比明里高半个头。她伸出手，轻轻地按在明里的肩膀上。]

**雪乃：** 也许会。

**明里：** 那你——

**雪乃：** 但凛一定会死。如果没有人去找她——她一定会死在那里。不是被敌人杀死。是被自己耗尽。

[明里的嘴唇在发抖。]

**明里：** 这个医疗站怎么办……伤兵怎么办……

**雪乃：** 你来管。

**明里：** 我——

**雪乃：** 你学了够多了。止血、缝合、配药、换药——你都会。高阶治愈你还差一点，但日常伤情你完全应付得来。

[她从怀里掏出一本薄薄的手册——封面用白布包着，上面写着"医疗站运营手册·雪乃手抄"。]

**雪乃：** 所有药方、操作流程、紧急情况的处理方案——都在这里面了。我昨晚写的。

[明里接过手册。手在发抖。]

**明里：** 你昨晚一整夜没睡，就是在写这个？

**雪乃：** 嗯。

**明里：** ……你什么时候回来？

[雪乃没有回答。]

**明里：** 雪乃姐！你什么时候回来？！

**雪乃：** 等我找到她。

[她背起背囊。]

**雪乃：** 等我找到她，把她的伤治好——我就带她回来。

[明里的眼泪终于掉了下来。她用力擦了一把脸。]

**明里：** ……你答应过的。你说过"只要手还在就能救人"。你的手——别让它——

**雪乃：** 不会断的。

[她微微笑了。那个笑容温柔而坚定，像是冬天最冷的那天清晨，第一道从地平线上升起的光。]

**雪乃：** 好好看着这里。等我回来。

[她转身走向营地入口。经过那块木板的时候，她停了一步。低头看了看上面的字——"此处收治一切伤者。不问阵营。不问身份。不问来路。"——然后继续向前走。]

[明里站在帐篷前，看着雪乃的背影越走越远。白色长袍在晨风中飘动，渐渐和地平线上的晨雾融为一体。]

[她走向了前线的方向。]

[走向了凛的方向。]

**明里：** （低声，声音发颤）……一定要回来啊。两个人一起。

[晨风将她的话吹散了。营地的白旗在风中猎猎作响。太阳刚刚升起来，光还很淡，但方向是确定的。]

### 场景描写

前线以南四十里。无名神殿。

这座神殿坐落在一片矮丘的顶部，石阶从丘底一路延伸到殿门——一共一百二十级，每一级都被岁月和脚步磨得光滑。但现在石阶上散落着碎瓦和焦黑的木料残片。神殿的屋顶塌了半边，穹顶上的彩色玻璃被震碎了大半，只剩几块还嵌在石框里，在风中发出危险的嘎吱声。正殿的大门被撞开着，一扇门板歪斜地挂在铰链上，另一扇已经不见了——大概被人拆去当柴烧了。

殿内曾经供奉着一尊和王都神殿同款的石质神像。但神像从腰部断裂，上半身倒在基座旁边，面容朝下扣在地砖上——看不清表情，也许也不需要看清了。基座上残留的半截身体像一根孤零零的树桩，在从破顶漏进来的天光下投出一道歪斜的影子。

空气中有石灰粉尘和焦木的气味。偶尔有风从穹顶的破洞灌进来，吹得满地的碎玻璃碴发出细碎的声响。

但神殿并不是真正空的。

在正殿的角落、侧室、甚至断裂的神像背后——蜷缩着一群人。老人、女人、孩子。大约三十来人。他们的衣服脏到看不出原本的颜色，脸上有灰尘和干涸的泪痕。孩子们不敢哭出声，只是无声地张着嘴。有几个老人看起来病得很重——呼吸带着哮喘般的哨音。

他们是从南面村落逃来的难民。暗潮的推进线碾过了他们的家园。他们一路北逃，发现了这座被遗弃的神殿，就躲了进来。不是因为信仰。只是因为石头墙比草棚结实。

### 剧情对话

#### 第一幕：空的圣地

[午后。雪乃沿着矮丘的石阶向上走。她已经在荒野中独自行走了两天。白色长袍的下摆沾满了泥和草屑，脸上有疲惫的痕迹，但脚步没有犹豫。]

[她在石阶的第六十级处停下了。因为她看到了殿门前的地面上有东西——几个小小的脚印。泥土印在石砖上，还没有被风完全吹散。是孩子的脚印。]

**雪乃：** ……

[她加快了脚步。]

[推开歪斜的殿门走进正殿的时候，三十多双眼睛同时看向了她。那些眼睛里的光是相同的——恐惧。极度的、已经麻木了的恐惧。]

[一个抱着婴儿的女人尖声叫了出来——]

**难民女：** 又来人了——不要——我们什么都没有了——

**雪乃：** 不要怕。

[她举起双手——空的。没有武器。掌心朝前。]

**雪乃：** 我不是兵。不是任何一方的兵。我是……

[她顿了一下。曾经她会说"我是祭司"。现在这个词不再属于她了。]

**雪乃：** 我是治愈的人。

[她蹲下来，和最近的一个孩子平视。那个孩子大概五六岁，抱着一只脏兮兮的布偶，嘴唇干裂，眼窝深陷——严重脱水的症状。]

**雪乃：** 渴了吗？

[孩子没有说话。但他点了点头。]

[雪乃解下背囊，取出水壶。她先倒了一点水在自己手心里喝了一口——表示水是安全的。然后她把水壶递给了孩子。]

[孩子犹豫了一下。然后他伸出手接过水壶，小口小口地喝起来。]

[其他难民看到这一幕，身体的紧绷松了一点点。一个老人拄着拐杖走过来——他的腿上有一道已经发炎的伤口。]

**老人：** 姑娘……你真的是……医者？

**雪乃：** 是。

## Event 11 — 角色事件·伍「墙上的炭笔字」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情（无战斗） |
| 情感基调 | 去找凛了，被涂掉的字比写下的字更沉 |
| 奖励 | 250 EXP, 炭笔碎片×1, CG「墙上的痕迹」 |

### 场景描写

前线纵深地带。无名废墟。

这里曾经是一个小镇——也许叫什么名字，但路标被炸断了，只剩半截木桩歪在路边，上面的字被火烧得只剩一个模糊的偏旁。镇子的建筑大部分已经坍塌，剩下的几面墙壁孤零零地立在瓦砾堆里，像是拔光了牙的牙床上残余的几颗门牙。

地面上有大量战斗的痕迹。刀劈在石墙上留下的白色切痕。箭矢嵌在木梁里，尾羽在风中瑟瑟发抖。泥地里踩满了脚印——军靴的、马蹄的，杂乱地交叠在一起，像是一幅被人踩烂了的画。有些脚印深得能看到鞋底的纹路。有些地方的泥土颜色比周围深——那是血渗进去的。

空气中弥漫着陈旧的焦糊味和铁锈味。这场战斗发生在至少三四天前——尸体已经被搬走了，但战斗留下的气味是最后消散的。

黄昏。天边的云被染成了暗红和紫色的交界，像是一块正在凝血的伤口。

雪乃走进这片废墟的时候，她的脚步慢下来了。不是因为疲惫——她已经独自行走了四天，疲惫是常态。她的脚步慢下来，是因为她看到了地面上的某种痕迹。

靴印。军靴的靴印。大部分靴印都是杂乱无章的，但有一组——从废墟东面延伸到西面的一组靴印——步伐极其均匀。每一步的间距几乎相同。步幅不大不小——是一个身高中等偏高的人在负伤状态下仍然保持着军人步态的痕迹。

雪乃认识这种走路方式。

### 剧情对话

#### 第一幕：战斗的残迹

[黄昏。废墟。雪乃沿着那组匀称的靴印走进小镇的中心地带。]

[她走得很慢。每走几步就蹲下来看一看地面。她不是在看路——她是在读。读这片废墟上写满的战斗。]

[一面墙上有三道平行的斩痕。极深。石墙被劈开了将近一寸。刃口锋利而笔直——不是普通士兵的刀法。]

**雪乃：** ……

[她伸出手，指尖触上了斩痕的边缘。石粉簌簌地落下来。]

*——三道。平行的。间距相等。这是凛的斩击。她习惯在面对多个敌人时用横扫三连斩开路。*

[她收回手。继续向前走。]

[地面上有一处凹陷——像是有人重重跪了一下。凹陷旁边的泥土颜色更深。深得发黑。]

*——跪了一下。然后又站起来了。血迹在膝盖的位置。*

*——凛。你在这里跪了一下。然后又站起来了。*

[她加快了脚步。]

[一段残存的围墙后面——大约是小镇广场的遗址——她看到了更多的战斗痕迹。这里是最激烈的交战区域。墙壁上、地面上，到处都是刀痕、魔法灼烧的黑印和箭矢的残骸。]

[广场中央有一棵树——曾经应该是一棵很大的树，现在被从中间劈成了两半。一半倒在地上，树干的截面焦黑，被雷击或高阶魔法直接命中过。另一半还立着，但树皮剥落了大半，露出下面白花花的木质，像是被剥了皮。]

[还立着的那半棵树的根部，插着一把断剑。剑从剑格处折断，只剩下握柄和一小截刃身。握柄上缠着深色的皮革——磨损严重，被汗水和血浸透后变成了近乎黑色的颜色。]

[雪乃在断剑前停住了。]

[她没有去拔那把剑。她只是蹲下来，仔细地看着剑柄。看着上面的磨损痕迹。看着皮革缠绕的方式——从下往上、左手习惯的缠法。]

**雪乃：** （极轻地）……不是你的剑。

[她松了一口气。极小的一口气。几乎不可闻。]

**雪乃：** 是苍的。苍惯用的缠法。

[她站起来，目光扫过广场的四周。寻找着什么。]

*——凛的剑不在这里。说明她带走了。说明她还在走。还在战斗。*

*——还活着。*

#### 第二幕：墙上的字

[广场西侧有一面保存相对完好的石墙——大概是某个商铺的外墙。墙壁约两人高，表面的灰泥大部分已经剥落，露出下面粗粝的石砖。]

[雪乃走到这面墙前时，她的脚步骤然停住了。]

[墙上有字。]

[用炭笔写的。写在大约一人高的位置——站着伸手就能够到的高度。字迹——]

[字迹被涂掉了。]

[不是擦掉的——是涂掉的。写字的人用同一根炭笔反复地、用力地在字上面画了无数道横杠，将原本的字盖得严严实实。炭粉被碾进了石砖的缝隙里，变成了一个漆黑的长方形区域——大约手掌宽、两个手掌长的黑色块面。]

[但涂得再厉害，炭笔和石砖之间的凹凸差异还是会留下痕迹。原本写字时笔画用力下按的地方，石砖表面会有轻微的沟痕。涂抹只是覆盖了颜色，没有填平沟痕。]

[雪乃站在墙前。一动不动地站了很久。]

[然后她慢慢地抬起右手。伸出食指。将指尖轻轻贴上了那片漆黑的区域。]

[她闭上眼。]

[指尖在黑色的炭粉层上缓缓移动。她在用触觉读那些被涂掉的字。感受石砖表面的沟痕——那些笔画落下时用力的地方、收笔时提起的地方、转折时犹豫的地方。]

**雪乃：** （闭着眼，声音几乎是气音）……第一笔。竖。很重。落笔的时候犹豫了一下——起笔处有一个小小的顿点。

[她的指尖向右移。]

**雪乃：** 横折。转折处很急。是用力写的——不是慢慢描的。这个人在写的时候……很急。或者很——

[指尖停住了。]

**雪乃：** ……很痛。

[她的手指继续移动。一笔一画地描摹。周围的废墟在黄昏中安静得像是被按下了静音键。只有风偶尔吹过残壁发出的呜呜声。]

[很久之后，她的手停了。]

[她睁开眼。手指停留在最后一个字的末笔上。指尖沾满了黑色的炭粉。]

**雪乃：** ……

*——我认得你的笔迹。在骑士团报告上认过。在军令状上认过。在你偷偷塞给我的纸条上认过。*

*——你的字永远很用力。横平竖直。像你这个人一样——不弯，不歪。*

*——你在这面墙上写了什么？*

*——然后你又涂掉了。*

*——你涂得那么用力。比写的时候还用力。*

[她将手从墙上收回来。低头看着自己沾满炭粉的指尖。黑色的粉末嵌在指纹的沟纹里，像是某种密码。]

**雪乃：** （低声，对着那面涂黑的墙壁）……你写了什么？

[墙壁不会回答。字已经被涂掉了。]

[但雪乃的指尖读到了那些沟痕。她知道那些字是什么。]

[她不说出来。有些字被写下它们的人涂掉了，就应该被涂掉。她不会替凛说出凛自己选择沉默的话。]

[但她知道了。]

*——你不是不打算活着回来。*

*——你只是不知道可以对谁说。*

**雪乃：** ……我来了。

[她将沾满炭粉的手指轻轻贴回墙面。不是在描摹。只是贴着。像是隔着一面石墙在触碰某个人的手。]

**雪乃：** 你不用写在墙上。你可以对我说。

[风又吹过来了。炭粉被风卷起一点点，飘散在黄昏的空气里。]

#### 第三幕：废墟中的星空

[夜晚。废墟。]

[雪乃靠着那面写过字的墙壁坐下来过夜。她将背囊垫在背后，白色长袍裹紧了身体——前线纵深的夜晚比后方冷得多。]

[她没有点火。火会暴露位置。在暗潮搜索队出没的区域，一点灯光就可能招来麻烦。]

[但今夜的星空极亮。没有月亮，反而让星星显得更加密集。银河从天幕的一端横贯到另一端，像是有人在黑色的天穹上洒了一把碎钻。]

[雪乃仰头看着星空。]

[她的后背贴着那面墙壁。贴着那些被涂掉的字。炭粉透过长袍的薄布料渗到她的背上——明天她的白袍背后会有一片黑色的印记。她不在意。]

**雪乃：** （对着星空，声音低得像是在念一段只有风能听到的祈祷）……你现在在哪里？

[沉默。星星不回答。但星星也不会跑掉。]

**雪乃：** 你在看同一片天空吗？

[她缩了缩身体。夜风从废墟的缝隙里灌进来，冷得刺骨。她将手伸进背囊里，摸到了那张记录着伤兵口述的纸——凛的消息。纸已经被翻看得起了毛。她没有拿出来看——太暗了，看不见。但她的手指摸着纸的边缘，像是在确认某种存在。]

**雪乃：** 我离你越来越近了。

[她的声音带着一种奇异的确信。不是盲目的乐观——是一个走了四天荒野、穿过了一座空神殿、在废墟里读到了墙上被涂掉的字的人，所积累起来的、不可动摇的确信。]

**雪乃：** 我看到了你的脚印。看到了你的刀痕。看到了你写的字。

[她闭上眼。]

**雪乃：** 你每一步都在告诉我你还活着。

[停顿。夜风呼呼地吹。远处有什么东西倒塌了——大概是废墟中某面终于撑不住的墙。轰的一声闷响。然后又安静了。]

**雪乃：** ……你知道吗。你涂掉那些字的时候，手一定很用力。炭笔都碾碎了——我在墙根下看到了碎片。很小的碎片。你把炭笔用到只剩指甲盖那么大一截，然后碾碎了。

[她睁开眼。望着银河。]

**雪乃：** 你在写那些字的时候——你想对谁说？

[风。]

**雪乃：** 你涂掉它们的时候——你是不是觉得没有人会听？

[星星。]

**雪乃：** ……我会听。

[她将手从背囊里抽出来。手指上还残留着黑色的炭粉——下午描摹那些字时留下的。她在星光下看着自己的指尖。]

**雪乃：** 你写什么我都听。你涂掉的我也听。你说不出口的——

[她将沾着炭粉的手指贴在自己的心口。]

**雪乃：** ——我替你收着。

[夜深了。气温降到了极低。雪乃缩在墙根下，闭上了眼。但她没有真正入睡——在前线纵深，真正入睡是危险的。她维持着一种浅浅的、随时可以醒来的休憩状态。]

[她的背贴着墙壁。墙壁另一面，那些被涂掉的字在黑暗中沉默着。]

[但它们不再是孤独的了。]

[有人读到了它们。有人的指尖触到了它们。有人在墙的这一侧，贴着它们睡了一夜。]

[黎明前最暗的时刻，雪乃睁开了眼。星星还在。银河已经移动了位置，但还横在天上。]

[她站起来。拍掉长袍上的灰尘——但背后那片炭粉的印记拍不掉。她低头看了看，没有去管它。]

[她弯腰从墙根下捡起了一小截炭笔碎片——凛碾碎的那根炭笔的残余。碎片只有指甲盖的三分之一大小，边缘粗糙，表面还能看到指纹的压痕——不是雪乃的指纹。]

[她将碎片小心地包在一块布里，放进了背囊的最内层。]

[然后她背起背囊。最后看了那面墙一眼——黑色的涂抹痕迹在晨光中显得不那么浓了。像是浓墨被水稀释了一层。]

**雪乃：** ……等我找到你，你就不用写在墙上了。

[她转身。继续向前线的方向走去。]

[身后，废墟在晨光中渐渐亮了起来。那面墙壁上的黑色痕迹被阳光照得发亮——像是一扇被涂黑了的窗。窗后面是什么，只有写字的人和读字的人知道。]

### 场景描写

前线纵深与后方缓冲地带的交界。河滩难民营。

一条曾经丰沛的河流如今只剩浅浅的水洼和裸露的卵石滩。河床两侧是低矮的土堤，堤上长着零星的芦苇——芦苇已经枯黄了，在风中发出干燥的沙沙声。

难民营就建在河滩的北岸。准确地说，不是"建"——是"堆"。帐篷用的是从废墟里捡来的帆布和木料拼凑的，没有一顶是完整的。最大的一顶用了三块不同颜色的布拼在一起，像是一面潦草的旗帜。营地周围用削尖的木桩和堆起来的土袋围成了一圈简陋的防御工事——不能挡住真正的军队，但能让野兽和零散的匪徒知难而退。

营地里大约有六七十人。大部分是从南面逃来的平民——农夫、匠人、商贩和他们的家眷。有几个人穿着破烂的军服，但军服上的纹章已经被撕掉了——退役士兵，或者逃兵。他们不想被认出来属于哪一方。在这里，没有人问来路。

营地中央有一口大锅架在篝火上，煮着稀薄的菜粥——能看到底的那种稀薄。但在这个时节，能有热粥喝已经算好的了。

雪乃在五天前的傍晚走进了这座难民营。

她当时几乎是被风吹进来的——独自行走了六天，干粮吃完了，水壶空了，灵力也在空神殿的战斗后一直没有完全恢复。她走到营地入口时，膝盖一软差点摔倒。是一个正在打水的老妇人扶住了她。

老妇人问她："你是谁？"

她说："我是治愈的人。"

然后她看到了营地里的伤者——被暗潮散兵袭击过的平民、生病的孩子、被感染的外伤。她坐都没坐下来就开始治疗。

五天了。她留在这里已经五天了。

她本该继续赶路去找凛。但她走不了——营地里的伤者太多了，每天都有新的。前天有一个孩子高烧不退，她彻夜守着用治愈术一点一点把体温压下来。昨天有两个壮丁出去觅食时被暗潮散兵的陷阱炸伤了腿，她花了三个时辰缝合骨头。

她走不了。她的脚想走，但她的手停不下来。

### 剧情对话

#### 第一幕：走不了的人

[清晨。难民营。]

[雪乃在营地最大的帐篷里——那顶三块布拼成的帐篷被临时改成了医疗帐。帐篷内摆着几张木板拼的床铺，上面躺着伤者。空气中混着草药的苦涩和汗水的酸味。]

[她正在为一个老人换药。老人的右手被流矢射穿了手掌，伤口经过治愈已经基本愈合，但需要每天换药防止感染。]

**老人：** 姑娘啊，你不是说还要赶路吗。

**雪乃：** 嗯。

**老人：** 都五天了。

**雪乃：** ……嗯。

**老人：** 别被我们拖住了。你有你的事。

[雪乃将干净的绷带一圈一圈地缠上老人的手掌。每一圈都均匀、贴合、不松不紧。]

**雪乃：** 你的伤口后天才能完全长好。如果今天我走了，明天换药谁来？

**老人：** 我自己来——

**雪乃：** 你用一只手给另一只手换药？缠得紧了血液不通，缠得松了绷带会掉。

[老人张了张嘴。没有反驳。]

**雪乃：** 后天。后天换完最后一次药我就走。

[她每天都这样说。"明天就走。""后天就走。"但每天都会有新的伤者，新的理由让她多留一天。]

[帐篷帘掀开，一个年轻女人冲了进来——她是营地里的巡逻队成员之一，退役女兵出身。]

**巡逻兵甲：** 雪乃姐！南边发现暗潮游击队！至少十个人！朝我们来了！

[雪乃缠绷带的手停了一瞬。然后她将最后一圈绷带打好结，站了起来。]

**雪乃：** 距离？

**巡逻兵甲：** 大概半个时辰的路程。他们沿着河滩过来的，速度不慢。

**雪乃：** 人数？

**巡逻兵甲：** 至少十个。有弓兵。可能还有术士——我看到了暗紫色的光。

[雪乃走出帐篷。她望向南面——河滩的远端。阳光在干涸的卵石滩上反射着刺眼的白光。在白光的最远处，有几个黑色的点正在缓缓移动。]

**雪乃：** ……游击队。不是主力。是来劫掠的。

**巡逻兵甲：** 跑吗？

**雪乃：** 六七十个人。老的小的、伤的病的。往哪里跑？跑得过游击队？

[巡逻兵甲咬着嘴唇。]

**雪乃：** 营地的防御工事——木桩和土袋——能撑多久？

**巡逻兵甲：** 如果只是挡刀剑……也许能撑几轮。但如果有术士——

**雪乃：** 防御工事前面站人。有人守就不一样。木桩挡不住，人能挡。

**巡逻兵甲：** 可我们只有四个巡逻兵——

**雪乃：** 够了。

[她转身面向巡逻兵。她的表情平静而笃定。]

**雪乃：** 你们守。我治。你们受了伤，我治到满。一直治。不会让你们倒。

**巡逻兵甲：** 但你自己——你没有武器，没有防具——

**雪乃：** 所以我需要你们不让他们靠近我。

[她的声音不大，但有一种不容置疑的力量——像是一根细细的钢丝，看起来柔软，拉起来才知道断不掉。]

**雪乃：** 这就是交易。你们保护我。我保护你们。谁也别倒。

#### 第二幕：治愈者的阵型

[半个时辰后。营地南面防御工事前。]

[四名巡逻兵和几名自愿参战的壮年难民站在木桩后面。他们的武器五花八门——两把短刀、一把猎弓、一根铁棍、几把削尖的木矛。铠甲几乎没有，最多是在衣服外面绑了几块木板。]

[雪乃站在防线后方约十步的位置——一个被土袋和翻倒的马车围成的半圆形空间里。这是她的"治愈阵位"。从这里她可以看到整条防线上的每一个人。]

[她已经做好了准备。长袍的袖口扎紧了，头发用一根布条束到脑后。面前的地上放着她的背囊——打开的，里面是药材和绷带。以防万一灵力耗尽，还可以用传统方式救治。]

[营地里的妇女和老人已经带着孩子转移到了北面最远的帐篷群中。那里离防线最远，但如果防线被突破——也只是多几分钟逃跑的时间而已。]

**难民营守卫甲：** （退役老兵，握着一把缺了口的短刀）姑娘。我打过仗。但那是十年前的事了。

**雪乃：** 身体还记得。刀怎么握、什么时候砍——身体会记得。你要做的只是不后退。

**难民营守卫甲：** 万一扛不住呢？

**雪乃：** 你扛不住的时候——看我的手。我的手亮着，你就还能站起来。

[老兵看着她。看了三秒。然后他点了点头。]

**难民营守卫甲：** ……明白了。

[远处河滩上的黑点越来越近。能看清人形了。能看清武器了。能听到脚步声了——很多脚步踩在卵石上的嘎吱声。]

**暗潮游击队长：** （远处，喊声被风送过来）——就是那个营地！物资、人口，全部带走！反抗的格杀！

[雪乃深吸一口气。将双手举在身前。掌心朝向防线的方向。]

[淡金色的光在她掌心凝聚。]

**雪乃：** （低声，只有自己能听到）……我不能挥剑。不能射箭。不能放火。

[光芒在她掌心越来越亮。]

**雪乃：** 但我可以让每一个站在前面的人——永远不倒。

*——这是治愈者的战斗方式。*

*——不是杀敌。是让守护者不死。*

### 战斗阶段

[战斗开始。地图为难民营南面防御工事及河滩区域。雪乃位于防线后方治愈阵位（不可移动，DEF为0）。]

[敌方从河滩方向进攻。我方在木桩防线处防守。]

*——八回合。守八回合。他们的补给有限。游击队不会在一个地方耗太久。*

*——八回合就够了。*

[第1回合。暗潮游击兵从河滩南侧冲来。弓兵在后方展开远程压制。]

**暗潮游击队长：** 一群难民而已！冲过去！

[我方在防线后迎击。难民营守卫甲挡住了第一波冲击，但被游击兵的刀砍中肩膀，HP骤降。]

[雪乃治愈发动——守卫甲HP恢复至满血。肩膀上的刀伤在金色光芒中愈合。]

**难民营守卫甲：** ——好了？！这就好了？！

**雪乃：** 别愣着！他来第二刀了！

[守卫甲猛地回神，举刀格挡。]

[第2回合。暗潮弓兵集中火力射击防线。箭雨落下，多名守卫受伤。]

[雪乃治愈发动——HP最低的守卫恢复至满血。]

**巡逻兵甲：** （拔掉肩膀上的箭，发现伤口已经愈合）——这感觉太诡异了！刚刚还疼得要死——

**雪乃：** 专心！箭还在飞！

[第3回合。暗潮突袭兵从防线东侧试图迂回包抄。玩家部队调兵应对。]

[雪乃治愈发动。]

*——第三回合了。还有五个。*

[第4回合。暗潮增援到达——精锐突袭兵×2。攻击方向突变，从防线西侧发起猛攻。]

**护卫单位：** 少说话！治我就行！

**雪乃：** 站起来。你还活着。继续守。

**雪乃：** （坐在地上，双手举着，声音沙哑但稳定）没事。别看我。看前面。

**雪乃：** 营地呢……孩子们……

**巡逻兵甲：** 都好！一个伤的都没有！

**雪乃：** ……那就好。

**巡逻兵甲：** 灵力耗尽了。她在后面治了我们八轮。每轮都是满血。

[雪乃喝了一小口水。]

**雪乃：** （微弱地，对孩子）……谢谢。

[孩子没说话。但他没有走开。他就蹲在那里，用两只小手捧着水壶，随时准备再让她喝一口。]

[但他们会盖毯子。会端粥。会遮太阳。会用两只小手捧着水壶。]

*——我治了他们。他们也在治我。*

## Event 13 — 角色事件·陆「追随之路」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情（无战斗） |
| 情感基调 | 自动加入触发，不是因为神的旨意，是因为自己的意志 |
| 奖励 | 350 EXP, 雪乃的决意×1, CG「两个人的祈祷」 |

### 场景描写

前线最深处。无名战场的残骸。

这里已经不像战场了——更像是大地被人用指甲抠开之后的伤口。地面上没有完整的平面，到处是被魔法轰出来的弹坑和被刀劈开的壕沟。壕沟里积着浅浅的雨水，水面上浮着一层灰色的粉尘，映不出天空的颜色。焦黑的树桩从泥土中戳出来，像是被打断的骨头。远处有几面残垣——曾经是阵地的一部分，现在只剩齐腰高的半截石墙，墙面上布满弹坑和刀痕。

空气中有一种很重的气味——不是血腥，血腥味早就散了。是泥土被反复翻搅之后释放出来的潮湿腐味，混着焦木和铁锈。蝇虫的嗡嗡声从某个看不到的角落传来。天空是灰白色的，不是阴天——是那种高远的、稀薄的、不关心地面发生了什么的白。

雪乃已经走了八天。

她的白色长袍不再是白色的了——下摆被泥浆浸成了灰褐色，衣襟上有草汁和灼烧的痕迹，右侧袖口在穿越荆棘丛时被撕开了一道口子，她用绷带随手扎了一下。头发没有来得及重新束好，一半散在肩上，碎发贴在脸颊和脖子上。脸颊凹进去了一些——这八天她没有吃过一顿完整的饭。

但她的眼睛是亮的。

不是精神焕发的那种亮——是某种更深的东西。是一个已经将全部储备燃烧殆尽、只靠目的地的引力还在往前走的人，眼睛里最后剩下的那种光。

她沿着战场的边缘行走。脚下踩过碎石、弹壳、断裂的箭杆。她的视线一直在扫地面——不是看路，是在找那组匀称的靴印。

但这里的地面被太多人踩过了。靴印交叠在一起，分不清哪一组是谁的。

她停下来。站在一座残垣的阴影里。

前方的开阔地上，战场的废墟延伸到视线的尽头。她不知道凛在哪个方向。八天来第一次——她找不到路标了。

### 剧情对话

#### 第一幕：在废墟中找到你

[午后。战场废墟。灰白色的天空下。]

[雪乃站在残垣后面，目光扫过前方的开阔地。弹坑、壕沟、倒塌的拒马、扭曲的铁丝——视线被这些东西切割得支离破碎。]

[她深吸一口气。闭上眼。]

*——不要慌。你走了八天。你穿过了空神殿。你在废墟里读到了她的字。你在难民营里守了五天。你没有在任何一个地方停下来。*

*——现在不要停。*

[她睁开眼。重新看向前方。]

[然后她看到了。]

[不是靴印。不是刀痕。是——]

[残垣群的最远处，一面断墙的根部，有一个人影。不是站着的——是靠着墙壁坐在地上的。身体微微前倾，头低着，像是睡着了，又像是——]

[雪乃的心跳停了一拍。]

[她开始走。然后走变成了跑。]

[她跑过弹坑。跑过壕沟——差点摔进去，手撑了一下地面继续跑。跑过散落的兵器残骸。白色长袍的下摆被地上的铁丝刮破了一道长口子。她没有感觉到。]

[越来越近了。]

[那个人影靠在断墙根部。穿着已经认不出颜色的铠甲——铠甲碎了。肩甲只剩左边的半片。胸甲上有一道横贯整个前胸的裂痕，裂痕的边缘翻卷着，像是被某种巨大的力量劈开的。铠甲下面的衬衣被血浸透了，干涸后变成了铁锈色和黑色交替的图案。]

[腿伸在地上。右腿的胫甲不见了，小腿上有两道深可见骨的切口，被人用布条潦草地缠过——布条已经被血浸透，变得僵硬。左手搭在身侧，手指微微蜷缩。右手握着一把剑——剑身完整，但刃口已经卷得不成样子，布满豁口和缺刻。]

[头低着。灰尘和血迹覆盖了大半张脸。头发散落下来，遮住了眼睛。]

[但雪乃认识这个人。]

[哪怕全身都碎了。哪怕面目全非。哪怕把她丢进一万个人影里——她也认得出来。]

**雪乃：** 凛——

[她跑到跟前。跪了下来。膝盖砸在碎石上，疼痛沿着骨骼传上来。她没有感觉。]

[她伸出手。颤抖着。手指悬在凛的面前——不敢碰。怕碰了之后发现是冷的。]

[三秒。]

[她的指尖触上了凛的脸颊。]

[——温的。]

[微弱的，快要被风吹散的，但确确实实存在的温度。]

**雪乃：** （声音碎了，碎成了气音）……活的。你是活的。

[凛没有反应。呼吸极浅——胸口的起伏几乎看不出来。嘴唇干裂到出血。面色灰白，像是被抽去了所有颜色的画布。]

[她失去意识了。已经不知道多久了。也许是几个小时。也许更长。]

[雪乃的手从脸颊移到凛的脖子侧面——找脉搏。有。极弱。极慢。像是一根快要断掉的弦在做最后的振动。]

*——还在。还在。还在。*

[雪乃深吸一口气。又一口。又一口。]

[她将碎掉的东西——恐惧、心疼、快要溃堤的眼泪——全部咽回去。现在不是哭的时候。现在是治的时候。]

**雪乃：** （声音仍然在抖，但每一个字都是清晰的）……我在了。我来了。我来治你了。

#### 第二幕：用自己的手治愈

[雪乃将背囊取下来。打开。手指飞快地翻找——绷带、高浓度伤药、止血粉、缝合针线。八天的行走已经消耗了大部分物资，但她一直留着最后一份。留着给这个人。]

[她先处理最危急的——凛的胸口。那道横贯前胸的裂痕不仅劈开了铠甲，还伤到了下面的肌肉和肋骨。雪乃小心地将碎裂的铠甲片从伤口上剥离，每一片都粘着干涸的血。凛在无意识中微微皱了一下眉——疼。即使失去意识了也疼。]

**雪乃：** （极轻地）忍一下。我知道疼。很快。

[她将双手覆上凛的胸口。掌心朝下。]

[闭眼。]

[淡金色的光从她掌心涌出。不急不缓——她已经学会了控制。八天前在难民营灵力耗尽后，她用了两天时间恢复，然后在行走的每一步中都在积蓄。现在这些积蓄全部涌向掌心，流向凛的伤口。]

[治愈的光渗入裂开的皮肉中。断裂的毛细血管开始重新连接。撕裂的肌纤维在金色光芒中缓缓靠拢、交织、愈合。那道横贯的裂痕从最深处开始向表面闭合，像是有人在用一根看不见的针，从里到外一层一层地缝合。]

[凛的呼吸深了一点。只深了一点。但雪乃感觉到了——掌下的胸腔起伏变大了。]

**雪乃：** ……对。呼吸。就这样。

[她将手移到凛的右腿——那两道深可见骨的切口。解开浸透了血的布条时，伤口的状态让她倒吸了一口气——已经开始感染了。伤口边缘红肿发热，有脓液渗出。如果再晚一天——]

[她不去想"如果再晚一天"。]

[治愈的光覆上伤口。感染被光芒逼退——红肿消褪，脓液被清除，然后新的肉芽开始从伤口底部生长。切口慢慢地、一层一层地闭合。]

[然后是左肩的脱臼。腰侧的淤伤。额角的擦伤。手背上密密麻麻的细小切口——那是握刀太久、刃口的震动传导造成的。]

[一处一处。一层一层。她的双手像是两支极细的笔，在凛的身体上逐字逐句地改写"毁坏"这两个字。]

[时间在流。治愈在流。灵力在流。]

[太阳从正午的位置滑向了西面。影子从短变长。光从白变黄。]

[雪乃的额头上布满了汗珠。灵力的消耗让她的手指开始发麻——那种麻是从指尖向手腕蔓延的，像是冰水慢慢漫上来。但她没有停。]

*——不用任何神殿的权柄。不用任何神明的祝福。*

*——用我自己的手。用我自己的力量。用我自己的意志。*

*——因为这双手是我的。这力量是我的。这个人——*

[她低头看着凛的脸。血污被她用水和绷带擦去了大半。面容渐渐清晰了——瘦了。棱角比记忆中更分明。眉心有一道浅浅的竖纹——和那些长老一样的竖纹。不是因为年龄，是因为太久太久地皱着眉。]

*——这个人是我想要守护的。*

[最后一处伤——右手虎口的撕裂。那是握剑过度造成的。伤口不大，但位置很深。雪乃将凛的右手轻轻从剑柄上掰开——手指已经僵了，以握剑的姿势僵在那里，像是即使失去意识了也不肯放手。]

[她花了很长时间才将凛的手指一根一根地展开。然后她将凛的手掌托在自己掌心里。淡金色的光流过虎口的裂伤，将它缝合。]

[治愈完成。]

[雪乃将凛的手轻轻放下。然后她自己的手臂一软——差点趴在凛身上。她用最后一点力气撑住自己。]

[她跪在凛面前。双手垂在膝盖上。掌心的光彻底消散了——灵力见底。整个人像是被拧干了的布。]

[但凛的脸色已经从灰白变成了苍白。呼吸从浅弱变成了平稳。嘴唇上的干裂开始愈合。]

[活过来了。]

[雪乃看着凛。看了很久。]

[然后她将凛散落的头发从脸上轻轻拂开。手指从额头掠过鬓角，指尖的触感从干燥的灰尘变成了温热的皮肤。]

**雪乃：** （低声，像是在对一个沉睡的人讲一个秘密）……你好重。伤得好重。让我治了好久。

[她的声音带着一种奇异的、疲惫到极点之后反而出现的温柔。像是水满了之后不再翻涌，只是平平地溢出来。]

**雪乃：** 但你活着。这就够了。

[黄昏的光从西面照过来。将两个人的影子拉得很长很长——一个跪着的、一个靠着墙坐着的。两道影子在碎石地面上交叠在一起。]

[凛的手指动了。]

[微微的。像是水面上被风吹出的最小的涟漪。]

[然后她的眉头皱了一下。嘴唇动了——没有声音。]

[雪乃屏住了呼吸。]

[凛的眼睛慢慢睁开了。]

[目光模糊。混沌。像是在一个很深很深的地方挣扎了很久才浮到水面上来。她的瞳孔花了几秒钟才聚焦——先是看到了灰白色的天空，然后是断墙的边缘，然后——]

[雪乃的脸。]

[逆着黄昏的光。面容疲惫。眼眶发红。嘴唇在微微发抖。]

[凛盯着她看了很久。久到雪乃觉得她可能还没有完全清醒。]

**凛：** （声音沙哑得像是砂纸在摩擦）……雪乃？

**雪乃：** 嗯。

**凛：** ……你怎么在这里。

**雪乃：** 来找你。

[凛又看了她几秒。像是在确认这不是濒死前的幻觉。]

**凛：** ……我在做梦吗。

**雪乃：** 不是。

[她伸出手，握住了凛的手。她的手也在抖——灵力耗尽后的生理性颤抖。但她握着。]

**雪乃：** 不是梦。你摸——我的手是热的。梦里的手是冷的。

[凛的手指在雪乃掌心里微微蜷动了一下。像是在确认。确认那个温度是真实的。]

[然后她的目光落在了自己身上——身上的伤全部被治愈了。胸口的裂痕只剩一道淡粉色的新疤。腿上的切口已经完全闭合。她能感觉到——身体里那种将要碎掉的感觉消失了。骨头不再磨骨头。肺不再漏气。]

**凛：** ……你治的。

**雪乃：** 嗯。

**凛：** 全部？

**雪乃：** 全部。

[沉默。凛的目光从自己身上移回到雪乃脸上。她看到了雪乃的状态——苍白的脸色、发颤的双手、灵力耗尽后的虚脱。]

**凛：** ……你把自己治空了。

**雪乃：** 嗯。

**凛：** 为什么。

[这两个字很轻。但落在废墟的寂静里，重得像石头。]

#### 第三幕：追随的理由

[黄昏加深了。天空从灰白变成了淡橘色。远处的地平线上有一层薄薄的云，被落日染成了深红和紫色交替的条纹。]

[雪乃没有立刻回答凛的问题。她将背囊里最后的水壶拿出来，拧开盖子，将壶嘴凑到凛的嘴边。]

**雪乃：** 先喝水。你脱水了。

[凛犹豫了一瞬。然后她微微张开嘴，让水流进来。水很少了——壶底的最后一点。她只喝了两口就推开了。]

**凛：** 你自己也喝。

**雪乃：** 我不渴。

**凛：** 骗人。你嘴唇都裂了。

[雪乃低下头。看了一眼水壶。然后她喝了一小口。真的只有一小口。]

[她将水壶放在一边。坐到凛身旁——靠着同一面断墙。两个人并肩。肩膀几乎贴着肩膀。断墙挡住了西面的风，但挡不住黄昏的光。]

[沉默了一会儿。]

**凛：** ……骑士团被拆了。

**雪乃：** 我知道。

**凛：** 苍——苍受了重伤。为了护我。

**雪乃：** 我知道。

**凛：** 你怎么知道的。

**雪乃：** 有个士兵从第七防线爬到了我的医疗站。他告诉我的。

[凛闭了一下眼。]

**凛：** ……你有医疗站？

**雪乃：** 有过。在"三不管地"建的。不分阵营。谁受伤都收。

**凛：** ……你一个人？

**雪乃：** 一开始是一个人。后来有帮手了。我走的时候留给了明里。

**凛：** 你走了——来找我。

**雪乃：** 嗯。

**凛：** 从"三不管地"走到前线最深处。

**雪乃：** 八天。

**凛：** ……你没有武器。没有护卫。没有——

**雪乃：** 凛。

[雪乃转头看着她。眼睛很红，但没有泪。声音很平，像是平静的湖水下面压着整片海。]

**雪乃：** 你问我为什么。

[凛没有说话。但她的眼睛在看着雪乃。在等。]

**雪乃：** 不是因为任何神告诉我要来。

[她低头看着自己的手。灵力耗尽后的手苍白而微颤，掌心的灼烧痕迹在黄昏的光里清晰可见。]

**雪乃：** 不是因为任何使命。不是因为任何教义。不是因为"圣光引导我"——圣光不引导任何人，我现在知道了。

[她转回头，直直地看着凛的眼睛。]

**雪乃：** 是因为我想来。

[三个字。]

[风从断墙上方吹过去。远处有什么倒塌了——大概是废墟中另一面终于撑不住的墙。轰的一声闷响。然后安静了。]

**凛：** ……你不该来。

**雪乃：** 嗯。

**凛：** 你会死在路上。

**雪乃：** 没死。

**凛：** 这里很危险。暗潮的主力——

**雪乃：** 凛。

[她的声音不大，但有一种不可动摇的东西——像是大地。你可以在上面打仗、流血、建造又摧毁，但大地不会动。]

**雪乃：** 你为什么一个人来这里。

[凛的嘴唇动了一下。没有声音。]

**雪乃：** 骑士团没了。苍倒了。王庭说你抗命。你回不了王都。你前面是暗潮主力。后面是追兵。你没有阵营了。没有人了。

[她的声音平稳如水。但每个字都准确地落在凛的沉默上。]

**雪乃：** 你来最前线——不是为了打仗。是因为你不知道还能去哪里。

[凛的呼吸停了一瞬。]

[然后她移开了目光。看向前方——灰色的废墟在黄昏中变得柔和了一些，像是一幅被水打湿的画。]

**凛：** （声音极轻）……你看到了。

**雪乃：** 看到什么？

**凛：** 墙上。废墟那面墙上。我写了——

[她停住了。]

**雪乃：** 你涂掉了。

[凛低下头。]

**凛：** ……嗯。

**雪乃：** 我读到了。

[凛猛地抬头看她。眼睛里有一瞬间的、几乎是恐惧的东西——被看穿的恐惧。那种把最软的地方暴露出来之后本能地想缩回去的恐惧。]

**雪乃：** 不用怕。我不会说出来。你涂掉了，我就当它是涂掉的。

[她伸出手，覆在凛握剑的那只手上。凛的手还僵着——握了太久的剑，手指的肌肉记忆太深。雪乃用自己的手轻轻包住它。]

**雪乃：** 但你不用再写在墙上了。

[凛的嘴唇在发抖。不是冷。是有什么东西在她喉咙深处撞着要出来，但她习惯了不让它出来。]

**凛：** （声音发紧，像是被拧紧的弦）……雪乃。我什么都没有了。骑士团没了。苍——不知道还在不在。我的剑都卷了。我的铠甲碎了。我连一个能回的地方都没有。

[她的声音碎了一道裂缝。从那道裂缝里漏出来的不是哭声——是一种比哭更深的、干涸的、已经连眼泪都挤不出来的疲惫。]

**凛：** 你跟着我——你能跟着一个什么都没有的人去哪里？

[雪乃没有立刻回答。]

[她看着凛。看着她碎裂的铠甲。看着她卷刃的剑。看着她干裂的嘴唇和深陷的眼窝。看着她灰尘和血迹下面那张瘦得棱角分明的脸。]

[然后她笑了。]

[极轻的。极淡的。像是冬天的最后一天、冰面上裂开的第一道缝——春水从缝隙里渗出来的那种笑。]

**雪乃：** 你还有我。

[三个字。]

[比"我想来"更轻。比废墟上方的天空更安静。但它落在凛身上的时候，像是有人在一座空了很久的房间里重新点燃了一盏灯。]

[凛盯着她。]

[一秒。两秒。三秒。]

[然后凛的眼眶红了。]

[不是突然的崩溃——是缓慢的、像冰在融化一样的、从眼角开始蔓延的红。她的嘴唇抿紧了，下颌的肌肉绷着，像是在做最后的抵抗。但那种红一直在蔓延。从眼角到整个眼眶。]

[一滴水从她的眼角滑了下来。]

[不是眼泪。只是水。在这个不会哭的人脸上，它只是水。]

[雪乃没有去擦。她只是握着凛的手。握得很紧。两只都在颤抖的手——一只是灵力耗尽后的颤抖，一只是忍耐了太久之后终于崩塌的颤抖——握在一起。]

**雪乃：** 我不是神派来的。没有任何使命。没有任何命令。神殿不要我了。祈祷珠被收走了。祭司袍被脱掉了。我的名字从神官录上被划掉了。

[她的声音平稳而清晰。每一个字都像是她走过的那八天路途中的一块石头——被脚踩实了，踩得坚不可摧。]

**雪乃：** 我也什么都没有了。

[停顿。]

**雪乃：** 所以——两个什么都没有的人。正好。

[她微微倾身。将额头轻轻靠在凛的肩膀上。凛的肩甲碎了，只剩衬衣的布料——汗渍和灰尘的味道。]

**雪乃：** 你去哪里，我去哪里。你打仗，我治伤。你站着，我站在你后面。你倒了，我扶你起来。

[她闭上眼。]

**雪乃：** 这不是誓言。不是祝福。不是任何仪式。只是——事实。

[凛的身体在发抖。微微的。不是因为伤——伤已经被治好了。是因为有什么东西从很深很深的地方涌上来，涌到了锁骨的高度，涌到了喉咙的位置，涌到了再也压不住的地方。]

[她空出来的左手——没有握剑的那只——缓缓抬起来。犹豫了一下。然后轻轻地、极轻地放在了雪乃的头上。]

[没有说话。只是放在那里。手指微微蜷着，碰着雪乃散落的头发。]

[这个不会说"谢谢"的人、不会说"我需要你"的人、在墙上写了字又涂掉的人——用一只满是茧子和伤疤的手，放在了另一个人的头上。]

[这就是她的回答。]

[黄昏的最后一点光将两个人的影子融成了一个。]

[断墙。碎石。废墟。]

[两个什么都没有的人，靠在一起。]

### 场景描写

前线后方。临时营地。黎明前。

战斗还没有结束——战争从来不会因为两个人的重逢就停下来。但在这一小块被帐篷和篝火围起来的空间里，有一种脆弱但真实的安宁。

营地建在一座矮丘的东坡上。选这个位置是因为东坡能最先看到日出——这是雪乃选的。凛说随便，哪里都行。雪乃说不随便，要能看到日出的地方。

帐篷只有两顶。一顶是雪乃的医疗帐——走到哪里都会先搭一个医疗帐，这已经变成她的本能了。帐篷前面挂着一面白旗，和"三不管地"医疗站上的那面一样，什么纹章也没有，只是纯白的布。另一顶是起居帐——两个人的行李加在一起少得可怜：一个背囊、一把卷刃的剑、几卷绷带、半壶水。

篝火已经烧成了灰烬。木炭还有微弱的红光，偶尔被风吹亮一下，又暗下去。空气中有木炭和露水混合的清冽气味。

东方的天际线还是深蓝色的。但深蓝的最下面——贴着地平线的那一条——已经有了极淡的灰白。日出前的预告。

### 剧情对话

#### 第一幕：黎明前的对话

[黎明前。矮丘东坡。]

[凛坐在帐篷前面的一块平石上。她的伤已经痊愈了——雪乃花了两天时间做了彻底的治疗和后续护理。但瘦削没有恢复。脸上的肉需要吃饭来补，不是治愈术能解决的。]

[她穿着一件干净的衬衣——雪乃在路过的废弃村庄里找到的，洗了晒干了才让她穿。铠甲在身边放着，碎片被雪乃用绳子和布条固定了一些，不再散架了，但离"完整"差得很远。]

[凛的手里没有握剑。这可能是很久以来第一次——她的手空着，放在膝盖上。手背上的细小伤疤在微弱的篝火余光中隐约可见。]

[雪乃从医疗帐里走出来。她换了一身——不是新的，但比之前那件泥浆浸透的干净。白色长袍，自己缝的。没有纹章，没有标记。只是白。她的头发重新束好了，用一根从绷带上裁下来的布条系在脑后。]

[她手里端着两个粗陶碗。走到凛身边，将一个碗递给她。]

**雪乃：** 喝。是药粥。加了补气的草药。味道不好。

**凛：** （接过碗，看了一眼碗里灰绿色的粥）……看起来确实不好。

**雪乃：** 不许剩。

[凛喝了一口。皱眉。但她继续喝了。]

[雪乃在她旁边坐下来。也端着碗。也喝了一口。也皱眉。]

**雪乃：** ……确实不好喝。

**凛：** 你自己配的。

**雪乃：** 药效和味道不成正比。这是医学的遗憾。

[两个人安静地喝粥。周围只有风声和远处不知什么鸟的几声短促的叫。]

[凛放下碗。看着东面的天际线。深蓝色正在一点一点地退去，灰白色的区域在扩大。]

**凛：** ……天快亮了。

**雪乃：** 嗯。

[沉默。]

**凛：** 你以前——在神殿的时候，每天日出前都会祈祷。

**雪乃：** 你记得。

**凛：** 你跪在回廊的窗户前面。面朝东。握着祈祷珠。嘴里念念有词。

**雪乃：** 念的是"晨颂"。神殿的必修功课。每天日出前做一次。

**凛：** ……你现在还做吗？

[雪乃低下头。看着自己手里的粗陶碗。碗面粗糙，能看到泥胎里混着的细小沙粒。]

**雪乃：** 不做了。晨颂的词——我还记得。每一个字都记得。但我不念了。

**凛：** 为什么？

**雪乃：** 因为那些词是念给神的。我——

[她停了一下。手指在碗沿上轻轻划着。]

**雪乃：** 我不确定有没有神了。也许有。也许没有。但就算有——他没有听过我。柿子村的三十个人是我救的，不是他。空神殿里的难民是我守住的，不是他。难民营的零伤亡是我治出来的，不是他。

[她抬起头。看着东面的天际线。灰白色的区域下面，出现了第一缕极淡的橘色。]

**雪乃：** 你的伤——是我治的。不是他。

[凛没有说话。她只是看着雪乃的侧脸。]

**雪乃：** 所以我不向神祈祷了。不是因为恨他。是因为——不需要了。

#### 第二幕：新的祈祷

[天际线上的橘色在扩大。深蓝色的天幕像是一张被人从底部点燃的纸——火焰从下面一点一点往上舔，蓝色退去，金色涌来。]

[雪乃放下碗。站了起来。]

[她走到矮丘东坡的边缘——那里视野最开阔。能看到地平线的全部。脚下是枯草和碎石，远处是绵延的丘陵和晨雾，更远处是天和地的交界线——正在被日出的前奏染成水彩般的渐变色。]

[她站在那里。面朝东。]

[然后她双手合十。放在胸前。]

[不是神殿的祈祷姿势——神殿的姿势要求掌心之间夹着祈祷珠，指尖朝上，对准穹顶的神像。她的姿势不一样。掌心贴掌心，空无一物。指尖朝前——朝着地平线。朝着即将升起的太阳。]

[凛看着她的背影。]

**凛：** ……你不向神祈祷了——那你向谁祈祷？

[雪乃没有回头。]

**雪乃：** 向日出。

[凛愣了一下。]

**雪乃：** 向日出。向还活着的人。向还在呼吸的人。向受了伤还没有被治好的人。

[她的声音很轻。但在黎明前的安静里，每一个字都清清楚楚。]

**雪乃：** 向正在走夜路的人——愿天快点亮。向正在流血的人——愿有人来包扎。向写了字又涂掉的人——愿有人读到。

[风吹过矮丘。吹起了她白色长袍的下摆。也吹起了医疗帐前面那面白旗——白布在风中猎猎作响。]

**雪乃：** 这就是我的新的祈祷。不念给神。念给活人。

[凛站了起来。]

[她走到雪乃身边。站在她右侧。面朝同一个方向——东。]

[她没有双手合十。她不是祭司。她不会祈祷。但她站在那里。站在雪乃旁边。]

**凛：** ……我不会念。

**雪乃：** 不用念。站着就够了。

**凛：** ……站着就算祈祷？

**雪乃：** 活着就算祈祷。

[地平线上，太阳的第一道光破了出来。]

[不是猛烈的——是极温柔的。一线金色从地平线的正中央渗出来，像是有人在厚重的帷幕下面伸出了一根手指。然后一根手指变成了一只手，一只手变成了整个掌心。金色的光从地平线上涌出来，铺过丘陵、铺过晨雾、铺过枯草和碎石，最终铺到了两个人的脚下。]

[光照在雪乃的白色长袍上——白布变成了金色。照在凛的衬衣上——灰白变成了暖黄。照在两个人的脸上——苍白变成了被日出染暖的淡金色。]

[雪乃闭上眼。双手合十，放在胸前。]

[她的嘴唇微微开合。极轻极轻地说着什么——不是晨颂。不是任何神殿教过的祷词。]

**雪乃：** （气音，只有旁边的人能听到）……愿受伤的人被治愈。

[停顿。]

**雪乃：** 愿迷路的人找到路。

[停顿。]

**雪乃：** 不为任何王国。不为任何胜利。

[她睁开眼。转头看着身边的凛。日出的光正好落在凛的侧脸上——瘦削的、棱角分明的、眉心有一道浅竖纹的侧脸。但那道竖纹比前天浅了一点。]

**雪乃：** 只为这些。够了。

[凛看着她。日出的光让雪乃的眼睛变成了一种浅琥珀色。]

[然后凛做了一件她从来没有做过的事——]

[她将双手合在一起。不是标准的。手指交叉的方式是错的——她是用握剑的方式在合十。但她合上了。]

[她闭上眼。]

[她不会念。她不是祭司。她这辈子大概唯一会的祈祷就是"别死"——在战场上对自己说的，或者对倒下的战友说的。]

[但今天她学了一句新的。]

**凛：** （声音极低极轻，像是怕说大声了就会被收回去）……愿受伤的人被治愈。愿迷路的人找到路。

[两个人站在矮丘的东坡上。面朝日出。一个双手合十，掌心空空的，不夹着祈祷珠。一个双手交握，指法不对，但合着。]

[日出的光将她们的影子投在身后的地面上——两道长长的影子，并排着，向西延伸，延伸到帐篷上、白旗上、延伸到更远的地方。]

[风将白旗吹得猎猎作响。纯白的布在金色的晨光中亮得发光。]

[两个人。两个什么都没有的人。但她们有日出。有彼此。有一句不念给任何神的祈祷。]

[这就是雪乃的新的信仰。]

[不在任何神像里。不在任何教义里。不在任何穹顶壁画里。]

[在治愈的双手里。在还活着的人里。在站在旁边的那个人里。]

[日出了。]
