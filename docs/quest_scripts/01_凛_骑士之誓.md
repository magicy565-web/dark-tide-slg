## Event 01 — 开战宣言「边境的号角」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.12-15 |
| 情感基调 | 凛然正气，不可撼动的信念 |
| 战斗条件 | 击败凛先锋部队（保留50%HP以上） |
| 特殊机制 | 凛的「圣光壁障」前3回合DEF+3；隘口地形每回合仅3单位可通行 |
| 敌方配置 | 凛(Lv.15) + 圣骑士×4(Lv.12) + 步兵×8(Lv.10) |
| 奖励 | 350 EXP, 800G, 碎裂的圣骑士徽章×1, 解锁凛档案第一页 |

### 场景描写

黄昏将天城王朝西部边境最险峻的关隘——灰谷隘口——浸入一片赤铜色的光辉之中。

两侧百丈高的断崖如同被巨神斧劈开的裂口，狭窄的谷道仅容三骑并行。岩壁上密密麻麻刻满了历代骑士的誓言——最古老的已被风蚀得只剩模糊凹痕，最新的一行墨迹未干，是今晨凛亲手刻下的。夕阳从谷口倾泻而入，将那些铭文一一点亮，仿佛百年间所有骑士的灵魂都在低声呢喃。

哨塔最高处，凛独自伫立。银白色的全身铠甲映着残阳，如同被鲜血浇铸的雕像。风从谷底灌上来，扯动她腰间的白百合纹章披风，猎猎作响。她的视线越过谷口，注视着远方地平线上缓缓逼近的黑色潮水——那是暗潮军团的旗帜，乌压压的，铺满了整个西面旷野，如同冬夜从地底涌出的浓墨。

她的手搭在剑柄上。没有颤抖。

### 剧情对话

[凛从哨塔缓步走下石阶，每一步铠甲的铿锵声在谷道中回荡。骑士团十二人列队等候，圣骑士们单膝跪地，步兵们挺直腰背，但每个人眼中都藏着同一个问题。凛走到队列正前方，停下脚步，目光缓缓扫过每一张面孔。]

**凛：** 都起来。跪着听不清我说话。

[圣骑士们站起身。副官苍站在最前，双手紧握长枪，指节微微发白。]

**凛：** 我不打算骗你们。暗潮先锋军已过磐石河，主力明日黄昏前抵达谷口。

**苍：** 团长，斥候回报的数字……

**凛：** 我知道。先锋军约三千，主力估计在一万二以上。我们这里，算上轻伤能战的，不到三百人。

[队列中传来压抑的吸气声。一名年轻步兵的枪杆碰到了身旁同伴的盾牌，发出一声脆响，在死寂中格外刺耳。]

**凛：** 五倍。

[她竖起一只手掌，五根手指张开。]

**凛：** 他们是我们的五倍。这是最好的估计。最坏的情况——

**苍：** ……八倍。

**凛：** 嗯。

[沉默。风从谷道穿过，像是远处战马的嘶鸣。]

**凛：** 援军方面。我昨夜收到枢密院回函。最近的机动部队在东都，急行军三日可达。三日。七十二个时辰。

**圣骑士甲：** 团长，那粮草——

**凛：** 够两天。水源倒不用担心，谷底有地下河。但两天之后……

[她停顿了一下。目光越过众人头顶，望向身后——谷道深处通往的方向。]

**凛：** 你们都知道身后是什么。桐叶村。落羽镇。白桦集。青石坊……一共七个村镇，尚未完成撤离。一千二百名平民。其中——

[她的声音微微沉了沉。]

**凛：** 其中有三百余名孩童。

[没有人说话。哨塔上的旗帜被风拉直，白百合纹章在赤铜色的光中显得庄严而孤绝。]

**步兵乙：** 团、团长……属下斗胆问一句。我们……真的守得住吗？

[凛转向那名年轻步兵。他大概十七八岁，脸上还有未褪去的稚气，嘴唇在微微发抖。凛走到他面前，近得几乎能看到他瞳孔中映出的自己的影子。]

**凛：** 你叫什么？

**步兵乙：** 属、属下柊。第三步兵连……

**凛：** 柊。你入伍多久了？

**柊：** 四……四个月。

**凛：** 四个月。

[凛轻轻点头，然后伸手拍了拍他的肩甲。那只手稳得像岩石。]

**凛：** 柊，我不会告诉你"一定守得住"。那是骗子才说的话。

[她转身面向全体，声音陡然拔高，在谷壁间激荡回响。]

**凛：** 我不会说这是一场轻松的仗！不会说我们人人都能活着回去！我尊重你们每一个人，所以我只说真话——

[她拔出长剑。剑身在夕阳中燃烧般闪耀，剑锋指向谷口外那片黑色的潮水。]

**凛：** 但灰谷隘口是天城西境最后一道天险。过了这里，暗潮铁骑半日可达桐叶村。那三百个孩子……跑不过战马。

[她将长剑竖在身前，剑尖触地，双手覆上剑柄，如同在祭坛前起誓。]

**凛：** 灰谷隘口——绝不失守。

[一瞬的静默后，苍第一个将长枪顿地。]

**苍：** 灰谷隘口，绝不失守！

[声音如涟漪般扩散。圣骑士们齐齐顿武器，步兵们用枪杆击盾。]

**全体：** 灰谷隘口——绝不失守！！

[回声在两侧百丈断崖间来回碰撞，久久不散。凛微微点头，转身面向谷口。暗潮的旗帜更近了。]

**凛：** 圣骑士列前阵。步兵分三组轮换。记住，谷道狭窄，每回合最多三个单位通过——这是我们唯一的优势。用地形，用壁障，用一切能用的东西。

[她举剑。]

**凛：** 列阵！

### 战斗阶段

[战斗开始。凛站在阵列最前方，「圣光壁障」展开——一面半透明的金色光幕笼罩前排，所有防御者DEF+3。]

*——灰谷的风很冷。但剑柄是热的。我的心跳是稳的。这就够了。*

[第一波暗潮士兵冲入谷道，被狭窄地形限制为三人一排。]

**凛：** 稳住阵线！不要被气势压倒！他们再多，这条路一次只能过三个人！

[战斗第2回合，暗潮投入重甲步兵。冲击力巨大，前排圣骑士后退半步。]

**凛：** 后退一步不丢人。但只许一步。第二排，枪阵补位！

[遭受重击时——]

**凛：** ……！

[她单膝跪地，盾牌上多了一道深深的裂痕。但她几乎立刻站起。]

**凛：** 这种程度……还差得远。

[战斗第3回合，圣光壁障开始闪烁——持续时间将尽。]

**凛：** 壁障即将消失。从现在起，用你们自己的盾。相信自己的身体！

[HP低于40%时——]

[凛的铠甲多处碎裂，左臂护甲完全脱落。鲜血从额角滑下，模糊了左眼视线。她用剑支撑身体，呼吸沉重但稳定。]

**苍：** 团长！退到第二排——

**凛：** 不退。

**苍：** 团长！！

**凛：** 我说了……不退。

[她缓缓直起身，将剑重新举至战斗姿态。]

**凛：** 我若退一步，后面的人就会退十步。然后百步。然后——那三百个孩子就没有"步"可以退了。

[她深吸一口气，圣光在破碎的铠甲缝隙中微微闪烁。]

**凛：** 来吧。下一个。

[战斗结束——玩家击败凛先锋部队。]

### 战后

[夕阳已完全沉入地平线。谷道中只剩下武器碰撞后的余韵和伤兵压抑的呻吟。凛拄剑站在最前方，身后是残破的阵线——十二名圣骑士倒下了四人，步兵伤亡过半。但阵线没有崩溃。]

[凛缓缓转身。她最后看了一眼谷口外的旷野——暗潮的先锋已经后撤重整，但主力的火光在地平线上连成一片，如同另一条银河。]

*——三日。只需要守三日。*

*——不。先守过今夜。*

[她将剑收回鞘中。金属入鞘的声音清脆而决绝。]

**凛：** 救治伤员。加固壁垒。轮班休息——

[她停了停。]

**凛：** 今夜的粥里多加些盐。天冷了。

[她最后回望了一眼刻满誓言的岩壁。在黑暗中，那些铭文已经看不清了。但她知道它们在那里。一直都在。]

> **获得奖励：** 350 EXP / 800G / 碎裂的圣骑士徽章×1 / 解锁凛档案第一页
>
> *「碎裂的圣骑士徽章」——从凛破碎的护肩上掉落的银色徽章。百合纹样被利刃斩成两半，但金属仍隐隐泛着圣光的余温。翻到背面，刻着一行小字："吾剑所指，即为正道。"*

## Event 02 — 前哨遭遇「圣殿骑士的壁障」

| 情感基调 | 钢铁意志中的第一丝裂痕 |
| 战斗条件 | 坚持8回合或击败凛 |
| 特殊机制 | 凛的「誓约之盾」每2回合恢复全体10%HP；百姓撤离进度每回合+12%；平原+壕沟DEF+2 |
| 敌方配置 | 凛(Lv.18) + 苍(Lv.16) + 圣骑士×3(Lv.14) + 枪兵×6(Lv.12) |
| 奖励 | 500 EXP, 1200G, 白河平原的泥土×1, 凛的断裂护臂×1, 解锁档案第二页 |

### 场景描写

三日后。白河平原。

天空是一种病态的灰白色，像是把所有颜色都榨干后剩下的渣滓。平原上的草已经被初冬的霜冻杀成枯黄，一直铺展到视野尽头，与低垂的天幕融为一体——分不清哪里是大地，哪里是天空，只有一条泛着铅灰色光泽的河流蜿蜒横亘其间，那是白河。

凛站在河岸的高坡上。

她已经三天没有睡觉了。眼窝深陷，瞳孔中那股凛然的光仍在，但多了一层玻璃般透明的脆感。银白铠甲上残留着灰谷隘口战斗的血迹——她没有时间清洗，也或许是不愿清洗。左臂的护甲在撤退途中用绷带草草固定，每次活动都会发出令人牙酸的金属摩擦声。

她的身后是三道新挖的壕沟。每一道都是她亲手参与挖掘的——骑士团幸存者不到百人，没有工兵编制，所有人包括凛自己都拿起了铁锹。壕沟的泥土还是湿的，混着白河的河水，散发着腥冷的土壤气息。

远处，白河镇的轮廓在灰色天幕下显得分外矮小。镇子里的炊烟还在升——那意味着还有人没有撤离。

### 剧情对话

#### 第一幕：援军不会来了

[一名传令兵骑着浑身泥浆的战马从东面官道疾驰而来。马蹄溅起的泥点打在壕沟边待命士兵的脸上，没有人去擦。传令兵翻身下马，单膝跪地，双手呈上一封火漆密函。火漆上盖着枢密院的鸢尾纹章。]

**传令兵：** 团长！枢密院急令——

[凛接过密函。她的手指在火漆上停了一瞬——指尖的泥土和干涸的血渍蹭上了鸢尾纹章的金色封蜡。她拆开信，目光从左到右缓缓扫过。]

[她的表情没有变化。]

[完全没有变化。]

[她将信折好，放入胸甲内侧的夹层。动作很轻，像是在安放一件易碎的东西。]

**苍：** 团长？援军的消息——

**凛：** …………

**苍：** 团长！

[凛抬起头。她看了苍一眼——那一眼里有太多东西，多到苍下意识后退了半步。]

**凛：** 援军不会来了。

[空气凝固。身后壕沟里传来铁锹戳入泥土的声音——有人还在挖掘，没有听到这句话。那声音此刻显得格外荒谬。]

**苍：** ……什么意思？

**凛：** 枢密院下令全面收缩防线至东都三十里圈。边境诸镇……被判定为非战略要地。不再增援。

**苍：** 非战略——

[苍的声音骤然拔高。他的长枪猛地戳入地面，枪身震颤不止。]

**苍：** 灰谷隘口死了多少人？！那些——那些士兵的命就是"非战略要地"？！白河镇还有上千平民——枢密院那帮坐在天鹅绒椅子上的——

**凛：** 苍。

[一个字。不大声。但像一把刀插入沸水，所有翻涌瞬间平息。]

**凛：** 够了。

[苍咬紧牙关，太阳穴的青筋跳动着。他低下头，双拳攥得骨节发白。]

**苍：** ……是。

[凛沉默了很久。风从白河方向吹来，裹挟着冬草枯萎的气息和远处炊烟的焦木味。她望向白河镇的方向。]

**凛：** 白河镇的撤离完成了多少？

**苍：** ……约六成。老人和腿脚不便的还有三百余人。镇长说最快还需要半日。

**凛：** 半日。

[她像是在咀嚼这两个字的重量。]

**凛：** 那就半日。

**苍：** 团长？

**凛：** 我们在这里再守半日。等白河镇全部撤完，再执行收缩命令。

**苍：** 可枢密院的命令是立刻——

**凛：** 我知道命令写的是什么。

[她转向苍，目光平静得近乎冷酷。]

**凛：** 半日。只需要半日。如果有人要追究违抗军令的责任——

[她将手按在胸甲上，密函所在的位置。]

**凛：** 由我一人承担。

[苍张了张嘴，最终什么也没说。他重新拔起长枪，走向壕沟阵地。]

**苍：** ……全体听令。备战。

#### 第二幕：白河防线

[暗潮追击部队抵达白河平原。尘埃在地平线上升起，像一堵缓缓推进的灰色墙壁。]

**凛：** 他们来了。

[她拔出长剑。剑身上灰谷之战留下的缺口还在——她同样没有时间修复。]

**凛：** 全员进入壕沟阵地。枪兵列第一壕，圣骑士列第二壕，我在第三壕策应。苍——

**苍：** 在。

**凛：** 你负责盯住撤离进度。每过一刻钟向我报告一次。百姓全部撤完之前，这条防线——

[她将剑指向白河。]

**凛：** 一步不退。

### 战斗阶段

[战斗开始。白河平原上，壕沟防线与暗潮追击部队正面交锋。]

[凛展开「誓约之盾」——一面巨大的金色光盾投影在天空中，覆盖整条防线。每隔两回合，光盾脉动一次，柔和的圣光洒落，恢复全体10%HP。]

**凛：** 誓约之盾——展开。

*——这道光能护住多少人？十个？二十个？……够了。只要能护住身后那条通往白河镇的路。*

[第1回合。暗潮先锋冲锋。壕沟地形DEF+2生效，枪兵以逸待劳。]

## Event 03 — 角色事件·壹「押送途中」

| 项目 | 数据 |

### 场景描写

数日后。通往王都的官道。

这条路凛走过无数次——每次进京述职、每次接受封赏、每次在千姬面前单膝下跪领命。官道两旁的白桦树在往年此时应该已经落尽叶子，只剩苍白的枝干指向天空，像无数瘦骨嶙峋的手臂。但今年的冬天来得格外早，树上还残留着几片枯叶，被风一吹就发出干燥的、像虫蛀木头碎裂的声音。

凛骑在马背上，率队护送一支运输队。十二辆马车排成长列，每辆车都被厚重的军用帆布严严实实地覆盖。官方的说法是"战争物资——前线急需的武器与军械"。

但有几件事不对。

马车的轮辙深得异常。凛在灰谷隘口见过运送攻城器械的辎重车，轮辙也没有这么深。帆布被绳索从外面系死，内侧似乎还加了锁扣。每辆车旁都有两名身穿黑衣的特务监察队队员随行——他们不归凛指挥，直接听命于王都情报局。

而最令凛不安的是声音。风停的时候，如果仔细听——马车厢里偶尔会传来一种压抑的、像是被布匹堵住嘴后发出的呜咽。

### 剧情对话

#### 第一幕：军械不会发出声音

[午后。队伍在官道上缓慢行进。凛驱马行至队伍中段，与特务监察队的带队官鸠场并行。鸠场是个面容苍白的中年男人，穿着王都情报局的黑色制服，胸前别着银色的鸢尾纹章——和枢密院同款，但多了一只展翅的鹰。他始终挂着一丝恰到好处的微笑，像是用尺子量出来的弧度。]

**凛：** 鸠场大人。

**鸠场：** 凛团长。辛苦了。边境之战的英勇，连王都都有所耳闻。

**凛：** 不敢当。

[沉默了一会儿。马蹄声在官道上单调地回响。]

**凛：** 我有一个问题。

**鸠场：** 请说。

**凛：** 这些军械——是运往哪条前线的？

**鸠场：** 这是机密。凛团长只需负责护送安全即可。

**凛：** 我明白。但作为护送指挥官，我需要了解货物性质以安排适当的保护方案。如果是火药或油脂类物资，行军路线需要避开——

**鸠场：** 不含危险品。凛团长多虑了。

[又一阵风停。寂静中，第七辆马车传来一声压抑的呜咽——像是婴儿的哭泣被强行捂住后漏出的尾音。凛的瞳孔骤然收缩。]

**凛：** ……鸠场大人。

**鸠场：** 嗯？

**凛：** 军械不会发出声音。

[鸠场的微笑凝固了大约半秒。然后恢复如初——甚至更加温和了一些。]

**鸠场：** 也许是车轴摩擦。这段路不太平整。

**凛：** 车轴摩擦不会呜咽。

[两人对视。凛的目光如同剑尖抵在鸠场的咽喉上。鸠场的笑容纹丝不动。]

**鸠场：** 凛团长。我再说一次——这是枢密院直辖的机密任务。您的职责是护送。仅此而已。

[他微微侧身，露出胸前的鸢尾鹰纹章。那枚纹章在阳光下闪了闪，像是某种无声的威胁。]

**凛：** ……

[凛没有睡。她独自坐在营地边缘的一棵倒木上，面朝马车方向。月光很亮，近乎惨白——那种能把一切颜色都洗掉只剩黑白灰的月光。]

[后半夜。篝火将灭未灭，只剩暗红色的余烬在灰烬中明灭。凛独坐在那棵倒木上。她摘掉了手套，赤裸的手指在夜风中微微发红。]

*——千姬陛下的御印。*

*——那枚我曾经在就任仪式上亲吻过的御印。*

这个地方曾经有名字——但凛已经不记得了。也许是某个村庄。也许是某个集镇。也许是某个孩子第一次学会走路的地方、某个老人每天清晨浇花的地方、某对恋人在桥头许下诺言的地方。

被烧毁的房屋只剩焦黑的木框架，像是巨兽的肋骨插在灰烬中。坍塌的石墙上还残留着生活的痕迹——被火烧得卷曲的门帘、熔成一坨的铁锅、散落在瓦砾间的碎瓷碗。空气中弥漫着焦木、灰烬和某种更深层的、令人作呕的甜腐气息——那是无人收殓的尸体在冬日冷空气中缓慢分解的味道。

凛站在废墟中央一栋坍塌建筑的二楼残壁上。月亮还没有完全升起，只有西边天际残留着一抹暗橘色的光。她一反常态地没有穿全套铠甲——右肩甲和左腿甲被她拆掉了，用轻便的皮革护具替代。这让她看起来更像一名突袭者而非圣骑士。

她的身后是五十人的突击队。精挑细选的老兵——但即便是老兵，此刻看着凛的背影也面面相觑。因为这不像凛的风格。凛一向是"盾"，不是"矛"。凛一向是守、不是攻。凛一向——

不是这样。

### 剧情对话

#### 战前

[凛蹲在残壁上，展开一张手绘的地图。地图上用红色标注了三个圆圈——暗潮军团的前线补给集结点。]

**凛：** 目标三个。东面粮草堆、北面军械库、中央指挥帐。全部摧毁。

**苍：** 团长，我有异议。

[凛没有抬头。]

**苍：** 这三个点分布在半里范围内，驻守兵力超过两百。我们五十人夜袭——就算打成功了，撤退路线会被切断。

**凛：** 不会。废墟地形复杂，他们展不开阵型。

**苍：** 就算如此——这种孤军深入的打法，风险太大了。我们可以等主力推进后再——

**凛：** 等？

[凛终于抬头。月光照在她的脸上——苍看到了什么，让他没有说出下一个字。]

[凛的眼神不对。不是往日那种平静如湖水的坚定。而是一种更灼热、更尖锐、带着某种近乎焦躁的光。像是一把过度淬火的刀——硬度极高，但随时可能碎裂。]

**凛：** 退到什么地方才算尽头？

[苍张了张嘴。]

**凛：** 灰谷退到白河。白河退到官道。官道退到……什么地方？退到王都城墙底下吗？退到千姬的御座前面吗？

[她的声音不大。但每一个字都像是从牙缝里挤出来的。]

**凛：** 不退了。今天开始，不退了。

[她站起身，将地图卷起塞入腰间。]

**凛：** 全员听令。分三组。第一组随我攻东面粮草堆。第二组由苍带领攻北面军械库。第三组——

**苍：** 团长。

[苍走近一步，压低声音。]

**苍：** ……你没事吗？

[凛停顿了一瞬。]

**凛：** 没事。

**苍：** 那天晚上……马车的事——

**凛：** 苍。

[她的声音陡然降温。]

**凛：** 现在不是谈这个的时候。

[苍沉默了。他认识凛十二年，从骑士学院到战场，他从未见过凛用这种语气和他说话。不是愤怒。不是冷漠。是——]

[是恐惧。凛在害怕。她害怕如果停下来思考那天晚上看到的东西，她会碎掉。所以她选择——]

[进攻。不停地进攻。用刀剑碰撞的声音淹没内心的质问。]

**苍：** （低声）……是。

**凛：** 第三组迂回至南面封锁撤退路线。六回合内完成全部目标。

[她拔出长剑。剑身上灰谷和白河两战留下的缺口仍在——但凛似乎完全不在意了。]

**凛：** 出发。

#### 战斗阶段

[夜袭开始。突击队如同三把尖刀插入暗潮补给线的心脏。]

[凛冲在第一组最前方——不是指挥位置，是突击位置。这又一次让苍皱起了眉头。]

[战斗开始。凛的ATK+3，但DEF-2——焦躁debuff「动摇之刃」生效。她攻击得比以往任何时候都更猛烈，但防御出现了破绽。]

**凛：** 粮草堆——点火！

[她一剑劈开守卫木栅栏的暗潮士兵，圣光沿剑刃爆发，将干燥的粮草引燃。火焰腾空而起，照亮了半个废墟。]

[凛击杀后，脚步不停——「连续行动」发动，她直接冲向下一个敌人。]

**凛：** 下一个。

[第2回合。火光引来暗潮增援。凛被三名敌兵包围。]

**凛：** 让开！

[她的剑法变了。不再是圣骑士团教科书般精准优雅的剑术，而是某种更原始、更暴力的东西——大开大阖，每一剑都带着要将面前一切劈成碎片的狂意。]

**凛：** 挡在我面前的——全部让开！！

[一名暗潮士兵的长刀砍中她拆掉肩甲的右肩——皮革护具被割开，鲜血飞溅。凛甚至没有看那个伤口一眼，反手一剑将对方击飞。]

[苍的视角——北面军械库方向。]

**苍：** （内心）*……不对。团长她不对。*

*她在用受伤来感受自己还活着。*

*她在用战斗来逃避某些东西。*

*从那天晚上——从那些马车——从那个孩子的眼睛——*

*团长……你到底看到了什么？*

[第3回合。苍带队成功引爆北面军械库。爆炸声震动了整片废墟。]

**苍：** 北面目标摧毁！

**凛：** ——中央指挥帐。最后一个。全员随我突进！

[第4回合。凛带队冲向中央指挥帐。沿途她以一己之力突破两道防线，连续行动三次——击杀、前进、击杀、前进。她的铠甲上全是血，分不清是敌人的还是自己的。]

**凛：** ……快了。就差一个。

[第5回合。中央指挥帐前，暗潮精锐列阵防守。凛停下脚步——不是因为畏惧，而是因为她的身体晃了晃。失血过多。]

**苍：** 团长！！你的肩——

**凛：** 没事。

**苍：** 在流血——

**凛：** 我说没事。

[她咬紧牙关。圣光从体内涌出，暂时封住伤口——但这不是治疗，只是强行止血。代价是更剧烈的疲劳。]

**凛：** ……最后一个。然后就可以停下了。

[那句话轻得几乎只有她自己听到。"然后就可以停下了"——不像是在说战斗。]

[第6回合。凛发动最后的冲锋。]

**凛：** 圣光——突进！！

[她全身爆发的圣光不再是温暖的金色——而是灼白的、近乎刺眼的白。像是一颗即将燃尽的恒星。她以不可阻挡之势穿过精锐防线，一剑劈开指挥帐的帐篷柱。帐篷轰然坍塌。]

[补给点三——中央指挥帐，摧毁。]

#### 战后

[战斗结束。]

[废墟中到处是火焰。粮草堆的火、军械库爆炸后的余烬、指挥帐被圣光引燃的残骸。橘红色的光摇曳不定，将所有人的影子拉得奇形怪状。]

[凛跪在废墟中央。]

[她用剑支撑着身体——剑尖插入碎石地面，双手覆在剑柄上，额头几乎贴到手背。她的呼吸粗重而不均匀，每一次吸气都伴随着细微的颤抖。右肩的伤口重新裂开，血沿着铠甲的纹路缓缓淌下，在碎石上汇成一小片暗红色的洼。]

[苍走到她身边。没有说话。只是站在那里。]

[凛抬起头。]

[她看到了墙。]

[一面半坍塌的石墙——也许是某户人家的外墙。被火熏黑了大半，但有一小块区域因为被坍塌的屋顶遮挡而幸免于难。那一小块干净的石墙上，有一幅画。]

[一个孩子用某种颜料——也许是浆果汁，也许是泥巴混着什么——画的画。]

[一个歪歪扭扭的圆。涂成黄色。周围伸出一圈长短不一的线条。]

[太阳。]

[一个孩子画的太阳。]

[凛盯着那幅画。]

[很久很久。]

[火光在她的瞳孔中跳动。那个歪扭的太阳在摇曳的光影中仿佛在微微转动——像是在看着她。]

*——那个孩子。马车里的那个孩子。*

*——她也曾经画过这样的太阳吗？*

*——在她被铁链锁住之前。在她的眼神变成那样之前。*

*——她也曾经拿着浆果汁，在自家的墙壁上歪歪扭扭地画过一个太阳吗？*

*——然后她的父母假装生气，说"又在墙上乱画"，但其实偷偷笑了？*

*——然后有一天，有人拿着盖了御印的文书来了。铁链来了。马车来了。*

*——太阳就灭了。*

[凛的视线模糊了。不是因为失血。]

[她迅速眨了眨眼。液体被逼退。没有落下。]

**苍：** 团长。该撤了。

**凛：** ……嗯。

[她试图站起来。手臂发抖。第一次没能成功。第二次，苍伸出手——凛顿了顿，然后握住了。]

[她站起身。最后看了那面墙一眼。]

**凛：** 苍。

**苍：** 在。

**凛：** ……没什么。走吧。

[她转身。脚步沉重。铠甲碰撞的声音不再铿锵——更像是拖行。]

[苍走在她身后，视线落在凛的背影上。那个曾经挺拔如松的背影，此刻微微弓着——像是肩上压着某种看不见的、远比铠甲沉重的东西。]

**苍：** （内心）*团长。你看到了什么，我不知道。*

*但你正在碎裂。*

*而我不知道该怎么接住你。*

[画面渐暗。废墟中的火焰在黑暗中渐渐熄灭。只有那面墙上歪扭的太阳，在最后一丝火光中，短暂地亮了一下。]

[然后，一切归于黑暗。]

### 场景描写

王都·天枢城。

凛已经两年没有回到这座城市了。记忆中的天枢城永远笼罩在一层柔和的金色光晕里——白石铺就的大道，两旁栽种着四季常青的桐叶树，树冠交错如穹顶，阳光透过枝叶洒下无数细碎的光斑。皇宫「千�的鸟御所」矗立在城市最高处，琉璃瓦顶在晴天折射出虹彩，远远望去如同漂浮在云端的宝石。

但今天的天枢城让凛觉得陌生。

大道上的桐叶树有三分之一枯死了——没有人修剪，干枯的枝干像骨骼一样伸向灰蒙蒙的天空。白石路面被无数辎重车的铁轮碾出深深的沟壑，缝隙间长满了杂草。街道两旁的商铺有一半关了门，剩下的也门可罗雀。偶尔有行人经过，脚步匆匆，目光低垂，谁都不看谁。

皇宫的琉璃瓦仍在。但凛抬头望去时，发觉正殿最高处的那面「千鸟金旗」——历代天城女王的象征——已经不见了。取而代之的是一面银灰色的旗帜，上面绣着枢密院的鸢尾纹章。

凛在宫门前停下脚步。她穿着洗净的银白铠甲——她花了整整一个晚上才把灰谷和白河的血迹全部擦去。百合纹章在冬日惨淡的阳光下泛着冷光。她的右肩仍然缠着绷带，但被肩甲遮住了。

宫门口站着两列卫兵——不是皇宫禁卫军的金甲，而是枢密院直属部队的黑甲。

凛的目光在那些黑甲上停留了一瞬。然后她迈步走入宫门。

### 剧情对话

#### 第一幕：嘉奖仪式

[千鸟御所·正殿「紫宸殿」。大殿空旷得令人不安。凛记得两年前来这里述职时，殿内至少有三十名朝臣列席。今天只有七人——全部穿着枢密院的紫色官服，站在御座两侧，表情如同石刻。]

[御座上坐着千姬。]

[凛已经两年没见到千姬了。记忆中的千姬永远端坐如莲——年轻的面容上带着一种超越年龄的、近乎神性的安详。她的声音像是从远处传来的风铃，轻柔却清晰，每一个字都恰到好处。]

[今天的千姬穿着全套朝服——十二层重叠的锦缎，从最内层的雪白到最外层的深紫，层层递进如暮色降临。金色的凤冠压在她乌黑的长发上，流苏垂下，遮住了大半张脸。]

[凛看不清千姬的表情。]

[她单膝跪地。铠甲与石板碰撞的声音在空旷大殿中回荡了很久。]

**凛：** 圣骑士团团长凛·艾斯特雷亚，奉召回京述职。

[沉默。很长的沉默。]

[一名枢密院官员——从位置判断应是枢密使——向前一步，展开一卷文书。]

**枢密使：** 圣骑士团于西境灰谷隘口至白河平原一线，抵御暗潮军团先锋，拖延敌军推进七日，为东都防线部署争取了关键时间。枢密院经审议，特颁嘉奖——

*——七日。他说的是"七日"。那七日里死了多少人，他知道吗。"拖延敌军推进"——我们不是在"拖延"。我们是在拿命填。*

**枢密使：** ——授予凛·艾斯特雷亚「忠勇之盾」勋衔，赐金百两，绢五十匹。圣骑士团幸存将士各升一阶，抚恤阵亡将士家属银二十两——

*——二十两。一条命值二十两银子。*

**枢密使：** ——此外，鉴于西境战况变化，圣骑士团即日起编入枢密院直辖序列，不再隶属皇宫禁卫体系。调令已下，凛团长即日赴东都防线报到。

[凛的眉心几不可察地跳了一下。]

*——编入枢密院直辖？圣骑士团自建团以来，一百二十年，从未脱离皇宫体系。我们直属于王冠。直属于——*

[她抬起目光，望向御座。千姬端坐不动。流苏遮住了她的眼睛。]

**凛：** ……臣领旨。

[枢密使满意地点头，卷起文书。其余官员面无表情。]

**枢密使：** 千姬陛下。嘉奖仪式已毕。是否有圣谕——

**千姬：** ……

[千姬动了。她抬起一只手——那只手极白，白到近乎透明，指节纤细得像是会折断的瓷器。]

**千姬：** 凛。

[一个字。声音比凛记忆中更轻。像是在风中随时会碎的薄纱。]

**千姬：** 辛苦了。退下吧。

[凛低下头。]

**凛：** 臣告退。

[她站起身，后退三步，转身。铠甲的声音在大殿中一下一下地回响。]

[她没有看到——在她转身的瞬间，千姬的手指在膝上轻轻蜷缩了一下。]

#### 第二幕：私下会面

[深夜。千鸟御所·后苑「月见亭」。]

[这是一座建在御苑池塘中央的八角亭。四面垂挂着薄纱帘幕，月光透过来变成一种柔和的银蓝色。池塘里的锦鲤早已在冬日里沉入池底，水面只有风偶尔掠起的细纹。]

[凛独自站在亭中。她接到了一封没有署名的短笺——只有三个字："月见亭。"字迹她认得。那是千姬的手书。]

[她等了一刻钟。然后听到脚步声。]

[千姬独自走来。没有随从，没有侍女。她换下了朝服，只穿一件素白的内衫，乌发披散在肩上，没有凤冠。月光下她看起来——]

[不像女王。像一个疲倦到了极限的年轻女子。]

**千姬：** 凛。谢谢你来。

**凛：** ……陛下。

[凛下意识想要跪下。千姬伸手阻止了她——那只手轻轻按在凛的肩甲上，力道小得几乎感觉不到。]

**千姬：** 不要跪。今夜没有君臣。

[凛僵了一瞬。然后缓缓直起身。]

**千姬：** 你瘦了。

**凛：** ……战事辛苦，无暇顾及。

**千姬：** 肩上的伤，还疼吗？

**凛：** 已无大碍。

### 场景描写

东都防线以西三十里。一片被称作「无人之境」的灰色地带。

这里曾经是天城王朝最肥沃的农业区之一——所谓「天城粮仓」的核心区域。但战争像一把巨大的锉刀，将一切磨成了灰色的粉末。田埂被碾碎，灌溉渠干涸后裂成龟甲状的硬块，冬麦的残茬从冻土中伸出来，像无数折断的手指。远处散落着几座农庄的残骸——不完全的废墟，而是一种更令人不安的半毁状态：房顶塌了一半，门还关着；墙壁倒了，窗台上的花盆还在，里面的枯草在风中摇摆。

天空是铅灰色的。不是阴天——没有云。只是天空本身似乎失去了颜色，像是把蓝色全部抽走后剩下的空白。

凛的残部——不到四十人——从东面缓缓行进。他们已经连续行军三天，中间穿插了两次小规模遭遇战。所有人都在极限边缘。马匹瘦得肋骨突出，蹄子在冻土上打滑。步兵的脚步越来越慢，间距越拉越大，像一串快要散架的珠链。

凛走在队伍最前方。她不再骑马了——她的战马在两天前的遭遇战中被流矢射伤了后腿，她把马让给了一个重伤无法行走的枪兵。她的靴子上满是泥浆和干涸的血，每一步都在冻土上留下浅浅的印痕。

前方的地平线上，出现了另一支队伍。

同样稀稀落落。同样疲惫不堪。同样在灰色的荒原上缓慢移动。

暗潮的旗帜。

**凛：** 嗯。和我们一样。

**苍：** ——平民！

**凛：** ……

[凛将长剑插入冻土。剑身微微颤动。]

**凛：** ……撤兵。

[苍愣了一下，但没有质疑。他转向部队。]

**苍：** ……全军后撤。收兵。

[暗潮一方也开始后撤。灰收起重刀，向凛投来最后一个目光。]

**灰：** 圣骑士。你叫什么？

**凛：** ……凛。

**灰：** 凛。我叫灰。记住这个名字。也许下次我们不会这么幸运。

**凛：** ……也许。

[灰转身。走出两步后又停下。]

**灰：** 那两个平民——老人和孩子。他们从我们的防区逃出来的。三天前暗潮后勤部队征粮，烧了他们的村子。

[凛的手指微微蜷缩。]

**灰：** ……我没能拦住。

[灰没有再说话。她带着残部向西撤退。凛带着残部向东撤退。两支队伍像是从同一个点被推开的两滴水，缓缓分离，最终消失在灰色荒原的两端。]

[凛走在队伍最后。她回头看了一眼——暗潮的旗帜已经变成地平线上一个模糊的黑点。]

*——暗潮的指挥官也在保护平民。也拦不住自己人伤害平民。*

*——和我一样。*

*——我们是敌人。*

*——我们有什么区别？*

[她转过头。继续走。靴子在冻土上发出沉闷的声响。]

[那个老人和孩子已经被安置在后方。路过时，孩子从老人怀里探出头，望着凛。]

[凛没有看他。]

[她不敢看。]

**【选择B·继续战斗场景】**

[凛举起长剑。]

**凛：** ……我没有停战的权限。

[灰看着她。沉默了很久。]

**灰：** ……可惜。

[战斗继续。在极度疲惫中，凛的部队凭借圣骑士的铠甲优势勉强占据上风。灰被凛击伤右腿，不得不下令撤退。]

[战后。凛站在空荡荡的战场上。脚边是暗潮士兵遗落的水囊。她弯腰捡起——水囊上刻着一朵粗糙的花纹。不是军用品。是某个士兵自己刻的。]

*——他们也是人。*

*——这个念头很危险。作为骑士，我不应该这么想。*

*——但它已经在了。拔不掉了。*

## Event 07 — 角色事件·叁「烧毁的家书」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情（无战斗） |
| 情感基调 | 骑士团的分裂，最后一丝归属感碎裂 |
| 奖励 | 350 EXP, 团长纹章（碎裂）×1, 解锁凛档案第七页, CG「深夜的火光」 |

### 场景描写

东都防线后方·临时驻地「枯井营」。

这个驻地得名于营地中央一口枯死的古井。井口的石沿被数百年的绳索勒出深深的槽痕，但井底早已干涸，只剩黑洞洞的深渊。有人往里面扔过石子，数了很久才听到触底的声音。此后再没有人往里面看。

驻地是由几排简易帐篷和一圈矮土墙围成的。帐篷用的是从废弃村庄搜集来的帆布和木料——每一根木柱上都还残留着某户人家的痕迹：一根门框、一段房梁、一块刻着"平安"二字的门楣。士兵们用这些碎片搭建临时的栖身之所，像是用别人生活的废墟拼凑出自己短暂的存在。

冬天的风从北面灌入营地，帐篷被吹得猎猎作响。天已经黑了，但大部分帐篷里没有灯火——油料紧缺，能省则省。只有营地东面靠近枯井的几顶帐篷透出暖黄色的光——那是圣骑士团的驻地。

凛坐在自己帐篷里。桌上放着一盏快要燃尽的油灯，火焰只有豆粒大小，在风中摇摇欲坠。灯光在她的脸上投下深深的阴影，让她的眼窝看起来像两个黑洞。

她面前摊着一张信纸。纸面上一个字也没有。毛笔在砚台上搁着，墨已经磨好了。

她已经在这张空白的信纸前坐了一个时辰。

### 剧情对话

#### 第一幕：解散令

[帐篷外传来急促的脚步声。帐帘被掀开，苍走进来。他的脸色——凛从未见过苍露出这种表情。不是愤怒，不是震惊，而是一种超越了这些情绪之后的、近乎空白的冷静。]

**苍：** 团长。枢密院的传令使到了。

[凛抬起头。]

**凛：** ……什么命令？

[苍沉默了两秒。然后从怀中取出一份文书，放在凛面前的桌上。]

**苍：** 解散令。

[凛没有立刻去拿文书。她看着苍的脸。苍的嘴唇抿成一条线，太阳穴的青筋在跳。]

**凛：** ……读给我听。

**苍：** （深吸一口气）"枢密院令。鉴于战况紧急，军事力量需高度统合。即日起，圣骑士团番号撤销。原团编制人员并入东都防卫军第三营。原团长凛·艾斯特雷亚交回团长纹章，降为第三营副官，听候东都防卫军统帅部调遣。"

[沉默。帐篷外的风把帆布吹得啪啪作响。]

**凛：** ……就这些？

**苍：** 后面还有一段。"圣骑士团所属武器、铠甲、纹章、旗帜等一切标识性装备，即日起上缴枢密院军备局。"

*——一切标识性装备。旗帜。纹章。铠甲。*

*——一百二十年的圣骑士团。从第一任团长铸造第一面百合纹章开始——一百二十年。*

*——一纸文书。就这样没了。*

**凛：** ……传令使在哪？

**苍：** 在营门外等着。带了一队枢密院的兵。大概是来"协助交接"的。

[凛听出了苍语气中的讥讽。]

**凛：** 让他等着。

[她站起身。拿起桌上的文书，仔细地看了一遍。从头到尾，每一个字。然后折好，放入胸甲夹层——和千姬的密函放在同一个位置。]

**凛：** 集合骑士团全体。

**苍：** ……全体？团长，现在——

**凛：** 现在。

#### 第二幕：分裂

[枯井营·圣骑士团驻地。所有幸存的骑士团成员被紧急集合。不到三十人——从灰谷隘口的三百人到现在的三十人。帐篷之间的空地上，他们列成了不整齐的队列。有人还在包扎伤口，有人被同伴搀扶着站立。]

[凛站在队列前方。油灯的光不够亮，但月光从帐篷间的缝隙中漏进来，勉强照亮了她的轮廓。]

**凛：** 我不绕弯子了。枢密院下令解散圣骑士团。即日生效。

[队列中传来压抑的骚动。有人低声咒骂，有人沉默不语，有人看着凛的眼神充满了不可置信。]

**圣骑士甲：** ……团长，这不可能。圣骑士团是千姬陛下亲建的——没有她的诏令，枢密院凭什么——

**凛：** 文书上有御印。

[她从胸甲中取出文书，高高举起。月光照在御印的殷红印泥上。]

*——又是这枚印章。千姬不知道的印章。盖在她不知道的命令上。*

*——但这不重要了。法律上——这就是合法的。*

**圣骑士乙：** 合法？！灰谷死的那些人合法吗？白河死的那些人合法吗？我们用命守——

**凛：** 够了。

[她的声音不大，但队列立刻安静下来。]

**凛：** 我知道你们的感受。但现在不是发泄情绪的时候。我需要你们做一个选择。

[她环视所有人。]

**凛：** 服从命令。交出装备。并入第三营。从此你们不再是圣骑士，但你们仍然是天城的军人。仍然有军饷、有编制、有后勤保障。

[她停顿了一下。]

**凛：** 这是正确的选择。我建议你们——所有人——都选这一个。

**苍：** ……团长。你说"所有人"。那你呢？

[凛没有立刻回答。她从腰间解下一枚银色的纹章——团长纹章。圣骑士团最高的象征。白百合在月光下泛着柔和的银光。她看了那枚纹章很久。]

**凛：** 我会交出纹章。以副官身份——

**苍：** 不。

[苍向前一步。他的声音不大，但硬得像铁。]

**苍：** 我问的不是你的官职。我问的是你。

[凛看着苍。]

**苍：** 你接受这个命令吗？

[长久的沉默。]

**凛：** ……我没有不接受的权力。

**苍：** 那不是我问的。凛。

[这是苍第一次在公开场合直呼凛的名字而不是"团长"。队列中的人都感觉到了空气中某种细微的变化。]

**苍：** 你心里——接受吗？

[凛闭上眼。再睁开。]

**凛：** ……不接受。

[那两个字像是从某个很深很深的地方被挖出来的。带着泥土。带着锈蚀。带着压抑了太久后终于暴露在空气中的金属的涩味。]

**凛：** 但这不重要。

**苍：** 不。这很重要。

[苍转向队列。]

**苍：** 听到了吗。团长不接受。我也不接受。

[骚动。]

**苍：** 我不是在煽动叛乱。我只是——

[他深吸一口气。]

**苍：** 我跟随凛团长十二年。从骑士学院到灰谷隘口。我效忠的从来不是枢密院的文书。不是御印。不是番号。我效忠的是这个人。

[他指向凛。]

**苍：** 如果她走，我跟着走。管他什么第三营。

[沉默。然后——]

**圣骑士甲：** ……我也是。

**枪兵丙：** 我——我跟团长。

[声音零零散散地响起。不是齐声的、热血沸腾的呼应——而是一个一个、犹豫的、颤抖的、但最终确定的声音。]

[但不是所有人。]

[队列后方，几名士兵低着头，没有说话。其中一人——凛认出是灰谷之战的老兵——终于开口了。]

**步兵丁：** 团长……对不起。我家里还有老母亲。军饷断了她——

**凛：** 不用解释。

[她的声音温和得让苍微微一愣。]

**凛：** 你们做了正确的选择。去第三营。好好活着。照顾好家人。

[那名老兵的眼圈红了。他单膝跪地，重重叩首，然后站起来转身走向营门方向。]

[又有几人跟着离去。]

[最终，队列中剩下十四人。不到一半。]

[凛数了数。十四。]

*——三百变三十。三十变十四。*

*——这就是圣骑士团了。*

**凛：** ……谢谢你们。

[她说不出更多。她将团长纹章握在掌心，用力——很用力——然后松开。纹章上留下了她掌纹的印痕。]

**凛：** 苍。把纹章送去给传令使。告诉他——圣骑士团已奉令解散。人员自行并入各部。不需要他"协助交接"了。

**苍：** ……那纹章——

**凛：** 交出去。

[她将纹章放入苍的手中。苍的手指接触到银色金属的一瞬间，微微发颤。]

**苍：** 这是一百二十年——

**凛：** 我知道。

[她的目光从纹章上移开。不再看。]

**凛：** 去吧。

[苍攥紧纹章，转身走向营门。他的背影在帐篷间的阴影中渐渐消失。]

[凛站在原地。十三名骑士——加上苍十四人——在她身后沉默地等待。]

[风从枯井的方向吹来，带着地底深处干燥的、空洞的气息。]

#### 第三幕：烧家书

[深夜。帐篷内。]

[所有人都睡了。或者在装睡。营地安静得只剩风声和帐篷帆布的拍打声。]

[凛坐在桌前。面前仍然是那张空白的信纸。毛笔。磨好的墨。油灯已经换了一盏——苍不知从哪里找来的。]

[她拿起笔。蘸墨。笔尖悬在纸面上方一寸处。]

*——父亲大人、母亲大人安鉴：*

*——女儿在前线一切平安……*

[她写不下去。]

[笔尖的墨汁滴落在信纸上，洇开成一个不规则的黑色圆点。]

*——一切平安。*

*——平安。*

*——三百人死了二百多。圣骑士团被解散了。我效忠的女王是个傀儡。我杀了不知道多少人，其中有些人和我一样只是在保护平民。我手下的士兵有一半选择离开。*

*——"一切平安。"*

*——我怎么写得出这四个字？*

[她将笔放回砚台上。手在发抖。不是冷的。]

[她拿起信纸。看着那个黑色的墨点。]

*——父亲。您教我拿起第一把木剑的时候说："骑士的剑是用来保护弱者的。"*

*——母亲。您每次收到我的家书都会回信说："照顾好自己，等你回家。"*

*——我不知道该对你们说什么了。*

*——如果我告诉你们真相——你们会怎么想？你们的女儿，曾经最骄傲的圣骑士团团长，现在是一个信仰碎了一地的人。*

*——我不想让你们知道。*

*——但我也写不出"一切平安"。*

*——所以——*

[她走出帐篷。]

[营地中央的枯井旁有一堆将灭的炭火——白天煮粥用的。凛蹲下身，拨开灰烬，里面还有几颗暗红色的余烬在微微发光。]

[她将信纸——空白的、只有一滴墨点的信纸——放在余烬上。]

[纸张的边缘迅速变黄、卷曲，然后燃起一朵小小的火焰。橘红色的光在凛的脸上跳动，映出一双干涸的眼。]

[火焰吞噬了信纸。那滴墨点在火中膨胀了一下，像是要说什么，然后碎成灰烬。]

[凛看着火光熄灭。灰烬在夜风中散开，向枯井的方向飘去——有几片落入了井口，无声地坠入那个黑洞洞的深渊。]

**苍：** （从帐篷阴影中走出）……团长。

[凛没有惊讶。也许她知道苍一直在看着。]

**凛：** ……苍。

**苍：** 家书？

**凛：** ……算不上。上面什么都没写。

**苍：** ……

[苍在凛身旁蹲下。两人一起看着那堆渐渐暗淡的余烬。]

**苍：** 我也很久没给家里写信了。

**凛：** 你应该写。你母亲会担心。

**苍：** 嗯。明天写。

[沉默。风从枯井方向吹来。]

**苍：** 团长。

**凛：** ……嗯。

**苍：** 就算没有纹章。没有番号。没有千姬的诏令。

[他看着凛。月光照在他的脸上，那张年轻的、棱角分明的脸上有一种坚定得近乎固执的神情。]

**苍：** 你还是我的团长。

[凛没有说话。她低下头，看着自己的手——没有了纹章的手。空荡荡的。]

[很久之后。]

**凛：** ……嗯。

[一个字。很轻。]

[但在枯井营寂静的深夜里，那一个字比任何誓言都沉重。]

[余烬最后的光芒在两个人的脸上闪烁了一下，然后熄灭了。]

[只剩月光。]

> **获得奖励：** 350 EXP / 团长纹章（碎裂）×1 / 解锁凛档案第七页 / CG「深夜的火光」
>

>
> *CG「深夜的火光」——枯井营深夜。凛蹲在将灭的炭火旁，手中一张空白信纸正在燃烧。火焰的橘红色照亮了她的下半张脸——嘴唇紧抿，下颌的线条绷得像弓弦。背景是枯井的黑色剪影。苍站在远处帐篷的阴影中，只露出半张侧脸，目光望向凛。天空中一弯残月如钩。*

### 场景描写

三日后。东都防线西端·「残阳岭」。

这片丘陵得名于它独特的地质——岩层中含有大量氧化铁矿物，在夕阳照射下整片山坡呈现出一种近乎血红的暗橘色。当地人说这里是远古战场，渗入土地的鲜血化成了矿石。传说真假难辨，但站在岭上往西望去，那种颜色确实让人无法不联想到血。

凛的残部——十四人——被部署在残阳岭的西坡。枢密院的命令措辞冠冕堂皇："圣骑士团余部英勇善战，特命镇守东都防线最前沿阵地，为主力部署争取时间。"

苍把那份命令读了三遍。

第一遍，他以为自己看错了。第二遍，他确认没看错。第三遍，他把文书揉成一团扔进了火里。

十四个人。守一座山岭。面对暗潮主力先锋——至少五百人的推进部队。

这不是部署。这是送死。枢密院把凛的残部当作诱饵——用十四条命去吸引暗潮先锋的注意力，为东都主力的调防争取半日时间。

凛站在岭顶，夕阳从她背后照来，将她的影子投射到血红色的山坡上——那个影子被拉得极长极细，像一把插入大地的剑。她的铠甲已经没有了圣骑士团的百合纹章——上缴了。肩甲上空荡荡的，只剩拆除纹章后留下的两个小孔。

风从西面吹来，裹挟着远处暗潮军团行进时扬起的尘土。那片尘土像一堵缓缓推进的土黄色墙壁，吞噬着夕阳最后的光线。

凛没有说话。她只是看着那堵"墙"越来越近。

### 剧情对话

#### 战前

[凛的十四人在岭顶列阵。没有壕沟——没有时间挖。没有壁垒——没有材料。只有裸露的血红色岩石和几块勉强能当作掩体的巨石。]

**苍：** ……五百人。

[他的声音平静得反常。像是已经把愤怒燃烧殆尽之后剩下的灰烬。]

**苍：** 枢密院给了我们十四个人，去拦五百个。

**凛：** 嗯。

**苍：** 你知道这意味着什么。

**凛：** 我知道。

**苍：** 你还要执行？

[凛看了苍一眼。]

**凛：** 我不是在为枢密院执行命令。

**苍：** ……

**凛：** 残阳岭后面是东都的西门。西门后面是民居区。如果先锋部队突破这里，他们会在主力调防完成之前冲入城内。东都有二十万平民。

[苍沉默了。]

**凛：** 又是平民。又是守。又是拿命填。

[她的嘴角浮现一个极淡的、没有温度的笑。]

**凛：** 也许这就是我们的命。骑士团不在了。但骑士——

[她将手放在肩甲上，那个空荡荡的纹章位置。]

**凛：** 骑士还在。

**苍：** ……是。

[他握紧长枪。枪尖在夕阳中闪烁。]

**苍：** 团长。如果今天——

**凛：** 不要说。

**苍：** ……让我说完。

**凛：** 苍。

**苍：** 如果今天是最后一天。我想让你知道——十二年来，我从未后悔过追随你。

[凛没有回答。她转身面向西方——暗潮先锋的尘墙已经近到可以看到旗帜的轮廓了。]

**凛：** ……全员。列阵。

[十四个人在残阳岭顶排成一条线。没有壕沟。没有壁障。没有援军。只有血红色的岩石、西沉的残阳，和十四把指向前方的武器。]

**凛：** 拔剑。

[她拔出长剑。夕阳的光沿剑身流淌，像是一道液态的火焰。]

**凛：** 不为王冠。不为枢密院。不为那些坐在天鹅绒椅子上签署命令的人。

[她将剑举至面前。]

**凛：** 为身后的二十万人。只为他们。

**全体：** ……是！

### 战斗阶段

[暗潮先锋抵达残阳岭西坡脚下。五百余人的推进部队看到岭顶只有十四个人时，前排的士兵甚至笑了。]

[凛没有笑。]

[战斗开始。]

[第1回合。暗潮先锋从三个方向同时登坡。凛的十四人利用岭顶的巨石作为支点，以三人为一组分散防守。]

**凛：** 不要对冲！利用高地——让他们爬上来！上坡消耗他们的体力！

[圣骑士和枪兵在巨石间穿插，利用地形优势逐个击退攀上岭顶的暗潮士兵。但数量差距太大——每击退一波，下一波立刻涌上。]

[第2回合。暗潮弓箭手在坡下列阵，开始仰射。箭矢从低处射来，角度刁钻。]

**苍：** 弓箭手！低头——

[一名枪兵被箭矢射中肩膀，倒地。苍冲过去将他拖到巨石后方。]

**凛：** 不要管弓箭手！盯住近战！只要阵线不破——

[她一剑劈退一名爬上岭顶的暗潮骑兵，力道大得将对方连人带盾推下了山坡。]

[第3回合。暗潮投入精锐骑兵。他们从西坡最缓的一段直接策马冲锋——铁蹄踏在血红色的岩石上，火星四溅。]

**凛：** ——骑兵！正面来了！所有人集中——

[冲击巨大。凛挡在最前方，长剑格住第一匹战马的冲锋——但冲击力把她推出了三步远，靴底在岩石上拉出两道深深的擦痕。]

**凛：** ……！

[她咬牙挡住。但第二匹、第三匹紧随其后。防线出现裂口。]

[第4回合。暗潮先锋将·铁面亲自带队冲入裂口。铁面是一个戴着铁制面罩的高大武将，手持一柄巨大的链锤。链锤呼啸着砸向凛——]

**苍：** 团长——！！

[苍从侧面冲出，用长枪架住了链锤的第一击。金属碰撞的声音震得整片岭顶都在颤抖。]

[但链锤的第二击——回旋。链条绕过枪杆，锤头直直砸在苍的胸甲上。]

[凛听到了一种声音。不是金属碰撞。是骨骼断裂的声音。]

**苍：** ——！！

[苍被击飞。他的身体在空中旋转了半圈，重重砸在一块巨石上。胸甲凹陷。鲜血从嘴角涌出。长枪从手中脱落，滚到了凛的脚边——枪杆从中间断成了两截。]

**凛：** 苍——！！

[她冲到苍身边。苍靠在巨石上，呼吸急促而微弱。胸甲的凹陷触目惊心——至少断了三根肋骨。他的眼睛还睁着，但焦距已经开始涣散。]

**苍：** （声音极弱）……团长……别管我……阵线——

**凛：** 闭嘴！不要说话——

**苍：** 阵线……要破了……

[凛回头。果然——苍的缺位让右翼的防守出现了巨大的缺口。暗潮士兵正如洪水般从缺口涌入。]

**苍：** 团长……去。

**凛：** 苍——

**苍：** 去！！

[苍用尽最后的力气吼了一声。鲜血从他嘴角喷出，洒在血红色的岩石上——几乎分辨不出颜色。]

[凛的眼睛在那一瞬间——变了。]

[不是悲伤。不是愤怒。是某种更原始的、更深层的东西——像是一头被逼入绝路的兽，在最后一刻咬断了锁链。]

[「狂暴」状态触发——凛ATK+6、DEF-5。每回合损失5%HP。]

**凛：** ……啊。

[她站起身。捡起苍断裂的半截枪杆。左手持枪，右手持剑。]

**凛：** 啊啊啊啊啊——！！！

[她冲入暗潮的人群。]

[不是战斗。是屠杀。是暴风。是一个彻底断裂的人将所有碎片化作刀锋向外倾泻的瞬间。]

[第5回合。凛一人冲入敌阵。长剑横扫，枪杆突刺——双手武器同时攻击，没有任何防御动作。她被砍中了背、被刺中了大腿、被箭矢射中了左臂——但她甚至没有减速。]

**凛：** 让——开——！！

[圣光从她体内爆发——但不是以往温暖的金色。是灼白的、刺目的、几乎带有攻击性的白。像是太阳在燃烧殆尽前最后的闪耀。]

[暗潮士兵被圣光灼伤，本能地后退。凛趁势突入更深——一剑劈开一面盾牌，枪杆刺穿一副铠甲，转身又是一剑。]

[她的动作不再有任何圣骑士剑术的痕迹。没有架势，没有套路。只有本能。杀意。和某种近乎自毁的决绝。]

[第6回合。暗潮先锋将铁面再次出现在她面前。链锤呼啸而来——]

[凛没有躲。她丢弃了左手的枪杆，双手握剑，迎着链锤劈下——]

[剑与锤在空中碰撞。冲击波将两人同时震退。凛的虎口崩裂，血从指缝间渗出。但她再次冲上去。一剑。又一剑。又一剑。]

**铁面：** ……疯了。

[铁面退后一步。他透过铁面罩看着凛——看到的不是一个骑士，是一团人形的火焰。正在燃烧自己。]

[第7回合。凛的HP跌破15%。她的身体终于到达了极限。]

[「燃尽」特殊演出触发。]

[凛的视线模糊了。世界在她眼中变成一片混沌的红——残阳的红、岩石的红、鲜血的红、圣光灼烧后视网膜上残留的白。她的双腿在发抖。不——不是发抖。是肌肉已经在抽搐。每一步都像是踩在碎玻璃上。]

*——身体……到极限了。*

*——但还差一点。还差一点——*

*——苍。苍还在后面。他还活着。我必须——*

[她举起剑。手臂的肌肉已经在撕裂。圣光从铠甲的每一道裂缝中涌出，将她整个人笼罩在灼白的光芒中——远远看去像一颗在地上缓缓坠落的流星。]

**凛：** ——还没有……结束……！！

[她发动最后一次冲锋。不是向敌人。是穿过敌人——像一枚灼热的箭矢射穿一张纸。她从暗潮阵型的正中央穿过，沿途所有被圣光触及的敌兵都被弹开。铁面被她的剑擦中面罩——铁面罩上多了一道从左到右的深深裂痕。]

[她从敌阵的另一端冲出来。]

[然后跪倒。]

[剑插入地面支撑着她。膝盖砸在血红色的岩石上。圣光熄灭了——像是吹灭了最后一根蜡烛。]

[暗潮先锋被这次突袭彻底打乱了阵型。铁面摸了摸面罩上的裂痕，沉默了很久，然后下令后撤重整。]

[残阳岭——守住了。]

### 战后

[夕阳沉入地平线。残阳岭上的红色渐渐暗去，变成一种深沉的暗褐色。像是血液干涸后的颜色。]

[凛跪在地上。她试了三次才站起来——双腿一直在抖。铠甲上到处是裂口和刀痕，血从每一道缝隙中渗出，在银白色的甲片上画出纵横交错的暗红纹路。像是某种扭曲的纹章。]

[她踉跄着走向苍倒下的地方。]

[苍还在那里。靠在巨石上。有人——一名枪兵——已经给他做了简单的包扎。胸甲被拆掉了，白色的绷带缠在胸口，已经被血浸透成了暗红色。]

[苍还活着。呼吸微弱但稳定。眼睛闭着。]

[凛在苍身边跪下。她伸出手——那只满是血的手——轻轻碰了碰苍的额头。]

**凛：** （几乎无声的耳语）……还活着。

[她的手在苍额头上停留了一瞬。然后缩回。]

[她环顾四周。十四人——还剩九个能站着的。五个倒下了，其中三个有生命危险。血红色的岩石上到处是倒下的暗潮士兵和散落的武器。残阳最后的光在地平线下发出一道暗淡的余晖，像是一只即将闭合的眼睛。]

*——守住了。*

*——又一次。又是拿命填。*

*——苍差点死了。*

*——因为我。因为枢密院把我们当诱饵。因为——*

[她低头看着自己的手。血。到处都是血。指甲里塞满了干涸的血块。分不清是谁的。]

*——这双手。刚才杀了多少人？*

*——我不记得了。暴走的时候——什么都是红的。什么都看不清。只有"往前"。"砍"。"不要倒下"。*

*——那不是骑士。*

*——那是野兽。*

[她攥紧拳头。又松开。]

**枪兵甲：** 团长……苍副官的伤——需要尽快送到后方。东都城内有军医——

**凛：** ……嗯。

[她站起身。摇晃了一下。]

**凛：** 做担架。最快速度把重伤员送回东都。

**枪兵甲：** 团长您也——

**凛：** 我没事。

[她显然不是"没事"。她的左腿在走路时明显拖着步子，左臂的箭伤还插着折断的箭杆——她甚至没有拔。但她的声音仍然平稳。机械的那种平稳。]

[她走到苍身边，俯下身，将苍的断枪——断成两截的那柄长枪——从地上捡起来。]

[她把两截断枪并在一起。断面不平整，无法重新接合。]

[她还是把它们拢在一起，用自己的披风——已经不再有百合纹章的披风——裹住，系在背上。]

**凛：** ……走。

[残阳彻底沉没。岭上只剩一片黑暗。九个还能站立的人，扛着五个倒下的同伴，缓缓离开了残阳岭。]

[他们走后，岭顶只剩风声。血红色的岩石在黑暗中看不出颜色了——和所有其他的石头一样灰暗。]

[但如果天亮后有人经过这里，他们会看到岩石上的血迹。那些血迹在阳光下会变得和岩石的颜色融为一体——再也分辨不清哪些是石头本身的红，哪些是人的红。]

## Event 09 — 角色事件·肆「被截获的密信」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情（无战斗） |
| 情感基调 | 千姬的真相——秩序与人性的最终抉择 |

### 场景描写

东都·北城区·废弃的织坊。

凛将这里选作临时据点。织坊的主人在战争初期就逃往南方了，留下了满屋子断裂的织机和成捆的未染白布。白布堆在角落里被潮气浸透，散发着一种微甜的霉味。月光从屋顶破洞的瓦缝中漏下来，在白布上留下一块块惨淡的银色光斑——远远看去像是一地未盖好的裹尸布。

凛的残部——九个还能行动的人——就栖身在这些织机之间。他们不再是任何编制内的军队。没有番号。没有军饷。没有补给线。枢密院的文书上，圣骑士团已经不存在了。这九个人，连同病床上的苍，是一群没有名字的幽灵——游离在天城王朝的秩序之外，靠着从战场废墟中搜集的残粮和东都贫民区善心人施舍的剩饭度日。

凛坐在织坊二楼的窗口旁。窗户的木框已经腐朽，一推就碎。从这里可以看到东都北城区的屋顶——灰色的瓦片层层叠叠，在月光下像鱼鳞一样泛着冷光。更远处是枢密院的塔楼，顶端的鸢尾纹章灯火通明，是整个北城区唯一的光源。

她面前的桌上放着一封信。信封上的火漆已经被拆开——不是她拆的。火漆上印着枢密院内阁的三环纹章，级别比普通的鸢尾纹章更高。信纸的边角被鲜血染红了一小块。

这封信是一个小时前送到的。送信的人是一名枢密院的低级信使——他在北城区的巷子里被凛的人拦截。信使受了伤，但还活着。凛让人把他绑在楼下的织机上，嘴里塞了布。

她读了那封信。读了三遍。

现在她坐在窗口旁，手指轻轻搁在信纸边缘，目光穿过破碎的窗框，望着远处枢密院塔楼上那盏永不熄灭的灯。

### 剧情对话

#### 第一幕：密信

[凛的手指在信纸上缓缓滑过。月光刚好照亮了信上的字迹——工整的楷书，墨色浓重，是枢密院公文惯用的官方字体。]

*——"致枢密使阁下：御医局已完成第三阶段药剂调配。临床试验证实，连续服用七日后，受试者意志力完全消退，仅保留基本生理反应与简单语言复述能力。面部表情可通过外部指令模拟，足以应对朝仪所需。建议于下月初一千姬陛下例行祈福仪式后实施替换程序。届时对外宣称陛下圣体抱恙，闭宫静养三十日。三十日后'康复'出宫之千姬，将完全符合枢密院施政所需。此方案不可逆。恭请阁下裁夺。"*

[凛的手指停在"不可逆"三个字上。指腹微微用力，信纸在那个位置凹下去了一点。]

*——不可逆。*

*——他们要把千姬变成——*

*——不。千姬已经是傀儡了。但至少她还有意志。还有眼泪。还有那天晚上在月见亭对我说"对不起"时颤抖的声音。*

*——他们连这个都要拿走。*

*——连"痛苦"都要拿走。连"作为人的资格"都要拿走。*

*——把她变成一具会说话的、会盖章的、会在祈福仪式上微笑的尸体。*

[她将信纸折好。手指很稳。但指尖的温度是冷的——比夜风还冷。]

[楼下传来脚步声。枪兵甲走上来，手里端着一碗稀粥。]

**枪兵甲：** 团长，该吃点东西了。您从昨天到现在——

**凛：** 信使审过了？

**枪兵甲：** ……审了。他只是跑腿的，什么都不知道。只知道要把信送到枢密使在北城的私宅。

**凛：** 下月初一。祈福仪式。还有几天？

**枪兵甲：** 八天。

[凛沉默了很久。粥的热气在夜风中散开，变成一缕细白的烟。]

**凛：** ……放那儿吧。

[枪兵甲将粥放在桌上，犹豫了一下。]

**枪兵甲：** 团长……那封信上写的——

**凛：** 你看了？

**枪兵甲：** ……截获的时候看了。对不起。

[凛没有生气。她甚至没有转头。]

**凛：** 没关系。你们都应该知道。

[枪兵甲站在原地，张了张嘴，最终什么也没说。他转身走下楼梯，脚步声在腐朽的木板上发出沉闷的嘎吱声。]

[凛独自坐在窗口旁。月光将她的影子投在身后堆积的白布上——影子的轮廓模糊不清，像一个正在溶解的人形。]

*——八天。*

*——八天后，千姬就不再是千姬了。*

*——然后天城王朝就真的只剩一个空壳了。一面旗帜。一个名字。一具坐在御座上的、会微笑的尸体。*

*——而枢密院会用那具尸体的手，盖下更多的御印。签署更多的命令。征用更多的百姓。解散更多的——*

[她闭上眼。]

*——这和我有什么关系？*

*——我已经不是圣骑士团团长了。不是天城的军人了。我只是一个带着九个残兵躲在废弃织坊里的逃兵。*

*——千姬的事——不是我能管的。*

[她睁开眼。望着那碗正在变凉的粥。]

*——不是我能管的吗？*

*——还是我不敢管？*

#### 第二幕：病床边

[织坊一楼的东侧角落被隔出了一小块区域，用白布帘子围起来。帘子后面是一张用木板和织机零件拼凑的简易床铺。苍躺在上面。]

[残阳岭之战后已经过去了五天。苍的伤比预想的更重——不只是三根肋骨。军医说是五根。其中两根刺穿了胸腔内壁，差一寸就扎进肺部。他需要静养至少两个月。但他们没有两个月。他们甚至没有足够的草药。]

[苍的脸色苍白。不是凛那种冷白——是一种失血后特有的、带着灰调的白。绷带从胸口一直缠到腰际，渗出的血已经不再是鲜红色，而是变成了暗褐色的干痂。他的呼吸很浅，每一次吸气胸腔都会微微起伏，然后在某个角度停住——那是断骨的位置在限制他的呼吸幅度。]

[凛掀开白布帘子走进来。她手里拿着那封信。]

[苍睁开眼。]

**苍：** ……团长。

**凛：** 别动。

**苍：** 我又没要起来。

[凛在床边蹲下。她将信递到苍面前。苍侧过头，用还能动的右手接过，吃力地展开。月光不够亮，凛从怀里掏出一小截蜡烛——不知从哪里找来的——用火石点燃，举在信纸旁边。]

[苍读信。他的眼球从左到右移动，速度很慢——也许是因为疼痛影响了注意力。但当他读到某一行时，眼球的移动停住了。]

**苍：** ……"不可逆"。

**凛：** 嗯。

[苍将信放在胸口。呼吸变得更浅了——不是因为伤，是因为愤怒在压迫他的胸腔。]

**苍：** ……畜生。

[一个字。牙齿咬得咯吱作响。]

**苍：** 千姬……她知道吗？

**凛：** 不可能知道。她身边全是枢密院的人。她连自己盖了什么章都不清楚——你觉得她会知道自己即将被——

[她没有把那个词说出来。]

[沉默。蜡烛的火焰在两人之间微微摇晃。苍的影子投在白布帘子上，因为他平躺着，影子的形状像一座低矮的坟丘。]

**苍：** 团长。你来找我——不只是给我看信的。

**凛：** ……

**苍：** 你想去救千姬。

[凛没有回答。但她没有回答本身就是回答。]

**苍：** 八天。你只有八天。要潜入千鸟御所——那里现在是枢密院直属部队把守的。至少三百人的驻军。加上情报局的暗哨。你手上九个人。其中四个带伤。

**凛：** ……我知道。

**苍：** 就算你救出了千姬——然后呢？带着一个毫无自保能力的女王逃亡？整个天城都在枢密院的控制下——你往哪里跑？

**凛：** 我——

**苍：** 而且。

[苍艰难地转过头，直视凛的眼睛。蜡烛的火光在他的瞳孔中跳动，映出一种凛很少在苍脸上看到的表情——不是愤怒，不是焦虑，而是一种异常清醒的、近乎残忍的冷静。]

**苍：** 你救千姬——是为了什么？

**凛：** ……什么意思？

**苍：** 你是为了"秩序"？为了"王冠"？为了"圣骑士的誓言"？

[凛的嘴唇动了动。没有发出声音。]

**苍：** 你已经不信这些了。灰谷之后你就不信了。白河之后更不信了。月见亭之后——彻底碎了。

**凛：** ……

**苍：** 那你为什么还要去？

[凛低下头。蜡烛的光照不到她的脸。只有下颌的线条在明暗交界处微微颤抖。]

**凛：** ……因为她是一个人。

[声音很轻。轻到蜡烛的火焰都比它响。]

**凛：** 千姬……不是一个符号。不是一面旗帜。不是一枚御印。她是一个人。一个会说"对不起"的人。一个在手帕上写"愿为田间女子"的人。一个——

[她的声音断了一瞬。]

**凛：** 一个被困在笼子里的人。而现在他们连她"作为人"的最后一点东西都要拿走。

[苍看着凛。很久。]

[然后他笑了。嘴角轻轻上扬——那个动作牵动了胸口的伤，让他微微皱了皱眉。但他还是在笑。]

**苍：** ……终于。

**凛：** ……什么？

**苍：** 你终于说出来了。

[他艰难地抬起右手，指向凛。]

**苍：** 凛。你知道吗——你保护的从来不是王冠。

[凛怔住。]

**苍：** 从来不是。从灰谷隘口开始——你守的不是关隘，是身后的三百个孩子。白河平原——你守的不是防线，是撤离的一千二百个人。残阳岭——你守的不是东都西门，是城里的二十万平民。

[他的声音越来越低。说话在消耗他有限的体力。但每一个字都清晰得像刻在石头上。]

**苍：** 你保护的从来不是王冠。是王冠下面的人。

[凛的呼吸停了一拍。]

[蜡烛的火焰在那一瞬间突然安静下来——不摇了。仿佛连空气都屏住了呼吸。]

*——王冠下面的人。*

*——是这样吗。*

*——灰谷的那三百个孩子。白河的那一千二百个人。残阳岭后面的二十万平民。马车里那个小女孩的眼睛。*

*——不是为了王冠。不是为了誓言。不是为了秩序。*

*——是为了人。*

*——一直都是。*

**苍：** 所以去。

[他闭上眼。呼吸变得更浅了。说这些话已经耗尽了他大半的精力。]

**苍：** 不是因为千姬是女王。是因为她是一个需要被保护的人。而你——

[他的嘴角又浮现了那个微弱的笑。]

**苍：** 你是凛·艾斯特雷亚。保护人，是你的本能。

[凛低着头。蜡烛的光在她睫毛上投下细碎的阴影。]

[很久之后。她伸出手，轻轻将苍握着信纸的手放回被褥上。]

**凛：** ……你好好休息。

**苍：** 嗯。

**凛：** 别死。

**苍：** ……这算命令？

**凛：** 算。

[苍轻轻笑了一声。那声笑牵动了胸口的伤，他的眉头皱了一下，但很快松开。]

**苍：** 遵命。团长。

[凛站起身。她将蜡烛留在苍的枕边。她拿走了信。]

[她掀开白布帘子走出去。帘子在她身后缓缓落下，遮住了苍苍白的脸和枕边那一小团跳动的烛光。]

#### 第三幕：决定

[凛走上二楼。织坊的残部——九个人中还醒着的五个——散布在织机之间。有人在擦拭武器，有人在包扎伤口，有人只是靠着墙壁发呆。听到凛的脚步声，所有人都抬起了头。]

[凛站在楼梯口。月光从屋顶的破洞中落下，恰好照亮了她的上半身。她的铠甲在月光下失去了银白色的光泽——被太多战斗磨损成了暗灰色。但她的眼睛是亮的。和之前不一样的亮。]

[不是灰谷时那种凛然正气的光。不是残阳岭前那种焦躁灼热的光。是一种更安静的、更深处的、像是地底暗河终于找到出口后涌出的光。]

**凛：** 都别睡了。

[靠墙发呆的两人直起身。擦武器的停下手。包扎伤口的系好了最后一圈绷带。]

**凛：** 我要做一件事。

[她展开那封信。月光照在枢密院的官方字体上。]

**凛：** 枢密院要对千姬动手。八天后。如果我们不做点什么——天城就真的没了。不是被暗潮打没的。是被自己人从里面掏空的。

[沉默。织机的影子在月光中像一排沉默的巨兽。]

**枪兵甲：** ……团长。您要我们做什么？

**凛：** 我不命令你们做任何事。

[她环视所有人。]

**凛：** 从骑士团解散那天起，你们跟着我，没有军饷，没有编制，没有任何人承认你们的存在。你们已经做得够多了。

[她将信折好，放入胸甲内侧。]

**凛：** 但我要去。

**枪兵甲：** ……

**凛：** 不是为了王冠。不是为了什么骑士的誓言。

[她顿了一下。苍的话在她脑中回响——"你保护的从来不是王冠。是王冠下面的人。"]

**凛：** 有一个人需要被救。就这样。

[沉默持续了大约十秒。然后枪兵甲站起身。他的动作有些僵硬——左腿的伤还没有完全好——但他站得笔直。]

**枪兵甲：** 团长。从灰谷到现在，我数过了，我已经死过三次了。每次都是您把我拽回来的。

[他将枪杆在地上顿了一下。]

**枪兵甲：** 第四次也无所谓。

[一个接一个。不是同时——是一个接一个地，犹豫着，但最终确定地，站起来。]

[最后，所有人都站着了。]

[凛看着他们。她的嘴唇动了一下——像是想说"谢谢"。但她没有说出来。她只是微微点了点头。]

**凛：** 明天开始侦察千鸟御所的防务部署。八天内制定潜入方案。

[她的声音恢复了那种清晰的、没有多余情绪的指挥语调。但所有人都听出了不同——那种语调下面有什么东西变了。不是为了别人的命令而战的机械。是为了自己的选择而动的决意。]

**凛：** 休息。明天天亮出发。

[她转身走向窗口。将那碗已经彻底凉透的粥端起来，一口气喝完了。]

[月光从破窗照在她的脸上。她望着远处枢密院塔楼上的灯火。]

*——八天。*

*——我会来的。*

*——千姬。等着我。*

[她放下碗。碗底还残留着粥的痕迹——在月光下像一个微小的、白色的圆。]

[像月亮。]

[她没有再看。转身。走向那张空荡荡的、没有信纸的桌子。从怀里取出密信，铺平，压在桌角。然后躺在桌旁的木板上，闭上眼睛。]

[三分钟后，她的呼吸变得均匀。]

[这是残阳岭之战以来，她第一次真正睡着了。]

### 场景描写

东都以北四十里。北方要塞——「落�的星堡」。

这座要塞建在一座孤立的台地上，三面悬崖，只有南面一条蜿蜒的山道可以通行。台地顶部被削平，筑起三重城墙，从外到内依次收窄，俯瞰如同嵌套的三个石匣。天城建筑师将这里设计成一座不可攻破的铁壁——至少在两年前它归属天城的时候，没有人想过有一天需要从外面往里打。

暗潮在半年前占领了落星堡。他们没有做太多改动——城墙上的天城纹章还在。白百合与交叉剑的浮雕被风雨侵蚀了一些，但轮廓仍然清晰。暗潮只是在纹章旁边竖了自己的黑色旗帜。两面旗帜在寒风中并排飘动——一黑一白，像是某种荒诞的和解。

凛站在山道起点的一块巨石后面。天刚蒙蒙亮——东方天际是一种混沌的青灰色，像是墨汁滴入清水后尚未散开的那一瞬间。要塞的轮廓在这种光线下显得格外沉重，三重城墙如同三道凝固的巨浪，压在台地之上。

她的身边不只有九个人了。

过去七天里，凛做了她从未做过的事——联络东都城外的散兵、逃兵、甚至被枢密院遣散的其他部队残余。一个、两个、五个、十个……零零散散地汇聚到织坊。有些人是冲着"圣骑士团团长"的名号来的，有些人只是走投无路需要一个容身之处，有些人听说"有人要对枢密院动手"就二话不说带着武器赶来。

现在她身后有五十三个人。

不是军队。是一群没有旗帜的人。

凛审视着落星堡的三重城墙。她的目光从第一道墙缓缓移向第二道，再移向第三道。每一道墙她都认识——她在骑士学院的战术课上研究过落星堡的防御体系。当时教官说："这座要塞只能从内部瓦解，正面强攻是自杀。"

现在她要正面强攻。

*——自杀般的强攻。用五十三个人攻一座三重城墙的要塞。*

## Event 11 — 角色事件·伍「废墟中的誓言石」

| 项目 | 数据 |
|------|------|

### 场景描写

落星堡内城。攻城战结束后的第二日。

落星堡的内城比外城更古老。建筑风格可以追溯到天城建国初期——粗犷的石块堆砌，没有后来那些精巧的雕饰和琉璃装饰。墙壁上覆满了苔藓和地衣，石缝间长出了倔强的蕨类植物，根系将石块一点一点撬开。在某些角落，建筑已经和自然融为一体——分不清是人造的废墟还是天然的岩壁。

战斗留下的痕迹到处可见。城墙上的弩箭射孔周围被烟熏黑了——那是火攻留下的。地面上散落着碎石、断箭和暗潮士兵遗弃的杂物。空气中弥漫着石粉和铁锈的气味，以及某种更古老的、从石缝深处渗出的潮湿的泥土味。

凛独自一人走在内城的废墟中。

她告诉所有人自己要"巡查防务"。但实际上她在找什么——她自己也说不清楚。也许什么都不找。也许只是需要一个人待一会儿。残阳岭以来，她几乎没有独处的时间。身边永远有人——需要下命令的人、需要包扎的人、需要安慰的人。她已经忘记了一个人走路是什么感觉。

她的靴子踩在碎石上，发出细碎的声响。左手的烫伤缠着绷带。右肩被弩箭擦过的伤口还在隐隐作痛。但她的步伐是轻的——轻到连她自己都觉得不真实。

她走到内城最深处——一个半坍塌的拱形建筑前。这里曾经是落星堡的礼拜堂。天城王朝信奉的"圣光教廷"在每一座重要要塞中都设有礼拜堂。但这座——

这座格外古老。

拱顶已经塌了大半，只剩骨架般的石肋条弯曲着指向天空。阳光从缺口中倾泻而入，在地面上投下一片片不规则的光斑。墙壁上的壁画已经模糊不清——只能依稀辨认出人形的轮廓和某种光芒四射的图案。地面上堆满了坍塌的石块和碎裂的雕像残片。

凛的目光被什么东西吸引了。

在礼拜堂最深处——原本祭坛所在的位置——有一块石头。

不是碎石。不是建材。是一块被人刻意放置的、打磨过的方形石碑。约两尺高，一尺半宽。表面被苔藓覆盖了大半，但在苔藓的缝隙间——有字。

### 剧情对话

#### 第二幕：誓言石

[凛的手指沿着石碑表面缓缓移动，一片一片地剥去苔藓。每剥去一片，就有新的字迹从下面浮现。]

[石碑上的字是凿刻的——不是后来天城官方那种工整的楷书，而是一种更古朴的、带有明显手刻痕迹的字体。笔画的深浅不一，有些地方用力过猛，凿痕溢出了字的边界。但每一个字都极大——比正常书写大三倍以上——仿佛刻字的人希望它们能被从很远的地方看到。]

[凛剥去了所有苔藓。]

[完整的碑文显露出来。]

[碑头刻着四个字——"圣骑士誓"。下方是正文。]

*——"吾等执剑之人，今于此石前立誓——"*

*——"守护苍生。"*

*——"不分贵贱。不论敌友。凡呼吸者，皆在吾等剑下之庇护。"*

*——"吾等不效忠王冠。王冠亦为凡人所铸。"*

*——"吾等不效忠权柄。权柄亦为凡人所握。"*

*——"吾等只效忠一事——苍生之安宁。"*

*——"若王冠守护苍生，则吾等为王冠之盾。"*

*——"若王冠背离苍生，则吾等为苍生之剑。"*

*——"此誓不刻于纸帛，纸帛可焚。"*

*——"此誓不刻于金石，金石可碎。"*

*——"此誓刻于吾心。心在，誓在。"*

*——"落星堡初代圣骑士团。建元七年。"*

[凛跪在石碑前。]

[她读了一遍。]

[又读了一遍。]

[又读了一遍。]

[阳光从拱顶的缺口中落下，恰好照在石碑上。字迹在光中投下微小的阴影——每一道凿痕都变得清晰可见，仿佛刻字的人就在一百二十年前的此刻放下了凿子。]

**凛：** ……"守护苍生。"

[她的声音在废墟中回荡。]

**凛：** "不效忠王冠。"

[回声从坍塌的拱顶碎片间反射回来，变成一个轻柔的、近乎叹息的余韵。]

**凛：** "若王冠背离苍生，则吾等为苍生之剑。"

[她低下头。双手撑在膝盖上。长发从肩侧垂下，遮住了她的脸。]

*——初代誓言。*

*——和我背诵的版本不一样。*

*——我背诵的版本是——"吾之剑，为王冠而鸣；吾之盾，为黎民而立。"*

*——什么时候改的？谁改的？为什么改的？*

*——"为王冠而鸣"——初代誓言里根本没有这句话。初代誓言明确地、毫不含糊地说——"不效忠王冠"。*

*——一百二十年间的某个时刻，有人把"守护苍生"改成了"效忠王冠"。把骑士从"苍生的守护者"变成了"王冠的工具"。*

*——然后这个被篡改的版本——被我在十四岁的时候刻进了心里。*

*——我用了十二年去信仰一个被篡改的誓言。*

[她的手指触碰石碑上"苍生"二字。指腹感受着一百二十年前的凿痕——粗糙的、用力的、充满信念的凿痕。]

*——但苍说得对。*

*——从灰谷到残阳岭——我保护的从来不是王冠。*

*——是苍生。*

*——一直都是苍生。*

*——我在不知不觉中——做了初代骑士做的事。尽管我背诵的是被篡改的誓言，但我的身体——我的本能——记得最初的那个版本。*

*——也许誓言本身从来不需要石头来记住。它刻在石头上可以被苔藓覆盖。刻在城门上可以被我一剑砸碎。*

*——但刻在心里——*

[她深吸一口气。]

*——心在，誓在。*

#### 第三幕：新誓言

[凛从腰间取出断剑。]

[她跪在誓言石前。右手握剑。剑刃上灰谷、白河、残阳岭、落星堡——无数次战斗留下的缺口在阳光中像一排参差不齐的牙齿。]

[她将剑尖抵在誓言石的底部——初代骑士的碑文下方，还有一小块空白的石面。苔藓被她刚才剥去了，露出的石头表面干净而光滑。]

[她开始刻字。]

[断剑的剑尖不适合凿刻——刃口太薄，角度不对。每刻一笔都需要极大的力气和极高的精准度。金属与石头摩擦发出刺耳的吱嘎声，在废墟的拱顶间回荡。石屑飞溅，有些落在她的手背上。左手的烫伤在握剑柄时传来一阵阵灼痛——但她没有换手。]

[她刻得很慢。]

[一笔。一笔。一笔。]

[阳光从头顶的拱顶缺口缓缓移动。她开始刻字时光柱照在石碑的右侧，等她刻完最后一笔时，光柱已经移到了左侧。至少过了一个时辰。]

[她放下剑。后退半步。]

[在初代骑士的誓言下方，新的字迹出现了。]

[没有初代那样的大字。没有工整的笔画。断剑凿出的字歪歪扭扭的，深浅不一，有些地方剑刃打滑留下了多余的刮痕。但每一个字都——用力极深。]

*——"凛·艾斯特雷亚。圣骑士团末代团长。"*

*——"今于先辈誓言之下，立新誓——"*

*——"旧誓已碎。非因背叛，而因觉醒。"*

*——"吾之剑不为王冠而鸣。王冠会腐朽。"*

*——"吾之盾不为秩序而立。秩序会扭曲。"*

*——"吾只为看得见的人、听得到的声音、触得到的痛苦而战。"*

*——"此誓不求天地为证。"*

*——"此誓只求此心不欺。"*

[凛跪在石碑前。双手覆在剑柄上。剑尖触地。姿势和十四岁在骑士学院第一次宣誓时一模一样——右拳抵胸，微微低头。]

[但意义完全不同了。]

[十四岁的凛跪在别人铸造的誓言前，将它刻入心中。]

[现在的凛跪在自己的誓言前——它是从心中生长出来、然后刻在石头上的。]

[阳光从拱顶的缺口中落下，照在石碑上。初代骑士的古老文字和凛刚刚刻下的新誓言并列在一起——一百二十年的间隔，两种笔迹，同一块石头。]

[凛站起身。]

[她没有急着离开。她站在那里，最后看了一遍石碑上所有的字——从初代骑士的"守护苍生"到自己的"此心不欺"。]

**凛：** ……谢谢。

[她对石碑说。]

[不知道在对初代骑士说，还是对十四岁的自己说，还是对苍说，还是对那个马车里的小女孩说。]

[也许都是。]

[她将断剑收回鞘中。转身走出礼拜堂的废墟。]

[阳光在她身后。她的影子投在前方的碎石地面上——不再是那个"正在溶解的人形"。是一个清晰的、完整的轮廓。]

[她走得很快。步伐坚定。铠甲碰撞的声音在废墟中回响——铿锵。]

[回到营地时，枪兵甲正在分配口粮。看到凛，他抬起头。]

**枪兵甲：** 团长，防务巡查——

**凛：** 嗯。没有问题。

[她从枪兵甲手中接过一块干粮，咬了一口。硬得硌牙。但她嚼得很认真。]

**枪兵甲：** ……团长，您看起来——

**凛：** 嗯？

**枪兵甲：** 不一样了。

[凛停下咀嚼。看了枪兵甲一眼。]

**凛：** ……也许吧。

[她没有解释。将剩下的干粮塞回口袋，走向苍的临时病房方向。]

[走到白布帘子前，她停下脚步。从胸甲内侧取出一块东西——不是密信，不是旧旗。是一块碎石。昨天从誓刻门的废墟中捡的那块，上面残留着半个"黎"字。]

[她将碎石翻过来。在背面——用断剑的剑尖轻轻地、几乎是温柔地——刻了两个字。]

[苍。生。]

[然后她将碎石放在帘子外的地上，苍一伸手就能够到的位置。]

[她没有掀开帘子。转身离开。]

[帘子内传来苍轻微的咳嗽声——他醒着。也许他听到了凛的脚步声。也许他会在某个时候伸手摸到那块碎石。也许他会翻过来看到背面的字。]

[也许他已经知道了。]

### 场景描写

落星堡。占领后第三日。黄昏。

暗潮的反攻比预想的更快。

凛原本以为至少有五天的窗口期——暗潮守将黒的残部需要时间后撤、报告、等待主力调遣。但她低估了暗潮对北方通道的重视程度。黒撤出落星堡不到两天，暗潮主力先锋就从北面压了过来。

旗帜铺满了北面的整个视野。

从落星堡的北墙望去，暗潮大军如同一片从地平线上涌来的黑色潮水——这不是比喻。那些旗帜、铠甲、战马、攻城器械汇聚在一起，在黄昏的光线下确实呈现出一种液态的、流动的黑。他们沿着北面的山道向落星堡推进，同时分出两支侧翼从东西两面迂回——试图将整座台地包围。

凛站在南门城墙上。

南门是落星堡唯一朝向东都方向的出口。如果南门被攻破，她的退路就彻底断了——五十三人（减去攻城战中的伤亡，还剩四十七人能战）将被困死在台地上。

但她更在意的不是退路。

南门后面，沿着蜿蜒山道下山再走二十里，是三个村庄。战前有人口约两千。战争开始后村民们大部分逃散了，但根据侦察，仍有三四百人滞留——老人、病人、来不及走的人。

*——又是这样。*

*——身后又有人。*

*——但这一次——*

[她低头看了看自己的手。左手烫伤的绷带已经换过，渗出的液体不再是血色，而是正在愈合的淡黄。右手握着断剑，剑柄被汗水和血浸润了无数次，裹着的皮革已经磨得发亮。]

*——是因为身后有三四百个人。他们需要时间。而我站在这里。*

*——就这么简单。*

### 战斗阶段

[暗潮大军从北面和两翼发起总攻。凛的部队集中防守南门。]

### 场景描写

落星堡。占领后第五日。深夜。

风停了。

自从凛抵达落星堡以来，北面的山风就没有停过——日夜不休地灌入城墙的缝隙，吹得旗帜猎猎、铠甲嗡嗡。士兵们早已习惯了在风声中入睡。但今夜，风忽然停了。那种突如其来的寂静反而令人不安——像是战场上箭雨停止的那一瞬间，你知道不是结束，只是敌人在换箭壶。

凛坐在南门城楼的墙垛上。双腿悬在城墙外侧，脚下是三十尺高的断崖。月光将台地下方的树林涂成一片银灰色的海洋。远处，暗潮后撤后留下的营火已经彻底熄灭——只剩下几缕极淡的烟痕，在无风的夜空中笔直地升起，像是大地写给天空的遗书。

她的铠甲已经不成样子了。灰谷、白河、废墟、残阳岭、落星堡——每一战都在上面留了痕迹。银白色的甲面被磕碰得斑驳不堪，好几处凹陷深到能插进一根手指。左肩的碎裂处用铁丝和皮绳勉强固定，右腿甲干脆换成了从暗潮士兵身上扒下来的黑色护胫——尺寸不合，走路时会磕碰膝甲发出叮当声。

只有胸口的位置还算完整。那里挂着叶子纹章——粗糙的、银线缠绕的、她亲手做的纹章。月光下看，银线已经暗淡了，但形状还在。

她的伤没有好。

不是没治。是没时间治。落星堡没有军医——攻城战死了两个，防守战又伤了一个。剩下的药物优先给重伤员。凛的伤不算最重——至少她还能站、能走、能挥剑。所以她把所有药物都让了出去。

左手的烫伤已经开始溃烂。绷带渗出的液体从淡黄变成了灰绿色，散发着一股令人牙酸的腥甜味。右肋被暗潮骑兵的马蹄踢中的位置，深呼吸时会传来一阵刺穿般的锐痛——也许是骨裂。额角灰谷之战留下的旧伤反复开裂，已经用同一块绷带缠了五次。

但她的脊背是直的。哪怕独自一人坐在无人看见的城墙上，脊背也是直的。

### 剧情对话

#### 第一幕：千姬的下落

[凛坐在墙垛上，双手放在膝盖上。她在等什么——也许什么也没等。也许只是睡不着。也许是不敢睡。每次闭眼，都会看到那些画面——马车里的铁链、傀儡女王空洞的眼睛、烧毁的家书、誓言石上的古老文字。]

[脚步声从城楼的石阶上传来。不是苍的步伐——苍的伤还没好，不可能这么快走石阶。是枪兵甲。]

**枪兵甲：** 团长。

**凛：** ……嗯。

[她没有转身。]

**枪兵甲：** 有消息。从王都方向来的。

[凛的背微微一僵。她转过身。月光照在枪兵甲的脸上——他的表情不对。不是战报时的紧张，不是好消息时的欣慰。是一种更复杂的东西——像是不知道该不该把手里的东西递出去的犹豫。]

**凛：** 什么消息。

**枪兵甲：** 散兵丁在南面山道截获了一个枢密院的传令兵。传令兵说……是关于千姬陛下的。

[凛从墙垛上跳下来。铠甲碰撞的声音在寂静的城楼上格外刺耳。]

**凛：** 说。

**枪兵甲：** ……团长，也许您应该——

**凛：** 说。

[枪兵甲吞咽了一下。]

**枪兵甲：** 传令兵说——枢密院三日前发布了一道密令。千姬陛下被……转移至东都地下的玉笼阁。

**凛：** 玉笼阁。

[她知道那个地方。天城王朝囚禁政治犯的秘密监牢。在地下三层。没有窗户。没有阳光。进去的人——很少出来。]

**枪兵甲：** 不只是……不只是囚禁。

[他深吸一口气。]

**枪兵甲：** 传令兵说……枢密院的御医对千姬使用了「鸦片露」。

[空气凝固。]

[鸦片露。天城王朝禁药中最残忍的一种。不是杀人——是灭魂。长期服用后，人的意识会被一点一点溶解，直到只剩下一具能呼吸、能进食、但不会思考、不会说话、不会记得任何事情的躯壳。]

[枢密院用它来处理那些不能杀、又不能留的人。不能杀——因为千姬的血统是王朝合法性的根基。不能留——因为清醒的千姬随时可能成为反枢密院势力的旗帜。]

[所以他们选择了第三条路。]

[让千姬活着。但不再是千姬。]

**枪兵甲：** 传令兵说……已经用了十天以上。千姬陛下现在——

[他说不下去了。]

**凛：** ……现在怎样。

**枪兵甲：** 已经……不认识任何人了。

[月光照在凛的脸上。]

[她的表情没有变化。]

[完全没有变化。]

[和白河平原读枢密院密信时一模一样——面无表情。]

[但枪兵甲注意到她的手。她的手在发抖。不是微微颤抖——是那种从手指根部开始、沿着手腕蔓延到前臂的、压抑不住的剧烈震颤。铠甲的关节处发出细碎的金属碰撞声——叮、叮、叮——像是某种倒计时的声响。]

**凛：** ……你下去吧。

**枪兵甲：** 团长——

**凛：** 下去。

[枪兵甲张了张嘴。最终什么也没说。转身走下石阶。脚步声渐远。]

[凛独自站在城楼上。]

*——千姬。*

*——那个在就任仪式上对我微笑的人。*

*——那个把御印盖在我的任命书上、对我说"天城的未来就交给你了"的人。*

*——那个——被我发现是傀儡之后，我仍然在心底某个角落抱有一丝希望的人。*

*——希望她还是她。希望她也在挣扎。希望总有一天我能站在她面前，问她一句——你知道吗？你知道你的名字盖在了什么样的文书上吗？*

*——那一丝希望。*

*——现在——*

[她低下头。月光在铠甲上流淌。她的手还在抖。]

*——现在连问的机会都没有了。*

*——她不认识任何人了。*

*——她不认识我了。*

*——那个曾经对我微笑的人——已经不存在了。*

*——枢密院没有杀她。他们做了比杀更残忍的事。他们把她变成了——*

*——一个活着的墓碑。*

[她缓缓蹲下。双手抱住膝盖。铠甲的关节发出吱嘎声。]

[她没有哭。凛·艾斯特雷亚从不哭。十四岁入骑士学院时父亲去世她没有哭。灰谷隘口战友倒下她没有哭。白河平原得知援军不会来她没有哭。马车里看到铁链锁着的孩子她没有哭。]

[她只是蹲在那里。像一座被掏空了地基的建筑，还维持着外墙的形状，但里面已经是空的。]

[很久。]

[也许一刻钟。也许一个时辰。月亮移动了一段距离。城楼上的影子改变了角度。]

[然后——]

[城楼石阶上传来新的脚步声。]

[不是枪兵甲的粗重步伐。不是苍的一瘸一拐。]

[是一种更轻的、更急促的、带着某种不顾一切气息的脚步。]

#### 第二幕：雪乃

[凛缓缓抬起头。]

[石阶口出现了一个人影。月光从侧面打过来，将那个人影的轮廓勾勒得格外清晰——身量纤细，一头长发在月光下呈现出近乎银白的淡色。她穿着旅行用的斗篷，布满灰尘和泥点，斗篷的下摆撕裂了好几处——一看就是长途跋涉、日夜兼程赶来的。]

[她的左手提着一个磨损严重的药箱。右手扶着石阶的墙壁——不是因为疲倦，而是因为太急了，跑上石阶时差点绊倒。]

[她站在石阶口。喘着气。大口大口地喘着气。]

[月光照亮了她的脸。]

**雪乃：** ……凛。

[凛怔住了。]

[雪乃。天城王朝第一药师。曾经是圣骑士团的随军医官——凛的随军医官。骑士团被解散后，她是少数没有被枢密院追捕的人——因为她的医术太重要了，枢密院需要她。但三个月前凛率残部出走后，雪乃也从王都消失了。没有人知道她去了哪里。]

[现在她站在落星堡的城楼上。浑身尘土。斗篷撕裂。药箱磨损。]

[她找过来了。]

**凛：** ……你怎么——

**雪乃：** 你受伤了。

[不是问句。是陈述。雪乃的目光快速扫过凛的全身——从溃烂的左手绷带到渗血的额角，从凹陷的胸甲到尺寸不合的右腿甲。药师的眼睛比任何人都敏锐——她在三秒内读完了凛身上所有伤痕的故事。]

**雪乃：** 左手烫伤溃烂——至少三天没换药了。右肋骨裂——你还在穿重甲负重。额角裂伤反复感染。右肩关节磨损——你用了多少次不正确姿势挥剑？还有——

[她走近一步。伸手按住凛的左肩——轻轻的，但凛下意识缩了一下。]

**雪乃：** 这里。锁骨有旧伤。灰谷留下的？

**凛：** ……你是怎么找到这里的。

**雪乃：** 我花了两个月跟着你的路线走。灰谷——白河——官道——残阳岭——落星堡。你每到一个地方都留下一堆伤兵——我沿着伤兵治过来的。

[她将药箱放在地上。打开。药箱里的瓶瓶罐罐排列得整整齐齐——和她此刻狼狈的外表形成鲜明对比。药是她的命。无论多狼狈，药箱永远是干净整齐的。]

**雪乃：** 坐下。

**凛：** ……雪乃，我——

**雪乃：** 坐下。

[语气和凛命令部下时一模一样——不容拒绝。凛认识雪乃八年了。八年来她只在两种情况下用这种语气：一是病人拒绝治疗时，二是凛拒绝治疗时。大多数时候，这两种情况是同一种情况。]

[凛坐下。靠在墙垛旁。]

[雪乃蹲在她面前。先从左手开始——她轻轻解开已经发硬的绷带。绷带粘在溃烂的皮肤上，扯动时凛的手指微微蜷缩了一下，但没有出声。]

**雪乃：** 三天？不对——五天没换药了。你不要命了？

**凛：** 药不够。优先给重伤员。

**雪乃：** 你自己不是重伤员？

**凛：** 我还能战斗。

**雪乃：** 再过两天你就不能了。这只手如果继续溃烂，最好的结果是截掉三根手指。

[凛沉默。]

[雪乃从药箱中取出一个墨绿色的小瓷瓶。拔开塞子，一股清冽的草药气息弥漫开来——和腐烂伤口的腥甜味形成刺鼻的对冲。她将药液滴在干净的棉布上，一点一点地清洗溃烂的创面。]

[凛的手指在棉布触碰时绷紧了——药液接触裸露的肉质时的灼痛让她的呼吸急促了一瞬。但只有一瞬。]

**雪乃：** 忍着。

**凛：** ……嗯。

[雪乃的手很稳。比凛挥剑的手还稳。她一层一层地清洗、上药、重新包扎。新绷带缠得紧而均匀——不像凛自己缠的那种皱巴巴的将就。]

[左手处理完之后是额角。雪乃将凛额前散落的头发拨开，检查反复开裂的伤口。]

**雪乃：** 这道伤缝合过？

**凛：** 自己缝的。

**雪乃：** ……用什么缝的？

**凛：** 铠甲修补用的粗线。

[雪乃闭了一下眼。像是在克制什么情绪。]

**雪乃：** 拆了重缝。你用那种线——伤口愈合后会留下缝线反应，整块皮肤都会坏死。

[她拆线。重新缝合。用的是药箱里的蚕丝线——细得几乎看不见，但韧性极强。凛感觉到针尖刺入皮肤的微痛，像蚊虫叮咬。和粗线缝合时那种撕裂般的痛完全不同。]

[然后是右肋。雪乃让凛解开侧面的甲扣，将手探入铠甲内侧触诊。]

**雪乃：** 第七肋骨裂。没有错位——但如果再挨一下重击，就会断。断了之后如果刺入肺——

**凛：** 我知道。

**雪乃：** 你知道还穿着重甲满战场跑？

**凛：** 不穿重甲怎么挡暗潮的链锤。

[雪乃没有回答。她从药箱中取出一卷宽绷带，在凛的肋骨处缠了三层——固定。然后在绷带外侧涂了一层硬化药膏，干燥后会变成类似石膏的硬壳。不能完全替代铠甲，但至少能分散冲击力。]

[全部处理完之后，雪乃将药箱合上。坐在凛身旁。]

[两人并排坐在城楼上。月光铺满整个台地。远处的树林在无风的夜里静止不动，像一幅画。]

[沉默。]

**雪乃：** ……千姬的事，我在路上听说了。

[凛的身体微微一僵。]

**雪乃：** 我经过东都外围的时候，有个逃出来的御医学徒告诉我的。鸦片露……十天以上……

[她的声音低了下去。]

**雪乃：** 凛。十天的鸦片露——以我所知——是不可逆的。

[不可逆。]

[凛已经猜到了。但从雪乃——天城第一药师——口中听到这三个字，最后那根稻草般细的希望也断了。]

**凛：** ……嗯。

[只有这一个字。]

**雪乃：** 你一直想去救她对吗？

**凛：** ……嗯。

**雪乃：** 你从灰谷开始——一路打到这里——心里一直想着，等打到王都，就能……

**凛：** ……嗯。

[三个"嗯"。每一个都比上一个更短、更轻、更接近于无声。]

**雪乃：** 凛。

[雪乃转过身。她面朝凛。月光从她背后打过来，凛看不清她的表情——但她看清了雪乃的手。那只手伸过来，覆在凛的右手上。温暖的。带着药草气息的。指尖有常年研磨药材留下的薄茧。]

**雪乃：** 你不需要再撑了。

[凛没有说话。]

**雪乃：** 你已经不是圣骑士团团长了。你已经不欠天城任何东西了。千姬……千姬也回不来了。那些曾经绑住你的绳子——全部都断了。

[凛的手在雪乃的掌心下微微颤抖。]

**雪乃：** 所以你不需要再撑了。这里没有部下在看你。没有敌人在等你崩溃。只有我。

[风仍然没有起。整个世界安静得像一个密封的瓶子。]

**雪乃：** 凛。你可以放下了。

[凛的呼吸急促起来。]

#### 第三幕：第一滴泪

[月光在铠甲上流淌。凛低着头。长发从两侧垂下，遮住了大半张脸。她的双手攥紧了膝盖上的铠甲边缘——指节发白。]

*——不能哭。*

*——骑士不哭。*

*——父亲说过。教官说过。千姬也说过——"凛，你是天城最坚强的骑士。"*

*——最坚强的骑士不哭。最坚强的骑士——*

[她的肩膀开始颤抖。]

*——千姬。*

*——你不记得我了。*

*——你再也不会叫我的名字了。你再也不会对我微笑了。你再也不会说"天城的未来就交给你了"。*

*——你变成了一个活着的——*

[她的呼吸急促到近乎痉挛。每一次吸气都带着压抑的、像被撕裂的布匹发出的声音。]

*——我从灰谷走到这里。每一步都在想——到了王都之后就能见到你。哪怕你是傀儡，哪怕你不是自愿的，哪怕一切都是枢密院的操纵——*

*——至少你还在。*

*——至少你还是你。*

*——现在——*

[雪乃没有说话。她只是将手从凛的右手移到她的肩上。轻轻地。]

[那个动作打碎了什么。]

[也许是铠甲。也许是城墙。也许是凛·艾斯特雷亚二十六年来从未倒塌过的那面墙。]

[凛发出了一个声音。]

[不是哭声。比哭声更原始。像是从胸腔最深处——从肋骨和心脏之间的缝隙里——被挤压出来的一个音节。没有词义。没有语言。只是纯粹的、无法再承受的痛苦。]

**凛：** ……啊。

[一个字。喉咙发紧。声带几乎无法震动。]

[然后是第二个声音——更长一些，尾音向上翘起，像一个被打碎的问号。]

**凛：** ……为什么……

[她不知道在问谁。问枢密院？问天城？问千姬？问自己？问这个把所有美好的东西都碾碎的世界？]

[然后——]

[泪水。]

[不是电影里那种缓缓滑落的、诗意的泪珠。是从紧闭的双眼中骤然涌出的、几乎是暴力的液体。热的。滚烫的。落在铠甲上时发出极轻的"啪"声——像雨滴打在铁皮屋顶上。]

[一滴。]

[凛·艾斯特雷亚二十六年来的第一滴泪。]

[落在胸甲上。]

[落在白百合纹章——不，叶子纹章——的位置。]

[然后是第二滴。第三滴。然后她不再数了——因为泪水已经连成了线，沿着铠甲的凹痕和裂纹向下流淌，在月光中闪着银色的细线般的光。]

[她没有发出大声的哭泣。没有嚎啕。只有那种压抑的、断断续续的、从牙缝和鼻腔间漏出的呜咽——像一头受伤的动物在夜里独自舔舐伤口时发出的声音。]

[雪乃将凛拉向自己。凛的头靠在雪乃的肩上——铠甲碰到斗篷的布料，发出一声沉闷的碰撞。凛的身体仍然是僵硬的——二十六年的习惯不是一瞬间就能卸下的。但她没有推开。她没有力气推开了。也许——也许她也不想推开。]

**雪乃：** ……哭吧。

[凛的呜咽声变大了一些。仍然不是嚎啕——更像是一扇生锈的门被一点一点地推开。]

**雪乃：** 我在。

[凛的手抓住了雪乃斗篷的前襟。指节发白。力气大到将布料攥出深深的褶皱。]

[她哭了。]

[不知道多久。月亮又移动了一段。城楼上的影子换了方向。]

[凛的呜咽声渐渐低下去。不是止住了——是哭到没有力气了。]

[她的脸从雪乃的肩上抬起来。月光照亮了她的脸——红肿的眼眶、尚未干涸的泪痕、被泪水浸湿后贴在面颊上的散发。那张脸不再是"面无表情"——也不是"坚毅"或"冷静"或任何可以用词语形容的东西。它只是——脆弱。纯粹的、毫无遮挡的脆弱。]

### 场景描写

落星堡。西面城墙外。黄昏。

太阳悬在地平线上方一拳的位置——像一枚被火烧红的铜币，边缘已经开始融化。整个西面的天空被染成一种浓烈到近乎暴力的橘红色，从天顶向四面八方蔓延，将云层、山脊、旷野、城墙全部吞没。落星堡的影子被拉得极长，像一只黑色的巨手伸向东方。

凛站在西面城墙外的空地上。

这块空地是落星堡建造时特意留出的演武场——三面被低矮的石墙围住，第四面正对西方，没有遮挡。一百二十年前，初代圣骑士团在这里操练。黄昏时分，夕阳会从正面打过来，骑士们逆光挥剑——教官说这是为了训练在"最不利视线条件下"战斗的能力。

现在凛站在演武场中央。夕阳正对她的脸。她没有眯眼。二十六年来她已经习惯了逆光。

她的铠甲在落日中呈现一种奇异的颜色——不是银白，不是暗灰，而是被夕阳浸透后的深铜色。像被鲜血浸泡了很久、又被时间慢慢氧化的古铜器。每一道伤痕、每一处凹陷都被夕阳填满，在铠甲表面形成一张由光与影编织的网。

胸口的叶子纹章——不，白百合与泪痕——在夕阳中几乎看不清了。银线被铜色的光完全掩盖，只留下一个隐约的轮廓。

她身后站着苍。

苍的伤已经好了大半——雪乃来了之后，所有人的伤都在以超出预期的速度愈合。苍的长枪重新打磨过，枪尖在夕阳中闪着锐利的光点。但她的脸上没有战意——只有一种沉静的、近乎悲伤的平静。

苍身后，是最后的残部。四名骑士，六名枪兵。十个人。从灰谷隘口的三百人走到这里——只剩下十个能战的了。

他们排成一列，面朝西方。逆光。十个影子被夕阳拖得很长很长，铺在身后的石板地上，像十条黑色的道路。

暗潮的军旗出现在西面旷野的尽头。

## Event 15 — 最終決戦「誓約尽头」

### 场景描写

圣骑士团创建之地——「初誓の丘」。

落星堡以东七里。一座低矮的丘陵。丘顶是一片被野草覆盖的废墟——石柱的残桩、倒塌的拱门、半埋在泥土中的浮雕碎片。一百二十年前，初代圣骑士团团长在这里竖起第一面白百合旗帜，十二名创建骑士在月光下立下誓言。后来天城在落星堡修建了正式的据点，初誓の丘便被遗忘了——只留在骑士学院的教材里，作为"圣骑士精神起源"的注脚。

没有人来这里了。杂草长到了腰高。石柱上的铭文被苔藓覆盖，只有极少数字迹还能辨认——"吾之剑"、"黎民"、"不背弃"——断断续续的词语，像一首被时间撕碎的诗。

天空是灰白色的。没有夕阳。没有月光。是那种暴风雨来临之前的、压抑的、铅灰色的天。云层低得像是要塌下来。空气中弥漫着泥土和雨水即将到来的潮湿气息。

凛站在丘顶。

她的身边没有苍。没有枪兵甲。没有任何人。

昨夜——捕获失败之后——她让苍带着所有人离开了。没有争吵。没有命令。只是说了一句话：

*——"你们先走。我还有一件事要做。"*

苍看着她。什么都没问。只是点了点头。然后带着残部消失在夜色中。

苍知道凛要做什么。她们认识太久了。有些事不需要说。

凛独自走了一夜。从落星堡到初誓の丘。七里路，走了四个时辰——因为她的腿在昨天的战斗后已经很难正常行走了。右膝的关节在每一步都发出令人牙酸的摩擦声。但她没有停。一步一步。像朝圣。

现在她站在丘顶。站在一百二十年前初代骑士们站过的位置。

她的手中有一把新的剑。

不是旧剑——旧剑已经断了。不是断剑——断剑在灰谷就碎了。

这把剑是她在落星堡的废墟深处——初代团长的私室中——找到的。一把被封存在石匣中的古剑。剑身已经不再锋利，钢铁因为年代久远而呈现一种暗沉的青灰色。但剑身上刻着完整的圣骑士誓言——那是初代团长的佩剑。

碎裂的誓約之剣。

不是因为它碎裂了——至少现在还没有。这个名字是凛给的。因为她知道——在这场战斗结束之后，无论胜负，这把剑、这段誓约、以及她作为"圣骑士"的一切——都会在这里终结。

### 剧情对话

#### 战前

[凛站在丘顶的废墟中央。风从四面八方涌来——灰色的天空下，野草被吹得东倒西歪，发出沙沙的声响。倒塌的石柱和拱门残骸在她身周围成一个不规则的半圆，像是某种天然的竞技场。]

[她将碎裂的誓約之剣横在身前。双手握柄。闭眼。]

*——一百二十年前。初代团长站在这里。*

*——他对着十二名骑士说——"吾之剑，为王冠而鸣；吾之盾，为黎民而立。直到最后一刻，骑士绝不背弃誓言。"*

*——十二个人在月光下举剑。那是圣骑士团的开始。*

*——现在，圣骑士团只剩一个人了。*

*——我。*

*——站在同一个地方。握着同一个人的剑。*

*——但我要做的不是重复那个誓言。*

*——我要结束它。*

[她睁开眼。灰色的天空映在瞳孔中。]

[暗潮的旗帜出现在丘陵下方的平原上。]

**凛：** ……你来了。

[她将剑尖指向前方。风将她的长发吹向身后——在铅灰色的天空下，她的轮廓像一幅用墨笔勾勒的白描。没有夕阳的铜色渲染。没有月光的银色衬托。只有灰白。只有她。]

**凛：** 上一次你赢了。但我没有交出答案。

[她向前迈出一步。石柱残桩旁的野草被她的铠甲碰倒。]

**凛：** 今天——在这片废墟里——我把所有的答案都给你。

[她举剑。碎裂的誓約之剣的剑身在灰色的光线下不反射任何光芒——一百二十年的岁月将所有锋芒都磨成了沉默。但那沉默本身就是一种重量。]

**凛：** 我是凛·艾斯特雷亚。不是圣骑士团团长。不是天城的骑士。不是任何人的剑与盾。

[她的目光没有动摇。]

**凛：** 只是一个需要确认——自己该把后背交给谁的人。

**凛：** 来。最后一次。

**凛：** 一百二十年前初代团长用这把剑建立了骑士团。它见证了每一代骑士的誓言。

**凛：** 也见证了每一次誓言被背叛。

**凛：** 你的攻击变快了。上一场你的左翼有破绽——现在没有了。你在成长。

[凛知道。]

**凛：** ……最后了。

[她将碎裂的剑举过头顶。和昨天「夕暮の一閃」一样的姿势——但这一次，她的双手在颤抖。不是因为恐惧。是因为她要把所有剩余的力量压缩进最后一击。]

**凛：** 「誓約の終焉」。

### 场景描写

初誓の丘。黎明前最黑暗的时刻。

天空是深蓝色的——不是夜晚的纯黑，也不是白昼的明亮。是介于两者之间的、最深最浓的蓝。没有星辰。没有月亮。云层在昨天的灰白之后散去了，但太阳还没有来。世界被浸泡在一种纯粹的、没有光源的幽蓝之中。

废墟比白天更安静。石柱的残桩在黑暗中变成了模糊的轮廓，像是一群沉默的守夜人。碎裂的誓約之剣的碎片仍然散落在丘顶——但在黎明前的光线下，它们不再是冰冷的灰色。它们吸收了泥土的湿气，每一枚碎片上都凝结了一层极薄的露水。在幽蓝的天光中，那些露水珠让碎片看起来像是在发光——一种微弱的、内敛的、不属于太阳也不属于月亮的光。

凛躺在废墟中央。

不是昏迷——是睡着了。战斗结束后，雪乃赶到了。她什么也没说——用最快的速度处理了所有伤口，然后将自己的斗篷盖在凛身上。凛在药物和疲惫的双重作用下沉沉睡去。

雪乃就坐在她旁边。整夜没有动。

现在——黎明前的最后一刻——凛醒了。

### 剧情对话

#### 第一幕：醒来

[凛的眼睛睁开。视野模糊。深蓝色的天空像一片倒置的海洋。]

[她试图动一下——全身的痛觉同时涌来。不是尖锐的刺痛——是那种大战之后骨头和肌肉深处的、酸胀的、沉重的钝痛。像是整个身体被泡在热水里太久。]

**凛：** ……嗯。

**雪乃：** 别动。

[雪乃的声音从右侧传来。近。很近。凛偏过头——看到雪乃坐在旁边。斗篷已经让给了凛，她自己只穿着里衣。在黎明前的寒气中，她的嘴唇微微发紫。但她的手放在凛的手腕上——在为凛把脉。]

**凛：** ……你在这里坐了多久？

**雪乃：** 不重要。

**凛：** 你冷。

**雪乃：** 不重要。

**凛：** ……雪乃。

**雪乃：** 你的脉搏稳定了。体温正常。肋骨没有进一步错位。左手的溃烂已经被控制住。额角缝合没有裂开。

[她像念清单一样报完所有伤势。然后——停了一下。]

**雪乃：** ……你把自己搞成什么样了。

[声音微微发抖。不是冷的。]

**凛：** ……赢了吗？

**雪乃：** ……你输了。

**凛：** 不是问胜负。是问——他够格吗。

[雪乃沉默了一会儿。]

**雪乃：** ……你都用了「誓約の終焉」他还站着。你觉得呢。

[凛闭上眼。然后重新睁开。深蓝色的天空映在瞳孔里。]

**凛：** ……够格。

[她试图坐起来。雪乃按住她的肩。]

**雪乃：** 再躺一会儿。

**凛：** 不。我要坐起来。

**雪乃：** 凛——

**凛：** 我要看日出。

[雪乃的手停住了。]

[凛用手肘撑起身体。每一寸移动都伴随着钝痛和铠甲碎片的叮当声。雪乃叹了一口气——然后伸手扶住她的背。将她扶起来。]

[凛靠在一根石柱残桩上。面朝东方。]

[天空的最东面——地平线上方——深蓝色开始变了。不是变亮。是在蓝色的最底层，渗入了一丝极淡的、几乎无法察觉的暖色。像墨汁滴入清水后，第一缕扩散的丝线——只不过方向相反。是光在侵入黑暗。]

**凛：** ……快了。

**雪乃：** 嗯。

[雪乃在凛身旁坐下。两人并排。面朝东方。]

[沉默。]

[风从丘顶掠过。轻。带着露水的湿润和泥土深处的凉意。凛的长发被风拂动——乱糟糟的，沾着血渍和泥土。雪乃伸手，将凛脸侧的一缕散发别到耳后。凛没有躲。]

#### 第二幕：新的誓言

**凛：** 雪乃。

**雪乃：** 嗯。

**凛：** 我想好了。

**雪乃：** ……什么？

[凛低头。看着自己的手——左手裹着雪乃换过的新绷带，白色的，干净的。右手空着——没有剑。手心里是老茧和伤疤。]

**凛：** 从灰谷到这里——我一直在想一个问题。骑士的誓言碎了之后，我是什么。

**雪乃：** ……

**凛：** 一开始我以为——什么都不是了。誓言碎了，骑士就不存在了。就像剑断了就不是剑了。

[她将右手翻过来。手背朝上。那一滴雨——昨天战斗结束时落在手背上的——早已干涸。但干涸后留下了一个极小的、圆形的水痕。]

**凛：** 后来我想——也许不是这样。

[她抬头。看着东方。天际线上的暖色更明显了——从一丝扩展成了一条极细的线。像是有人用极淡的橘色在地平线上画了一笔。]

**凛：** 剑断了还是剑。只是不能用旧的方式挥了。

**雪乃：** ……

**凛：** 誓言碎了——但"想要保护的心"没有碎。那个心不需要誓言。不需要骑士团。不需要王冠来证明。

[她转过头。看着雪乃。]

[在黎明前的幽蓝中，雪乃的脸只有模糊的轮廓——但凛看清了她的眼睛。那双眼睛里有疲惫。有心疼。有一整夜守夜不眠的红血丝。还有别的什么——凛不知道该叫什么。也许是信任。也许更多。]

**凛：** 我要跟着暗潮的指挥官走。

[雪乃没有说话。]

**凛：** 不是因为我输了。不是臣服。

[她低头。看着膝盖上散落的铠甲碎片。]

**凛：** 是因为——他能保护我保护不了的人。而我能保护他保护不了的人。

[她伸出右手——空的，没有剑的手——在黎明前的空气中摊开。]

**凛：** 一把剑保护不了所有人。但两把——也许可以多保护一些。

[雪乃看着凛的手。沉默了很久。]

**雪乃：** ……你终于学会不一个人扛了。

**凛：** ……被你和苍骂了太多次。总要学会的。

[雪乃的嘴角动了一下。不是笑——更像是某种长久紧绷之后的松弛。]

**雪乃：** 那我也去。

**凛：** ……你不必——

**雪乃：** 闭嘴。

[语气和"坐下"时一模一样。不容拒绝。]

**雪乃：** 你以为我跑了两个月、沿着你留下的伤兵一路治过来，是为了看你去给别人当部下然后我回家种药材？

**凛：** ……

**雪乃：** 你走哪儿我走哪儿。你打仗我治伤。你再用铠甲修补线缝自己的伤口我就把你的手绑起来。

[凛看着雪乃。]

[在黎明前最后的蓝色中，两个人对视。一个浑身是伤靠在石柱上，一个冻得嘴唇发紫坐在冰冷的石板上。]

[凛伸出右手。不是握拳。不是举剑。只是伸出——像昨晚雪乃在城楼上伸出的那只手一样。]

**凛：** ……那立个新约吧。

**雪乃：** ……约？

**凛：** 不是骑士的誓言。不是对王冠的效忠。只是——

[她看着东方。地平线上的那条橘色细线正在膨胀——缓慢地、不可阻挡地向上方蔓延。深蓝色的天空被一点一点地吞噬。]

**凛：** ——我会活着。我会保护身边的人。不是因为谁的命令。是因为我想。

[她的右手在空中等着。]

**凛：** 你呢。

[雪乃看着那只手。看了很久。]

[然后她伸出自己的手——带着药草气息的、有薄茧的、温暖的手——握住了凛的。]

**雪乃：** ……我会让你活着。不是因为你是我的病人。是因为——

[她没有说完。]

[不需要说完。]

[两只手握在一起。在黎明前最后的黑暗中。]

#### 第三幕：日出

[东方。]

[地平线上的暖色不再是一条线了。它膨胀成了一片——从橘色变成金色，从金色变成白色。白色的光从地平线上涌出，像液体一样沿着大地的表面向四面八方蔓延。]

[第一道阳光照上了初誓の丘。]

[光从东面的废墟拱门穿过——那个倒塌了一半的拱门恰好形成了一个框。阳光从这个框中射入，笔直地照在丘顶的中央——照在散落的剑碎片上。]

[碎片上的露水在阳光中亮起。]

[数十枚青灰色的钢铁碎片——碎裂的誓約之剣的遗骸——每一枚上面的露水珠都变成了一个微小的棱镜。阳光穿过露水，在碎片表面折射出细碎的彩虹色光斑。那些光斑密密麻麻地铺在泥土上、石板上、杂草上——像是废墟开满了一种只在黎明时分才会绽放的、由光和水构成的花。]

[凛看着那些光斑。]

[她低头——然后看到了。]

[在她身旁。在石柱残桩的根部。在泥土和碎石的缝隙间。]

[一株草芽。]

[极小的。只有小指的一半高。两片嫩绿色的子叶刚刚展开，像一双微小的手掌。茎秆细得几乎透明——在晨光中可以看到内部浅绿色的纹路。它生长的位置不可思议——石柱的基座与地面的接缝处。周围全是石头和碎砖，没有可见的土壤。但它就是长出来了。从不可能生长的地方。]

[凛看着那株草芽。很久。]

**凛：** ……雪乃。你看。

[雪乃顺着凛的目光看过去。]

**雪乃：** ……嗯。看到了。

**凛：** 从石头缝里长出来的。

**雪乃：** 嗯。

**凛：** ……很倔。

**雪乃：** ……像谁。

[凛没有回答。但她的嘴角——在血污和晨光之间——弯起了一个弧度。不是苦笑。不是疲倦的抽搐。是笑。小的。安静的。但真实的。]

[也许是凛·艾斯特雷亚很长时间以来的第一个真正的笑容。]

[阳光越过废墟的拱门，将整座丘顶慢慢照亮。深蓝色退去。金色涌来。凛铠甲上的每一道伤痕、每一处泪痕都被晨光填满——不是夕阳的铜色，不是月光的银色，而是黎明特有的、温暖的、淡金色的光。]

[她靠在石柱上。握着雪乃的手。看着日出。]

[她没有站起来。不需要。太阳自己会升上来的。]

[风从东面吹来。带着阳光的温度。带着泥土苏醒的气息。废墟中的杂草在风中摇摆——它们经历了昨天的战斗、碎片的坠落、整夜的寒冷，但清晨一到，它们又直起来了。]

[凛看着远方。地平线上的光已经连成了一条完整的、明亮的弧线。太阳的上缘出现在地平线的边际——一小片灼目的白金色。]

[她深吸一口气。肋骨疼了一下。但疼痛正在变得可以忍受——或者说，她已经习惯了。]

**凛：** ……走吧。

**雪乃：** ……去哪儿？

**凛：** 去暗潮的指挥官那里。该正式见个面了。上两次都在打架——还没好好说过话。

[雪乃站起身。然后弯腰，将凛拉起来。]

[凛站立。摇晃了一下。稳住。]

[两人并排站在初誓の丘的丘顶。面朝东方。身后是废墟、碎片、散落的誓言。面前是正在升起的太阳。]

[凛低头看了一眼胸甲上的白百合与泪痕纹章。晨光照在上面——泪痕氧化的青灰色和银线的暗淡在阳光中反而呈现出一种独特的质感。像旧银器。像被时间打磨过的、不再锋利但更加温润的东西。]

[她没有摘下它。]

[也没有换上新的。]

[这枚纹章就是她现在的样子——旧的、碎裂的、被泪水浸过的、不完美的。但还在。]

[她迈出第一步。走下丘顶。雪乃跟在她身旁——不是身后。是旁边。]

[身后，初誓の丘在晨光中渐渐远去。石柱的残桩、倒塌的拱门、碎裂的剑片——一百二十年的历史安静地留在那里。]

[石柱根部那株草芽在晨风中微微摇晃。两片子叶迎着太阳。]

[凛没有回头。]
