## Event 01 — 开战宣言「不可逾越之墙」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.12-15 |
| 情感基调 | 铁壁的沉默——无言的威压，不可撼动的守护意志 |
| 战斗条件 | 攻城战，突破银冠要塞外城门（在15回合内） |
| 特殊机制 | 城墙防御加成DEF+5；冰華「绝对防御」被动——受到致命伤害时HP保留1并免疫下一击；攻城器械伤害×2 |
| 敌方配置 | 冰華(Lv.15) + 重装兵×6(Lv.13) + 弓兵×4(Lv.12) + 城墙固定弩×2 |
| 奖励 | 250 EXP, 500G, 银冠城砖碎片×1, 解锁冰華档案第一页 |

### 场景描写

银冠要塞。

天城王朝北部边境最后一座未陷落的堡垒，因主塔顶端覆盖的银白色合金在阳光下如同一顶冠冕而得名。它不像南方的城堡那样追求华美——没有飞扶壁，没有彩绘玻璃，没有旗帜猎猎的尖塔。它只是一面墙。一面高三十丈、厚八丈、以花岗岩和铁矿石混合浇铸的墙。

冬日的晨光苍白而稀薄，落在城墙上如同一层霜。要塞前方的旷野已被暗潮军团的营帐覆盖——黑色的帐篷从地平线一直铺到护城河边，炊烟升起又被北风压回地面，如同无数灰色的手指在泥地上爬行。

城墙之上，一个人站在最高处的雉堞之间。

冰華。

银白色的长发在北风中散开，如同从城墙上倾泻而下的冰瀑。她的全身重甲是哑光的铁灰色——没有任何装饰，没有纹章，没有镶嵌的宝石。只有厚实的钢板、密集的铆钉、以及左臂上那面几乎与她等身的巨型方盾。盾面同样没有任何图案，只有无数道被利刃、箭矢、投石砸出的凹痕和划痕，层层叠叠，如同一棵古树的年轮。

她的面容与她的名字一样冰冷。眼瞳是极淡的灰蓝色，像是冻结的湖面。没有愤怒，没有恐惧，没有傲慢，甚至没有战意——只是一种近乎矿物质般的、无机质的平静。

她望着城下那片黑色的海洋。一言不发。

### 剧情对话

[城墙上，守军列队站在冰華身后。重装步兵们顶盔掼甲，手持长矛与塔盾，沿城墙垛口一字排开。没有人交头接耳，没有人面露惧色——银冠要塞的守军经历过太多次围城，恐惧早已被磨成了日常。]

[副官·霜站在冰華身后两步的位置，手中捧着一份刚送到的战报。她等了许久，冰華始终没有转身。]

**霜：** 女伯爵大人……暗潮军团先锋已完成合围。东、西、南三面城门均被封锁。北面悬崖虽无驻军，但斥候发现了登山绳索的痕迹。

[冰華没有回头。]

**霜：** 预估兵力……前锋约五千。后续主力——

**冰華：** 多少都一样。

[她的声音低沉、平稳，像石头落在石头上。没有起伏，没有情绪。]

**霜：** ……是。

**冰華：** 城墙高三十丈。他们的云梯最高二十五丈。差五丈。攻城锤需要至少三十人协力推进，护城河宽十二丈，冰层厚度不足以承重。投石车有效射程内有七个预设伏击点。

[她终于转过身来。灰蓝色的眼睛扫过每一名守军。]

**冰華：** 银冠要塞建成一百四十七年。经历攻城战四十三次。未失守。

[她举起左臂的巨盾。]

**冰華：** 今天不会是第一次。

[她没有再多说一个字。只是将巨盾竖在身前，盾底触地，发出沉闷的「咚」声——如同一记钟鸣。]

[下一秒，城墙上所有守军同时举盾。]

[几百面盾牌几乎在同一瞬间砸落城墙垛口的石面。]

**咚。**

[那声音不是金属碰撞。是大地在震动。是银冠要塞本身在回应。]

[暗潮阵营中，最前排的士兵不由自主地后退了半步。]

*——墙不需要说话。墙只需要在那里。*

*——我也是。*

[冰華将盾靠在雉堞上，面朝城外，一动不动。风将她的银白长发吹向身后，如同城墙上唯一的旗帜。]

**霜：** （低声，对身旁的百夫长）……女伯爵从昨夜起就站在那里了。一整夜。

**百夫长：** （低声）她永远是第一个站上城墙的，最后一个离开的。

**霜：** 她……不累吗？

**百夫长：** 我在银冠驻守十一年。从没见她说过"累"。也没见她说过"怕"。偶尔……

**霜：** 偶尔？

**百夫长：** 偶尔我会想，她到底是在守护这座城，还是她自己就已经变成了城墙的一部分。

[远处，暗潮军团的战鼓响起。低沉的节奏如同大地的心跳，从旷野尽头一波一波地传来。]

[冰華依然没有动。]

[鼓声越来越近。地面开始微微震颤——那是数千双脚同时踏步的节奏。暗潮的先锋部队开始推进，黑色的方阵如同潮水般涌向护城河。]

**暗潮指挥官：** （远处扩音）银冠守军听着！天城王朝气数已尽！交出要塞，可免一死！

[城墙上一片沉默。]

**暗潮指挥官：** 铁壁女伯爵！你的国王已经放弃了你！枢密院的援军不会来了！你在为谁守这座城！

[冰華依然没有动。]

[霜看了看她，又看了看城下，犹豫着是否该代为回应。]

[冰華终于动了。]

[她只做了一个动作——将巨盾从雉堞上提起，向前迈了一步，站到了城墙最外沿。北风正面灌来，几乎能把普通人吹下城去。她的重甲和巨盾加在一起超过两百斤，在狂风中纹丝不动。]

[她低头俯视城下密密麻麻的暗潮大军。]

[然后——]

[她将巨盾缓缓横在身前。]

[就这样。]

[没有回话。没有宣言。没有嘲讽。甚至没有拔出背后的阔剑。]

[只是一面盾。挡在那里。]

[城下，暗潮指挥官沉默了很久。]

**暗潮指挥官：** ……全军进攻。

### 战斗阶段

[攻城战开始。暗潮军团从三面同时发起进攻。]

[城墙防御加成生效——所有守军DEF+5。冰華站在正面城门上方，「绝对防御」被动激活：受到致命攻击时，HP保留1并免疫下一次伤害。]

*——来吧。*

*——不管是一千人，还是一万人。*

*——这面墙，不会倒。*

[第一波：暗潮步兵架设云梯。云梯顶端距城墙顶仍有五丈距离——正如冰華所计算的。攀爬的士兵在最后五丈暴露在守军弓矢之下，几乎无法还手。]

**冰華：** 弓兵，自由射击。不用节省箭矢。

[第二波：暗潮投入攻城锤。三十人的推进队伍扛着包铁的巨木冲向城门。护城河的冰面在重压下发出不祥的龟裂声。]

**冰華：** 投石。瞄准冰面。

[守军从城墙上推下预先准备的巨石。冰面在第三块巨石落下时彻底碎裂，攻城锤连人带木坠入刺骨的护城河中。惨叫声在冰水中迅速消失。]

[冰華面无表情地看着这一切。]

*——不是我冷酷。*

*——是战争冷酷。*

*——我只是……站在该站的位置上。*

[第三波：暗潮调动投石车。巨石呼啸着飞向城墙——第一发偏了，砸在城墙外壁上，崩落一片碎石。第二发越过城墙，落入要塞内的空地。第三发——]

[正中冰華所在的城段。]

[巨石撞击产生的冲击波掀翻了周围三名守军。烟尘弥漫之中——]

**霜：** 女伯爵大人！！

[烟尘散去。]

[冰華还站在那里。]

[她的巨盾正面多了一个巨大的凹陷——那是她用盾硬接了投石。双脚在石板地面上犁出两道半尺深的沟痕。左臂微微颤抖，但盾没有放下。]

**冰華：** ……比我预想的轻。

[她的声音依然平稳。仿佛刚才那一击只是一阵微风。]

**冰華：** 弓兵转移目标。优先摧毁投石车。

[HP低于30%时——]

[冰華的重甲已多处碎裂，左肩的护甲完全脱落，露出内衬被鲜血浸透的锁子甲。巨盾的边缘被削去一角，露出了内部的多层钢板结构。她的呼吸变得沉重，呼出的白气在寒风中如同浓雾。]

[但她没有后退一步。]

[她脚下的位置，从战斗开始到现在，一寸都没有变。]

**霜：** 大人，请退到内墙——

**冰華：** 我退了，谁站在这里？

**霜：** 可是您的伤——

**冰華：** 城墙裂了会去休息吗。

**霜：** ……

**冰華：** 我还站得住。那就还不到退的时候。

[她将碎裂的巨盾重新举起。盾面上的无数伤痕——新的叠在旧的上面——在冬日惨淡的阳光下如同一部无字的战争史书。]

*——我不知道该守到什么时候。*

*——但"什么时候放弃"这个问题，从来不在我的选项里。*

[战斗继续。冰華如同城墙本身一样——沉默、坚固、不可动摇。]

[战斗结束——玩家突破外城防线后，冰華率残部有序撤入内城。没有溃败，没有混乱。她是最后一个离开城墙的人。]

### 战后

[暮色降临。玩家占领了银冠要塞外城。城墙上的战斗痕迹触目惊心——碎裂的石板、折断的箭矢、凝固的血迹。但最令人印象深刻的，是城墙正面那个盾形的凹痕——冰華站立的位置，地面被双脚犁出的沟痕，以及盾牌接住投石时留下的放射状裂纹。]

[那看起来不像一个人站过的地方。那看起来像一座碉堡曾经矗立过的地基。]

[远处，内城的城门缓缓关闭。冰華的身影在门合拢前最后一刻消失在黑暗中。]

[玩家的副官走过来，看着那片痕迹，沉默了很久。]

**玩家副官：** ……她一个人挡了多少轮？

**斥候：** 属下清点了城墙上的投石弹痕。她接下了——

[斥候低头看了看手中的记录板，表情有些微妙。]

**斥候：** 十七发。她一个人用盾接下了十七发投石。

**玩家副官：** ……

**斥候：** 而且她脚下的位置——从第一发到最后一发——没有移动超过一尺。

[玩家副官回头望向已经关闭的内城城门。]

**玩家副官：** 这就是"铁壁女伯爵"吗。

*——不。这不只是铁壁。*

*——这是一个把自己活成了城墙的人。*

> **获得奖励：** 250 EXP / 500G / 银冠城砖碎片×1 / 解锁冰華档案第一页
>
> *「银冠城砖碎片」——从外城城墙上剥落的一块花岗岩碎片。表面光滑得异常——不是人工打磨的，而是被无数次攻城战中飞溅的碎石和箭矢冲击后自然形成的。仔细观察可以发现，碎片的截面上嵌着一小片铁屑——那是冰華巨盾碎裂时飞入城墙裂缝中的。墙与人，已分不清界限。*

## Event 02 — 前哨遭遇「铁壁女伯」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.15-18 |
| 情感基调 | 三道防线——层层递进的压迫感，沉默将领的绝对掌控力 |
| 战斗条件 | 依次突破三道防御阵地，在最终阵地击败冰華 |
| 特殊机制 | 三段式战斗，每突破一道防线后可选择休整（恢复30%HP）或强攻（经验+20%）；冰華「不动之山」被动——每3回合触发一次完全免疫（持续1回合） |
| 敌方配置 | 第一防线：重装兵×6(Lv.14) + 弓兵×2(Lv.13)；第二防线：重装兵×4(Lv.15) + 枪兵×4(Lv.15) + 弩车×1；第三防线：冰華(Lv.18) + 精锐亲卫×4(Lv.16) |
| 奖励 | 350 EXP, 800G, 铁壁阵形图×1 |

### 场景描写

银冠要塞外围，三道防线横亘在通往内城的唯一通道上。

这不是临时搭建的路障。这是冰華用十二年时间精心设计的纵深防御体系——每一道防线都利用了天然地形：第一道依托一条干涸的河床，防御者据高临下；第二道嵌入两座矮丘之间的隘口，宽度仅容四人并行；第三道则直接建在内城城门前的开阔地上，背靠城墙，四面无遮——看似没有地利，实际上正因为无处躲藏，进攻者不得不在冰華的视野中完全暴露。

晨雾未散。三道防线在薄雾中若隐若现，像三条沉睡的巨蛇盘踞在通道上。

第一道防线上，守军已列阵完毕。塔盾排成一面铁墙，长矛从盾隙中探出，如同一只浑身是刺的铁刺猬。

第二道防线上，隘口两侧的矮丘上架设了弩车和投石篮。居高临下的枪兵们安静地等待着。

第三道防线上——只有冰華和四名亲卫。

她坐在一把简陋的木椅上，巨盾竖在身旁，靠着城墙。双眼闭着。如果不是铠甲随呼吸微微起伏，她看起来就像一座雕塑。

### 剧情对话

[暗潮军团——即玩家的先锋侦察队——抵达第一道防线前方。薄雾中，防线的全貌逐渐显现。]

**玩家副官：** ……三道？

**斥候：** 是的。三道防御工事，纵深约两里。每道间隔约四百步，刚好是弓箭有效射程之外——也就是说，突破一道之后，在到达下一道之前有一段无人区。看似可以喘息……

**玩家副官：** 但实际上？

**斥候：** 实际上，那段无人区完全在第二道防线弩车的覆盖范围内。冲过去等于进入死亡地带，停下来等于被弩箭当靶子。

**玩家副官：** 那绕路呢？

**斥候：** （展开地图，摇头）两侧是峭壁和冻结的沼泽。除非有飞行单位，否则这条通道是唯一的路。

**玩家副官：** ……她把地形研究透了。

**斥候：** 据俘虏交代，这套防御体系是冰華亲自设计的。她花了三年勘测地形，四年建造工事，五年反复演练。每一块石头的位置，每一个射击角度，都是她一个人计算的。

**玩家副官：** 一个人？没有参谋？

**斥候：** 银冠要塞的参谋团在七年前的一次瘟疫中几乎全灭。之后冰華没有要求补充。她说——

[斥候看了看手中的审讯记录。]

**斥候：** 她说"城墙不需要参谋。城墙只需要够厚。"

[远处，第一道防线上传来金属碰撞的整齐声响——守军正在做最后的备战调整。]

**玩家副官：** 准备进攻。先集中兵力突破第一道。

#### 第一防线：干河之壁

[战斗开始。玩家军队推进至第一道防线——干涸河床。守军居高临下，塔盾阵密不透风。]

[进攻方需要从低处向高处突击。河床的碎石地面严重影响行军速度（SPD-2），而防御方的弓兵可以从塔盾间隙中自由射击。]

**重装兵队长：** 稳住！盾墙不许松动！女伯爵说了，第一道防线的任务是——

**重装兵甲：** "消耗，不是歼灭。"我们知道。

[无论玩家选择强攻还是休整，推进至第二防线时，隘口的压迫感扑面而来。两座矮丘如同两只蹲伏的巨兽，隘口窄得几乎要侧身才能通过。弩车的黑色轮廓在丘顶若隐若现。]

**玩家副官：** 这个宽度……一次最多四人通过。

**玩家副官：** 那就先拿下两侧高地——

**斥候：** 高地斜面坡度接近七十度，表面覆冰。爬上去需要至少——

**斥候：** 能活着走的人通常都觉得够了。

[冰華。]

[她已经从木椅上站了起来。巨盾在左手，阔剑——她极少使用的阔剑——在右手。四名亲卫站在她身后，间隔均匀，如同四根柱子。]

[她看着对面。]

**冰華：** 你们突破了两道防线。

[她的声音不大，但在空旷的石板广场上传得很远。]

**冰華：** 比我预想的快了半个时辰。

**冰華：** 但第三道防线没有地形优势。没有弩车。没有隘口。

**冰華：** 只有我。

*——但也消耗了我大半兵力。*

**亲卫甲：** （战斗中）女伯爵！右翼压力过大！

**冰華：** 收缩阵线。靠近我。

**冰華：** 退到我身后。

**亲卫甲：** 属下还能战——

**冰華：** 退到我身后。

[亲卫退到她身后。冰華独自面对右翼的攻势。]

[「不动之山」触发——冰華全身泛起淡蓝色的光芒，所有攻击在接触她时如同水波撞上礁石，无声消散。]

*——三回合。只是三回合。*

*——三回合之后我又会受伤。*

*——但三回合……够了。*

*——够让身后的人喘一口气。*

[HP低于40%时——]

[冰華的铠甲碎裂得更加严重。巨盾已经看不出原来的形状——它被砸成了一块扭曲的铁板，只是还勉强能挡在身前。她的右膝微微弯曲，是整场战斗中第一次显露出疲态。]

[但她的脚——依然没有后退一步。]

**霜：** （从城墙上大喊）大人！！城门可以打开！请撤入内城！！

**冰華：** 关着。

**霜：** 大人！！

**冰華：** 城门打开，他们就会冲进来。然后内城的百姓——

[她挡下一记重击，肩甲彻底碎裂。]

**冰華：** ——就没有墙了。

[她深吸一口气。将碎裂的巨盾向前推出——不是格挡，是推。盾的破碎边缘在石板地面上犁出火花。]

**冰華：** 我还站着。门就不用开。

[战斗结束条件：击败冰華（HP归零时「绝对防御」触发，保留1HP，但冰華单膝跪地，阔剑插在地上支撑身体）。]

### 战后

[冰華跪在城门前。巨盾终于从手中滑落，砸在石板上发出沉闷的巨响。那面盾已经不成形了——更像一块被反复锻打过的废铁。]

[她的呼吸沉重而缓慢。银白色的长发散落在肩头和地面，被汗水和血迹黏成一缕一缕。]

[但她的右手还握着插在地上的阔剑。身体微微摇晃，却始终没有倒下。]

[城门在她身后缓缓开启一条缝——霜带着几名士兵冲出来，将她架起。]

**冰華：** 门关上了吗。

**冰華：** ……嗯。

## Event 03 — 角色事件·壹「废墟中的花」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情 |
| 推荐等级 | — |

### 场景描写

银冠要塞以南三里。曾经的桦木村。

"曾经的"——因为现在这里什么都没有了。

战火经过时没有特别瞄准这个村庄——它只是恰好在两军交锋的路径上。暗潮军团从南面推进，天城王朝的游击部队从北面反击，桦木村夹在中间，像两块磨石之间的一粒麦子。没有人下令烧毁它，但也没有人下令保护它。它只是——被碾碎了。

木屋的残骸东倒西歪，梁柱焦黑，还散发着隐隐的烟味。村口的水井被碎石填塞，井沿上坐着一只瘦得几乎透明的猫，用浑浊的眼睛注视着偶尔路过的人。田地里还残留着秋收后的麦茬，但大部分已被马蹄和车辙碾入泥中。

冬日下午的阳光毫无温度，只是机械地照亮了这片废墟的每一个细节——倒塌的篱笆、散落的陶器碎片、一只孩童的木偶马，漆面已被烟熏得面目全非。

在村庄的最深处——半坍塌的村长屋后方——传来微弱的声音。

### 剧情对话

[冰華独自一人，没有穿全套重甲，只着内衬的锁子甲和一件灰色的旧斗篷。巨盾背在身后，几乎遮住了她整个背影。她正在巡查外围防线时偏离了预定路线——因为她看到了烟。]

[不是战火的烟。是炊烟。极细的、几乎不可见的一缕。从废墟深处升起。]

*——有人在这里。*

*——战火过后三天了。还有人在这里。*

[她的步伐变了。不再是巡逻时那种稳重均匀的节奏，而是更快、更轻——一种刻意避免发出声响的走法。她的手没有碰武器。]

[绕过倒塌的村长屋，她看到了。]

[一个临时搭建的窝棚——用烧焦的木板和一块沾满泥的帆布拼凑而成，勉强能遮风。窝棚下面挤着十一个人。三名老人，两名妇人，六个孩子。最大的孩子看起来十岁左右，最小的还在襁褓中。]

[一名妇人正用一只破铁锅煮什么东西——锅里只有水和几根不知道从哪里扒来的野草根。那缕炊烟就是从这里升起的。]

[冰華停在十步之外。]

[窝棚里的人察觉到动静，所有人同时僵住。那名煮东西的妇人本能地将身体挡在孩子们前面，另一名妇人抱紧了怀中的婴儿。老人们缓缓站起来——或者试图站起来，但他们的腿已经支撑不住。]

[一个约七八岁的女孩从妇人身后探出头来。她的脸上有烟灰和泥土，左臂用一条脏布吊着——是骨折还是脱臼，看不出来。但她的眼睛很亮。]

**女孩：** ……是天城的兵？

[冰華没有立即回答。她的目光缓慢地扫过每一个人——评估伤情，计算人数，估量剩余物资。这是军人的本能反应。]

[然后她开口了。]

**冰華：** 你们是桦木村的人？

**妇人甲：** （声音颤抖）是……是的。女、女伯爵大人……

[她认出了冰華。银冠要塞的铁壁女伯爵——在这一带没有人不认识那面巨盾。]

**冰華：** 为什么还在这里？疏散令三天前就下达了。

**妇人甲：** 我们……我们去了集结点。但、但是——

[她的声音断了。眼泪突然涌出来，但她使劲忍住了——不是因为坚强，而是因为太累了，连哭的力气都要省着用。]

**妇人乙：** （抱着婴儿，声音更低）到集结点的时候……驻军说名额满了。说要等第二批转运。让我们在原地等。

**冰華：** ……第二批转运是什么时候？

**妇人乙：** 他们说……两天后。

[沉默。]

**妇人乙：** 那是五天前的事了。

[冰華的眼神没有变化。但她的右手——垂在身侧的右手——缓缓握紧了拳头。锁子甲的铁环在指节处被挤得发出细微的声响。]

**冰華：** 你们自己的领主呢。桦木村属于东谷伯爵的封地。

**老人甲：** （苦笑）东谷伯爵……三天前带着家眷和全部护卫撤去东都了。走之前把村里的存粮全部征用了。说是"战时紧急征调"。

**老人乙：** 他走的时候马车装了十七辆。粮食、金器、家具……连花园里的盆栽都带走了。

[冰華沉默了很久。]

[那个受伤的女孩一直在看她。看她的巨盾，看她的银白长发，看她几乎没有表情的脸。]

**女孩：** 女伯爵大人……你是来带我们走的吗？

[冰華低头看向她。]

[女孩的眼睛很亮。不是因为希望——那种东西在五天前就已经耗尽了。是因为纯粹的、生物本能的求生欲望。那种光芒不会熄灭，直到生命本身熄灭为止。]

[冰華蹲了下来。]

[她从未在任何人面前蹲下过——至少银冠守军中没有人见过。铁壁女伯爵永远是站着的，永远是从高处俯视的，永远是城墙一样的存在。]

[但现在她蹲在一个七岁女孩面前，视线与她平齐。]

**冰華：** 你叫什么名字？

**女孩：** ……小椿。

**冰華：** 小椿。你的手臂怎么了？

**小椿：** 房子塌的时候被压了一下……不太痛了。

[冰華伸出手，极轻地碰了碰那条吊着的手臂。小椿吸了一口凉气但没有躲开。]

**冰華：** 是脱臼。需要复位。会痛。

**小椿：** ……会很痛吗？

**冰華：** 会。但只痛一下。

[小椿看了看她。然后点了点头。]

[冰華的手法很准确——军人都会简单的战场急救。一声轻脆的「咔」，小椿尖叫了一声，然后紧紧咬住嘴唇。冰華从自己的斗篷上撕下一条布，重新为她固定手臂。动作出乎意料地轻柔。]

**小椿：** （眼眶泛红但没有哭）……谢谢。

[冰華站起身。]

[她解开自己的斗篷。]

[北风立刻灌进锁子甲的缝隙。她连眉头都没皱一下。将斗篷裹在小椿身上——斗篷太大了，几乎把女孩整个人都包了进去，只露出一张脸。]

**冰華：** 裹紧。

[她转向那名煮野草根的妇人。]

**冰華：** 锅里的东西倒掉。野草根有些有毒，你分不出来。

[她从腰间的行军包里取出干粮——那是她自己今天的口粮。一份。不多。但对面有十一个人。]

[她把干粮递给妇人。然后犹豫了——]

[这是她作为军人极少犹豫的时刻之一。]

[然后她把行军水壶也一起递了出去。]

**冰華：** 先给孩子们吃。大人分剩下的。

**妇人甲：** （双手颤抖着接过）女伯爵大人……我们、我们怎么报答——

**冰華：** 不用。

[她直起身。看着这十一个人——被自己的王国遗弃的人。不是被暗潮军团伤害的，是被自己人遗弃的。征调了粮食然后逃跑的领主，承诺了转运然后消失的驻军。]

*——我守着银冠要塞。*

*——我以为我在守护他们。*

*——但我守的墙把他们关在外面。而他们自己的领主——我的同僚——把他们扔在废墟里等死。*

[她闭上眼睛。又睁开。]

*——如果我们不保护他们——*

[小椿裹着她的斗篷，正在分干粮给更小的孩子。那件灰色斗篷上还残留着冰華的体温。]

*——要塞有什么意义？*

**冰華：** （低声，像是对自己说）如果我们连他们都不保护……城墙又是为了什么。

[霜从远处跑来——她一直在寻找偏离巡逻路线的冰華。看到眼前的场景后，她愣住了。]

**霜：** 大人……这些人——

**冰華：** 桦木村的平民。被遗弃的。

**霜：** 被遗弃？被谁？

**冰華：** 被我们。

[霜张了张嘴，没能说出话来。]

**冰華：** 霜。回要塞调一辆辎重车来。带毛毯。带粮。带军医。

**霜：** 可、可是大人——要塞的物资配额已经很紧张了。枢密院规定——

**冰華：** 从我的口粮里扣。

**霜：** ……大人。

**冰華：** 从我的口粮里扣。我说的。

[霜看着她——没有穿斗篷、只着锁子甲站在北风中的冰華。看着那十一个蜷缩在窝棚里的人。看着那个裹着女伯爵斗篷的小女孩。]

**霜：** ……是。属下马上去办。

[霜转身快步离开。]

[冰華站在原地。北风灌过锁子甲，冷得像刀割。但她一动不动。]

[小椿分完了干粮，抬头看她。]

**小椿：** 女伯爵大人……你不冷吗？

[冰華低头看了她一眼。]

**冰華：** 不冷。

[她在说谎。但她说谎的方式和她说真话的方式完全一样——平淡、简短、不容置疑。]

[小椿歪了歪头，似乎不太相信。然后她从斗篷里伸出没有受伤的那只手，从窝棚旁的废墟缝隙中摘了一朵小小的花——不知道是什么品种，在冬天的废墟中顽强开放的、指甲盖大小的白色花朵。]

[她踮起脚尖，把花递给冰華。]

**小椿：** 给你。你帮了我们，我没有别的东西可以给你。

[冰華低头看着那朵花。]

[白色的，极小的，在冬天的废墟中开出来的花。根系扎在碎砖和灰烬之间。花瓣薄得近乎透明，在北风中颤抖但没有凋落。]

[她伸出手。那只能举起两百斤巨盾的手，此刻极其缓慢地、极其小心地接过了那朵花。]

[她的手指甲缝里还有上一场战斗留下的血垢。粗糙的、满是老茧和旧伤疤的手指，捏着一朵几乎没有重量的白花。]

*——废墟中的花。*

*——不需要城墙保护。*

*——不需要任何人保护。*

*——它只是……自己长出来了。*

*——在所有人都觉得什么都不可能生长的地方。*

[冰華将那朵花小心地放入胸甲内侧的夹层中——那个夹层原本是放急救纱布的。]

**冰華：** ……谢谢。

[这是银冠要塞的守军们从未听过冰華说出的两个字。]

[CG触发：「废墟中的花」——冬日午后，废墟之中。冰華蹲在小椿面前，锁子甲上反射着苍白的阳光。小椿裹着灰色斗篷，踮起脚尖将一朵白色小花递向冰華。冰華伸出的手占据了画面中央——粗糙的、伤痕累累的大手，与花朵的柔软脆弱形成强烈对比。背景是焦黑的废墟和灰色的天空，但那朵花和小椿的眼睛是画面中仅有的两个亮点。冰華的表情——第一次——不再是矿物质般的冰冷。那不是微笑。只是冰面上出现了第一道极细的裂纹。]

## Event 04 — 防御反击·一「银冠之怒」

| 项目 | 数据 |

| 类型 | 战斗 |
| 推荐等级 | Lv.18-22 |

### 场景描写

黄昏。银冠要塞以东的旷野上，一条蜿蜒的泥路连接着几个被战火波及的村落。

暗潮军团——也就是玩家——的哨兵首先注意到了异常：地平线上出现了移动的黑影，极慢，如同蚁群在冻土上爬行。

等到斥候抵近观察后，传回的报告让所有人都沉默了。

那是冰華。

铁壁女伯爵亲自率领六名重装护卫，组成了一个方形防御阵列。阵列的中心不是指挥官，不是辎重——是三辆牛车，上面挤满了平民。老人、妇女、孩子。还有伤员——被简单包扎后躺在草席上的农夫，断了腿的牧童，烧伤的铁匠。

冰華走在阵列最外侧，巨盾朝外，面对一切可能的威胁方向。她的步伐极慢——不是因为她走不快，而是因为牛车走不快，伤员走不快，抱着孩子的母亲走不快。

她把自己的速度降到了最弱者的速度。

然后用盾将他们全部遮住。

### 剧情对话

[玩家指挥部。斥候送回侦察报告。]

**斥候：** 报告——银冠守军出城。铁壁女伯爵亲自带队。

**玩家副官：** 出城？反击？

**斥候：** 不是。她在……护送平民。

[副官沉默了一瞬。]

**斥候：** 从东面几个被毁的村庄收拢了约八十名平民，用三辆牛车转运。冰華和六名护卫组成方阵包围车队。行进速度极慢——目测时速不到两里。

**玩家副官：** 两里？那她要走到天黑才能回到要塞。

**斥候：** 是的。而且——

[他犹豫了一下。]

**斥候：** 她的阵列完全是防御型。所有盾朝外。没有人拿着进攻武器。她甚至没有拔出阔剑。

**玩家副官：** ……她是在赌我们不会攻击平民。

**斥候：** 或者她不在乎我们攻不攻击。她只是——在保护该保护的人。

[玩家副官看着地图上标注的冰華部队位置。距离银冠要塞还有约四里。以她目前的速度，至少需要两个时辰。两个时辰的暴露行军，完全在玩家军队的打击范围内。]

**玩家副官：** 指挥官，您的命令？

[选择界面弹出。]

> **【选择】**
> - **发起攻击** — 拦截冰華车队。平民不受伤害（战斗中平民车队为不可攻击单位），但击败护卫后可缴获车队物资。（额外获得 500G）
> - **允许通行** — 撤回哨兵，让冰華车队安全返回要塞。（声望+10，冰華好感+3）

---

#### 路线A：发起攻击

[玩家选择拦截。暗潮军队从侧翼包抄，切断冰華车队的行进路线。]

[冰華看到暗潮旗帜出现在两侧丘陵上时，没有表现出惊讶。她只是停下脚步，转向护卫队。]

**冰華：** 缩小阵形。盾墙合拢。平民居中。

**重装护卫甲：** 女伯爵，敌军至少有三个中队——

**冰華：** 多少都一样。

[这是她的口头禅。在不同的语境下有不同的含义。此刻的含义是——]

*——不管有多少。我的任务不变。*

*——保护身后的人。*

[冰華将巨盾挡在车队面前，面向玩家军队。]

[战斗中，冰華部队SPD-3，机动性极差。但防御极高——冰華的DEF在护送状态下额外+2（「守护之壁」被动：身后有需要保护的平民时，DEF额外提升）。]

[车队中传来孩子们的哭声。一名母亲用手捂住孩子的眼睛。]

**冰華：** （不回头，声音平稳）不要怕。

[她说这句话的语气和说"城门关了吗"一模一样。平淡、确定、不容置疑。]

**冰華：** 有我在。

[这是一句极其简单的话。但从冰華口中说出来，它的重量和她的巨盾一样。]

[战斗进行中。冰華几乎不还击——她所有的行动都是防御：格挡、推开、移位挡箭。偶尔有攻击越过她的盾——她会用身体去挡。不是用甲。是用身体。]

**重装护卫乙：** 女伯爵！您的侧腹——

**冰華：** 擦伤。继续防御。

[那不是擦伤。锁子甲被刺穿了两环，鲜血正从缝隙中渗出。但她的盾没有移动一寸。]

*——让他们打我没关系。*

*——不能让任何一击落在车队上。*

[战斗结束。冰華部队被击败——护卫兵全部倒下，冰華单膝跪地。但牛车上的平民毫发无损。]

[冰華跪在地上，巨盾依然挡在身前。她抬头看向玩家军队，灰蓝色的眼睛里没有愤怒，没有恨意——只有一种近乎恳求的、但绝不低声下气的平静。]

**冰華：** ……平民。不要伤害平民。

[这是她开口说的唯一一句话。不是求饶，不是谈判。只是一个事实性的陈述，一个不应该需要说出口的底线。]

#### 路线B：允许通行

[玩家选择放行。暗潮军队的哨兵撤离路线两侧，让出通道。]

[冰華的车队缓缓经过玩家军队的观察范围。她显然知道自己正被注视着——巨盾始终朝向暗潮方向，一刻也没有放下。]

[但当车队安全通过最后一个暴露区域时，她做了一个细微的动作。]

[她的盾——那面从未向任何人低下的盾——微微倾斜了一下。]

[不是致敬。不是感谢。更像是——]

[一面墙，承认了另一面墙的存在。]

**霜：** （小声）大人……他们为什么没有攻击？

**冰華：** ……

**霜：** 大人？

**冰華：** 也许他们觉得不该攻击带着平民的队伍。

**霜：** 那……我们的人——东谷伯爵——

[她没有说完。但意思很明确：我们自己的贵族领主把平民扔下逃跑了，而敌人却放行了护送平民的队伍。]

[冰華没有回答。]

*——敌人比盟友更像守护者。*

*——这是什么样的世界？*

[车队继续向要塞行进。冰華走在最后。暮色将她的影子拉得很长，巨盾的轮廓在夕阳中如同一扇移动的大门。]

[车队上的小椿——Event 03中那个受伤的女孩——从牛车上探出头，回头看了一眼暗潮军队的方向，又看了看冰華的背影。]

**小椿：** 女伯爵大人……那些人，是坏人吗？

[冰華沉默了很久。]

**冰華：** ……他们是敌人。

**小椿：** 敌人和坏人不一样吗？

[冰華没有回答。]

[牛车的轮子在冻土上嘎吱作响。北风将她的银白长发吹向要塞的方向。]

*——敌人和坏人不一样吗。*

*——七岁的孩子问出的问题。*

*——我活了三十二年，回答不了。*

### 战后（两条路线汇合）

[银冠要塞。内城。]

[冰華将所有平民安置在内城的仓库区——那里原本堆放着军备物资，现在被腾出一半用作临时住所。]

[霜看着物资被搬走，面露难色。]

**霜：** 大人，这批物资是下个月的防御储备——

**冰華：** 人先活着。储备以后再说。

**霜：** 可是如果暗潮发动下一轮攻击——

**冰華：** 那就用少一半的物资去守。

[她走进仓库区。平民们正在铺设简易床铺。看到她进来，一些人站起来想要行礼，被她抬手制止。]

[她在仓库角落找到了小椿。女孩正用没受伤的那只手在墙壁上用炭条画画。画的是一面很大很大的盾——和一个很小很小的人。]

[冰華看了一会儿。]

[然后默默转身离开。]

*——墙是为了保护人。*

*——但如果墙把人关在外面呢。*

*——如果墙保护的不是人，而是墙自己呢。*

[她闭上眼。城墙之上，风声如泣。]

## Event 05 — 角色事件·贰「另一种守护」

### 场景描写

银冠要塞以南，暗潮军团占领区。

三个月前这里还是战场。弹坑和壕沟的痕迹依稀可辨。但现在——

道路被修整过了。不是军用的硬化路面，而是用碎石和黄土夯实的民用道路，路旁甚至栽了树苗——虽然在冬天只剩光秃秃的枝干，但那些树坑整齐划一，显然是按规划种植的。

被毁的村庄正在重建。不是原样修复，而是更合理的布局——房屋之间留出了防火间距，水井从村边移到了村中央，公共粮仓被建在了地势最高的位置。木材是新的，散发着松脂的清香。

田地已经翻耕完毕，等待春播。冬小麦的绿苗整齐地排列在田垄上，被稻草覆盖保暖。有人在田埂上设置了简易的灌溉水渠——虽然粗糙，但功能完整。

最显眼的是一个临时搭建的集市。暗潮军团在此设立了物资分发点，定期向平民发放粮食、棉衣、药品。领取的队伍排成长龙，但秩序井然——因为每个人都知道物资充足，不需要争抢。

一名暗潮军官正在集市入口处与一名老农交谈。老农指着自家被重建的屋顶，说了什么，军官点点头，在册子上记了一笔。没有呵斥，没有跪拜，没有任何封建领地里常见的卑躬屈膝。只是两个人在讨论一件需要被完成的事。

### 剧情对话

[银冠要塞。冰華的私人营房。深夜。]

[营房极其简朴——一张行军床、一张桌子、一把椅子、一个武器架。武器架上只有阔剑和备用盾。桌上摊开着防御工事的图纸和巡逻日志。墙上没有任何装饰——没有画，没有旗帜，没有家族徽章。只有一张手绘的银冠要塞全图，上面密密麻麻标注着每一处城墙的厚度和裂缝位置。]

[唯一不属于军事范畴的物品，是桌角一只粗陶小瓶里插着的那朵白花——Event 03中小椿送的。花已经干枯了，但冰華没有扔掉。]

[敲门声。]

**冰華：** 进来。

[霜推门进入。她的表情复杂——像是有话想说但不知道该怎么开口。]

**霜：** 大人……今日的情报汇总。

[她将一沓羊皮纸放在桌上。冰華拿起最上面一份，是关于暗潮占领区的侦察报告。]

**冰華：** （边看边问）他们在修路？

**霜：** 是的。碎石黄土路，连接了三个被毁村庄。斥候还观察到他们在建造公共粮仓和水利设施。

**冰華：** 粮食分配的情况？

**霜：** ……每人每日两斤粟米，儿童额外配给半斤豆粉。受伤或生病的平民可以领取草药。没有强制劳役——重建工作是有偿的，以粮食结算。

[冰華的手停在报告上。]

**冰華：** 每人每日两斤。

**霜：** 是。

[沉默。]

[银冠要塞内，因为物资紧张，守军的口粮已经削减到每人每日一斤半。平民收容区的配给更低——一斤。孩子们每三天才能吃一次热食。]

[冰華知道这些数字。她每天亲自审核物资分配表。]

**冰華：** 他们给出去的……够吗？

**霜：** （愣了一下）……够？大人是在问——

**冰華：** 暗潮给那些平民的粮食。够吗。够他们过冬吗。

**霜：** 根据斥候的估算……如果维持目前的分配标准，那些平民不仅能过冬，春播后的收成还能有余粮。

[冰華将报告放下。她的动作很轻，但纸张落在桌面上的声音在安静的营房中格外清晰。]

**冰華：** ……我们呢？

**霜：** ……

**冰華：** 银冠要塞内收容的平民。够吗。

**霜：** （低声）……以目前的存粮和消耗速度，如果枢密院不补给，我们能撑到开春。但——

**冰華：** 但只是"撑"。

**霜：** 是。饿不死。但也仅此而已。大人您自己的口粮已经削减了三次——

**冰華：** 那不重要。

[她站起身，走到窗前。]

[银冠要塞的窗户都很小——不是为了采光，而是为了防箭。冰華从狭长的窗缝中望出去，能看到远处旷野上的一点光——那是暗潮占领区的灯火。温暖的、安定的、日常生活的灯火。]

[要塞内城则是一片黑暗。为了节省灯油，宵禁后只有城墙上的哨位允许点灯。]

*——墙内是黑暗。*

*——墙外是灯火。*

*——我守着一座黑暗的城，挡住了一群带着灯火的人。*

**冰華：** 霜。

**霜：** 在。

**冰華：** 你觉得……城墙的意义是什么？

[霜没有想到冰華会问这种问题。在她认识女伯爵的八年中，冰華从未问过任何抽象的、哲学性的问题。她问的永远是——城墙厚度多少，箭矢库存多少，下次巡逻几点出发。]

**霜：** 城墙的意义……是保护墙内的人？

**冰華：** 如果墙内的人过得比墙外的人更差呢。

**霜：** ……

**冰華：** 如果城墙不是在保护人，而是在关住人呢。

[霜张了张嘴。没有说出话。]

[冰華转过身来。月光从狭窄的窗缝中射入，在她脸上划出一道苍白的光线——刚好落在眼睛上。灰蓝色的瞳孔在月光中几乎变成了银色。]

**冰華：** 你知道桦木村那些被遗弃的人——小椿她们——现在住在哪里吗？

**霜：** 在内城仓库区——

**冰華：** 住在仓库里。睡在地上。吃着一天一斤的配给粮。冬天没有足够的柴火，只能挤在一起取暖。

[她停了停。]

**冰華：** 而如果她们在墙外——在暗潮的占领区——她们会住在重建的木屋里。每天吃两斤粟米。孩子们有额外的豆粉。受伤的人有草药。

**霜：** 大人……您是说我们应该——

**冰華：** 我不知道我在说什么。

[她闭上眼睛。]

**冰華：** 回去休息。明天的巡逻照常。

**霜：** ……是。大人您也早些休息。

[霜转身走向门口。手搭在门把上时，她犹豫了一下，回头看了一眼。]

[冰華依然站在窗前。面朝那点远处的灯火。银白色的长发在月光中近乎透明。]

[霜出去了。门关上。]

[冰華独自站在黑暗中。]

[许久之后，她走回桌前，坐下。拿起桌角那只粗陶小瓶——里面那朵干枯的白花在月光中像一片薄纸。]

[她看了很久。]

*——小椿问我，"敌人和坏人不一样吗？"*

*——我回答不了。*

*——现在我更回答不了了。*

*——因为答案可能是——*

*——是我们。我们才是坏人。*

*——不。不是"坏人"。是"没用的人"。*

*——守着一座越来越冷的城。关着一群越来越饿的人。挡住一群想让他们吃饱的人。*

*——这就是我的守护。*

*——这就是铁壁。*

[她将花瓶放回桌角。站起身。走到武器架前，拿起巨盾。]

[她将盾面转向自己。]

[无数道伤痕——旧的新的，深的浅的——如同一部用金属和暴力书写的编年史。每一道伤痕背后都是一场战斗，每一场战斗背后都是一个"守护"的理由。]

[她从未质疑过这些伤痕的意义。]

[直到今天。]

[她将盾放回架上。转身。走回行军床，和衣躺下。闭上眼睛。]

[但很久很久之后，她的呼吸都没有变成睡眠的节奏。]

[城墙上，哨兵换岗的脚步声规律地传来。远处，暗潮占领区的灯火依然亮着。]

[更远处，一只不知名的夜鸟在空旷的天地之间叫了一声，孤零零的。]

## Event 06 — 遭遇战·转折「难民的选择」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.20-24 |
| 情感基调 | 敌人在哪一边——难民想要离开要塞，冰華被命令阻拦，信念撕裂 |
| 战斗条件 | 冰華拦路，玩家可选择战斗突破或谈判通过 |
| 特殊机制 | 冰華战斗力严重下降（ATK=1），内心抗拒导致无法全力战斗；玩家选择：【战斗突破】正常战斗 / 【谈判】跳过战斗，声望+5 |
| 敌方配置 | 冰華(Lv.22, ATK强制=1) + 重装兵×4(Lv.18) |
| 奖励 | 450 EXP, 1200G |

### 场景描写

银冠要塞南门外。拂晓。

天还没有完全亮，灰蓝色的天空与灰白色的大地几乎融为一体，分不清地平线在哪里。薄雾从护城河的冰面上升起，在城门前弥漫成一片朦胧的帷幕。

城门缓缓打开了一条缝——仅容一人侧身通过的宽度。

然后，人开始从缝隙中挤出来。

一个，两个，五个，十个……最终，大约四十名平民聚集在城门外的空地上。他们裹着破旧的棉衣，背着简陋的包袱，有些人手里牵着孩子。每个人的脸上都写着同一种表情——不是恐惧，不是犹豫，而是一种已经想了很久、终于下定决心的坚定。

他们要走了。

离开银冠要塞。离开天城王朝的保护。去往暗潮军团的占领区。

因为在那里——他们能活下去。

消息是从一个月前开始传开的。关于墙外的村庄被重建的消息，关于粮食分配的消息，关于暗潮军队不强制劳役的消息。起初没有人相信。后来有人偷偷翻墙出去看了一眼，回来之后就再也坐不住了。

"他们比我们吃得好。他们的孩子不用在仓库地板上挤着睡觉。他们有药。他们有柴火。"

这些话像火种一样在平民收容区传播。最终，四十个人决定行动。

他们没有选择偷偷翻墙——那太危险了，而且有老人和孩子。他们凑齐了一些物资贿赂了南门的值夜哨兵，让他在换岗前"忘记"锁门。

但他们没有料到的是——

冰華就站在城门外面。

### 剧情对话

[四十名平民从城门缝隙中挤出来，在薄雾中看到了冰華的身影。所有人都停住了。]

[冰華站在城门外二十步的位置。全套重甲，巨盾在手。身后四名重装兵列成一排。]

[她显然已经在这里等了很久——铠甲上有一层薄薄的霜，说明她至少站了一整夜。]

*——我知道他们会来。*

*——南门哨兵被收买的事，我三天前就知道了。*

*——我没有惩罚那个哨兵。因为——*

*——因为我不确定他做错了。*

[人群中最前面的是一个中年男人——桦木村出身，逃难时失去了一条胳膊的前农夫。他是这次出走的发起人。]

**独臂农夫：** （声音发抖但没有后退）女……女伯爵大人。

[冰華看着他。]

**冰華：** 你们要去哪里？

[独臂农夫咽了口唾沫。身后的人群挤成一团，有孩子的母亲将孩子往身后藏。但没有人转身回去。]

**独臂农夫：** 大人……您知道我们要去哪里。

**冰華：** 我问的是你们。亲口说。

[独臂农夫直了直腰。他只剩一条胳膊，但那条胳膊握拳放在身侧，指节发白。]

**独臂农夫：** 我们要去暗潮那边。

[人群中传来压抑的抽气声。一个老妇人开始低声啜泣。但独臂农夫继续说了下去。]

**独臂农夫：** 不是因为我们想背叛天城。不是因为我们不感激您收留我们。是因为——

[他的声音开始颤抖。]

**独臂农夫：** 是因为我的孩子昨天饿晕了。三岁。三岁的孩子，饿到晕过去。我去粮仓排队领配给，排了两个时辰，拿到了半斤发霉的粟米。半斤。我一个大人吃都不够，何况——

[他闭上眼睛。深吸一口气。再睁开时，眼眶是红的。]

**独臂农夫：** 我不怕死。我逃难的时候一条胳膊都丢了，还有什么可怕的。但我怕我的孩子饿死。在城墙里面。在天城王朝的保护下。饿死。

[冰華没有说话。]

**独臂农夫：** 墙外面……暗潮那边的人，每天能吃两斤粮食。孩子有豆粉。有药。有柴火。我们不求多——只求活着。

[人群中越来越多的人开始点头。有人压抑不住，喊了出来——]

**妇人：** 女伯爵大人！我们不是叛徒！我们只是想让孩子活下去！

**老人：** 大人……求您放我们走吧……

**少年：** （怀里抱着一个婴儿）他才四个月大……再不找到奶，他活不过这个冬天……

[哭声、恳求声、孩子的啼哭——混杂在拂晓的薄雾中。]

[冰華站在原地。]

[她的表情没有变。灰蓝色的眼睛依次扫过每一张面孔——和她在城墙上审视敌军时一模一样的目光。但如果有人足够仔细，会发现她握盾的手——左手——在微微颤抖。]

[那面巨盾从来没有抖动过。在投石车的重击下没有抖动过。在千军万马面前没有抖动过。]

[现在，在四十个饥饿的平民面前——它抖了。]

*——我接到的命令是：不允许任何平民离开要塞。*

*——理由是：防止暗潮利用平民获取要塞情报。*

*——合理的命令。*

*——合理的理由。*

*——但——*

*——那个三岁的孩子。饿晕的三岁孩子。他知道什么情报？*

*——那个四个月大的婴儿。他能泄露什么军事机密？*

*——我要用这面盾，挡住他们吗？*

*——挡住他们去活下去的路？*

**霜：** （从身后快步走来）大人！枢密院的命令——

**冰華：** 我知道命令。

**霜：** 那——

**冰華：** 我知道命令。

[她重复了一遍。但这一次，那句话的语气变了。不再是确认，而是——]

[一种极其沉重的陈述。像一块石头落进了深井，过了很久才听到水声。]

[冰華向前迈了一步。]

[人群本能地后退。一个孩子尖叫了一声，被母亲捂住嘴。]

[冰華停下来。]

[她看到了人群中的小椿。]

[那个七岁的女孩站在独臂农夫的身后，左臂已经痊愈了——冰華帮她复位的。她身上还穿着那件灰色斗篷——冰華的斗篷。太大了，拖在地上，沾满了泥。但小椿显然每天都穿着它。]

[小椿看着冰華。没有害怕。]

**小椿：** 女伯爵大人……

[冰華的嘴唇动了一下。没有声音。]

**小椿：** ……我们想去有饭吃的地方。

[五个字。]

[每一个字都像一颗钉子，钉进了冰華身上那层比铠甲更厚的东西——信念。]

[她闭上眼睛。]

[再睁开时——]

**冰華：** （声音极低）……我不能放你们走。那是命令。

[人群发出绝望的声音。]

**冰華：** 但——

[她举起盾。]

[人群又一次后退。]

[冰華将盾横在身前——不是挡在人群前面。是挡在身后的四名重装兵前面。]

[四名重装兵愣住了。]

**冰華：** 我不能放你们走。

[她的声音没有提高。但每一个字都像是从牙缝里挤出来的。]

**冰華：** 但如果——有人用武力突破了我的防线——那就不是"放走"。那是"失守"。

[沉默。]

[独臂农夫愣了很久。然后——他听懂了。]

*——她不是在挡我们。*

*——她是在挡她自己的兵。*

*——她是在给我们一个"合理的"离开方式——她"防守失败了"，我们"突破了封锁"。*

*——她在违抗命令。*

*——以不违抗命令的方式。*

[此时，玩家军队的侦察队恰好出现在南门外围。]

[玩家选择界面弹出。]

> **【选择】**
> - **战斗突破** — 协助难民"突破"冰華的防线。冰華会象征性地抵抗（ATK=1），战斗极为轻松。
> - **谈判通过** — 直接与冰華对话。冰華会以"防线遭到暗潮军队强攻"为由"撤退"，难民顺势离开。（声望+5，跳过战斗）

#### 路线A：战斗突破

[战斗开始。冰華和四名重装兵摆出防御阵形。]

[但——]

[冰華的攻击几乎没有伤害。ATK=1。她的每一次挥剑都慢得异常，每一次盾击都轻得像是在推门而不是在战斗。]

[四名重装兵看着自己的指挥官，满脸困惑。]

**重装兵甲：** 女伯爵……？

**冰華：** 专心战斗。

[她的声音是认真的。但她的动作不是。]

[这是冰華——铁壁女伯爵——第一次，也许是唯一一次，故意输掉一场战斗。]

*——我在做什么？*

*——我在把城墙打开一个洞。*

*——我守了一辈子的墙。我自己凿开的。*

*——但那个洞——*

*——那个洞是给人走的。*

[战斗在几个回合内结束。冰華"战败"，率部后撤至城门内。难民们从"被突破的防线"中涌出，向暗潮占领区方向奔去。]

[小椿经过冰華身边时停了一下。她从斗篷里伸出手——手心里攥着什么东西。]

[是一朵干花。和Event 03中那朵一模一样——白色的，极小的。小椿不知道从哪里找到了第二朵，一直留着。]

[她把花放在冰華脚边的地面上。]

**小椿：** ……谢谢你，女伯爵大人。

[然后她转身，小跑着追上了人群。灰色的斗篷在薄雾中飘动。]

[冰華低头看着地上那朵花。]

[没有捡起来。]

[花在北风中微微颤动，被一颗碎石压住了边缘，所以没有被吹走。]

#### 路线B：谈判通过

[玩家选择谈判。玩家方代表走向冰華。]

**玩家代表：** 铁壁女伯爵。

[冰華看着他。盾依然举在身前。但盾面对着的方向——是自己人的方向。]

**玩家代表：** 我们不想和你打。

**冰華：** ……

**玩家代表：** 这些人想活下去。我们能给他们活路。你知道。

[冰華沉默了很久。北风在她和对方之间穿过，卷起地面的霜尘。]

**冰華：** 我的防线……遭到了暗潮军队的强攻。

[玩家代表愣了一下。然后理解了。]

**冰華：** 敌军兵力远超预估。我不得不率部撤回城内。在撤退过程中……平民趁乱离开了。

[她的声音平稳如故。但她在说谎。一个非常明显的谎。明显到她身后的四名重装兵都知道这不是真的。]

[但没有人反驳。]

**冰華：** （转向霜）记录在案。南门防线于辰时遭暗潮强攻，我部寡不敌众，撤回内城。部分平民在混乱中脱离要塞。

**霜：** （声音微颤）……是。

[冰華转身，带队返回城内。难民们从她让出的道路上通过，向暗潮方向涌去。]

[小椿经过时停了一下。她没有说话——只是从斗篷里掏出一朵干花，放在冰華身旁的石板上。]

[霜站在她身后，手里握着那份谎报——或者说，冰華口述的"撤退记录"。]

**霜：** 您会被……

**冰華：** 城墙裂了，可以修。

## Event 07 — 角色事件·叁「城墙上的裂痕」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情 |
| 推荐等级 | — |
| 情感基调 | 铁壁的动摇——城墙的裂痕与信念的裂痕重叠，隐喻式的觉醒 |
| 触发条件 | Event 06 完成后自动触发 |
| 特殊机制 | 无战斗；CG解锁；冰華内心转折的关键节点 |
| 奖励 | 250 EXP, CG「城墙上的裂痕」 |

### 场景描写

深夜。银冠要塞外城墙。

月亮被厚重的云层遮住了大半，只漏出一弯苍白的边缘——像一只快要闭上的眼睛。城墙上的火把间隔十步一支，在夜风中摇曳，将守军的影子投射在石板上，忽长忽短，如同一群不安的鬼魂。

冰華独自一人走在城墙的巡逻道上。

这不是排班内的巡逻。排班表上今夜的巡逻属于霜和第二中队。但冰華睡不着——已经连续三夜了。每次闭上眼睛，她就会看到那四十个人走向南方薄雾中的背影。小椿的灰色斗篷。独臂农夫的弯腰。抱着婴儿的少年的跑姿。

她的行军靴踩在石板上，沉重而规律，如同一架精确的钟摆。左手习惯性地搭在腰间——但今夜她没有带巨盾。这是极其罕见的。银冠守军几乎没有见过不带盾的冰華，就像没有见过没有城墙的要塞。

她走到外城北段——这是要塞最古老的一段城墙，建于一百四十七年前。花岗岩被风雨侵蚀得表面粗糙如老树皮，铁矿石的锈痕在石缝中留下暗红色的血管般的纹路。

她停下来。

因为她看到了裂痕。

### 剧情对话

[冰華蹲在城墙垛口旁，手指抚过一条从城墙顶部延伸至外壁的裂缝。裂缝约两指宽，深度无法判断——她用匕首探了探，金属触底的声音来得比预想更晚。]

*——深。比我以为的深。*

[她站起身，沿着裂缝走了几步。裂缝在火把光中像一条黑色的蛇，从垛口蜿蜒到城墙内侧，分出三四条支线，如同河流的支流。最长的一条几乎延伸到城墙基座。]

*——这条裂缝什么时候出现的？*

*——我每天巡逻，为什么之前没有发现？*

[她再次蹲下，将手掌贴在裂缝旁的墙面上。石头的温度冰冷刺骨——冬夜的花岗岩比铁还冷。但她的手没有缩回。]

*——不。*

*——不是"没有发现"。*

*——是我不想看到。*

[她闭上眼睛。手掌贴着城墙。感受着那条裂缝的边缘——粗糙的、锯齿状的断面。]

*——一百四十七年了。这面墙承受了四十三次攻城。投石车、攻城锤、火油、地道。每一次它都挺过来了。*

*——但裂缝一直在那里。在每一次冲击中加深一点。在每一个冬天因冰冻膨胀扩大一点。在每一场雨中被水侵蚀延长一点。*

*——不是某一次攻击打裂了它。*

*——是一百四十七年的所有攻击，加在一起，一点一点地——*

*——让它裂开了。*

[她睁开眼。盯着那条裂缝。]

[然后她做了一件从未做过的事——她将自己的手伸进了裂缝里。]

[裂缝刚好能容纳她的手指。粗糙的岩石边缘剐蹭着她的皮肤，但她没有在意。她的手指在裂缝深处摸到了什么——松软的，潮湿的。]

[她将手抽出来。]

[指尖沾着泥土。湿润的、黑色的、有机质丰富的泥土——不是花岗岩的碎屑，而是真正的、能长出东西的泥土。不知道从哪里被风和雨带进了裂缝深处，在黑暗和缝隙中积累了不知道多少年。]

*——裂缝里有泥土。*

*——泥土里能长出东西。*

*——城墙的伤口里……藏着生长的可能。*

[一阵夜风吹过。火把的光摇晃了一下。冰華低头看着指尖的泥土，久久没有动。]

[脚步声。霜从巡逻道另一端走过来。]

**霜：** 大人？您怎么在这里——夜巡不是您的班次。

**冰華：** 睡不着。

[霜走到她身旁，看到了裂缝。表情立刻变得严肃。]

**霜：** 这道裂缝……我之前巡逻时没有注意到。需要安排工兵修补吗？

**冰華：** ……不急。

**霜：** 不急？大人，如果裂缝延伸到承重结构——

**冰華：** 霜。

**霜：** 在。

**冰華：** 你觉得……城墙为什么会裂？

[霜愣了一下。作为工程方面有一定知识的副官，她的回答很专业：]

**霜：** 热胀冷缩，地基沉降，还有长年累月的外力冲击。花岗岩虽然坚硬，但它没有韧性。受力超过极限就会产生微裂纹，微裂纹在反复应力下扩展——

**冰華：** 不是问工程。

**霜：** ……？

**冰華：** 我是说——城墙为什么会裂。不是材料力学的"为什么"。是……

[她找了很久的措辞。冰華不擅长这个——她一辈子都在和数据、厚度、角度打交道，从来不需要表达抽象的想法。]

**冰華：** 如果城墙一直在保护对的东西……它会裂吗。

[霜沉默了。]

**冰華：** 我守了这面墙十二年。每一块砖的位置我都知道。每一道旧伤我都清楚。我一直以为——只要够厚，够硬，就不会被打破。

[她再次将手贴在裂缝旁的墙面上。]

**冰華：** 但它裂了。不是因为外面的攻击。是因为——

[她停下来。]

*——是因为里面。*

*——城墙把人关在里面。关久了，人要出去。人出去的力量——*

*——和外面的攻城锤一样大。*

**冰華：** ……是因为它关住了不该关住的东西。

**霜：** 大人……

**冰華：** 一百四十七年。四十三次攻城。城墙没有倒。但裂缝一直在加深。不是因为敌人太强——

[她站起身。目光沿着裂缝，从垛口一直看到基座。那条裂缝在月光的边缘处若隐若现，如同城墙上一条缓慢流淌的黑色河流。]

**冰華：** 是因为墙本身太硬了。太硬的东西不会弯。不会弯的东西——只会裂。

[霜看着她。从这个角度看——月光只照亮了冰華半张脸——她的表情第一次显得不像城墙。]

[那是一种……困惑。一种三十二岁的人在某一个瞬间意识到自己信了一辈子的东西可能是错的那种困惑。不是崩塌，不是绝望。只是——裂缝。信念上的裂缝。和城墙上的裂缝一模一样的走向。]

**冰華：** 霜。

**霜：** 在。

**冰華：** 你有没有想过……如果这面墙不在了。

**霜：** ……不在了？

**冰華：** 如果银冠要塞的城墙不在了。这些人——墙内的人，墙外的人——会怎样？

**霜：** 那……没有城墙保护，平民就会——

**冰華：** 会怎样？

**霜：** 会暴露在危险中。

**冰華：** 什么危险？

**霜：** 暗潮军团的……

[她说到一半，停住了。]

[因为她想到了——暗潮军团占领区的平民。那些每天吃两斤粟米的平民。那些住在重建木屋里的平民。那些不需要城墙保护、但过得比城墙内更好的平民。]

**霜：** ……

**冰華：** 你也想到了。

[霜低下头。]

**冰華：** 城墙挡住的到底是什么。如果它挡住的不是"危险"……那它挡住的是什么？

[她没有等霜回答。因为她知道霜回答不了。就像她自己也回答不了。]

[她最后看了一眼那道裂缝。然后做了一件奇怪的事——]

[她没有标记裂缝的位置以便安排修补。]

[她只是……转身走了。]

[把那道裂缝留在那里。在月光下。在夜风中。慢慢地、安静地、不可逆转地——]

[继续延伸。]

*——城墙上的裂痕。*

*——和我心里的裂痕。*

*——一模一样的形状。*

*——一模一样的深度。*

*——一模一样的——无法修补。*

[CG触发：「城墙上的裂痕」——深夜。银冠要塞城墙之上。冰華独自站在一段古老的城墙旁，右手手掌贴在墙面上，指尖恰好落在一条蜿蜒的裂缝边缘。月光从云层的缝隙中漏下来，落在她的手和裂缝上，形成一道苍白的光带。她的银白长发在夜风中散乱地飘向一侧。表情不再是冰冷——是某种比冰冷更复杂的东西。裂缝从她指尖的位置向画面两端延伸，穿过城墙，像一条将整幅画面劈成两半的断层线。裂缝的一侧是城墙、铠甲、月光下的哨塔——秩序的、坚硬的、冰冷的。另一侧是远处模糊的灯火、风中隐约的烟的气味——温暖的、柔软的、人的。冰華站在裂缝的正中间。两个世界的交界处。]

## Event 08 — 防御反击·二「最后的巡逻」

| 项目 | 数据 |

| 类型 | 战斗 |

### 场景描写

银冠要塞以西。冬日午后。

天空是一种死灰色——不是阴天的灰，而是冬日特有的、仿佛整个天穹都被磨砂玻璃覆盖的那种苍白。阳光透过云层但毫无温度，将大地照得清晰却寒冷，如同一张过度曝光的照片。

这条西向的山谷小径是银冠要塞最后一条与外界联通的陆路——北面是悬崖，东面和南面已被暗潮军团控制，只有西面这条翻越矮山的小径还勉强能走人。

冰華带着四名巡逻兵走在小径上。这是例行巡逻——每三天一次，确认西路是否安全，是否有暗潮部队渗透。

她走得很慢。不是因为地形——这条路她走了十二年，闭着眼睛都不会摔。是因为她在想事情。

自从那天夜里在城墙上发现裂缝以后，她想了很多。想的时候走路就慢。这对一个一辈子用"不思考"来维持效率的人来说，是一种陌生的、令人不安的变化。

巨盾背在身后。阔剑挂在腰间。铠甲上的旧伤痕在灰白色的光线下如同地图上的等高线——层层叠叠，记录着每一次她替别人挡下的攻击。

但今天，那些伤痕看起来不像是勋章。

看起来像是——疤。

### 剧情对话

[巡逻队沿小径西行约两里后，前方斥候突然举手示意停止。]

**斥候：** （低声）前方五百步……有人。

[冰華走到队伍前方，目光越过斥候肩头，看向小径远处。]

[她看到了。]

[一支小型商队正沿着小径向东行进——与冰華的巡逻方向相对。三辆牛车，装载着粮食袋和棉布捆。十几名平民跟随在车旁，有男有女，有老有少。车上还坐着几个走不动路的老人和幼儿。]

*——他们在护送平民。*

*——护送平民去哪里？*

[她仔细观察。商队行进的方向是东——向着银冠要塞附近的几个村庄。车上装的是粮食和棉布——是物资补给。]

*——他们在给平民送物资。*

*——给那些住在两军之间、无人管辖地带的平民。*

*——那些平民……既不在我的城墙内，也不在暗潮的占领区。他们在灰色地带。*

**冰華：** ……

**斥候：** 女伯爵？

[冰華没有回答。她看着远处的商队缓缓接近。]

[商队中，一名暗潮士兵弯腰将一个走累了的小男孩抱上了牛车。男孩的母亲对他说了一句感谢的话，士兵摆了摆手，继续走。]

**巡逻兵甲：** （低声）女伯爵……我们是不是该——

**冰華：** 迎上去。

**巡逻兵甲：** 迎……？是进攻吗？

**冰華：** 迎上去。正面。不要伏击。

[她从身后取下巨盾，左手握住。右手——没有拔阔剑。只是放在剑柄上。]

[她带着四名巡逻兵，走上了小径的正中央。]

---

[暗潮护送队看到了前方出现的银冠守军。一阵短暂的混乱后，护送队队长——一个三十出头的年轻军官——迅速做出反应。]

**暗潮队长：** 平民后撤！护卫前置！

**暗潮队长：** ……铁壁女伯爵。

**冰華：** 你们在做什么？

**暗潮队长：** （谨慎地）我们在执行任务。

**冰華：** 什么任务。

**冰華：** ……

### 战斗阶段

[战斗开始。冰華全属性debuff生效：ATK-3, DEF-4, SPD-1。]

*——不是真的重。是——*

*——对着他们。举不起来。*

**冰華：** 继续战斗。

**巡逻兵甲：** 女伯爵……我们需要追击吗？

**冰華：** ……不需要。

**巡逻兵甲：** 但那些物资——

**冰華：** 那些物资是粮食和棉布。不是武器。

**巡逻兵甲：** 可是军令——

*——我刚才……*

*——在战场上——*

*——放水了。*

*——铁壁女伯爵。十二年没有输过一场防御战的人。*

**冰華：** 今天的巡逻报告——

**巡逻兵甲：** 是。

**巡逻兵甲：** ……是。未见异常。无敌情。

[冰華点了点头。继续走。]

[这是她第二次伪造报告了。]

[一辈子恪守军纪的铁壁女伯爵，在不到一个月的时间里伪造了两次报告。]

*——城墙上的裂缝在扩大。*

*——我心里的裂缝也在扩大。*

*——有些东西——守不住了。*

*——不是城墙守不住。*

[冰華看着它。第一次觉得——那不像家。]

[那像牢笼。]

## Event 09 — 角色事件·肆「孩子的画」

### 场景描写

银冠要塞。内城仓库区——如今的难民临时安置所。

傍晚。天色是一种介于铅灰与靛蓝之间的暧昧颜色，冬日最后一点光线从仓库高处的气窗中斜射进来，将浮尘照成一片缓缓旋转的金色微粒。仓库里弥漫着炭火、干草、粗布和许多人挤在一起生活的气味——不算好闻，但莫名让人觉得温暖。

安置所的状况比一个月前好了很多。冰華用自己的口粮配额和军备储备腾出了空间和物资，霜又从后勤处"借"了几批毛毯和灯油——"借"字加了引号，因为后勤官心知肚明这些东西不会归还，但看在女伯爵的面子上没有追问。

仓库的角落里，孩子们的"区域"被一排空木箱隔出来。几个年长的妇人轮流看管。孩子们正在做孩子该做的事——用炭条和碎石在地上画画，用麻绳跳房子，围着一只仓库里的老猫叽叽喳喳。

小椿不在那群孩子中间。

她一个人坐在仓库最里侧的窗台下面。气窗漏进来的最后一缕天光刚好落在她膝盖上铺着的一张纸上——那纸是冰華行军地图的背面，空白的那一面。小椿正用一截烧焦的细木棍在上面画画。

她画得非常专注。左臂已经拆了绷带——骨折在冰華安排的军医照料下愈合良好——但她仍然习惯性地用右手做一切事情。木棍在纸面上划出细密的沙沙声，像蚕食桑叶。

冰華来安置所是送今天的口粮分配单的。这是她每三天亲自做一次的事——不交给霜，不交给任何人。她说这是"核实人数"。霜知道这是借口。

### 剧情对话

[冰華穿着常服——没有全甲，只有内衬锁子甲和灰色军袍。巨盾没有带。她走进仓库时，几个靠近门口的大人站起来想行礼，被她抬手制止——这已经成了每次来的固定开场。]

[她将口粮分配单递给负责分发的妇人甲，低声交代了几句。然后习惯性地扫视了一圈安置所——清点人数，检查有没有新增的伤病员，确认炭火是否充足。]

[一切正常。她转身准备离开。]

[这时小椿从窗台下跑了过来。]

**小椿：** 女伯爵大人！

[冰華停步。低头看她。]

[小椿的脸比一个月前圆了一点，脸颊上不再是灰扑扑的泥色，而是有了一层薄薄的血色。她的眼睛还是那么亮——但亮的性质变了。不再是求生本能的光，而是一种更柔软的、属于日常生活的光。]

**小椿：** 我画了一幅画！给你的！

[她把那张地图背面的画双手举到冰華面前。]

[冰華低下头，看向那幅画。]

[画面中央是一座城堡。用炭条画的——线条歪歪扭扭、比例完全不对，城墙画得太厚，塔楼画得太矮，护城河像一条弯弯曲曲的蚯蚓。但它毫无疑问是银冠要塞——因为最高的塔顶被涂成了深色，小椿在旁边标注了歪歪斜斜的三个字："银色的"。]

[但这座城堡和真正的银冠要塞有一个巨大的不同。]

[它有一扇门。]

[一扇很大的门。画在城墙的正中央。门是打开的——两扇门板向两侧敞开，像张开的双臂。]

[门的里面和外面都画了人。里面的人在走出来，外面的人在走进去。所有的人都是简单的火柴人，但每个人的头上都有一个圆圆的圈——小椿画的笑脸。]

[门的底部——门槛的位置——画了花。很多花。红的、黄的、紫的。花从门槛下面长出来，沿着打开的门板攀爬而上，一直爬到城墙的顶端。]

[城墙顶上站着一个人。那个人比其他火柴人大得多，旁边画了一面巨大的方形——盾。银白色的长头发被画成了一条飘在风中的长线。]

[那是冰華。]

[冰華看着这幅画。很久。]

**冰華：** ……这是银冠要塞。

**小椿：** 嗯！我画了你的城堡！

[冰華的目光停留在那扇打开的门上。]

**冰華：** 门……是开着的。

**小椿：** 嗯！

**冰華：** 银冠要塞的门不是这样的。真正的城门——在战时是关着的。

[小椿歪了歪头。]

**小椿：** 我知道呀。但是关着的门不好看。

**冰華：** ……

**小椿：** 关着的门看不到里面的人。外面的人也进不来。

**冰華：** 那是为了保护——

**小椿：** 可是花也进不来呀。

[冰華的话断在了半句。]

[小椿指着画上门槛处盛开的花。]

**小椿：** 你看，我画了好多花。花要从门底下长进来，还要从门上面爬上去。如果门关着的话，花就只能在外面。城堡里面就没有花了。

**冰華：** ……

**小椿：** 没有花的城堡，住在里面会很无聊的吧？

[她抬头看着冰華，表情是那种孩子特有的、完全不觉得自己在说什么了不起的话的天真。]

**小椿：** 所以我把门画成开着的。这样就好看了。我把你的城堡画得更好看了哦！

[冰華没有说话。]

[她的目光从画上的花移到打开的门，从打开的门移到门内外来来往往的笑脸火柴人，从火柴人移到城墙顶端那个长头发的人——自己。]

[画中的自己站在城墙上，脚下是花，身后是笑脸，面前的门是敞开的。]

[和现实中的自己完全相反。]

[现实中，她站在城墙上，脚下是裂缝，身后是冰冷的要塞，面前的门是紧闭的。]

*——孩子画的世界。*

*——没有关着的门。*

*——门是开着的。花从门缝里长出来。*

*——多简单的事情。*

*——为什么一个七岁的孩子觉得理所当然的事，我花了三十二年都没有想过？*

[旁边的妇人甲走过来，看到了小椿手中的画，面露歉意。]

**妇人甲：** 女伯爵大人，实在抱歉——小椿不该浪费您的地图纸——

**冰華：** 不是浪费。

[她的声音比平时更轻。不是刻意压低——是自然而然地变轻了。]

**冰華：** ……可以给我吗？

[小椿用力点头。]

**小椿：** 本来就是给你的呀！

[冰華伸出手。那只能举起两百斤巨盾的手，再一次极其小心地——如同Event 03中接过白花时一样——接过了那张薄薄的纸。]

[纸很轻。比箭矢轻。比碎石轻。比风轻。]

[但冰華接过它的动作，仿佛它比巨盾还重。]

**冰華：** ……谢谢。

[第二次。这是银冠守军们从未听过——现在听过两次的两个字。]

[小椿笑了。露出缺了一颗门牙的笑容。然后转身跑回窗台下面，大约是要开始画下一幅画了。]

[冰華站在原地。手中捧着那幅画。仓库里的炭火噼啪作响。孩子们在木箱后面追着老猫跑。妇人们在分发晚餐的口粮。灯油的光芒温暖而昏黄。]

[深夜。冰華的私人居室。]

[说是居室，不如说是哨位。一间不到十个榻榻米大小的石砌房间，靠近内城城墙最高处。除了一张硬板床、一把木椅、一张桌子，几乎没有任何私人物品。墙壁上钉满了地图、防线布置图、巡逻轮值表、物资清单——银冠要塞的防务全貌，被冰華用墨线和图钉密密麻麻地标注在每一寸可用的墙面上。]

[这间屋子看起来不像一个人住的地方。它像一座要塞的大脑。]

[冰華坐在桌前。桌上摊开着今天的巡逻报告和明天的防务安排。但她没有在看那些。]

[她在看那幅画。]

[她将画展开，铺在巡逻报告上面。炭条画的线条在烛光下显得柔和而模糊——和周围墙壁上精确到步数的防线图形成了奇异的对比。]

[然后她站起身。]

[她走到床头的墙壁前。那面墙是整个房间里文件最密集的区域——银冠要塞的核心防御配置图就钉在那里。内城城门的结构图、城墙薄弱点分析、紧急撤退路线规划……全部用冰華的笔迹标注，每一个数字都精确到小数点后两位。]

[她从桌上拿起一条细麻绳。]

[将小椿的画挂在了核心防御配置图的旁边。]

[右边是一个七岁女孩画的城堡——歪歪扭扭的、花从门底下长出来的、门是打开的。]

[它们并排挂在一起。]

[冰華退后一步，看着这面墙。]

*——同一座城堡。*

*——一幅画了怎么守住它。*

*——另一幅画了怎么打开它。*

*——十二年来我只看过第一幅。*

*——今天开始……第二幅也在了。*

[她躺到硬板床上。侧过身。]

[视线的尽头，烛火的光芒刚好落在那幅画上——落在那扇打开的门、那些攀爬着的花、和那个站在城墙顶端长发飘飘的火柴人身上。]

[不知道过了多久，她闭上了眼睛。]

[嘴角有一丝极其微弱的弧度。不是笑。只是——冰面上的裂痕又深了一点。]

[而这一次，裂痕里透出来的不是冷风。]

[是光。]

[CG触发：「孩子的画」——深夜。冰華的居室。画面采用暖色调，烛光从画面右下角照来。墙壁上密密麻麻的军事图纸占据了背景，线条精密而冰冷。正中央，小椿的画被一条麻绳挂在防御配置图旁边——色彩斑斓的孩童笔触与周围单色的军事文件形成强烈反差。画中那扇打开的门、攀爬的花朵、来往的笑脸火柴人，在烛光中仿佛有了自己的生命。冰華的手出现在画面下方——她正伸出一根手指，极轻地触碰着画中那扇敞开的门。手指上有旧伤疤和老茧，指甲剪得极短。窗外是银冠要塞深蓝色的夜空和城墙的剪影。在军事地图与孩子的画之间——在坚硬的世界与柔软的世界之间——冰華的手是唯一的桥梁。]

## Event 10 — 据点攻略「银冠要塞」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.26-30 |

### 场景描写

银冠要塞。内城。

这是最后一战。

暗潮军团的旗帜已经插遍了外城的每一面墙壁。外城的三道防线——冰華花十二年设计、建造、演练的纵深防御体系——已在此前的战役中被逐一突破。护城河干涸了，投石车的残骸散落在城门前的空地上如同巨兽的遗骨，城墙外壁上密密麻麻的攻城痕迹让花岗岩看起来像被无数只手撕扯过的旧布。

但内城还在。

银冠要塞的内城是这座堡垒真正的核心——城中之城，墙中之墙。外城是皮肉，内城是骨骼。内城墙比外城墙矮五丈但厚三丈，每一块石砖都用铁箍加固，墙面上覆着一层薄薄的银白色合金——这就是"银冠"之名的由来。在冬日的晨光中，内城闪烁着寒冷的金属光泽，如同一座被冻结的王冠。

晨雾中，玩家的军队在内城门前集结。攻城器械已就位，弓箭手拉满了弓弦，步兵方阵的盾墙在薄雾中整齐排列。

城墙上——

出奇的安静。

没有往日那种密不透风的盾墙。没有层层叠叠的长矛。没有从每一个垛口探出的弩箭。

城墙上的守军——少得不正常。

### 剧情对话

[玩家指挥部。斥候急匆匆送来最新侦察报告。]

**斥候：** 报告——内城守军数量异常。

**玩家副官：** 什么意思？

**斥候：** 根据此前情报，冰華的内城守军应有精锐亲卫四十人、弓兵二十人、重装步兵六十人，加上固定防御设施和城墙弩。但目前观测到的城墙上……

[他看了看手中的报告，又看了看城墙方向，好像自己也不太相信。]

**斥候：** 不到二十人。

**玩家副官：** 二十？一百二十变成二十？

**斥候：** 属下反复确认了三次。城墙上确实只有不到二十名守军。其中半数以上是轻装——没有穿全甲。弩车只剩一架。固定弩全部被拆除了。

**玩家副官：** 拆除？被我们摧毁的？

**斥候：** 不是。是被他们自己拆的。拆卸痕迹是从内侧开始的——螺栓是用工具取下的，不是被炮火打断的。

[副官沉默了一会儿。]

**玩家副官：** ……其余的士兵去哪了？

**斥候：** 据我们在西路的眼线报告——过去五天内，有大批银冠守军以小队形式从西面山路离开。没有携带武器，只带了个人物品和少量干粮。看起来不像是突围或转移阵地……

**玩家副官：** 像什么？

**斥候：** （犹豫）像……回家。

[长久的沉默。]

**玩家副官：** 她放他们走了。

**斥候：** 是的。铁壁女伯爵——遣散了自己大部分的守军。

**玩家副官：** 在最后一战之前。

**斥候：** ……是。

[副官走到地图前，看着银冠要塞内城的布防图。此前密密麻麻的兵力标注，现在大半需要被划掉。]

**玩家副官：** 她为什么这么做？

**斥候：** 审讯被遣散的守军后……他们说女伯爵召集了全体会议。她说——

[斥候翻了翻审讯笔录。]

**斥候：** 她说："这场战争不是你们的。银冠要塞的兵，大多是附近村镇征召的农夫和工匠的儿子。你们的父母还在等你们回去春耕。走吧。城墙不需要这么多人。"

**玩家副官：** ……然后呢？

**斥候：** 然后有人问："那女伯爵您呢？"她说——

[斥候又看了看笔录。]

**斥候：** "我是城墙。城墙不走。"

[副官闭了闭眼睛。]

**玩家副官：** 她留了多少人？

**斥候：** 留下的都是自愿的。那几个亲卫——跟了她最久的人。还有霜。其余全部遣散。

**玩家副官：** ……她知道我们今天会攻城。

**斥候：** 她知道。

**玩家副官：** 她知道这些人留下来会死。

**斥候：** 她知道。所以她不让他们留。

[城墙上，冰華的身影出现在最高处的雉堞之间。和Event 01一模一样的位置。银白色长发在北风中散开。巨盾竖在身旁。]

[但这一次，她身后的城墙几乎是空的。]

[没有举盾的守军。没有密不透风的防线。没有四十三次攻城战未失守的底气。]

[只有她，和几个不肯走的人。]

*——最后一次了。*

*——一百四十七年。四十三次攻城战。*

*——第四十四次。*

*——也是最后一次。*

[冰華将巨盾从地面提起。盾面上的伤痕比Event 01中更多、更深——这些天的战斗在旧伤痕上叠加了新伤痕，层层叠叠，已经看不出原本的钢面颜色。]

[她只是看着城下的暗潮大军——看了很久——然后转头看了一眼身后的内城。]

[内城里，安置所的方向隐约传来人声。是平民们。小椿们。那些被她收容在城墙之内的人。]

[她转回头。]

**冰華：** （对身旁的霜，声音很轻）最后守一次。

**霜：** ……是。

**冰華：** 不是为了城墙。

**霜：** 是。

*——是为了让他们再多安全一会儿。*

*——在他们决定下一步怎么走之前。*

*——「不动之山」。我最后一次用这个了。*

*——不是因为用不了。是因为——*

*——山也该动了。*

[第一回合。暗潮军团的攻城锤撞击内城城门。城门剧烈震动，铁箍发出刺耳的呻吟。冰華站在城门正上方，巨盾挡下了瞄准城门指挥的箭雨。]

[第二回合起，冰華的防御回落到正常水平。箭矢开始有零星穿过她的盾防——以前这不可能发生。守军太少，覆盖面不够，侧翼是空的。]

**冰華：** 我知道。

[她的战斗方式和以前完全不同了。以前的冰華是城墙——站在一个位置，挡住一切。今天的冰華是一个人——在整面城墙上来回移动，哪里缺人就堵哪里，像一块不断被挪动的砖。]

**冰華：** 再撑三回合。

**重装亲卫甲：** 三回合——

**冰華：** 三回合够了。够他们做出选择。

[亲卫不明白"他们"是谁，"选择"是什么。但他没有多问。跟了冰華八年，他学会了一件事：女伯爵说够了，就是够了。]

[HP低于40%时——]

[冰華的铠甲已是千疮百孔。右肩甲完全脱落，露出被鲜血浸透的衬甲。左腿膝甲被砸歪，每走一步都会发出金属摩擦的刺耳声。巨盾的表面已经凹陷得看不出原来的形状——那不再是一面盾，更像一块被反复锤打的废铁。]

[但她还站着。]

[站在城墙上。]

*——不是"守不住了"的够了。*

*——是……该够了。*

[但这一次的「咚」声不同了。]

[不是"不可撼动"的声音。]

[是"落幕"的声音。]

### 战后

[银冠要塞内城门被突破。暗潮军团涌入。]

**冰華：** 城墙是我的。陷落也是我的。但城里的人——

[她闭了闭眼睛。]

**冰華：** 城里的人不是城墙的一部分。他们是人。请当他们是人。

## Event 11 — 角色事件·伍「打开的城门」

| 项目 | 数据 |
|------|------|
| 类型 | 剧情 |
| 推荐等级 | — |
| 情感基调 | 百姓的选择——不是将军放弃了城堡，是百姓亲手打开了城门 |
| 触发条件 | Event 10 战斗进行中自动触发（第七回合开始时插入） |
| 特殊机制 | 无战斗；CG解锁；此事件为Event 10攻城战中的剧情插入段，触发后攻城战直接结束 |
| 奖励 | 350 EXP, CG「打开的城门」 |

### 场景描写

银冠要塞内城。攻城战第七回合。

城墙上的战斗仍在继续——但城墙之下，内城的街道中，一场没有人预料到的事情正在发生。

安置所的大门被推开了。

不是从外面打开的。是从里面。

三百多名平民——老人、妇人、孩子、曾经的农夫、铁匠、牧羊人——从东区仓库中涌出，沿着内城的石板街道向城门方向汇聚。他们没有武器。没有铠甲。很多人甚至没来得及穿外衣，裹着安置所的粗毛毯就出了门。

但他们每个人手里都拿着什么东西。

白色的布条。有人撕开了床单，有人拆了绷带，有人用面粉袋翻了个面——白色的那面朝外，举在手中，举过头顶。

还有纸。很多纸。歪歪扭扭的字写在各种能找到的平面上——木板、陶片、布条、地图的背面。

"别打了。"

"我们想活。"

"开门。"

这不是军队组织的。不是谁下达的命令。这是三百多个被战争碾碎又被收容在城墙内的普通人，自己做出的决定。

城门。

### 场景描写

银冠要塞以南。开阔的旷野。

城门打开后的第二天。

消息传得很快——银冠要塞陷落、铁壁女伯爵投降的消息如同野火一般蔓延。对暗潮军团来说，这是一场无需流血的胜利。对天城王朝来说，这是北方防线最后一块拼图的崩塌。

但对枢密院来说——这是叛国。

天城王朝的枢密院在得知冰華放下武器、任由城门打开的消息后，第一反应不是哀叹要塞的丢失——而是愤怒。铁壁女伯爵，北境最后的守护者，未经允许便将王朝的堡垒拱手让人。在枢密院的文牍中，这叫做"弃城叛逃"。

一支讨伐军被紧急派出。不多——三百人左右的快速机动部队，由枢密院直属的督战官率领。他们的任务不是收复银冠要塞——那已经不可能了。他们的任务是"惩处叛将冰華"，以及——

"处理与叛将合谋的平民。"

这是枢密院的用词。"处理"。

讨伐军在黎明前抵达了银冠要塞以南的旷野。此时平民们正在暗潮军团的协助下从要塞中有序撤出，临时营地搭建在要塞南门外的平地上——帐篷、篝火、牛车。三百多人加上随行的暗潮后勤人员，乱哄哄地分配物资、安置老幼。

没有人预料到王朝的讨伐军来得这么快。

冰華预料到了。

[玩家部队在她身后展开防御阵形，协助拦截侧翼渗透和远程攻击。这是第一次——冰華和暗潮军团并肩作战。]

*——从来没想过会有这一天。*

*——和攻了我十二年城的人站在一起。*

*——保护被我守了十二年的人。*

*——但这感觉——*

*——比站在城墙上轻。*

[第一回合。讨伐军骑兵率先冲锋。四匹战马并排冲向冰華——马蹄践踏冻土的轰鸣如同密集的鼓点。]

[冰華没有移动。盾朝前。双脚钉在冻土中。]

[骑兵的长枪刺向盾面——冰華用巨盾格开两杆，第三杆擦过盾沿刺入她右臂——她咬紧牙关，用身体的旋转将长枪的力道卸向一侧。第四匹马试图从左侧绕过——]

**冰華：** 不许过。

[她一步横移，巨盾如同一扇移动的闸门，挡住了骑兵的迂回路线。马匹受惊后仰，骑兵摔落。]

[第三回合。枪兵方阵推进。六杆长枪同时刺来——冰華无法用盾挡住所有方向。三杆被格开，两杆刺中了她的侧腹和大腿，一杆被她用空出来的右手——没有拿武器的右手——直接握住了枪杆。]

[手掌被枪杆的倒刺割开。鲜血从指缝中流下来。]

[但那杆枪没有再前进一寸。]

**讨伐军枪兵：** （震惊）她用手——！

**冰華：** （握着枪杆，低声）空手接白刃。军中都学过。只是没人真的用过。

[她将枪杆折断。]

**冰華：** 因为很痛。

[第六回合。冰華的伤越来越多。没有城墙加成的她，防御力虽然极高但不是无限的。讨伐军从三面包抄——她只有一面盾。]

[但「活墙」的嘲讽效果让所有敌人都在攻击她——不是玩家部队，不是远处的营地。只有她。]

[她在用自己的身体吸收本该落在所有人身上的伤害。]

**玩家副官：** （在后方，对玩家）指挥官——她在替我们挡！我们必须协助！

[玩家部队协同冰華防守——弓箭手从后方覆盖射击，步兵封锁侧翼渗透。第一次真正的协同作战。]

[第九回合。讨伐军指挥官亲自率领精锐突击冰華的位置。他是一名骑兵出身的军官，枪术精湛。长枪连刺三击——第一击被盾挡，第二击冰華侧身闪开，第三击——]

[刺穿了她左肩的锁子甲。]

[枪尖从肩甲后方穿出。鲜血沿着枪杆流下来。]

[冰華低头看了一眼穿过自己肩膀的长枪。]

[然后——]

[她用左手握住了穿过自己身体的枪杆。]

[不让他拔出来。]

**讨伐军指挥官：** ——！！

**冰華：** （握着穿过自己肩膀的枪，声音不变）你的枪……现在是我的盾的一部分了。

[讨伐军指挥官用力拔枪——拔不出来。冰華的左手如同铁钳，将枪杆死死钳住。他越用力，枪杆就越往冰華的肩膀里嵌——但她的手没有松开。]

[她用自己的身体困住了敌人的武器。]

*——痛。*

*——很痛。*

*——但比起那些人受到的痛——*

*——这种痛，我扛得住。*

[第十一回合。讨伐军的攻势开始减弱——不是因为伤亡，而是因为士气。他们是王朝的正规军，训练有素，服从命令。但命令告诉他们的是"缉拿叛将"——眼前这个满身鲜血、一个人挡在旷野中、保护着三百平民的女人，看起来不像叛将。看起来像——]

[像一面墙。]

[一面没有城堡、没有地基、只凭自己的意志矗立在那里的墙。]

**讨伐军士兵甲：** （低声，对同伴）她一个人挡了我们十一轮……

**讨伐军士兵乙：** （低声）她身后都是老人和孩子。

**讨伐军士兵甲：** ……我们在做什么？

[第十二回合。讨伐军指挥官下令撤退——不是因为战败，而是因为继续进攻意味着必须踏过冰華的尸体，而他不确定自己的部下还愿意这么做。]

[战斗结束。]

[冰華还站着。]

[她的铠甲——不，她没穿铠甲。她的锁子甲和军袍已经被割得千疮百孔。肩膀上的枪伤在讨伐军撤退后被霜紧急包扎，但鲜血仍从绷带下渗出。右手因为徒手折枪而几乎无法握拳。左腿有两处刺伤。脸上有一道被飞石擦过的伤口，血迹从颧骨延伸到下巴。]

[但她的盾——巨盾——还举在身前。]

[从第一回合到最后一回合，没有放下过。]

[在没有城墙的旷野上，她用一面盾和自己的身体守住了所有人。]

### 战后

[讨伐军撤退后。旷野重归寂静。冬日的阳光终于穿透了云层，苍白但真实地落在冻土上。]

[冰華的膝盖终于弯曲了——她缓缓跪倒在地，巨盾支撑着她没有完全倒下。盾的底部深深嵌入冻土中，如同一棵在风暴后依然直立的枯树。]

[霜跑过来。身后是营地中被惊醒的平民们——他们此刻才意识到发生了什么。]

**霜：** 大人！您的伤——

**冰華：** （低声）他们……没过来吧？

**霜：** 没有。讨伐军撤了。平民都安全。

[冰華闭上眼睛。嘴角有一丝极微弱的弧度——也许是笑，也许是痛到极致的扭曲，分不清楚。]

**冰華：** ……那就好。

[营地的人们围过来。有人拿了毛毯，有人烧了热水，有人在找军医箱。]

**小椿：** 女伯爵大人——！你流了好多血——！

**冰華：** （睁开一只眼睛）……没关系。不痛。

[她又在说谎了。]

[小椿蹲下来，用袖子去擦冰華脸上的血——擦不干净，反而把自己的袖子也染红了。她的眼泪掉了下来。]

**小椿：** 你说过城墙不会痛的——

**冰華：** 我不是城墙。

[小椿抬头看她。]

**冰華：** 我只是一个人。人会痛。但——

[她伸出还能动的右手——手掌上全是枪杆割出的伤口——轻轻碰了碰小椿的头顶。]

**冰華：** 人也会选择替谁痛。

[周围安静了下来。平民们看着跪在地上的冰華，看着她身上数不清的伤口。]

[一个妇人突然哭了出来。]

[然后更多的人开始哭。]

[不是因为恐惧。是因为——]

[有人为了保护他们，把自己当成了城墙。而城墙——碎了。]

*——没有城墙了。*

*——但人还在。*

*——只要人还在——*

*——墙在哪里都一样。*

## Event 13 — 角色事件·陆「融冰」

### 场景描写

银冠要塞以南。临时营地。

黄昏。

讨伐军撤退后的第三天。冰華的伤已经被军医处理过了——左肩的贯穿伤最为严重，用了止血草和铁针缝合，绷带从肩部一直缠到胸口。军医说至少需要卧床二十天。冰華第二天就下了床。军医说你再不躺下伤口会裂开。冰華说："裂了再缝。"

但今天，她终于安静下来了。

不是因为听了军医的话。是因为——没有什么需要守的了。

讨伐军走了。暗潮军团的后勤部队已经开始接管银冠要塞的物资分配。平民们的安置方案正在落实——有人选择跟随暗潮军团前往南方的安全区域，有人选择回到被战火波及但已恢复平静的家乡，也有人决定留在银冠要塞附近的村镇重建家园。

一切都在有条不紊地进行。不需要城墙。不需要防线。不需要铁壁女伯爵。

冰華坐在营地中央的篝火旁。

篝火是给所有人的——不分军民，不分暗潮与银冠。天冷的时候，火不认识旗帜。谁靠近，谁就暖和。

她的坐姿很奇怪——对她来说很奇怪。不是军人的端坐，不是巡逻时的警觉姿态，也不是城墙上那种如同雕塑般的直立。她盘着腿，后背微微弓起，双手搭在膝盖上。铠甲已经完全脱掉了——不是锁子甲外面的全甲（那在Event 11中就脱了），而是连锁子甲也脱了。她穿着一件灰白色的粗布内衬——那件衬衣太薄了，能看到肩部绷带的轮廓。

没有铠甲的冰華——看起来比所有人想象的都要瘦。

巨盾靠在她身旁的地面上。阔剑不在——阔剑在Event 11中被放在了城门口的地上，之后她没有取回。

篝火的光芒在她脸上跳动。银白色的长发散在肩后，没有扎起来——第一次。火光将她苍白的脸颊映出一层温暖的橙色，如同冬天的冰面上倒映着的夕阳。

周围有很多人。平民们围着篝火取暖、聊天、给孩子喂饭。几名暗潮士兵坐在不远处啃干粮。霜在帮妇人们整理明天出发的行李。

冰華坐在他们中间。不是在人群的外围，不是在高处俯视，不是站在任何人的前面挡着什么。

就是——坐在中间。

和所有人一样。

### 剧情对话

[篝火噼啪作响。有人往火堆里加了新的柴——是拆掉的营帐支架，干燥的松木，烧起来有淡淡的松脂香气。]

[冰華盯着火焰。橙色的光在她灰蓝色的眼睛里跳动，像是冰面下有什么东西在缓慢地、不可逆转地融化。]

[一个老妇人从篝火另一侧走过来。冰華认出她——桦木村的人。Event 03中窝棚里的十一人之一。]

[老妇人手里端着一只粗陶碗。碗里是热汤——不是什么好汤。军用干肉、野菜、一点盐。但冒着白腾腾的热气，在冬天的傍晚，那热气本身就是一种奢侈。]

**老妇人：** 女伯爵大人。喝点汤吧。

[冰華抬头看她。]

**冰華：** ……给我的？

**老妇人：** 给你的。你受了那么重的伤，不喝点热的怎么行。

[冰華看着那碗汤。]

[她想起了Event 03——她把自己的干粮和水壶给了桦木村的人。"先给孩子们吃。大人分剩下的。"那时候她是给予者。施舍者。保护者。]

[现在有人给她一碗汤。]

[不是因为她是女伯爵。不是因为她是铁壁。不是因为她保护了他们。]

[只是因为天冷了，她受了伤，应该喝点热的。]

[这么简单的理由。]

[冰華伸出双手。]

[那双手——握过两百斤巨盾的手、徒手折断过长枪的手、在城墙上挡过十七发投石的手——此刻极其缓慢地、极其小心地接过了那碗汤。]

[陶碗的热度透过碗壁传到她的手掌。粗糙的、满是老茧和伤疤的手掌。右手手心还缠着绷带——那是Event 12中徒手抓枪杆留下的伤。绷带被汤碗的热气蒸得微微潮湿。]

[她的手在微微发抖。]

[不是因为伤。不是因为冷。]

[是因为——很久没有人给她递过东西了。]

[从她成为铁壁女伯爵的那天起——十二年——她永远是那个给予的人、保护的人、挡在前面的人。没有人给她递过东西。因为城墙不需要别人递东西给它。城墙只需要站在那里。]

[但此刻有人给她一碗汤。]

[她将碗捧到嘴边。喝了一口。]

[热汤从喉咙一直暖到胃里。那种温度——不是篝火的温度，不是铠甲内闷出的体温——是另一种温度。是别人的好意传递过来的温度。是"有人在乎你冷不冷"这件事本身的温度。]

[她喝了第二口。]

[第三口。]

[碗沿抵在她嘴唇上。热气模糊了她灰蓝色的眼睛。]

[旁边有个孩子——不是小椿，是另一个更小的男孩——走过来，好奇地看着她。]

**男孩：** 女伯爵大人在哭吗？

[冰華放下碗。]

[她没有在哭。]

[但她的眼眶——在篝火的映照下——确实比平时更亮了一些。不是泪。只是——冰在融化时会反光。]

**冰華：** 没有。汤太热了。

[她又在说谎了。]

[男孩歪了歪头，不太相信，但没有追问。他转身跑开了。]

[冰華继续喝汤。一小口一小口地。]

[这是她三十二年来第一次——真正意义上的第一次——不是在吃"行军口粮"或"战斗补给"。]

[她在喝一碗有人特意为她端来的热汤。]

[篝火的光芒在她脸上跳动。周围的人声渐渐变成了一种模糊的、温暖的背景音——像河水。像风穿过春天的树林。像很久以前、在她成为城墙之前的某个冬天。]

[小椿跑过来了。手里抱着一大卷东西——是冰華的灰色斗篷。从Event 03开始就一直在小椿身上的那件斗篷。]

**小椿：** 女伯爵大人！你的斗篷！还给你！

[她把斗篷铺开——大人尺寸的斗篷在她手里展开如同一面旗帜。]

**小椿：** 你受伤了，会冷的。我不冷了——他们给了我新毛毯！

[她指了指营地里暗潮后勤分发的灰色军毯。]

[冰華看着那件斗篷。它比Event 03中干净了一些——有人帮小椿洗过——但磨损得更厉害了。衣角被踩得发毛，袖口有缝补过的针脚，领口处有一块不明来源的污渍。]

[它看起来不像一件女伯爵的斗篷了。它看起来像一件被一个小女孩穿了一个多月的旧衣服。]

[冰華接过斗篷。将它披在肩上。]

[斗篷的温度和她记忆中不一样了。上次穿它的时候是Event 03——北风中脱下给小椿的时候。那时它带着她自己的体温。现在，它带着小椿的体温。]

[一种更小的、更软的温度。]

**冰華：** ……谢谢。

[第三次了。这个从不说"谢谢"的人，现在说了第三次。]

[小椿在她身旁坐下来。靠着她没有受伤的右侧。篝火的热度和孩子的体温同时传过来。]

[冰華没有动。]

[她只是坐在那里。捧着快要喝完的汤。披着带着小椿体温的斗篷。坐在篝火旁。身边是她曾经用城墙保护、如今用身体保护、此刻不需要保护只需要坐在一起的人。]

[霜走过来，想叫冰華回去休息。但看到那个画面后，她停住了脚步。]

[冰華坐在篝火旁。小椿靠着她的肩膀半睡半醒。碗里的汤喝完了，空碗搁在膝盖上。火光在她银白色的长发上镀了一层温暖的金。]

[然后霜看到了一样她从未见过的东西。]

[冰華在笑。]

[不是事后才意识到的那种"嘴角弧度"。是一个真正的、此刻正在发生的、霜亲眼目睹的笑容。]

[那不是战士完成任务后的冷硬满足。不是铁壁女伯爵压制住攻势后的微微点头。]

[那是一个——温柔的、几乎害羞的微笑。]

[嘴角上扬的弧度很小。眼睛微微眯起——不是因为火光刺眼，而是因为笑的时候眼睛就是会眯起来的。那个表情让她看起来不像三十二岁的铁壁女伯爵，像一个——]

[像一个普通人。一个在寒冷的冬夜喝到热汤时会感到幸福的普通人。]

[霜的眼泪毫无预兆地涌出来。她连忙转过身，用手背使劲擦了擦眼睛。]

**霜：** （小声，对自己）……十二年。我跟了她十二年……第一次看到她笑。

[篝火继续燃烧。风小了。星星从云层的缝隙中露出来——冬天的星星，冷而亮。]

### 场景描写

银冠要塞。

春天。

准确地说——是冬天和春天的交界处。那种白天已经开始变长、但夜里仍然会结冰的时节。空气中有一种奇特的混合气味：残雪的清冽、泥土解冻后的腥气、以及从远处山谷中随风飘来的、极淡极淡的——花的气息。

银冠要塞变了。

从外面看，最先注意到的是城墙——或者说，曾经是城墙。内城城墙的正面已经被拆去了一半。高三十丈的花岗岩墙体如今只剩下两侧的基座和底部几层石砖——像一排被拔掉了大半牙齿的下颌骨。拆下来的石砖被整齐地码在城门外的空地上，摞成一个个方方正正的石堆，表面还残留着银白色合金的光泽。

拆墙的工程还在继续。不是用攻城锤，不是用投石车——是用人手。一砖一砖地拆。每块砖都被小心地取下、编号、码放。不是破坏。是拆解。是一种近乎温柔的解构。

而主持这项工程的人——

冰華。

她站在半拆的城墙上方——那个曾经是她站岗位置的地方。脚下的石板还是那块石板，但视野已经完全不同了。以前从这里望出去，目光会被城墙挡住，只能看到城墙外面的敌人。现在城墙矮了，视线越过残存的墙基，可以看到更远的地方——旷野、远山、蜿蜒的小路、路上行走的人。

她没有穿铠甲。穿着一件新的灰蓝色粗布外衣——是暗潮后勤部发放的普通冬装。没有纹章，没有军衔标识。外衣下面是白色衬衫，领口松松地敞着。左肩的伤已经愈合，但隔着衣服还是能看到绷带的轮廓——军医说疤会一直在。

她的头发第一次扎起来了。不是军人的发髻——是一根简单的麻绳在脑后松松地束了一下。银白色的马尾在春风中轻轻摆动。

她的表情——

和一百四十七年的城墙一样——变了。

不再是矿物质般的冰冷。不再是无机质的平静。也不是Event 13中那个初生的、羞涩的微笑。

是一种更日常的表情。一种在干活干到一半、擦了擦汗、抬头看看天色的普通人会有的表情。专注，但不紧绷。平静，但不冰冷。嘴角没有明显上扬，但整张脸的线条都比以前柔和了——像一块棱角分明的石头被水流冲刷了很久之后，变得圆润。

["城堡"和城墙的区别——孩子们搭的"城堡"全部都有门。而且门是开着的。]

[冰華站在城门口。看着这一切。]

[银白色的马尾在春风中摆动。灰蓝色的眼睛里映着夕阳、花苗、来往的人群、和远处半拆的城墙。]

[她的脸上没有特别的表情。不是冰冷——冰已经融了。不是微笑——微笑是偶尔的事件。]

[是平静。]

[真正的平静。]

[不是城墙般的——硬的、拒绝的、因为无法被触及所以看起来平静的——那种假平静。]

[是水的平静。融化后的冰变成的水。可以被捧起来的、可以流淌的、可以灌溉花朵也可以倒映天空的——水的平静。]

*——银冠要塞。一百四十七年。四十四次攻城战。*

*——最后一次——不是攻城。是拆城。*

*——不是敌人拆的。是我自己拆的。*

*——一砖一砖。*

*——每拆一块，就轻一点。*

*——现在——*

*——好轻。*

*——风从那些缺口中穿过来。*

*——不是冬天的风了。*

*——是春天的风。带着泥土和花的气味。*

*——城墙挡了一百四十七年的风。*

*——原来风是这个味道的。*

[小椿跑来拉她的手。]

**小椿：** 冰華！吃饭了！今天有鱼汤！

[冰華低头看她。]

[牵着她的手。]

[走进那扇再也不会关上的城门。]

[门槛两侧的花在春风中轻轻摇摆。黄色的花瓣上沾着午后的露水——或者是夕阳的碎片，分不清楚。]

[身后，巨盾矗立在城门入口。]

[盾面上的字在夕阳中闪着微光——]

**此处无墙，此处有人。**

[——城墙拆了。门打开了。花长出来了。]

[——冰融了。]

[——但水还在。]
