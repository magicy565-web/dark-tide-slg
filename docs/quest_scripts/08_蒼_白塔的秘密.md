## Event 01 — 开战宣言「七塔共鸣」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.30-34 |
| 情感基调 | 大法师的绝对理性，学者审视标本般的冷漠 |
| 战斗条件 | 突破七塔外围结界（击破3座小型结界核心后进入主塔射程范围） |
| 特殊机制 | 蒼「七塔共鸣」每回合对全体造成INT×0.4魔法伤害；七塔地形：魔力场使非术师单位SPD-2，蒼自身INT+3 |
| 敌方配置 | 蒼(Lv.34) + 白塔法师×6(Lv.31) + 结界术士×4(Lv.29) + 奥术傀儡×3(Lv.30) |
| 奖励 | 500 EXP, 1200G, 蒼之书页×1, 解锁蒼档案第一页 |

### 场景描写

白塔同盟领地——大陆中央高原上矗立着七座白色尖塔。

它们并非人类建造的。四百年前，第一代大法师们在这片魔力浓度最高的高原上发现了天然的魔力结晶柱，然后围绕它们修建了塔身。七座塔呈北斗七星之形排列，塔尖之间有肉眼可见的蓝色魔力弧光相互跳跃、连接，如同一张覆盖整个高原的闪电之网。

空气中弥漫着臭氧与古老羊皮纸混合的气味——那是高浓度魔力与知识的味道。地面上的石板被刻满了符文，每一块都是一个微型法阵，踩上去会发出微弱的蓝色荧光。草木在这片土地上无法生长，取而代之的是一种被称为"思维苔"的蓝色菌类，它们会根据周围魔力的波动改变颜色和形态。

此刻，七座塔同时亮起了蓝光。

从塔尖到塔基，符文逐层点亮，像是有人从最高处倒下一杯蓝色的液体，让光芒沿着塔壁缓缓流淌而下。七道蓝色光柱冲天而起，在高空汇聚成一个巨大的七芒星法阵。天空被染成了深蓝色，仿佛白昼骤然变为黄昏。

在主塔——也是最高的那座塔的塔顶平台上，一个身影悬浮在半空中。

她盘腿而坐，距离平台地面约三尺。深蓝色的法师长袍在无风中飘动，袍角绣着的银色符文自行发光。她的黑发极长，垂到腰际以下，发丝间缠绕着微小的蓝色光点——那是渗入发丝的纯化魔力。一本翻开的古籍悬浮在她面前，书页在自动翻动。

蒼·阿尔卡迪亚。白塔同盟大法师。七塔之主。

她抬起眼帘。深蓝色的瞳孔中映出了高原边缘逐渐逼近的暗潮军队，如同显微镜下观察到一群向培养皿边缘爬行的微生物。

### 剧情对话

[蒼悬浮在主塔顶端。她面前的古籍自动合上，缓缓飘到她身侧。她伸出右手，食指和中指并拢，在空中划出一道光弧——远视术。]

[高原边缘的景象被放大数十倍，投影在她面前的空气中。暗潮军队的旗帜、阵型、兵种构成——一切尽收眼底。]

**蒼：** ……

[她注视了大约十秒，然后轻轻偏了偏头。不是惊讶，不是紧张——是一种学者特有的评估。]

**蒼：** 有意思。

[身后传来急促的脚步声。一名穿着白袍的年轻法师匆匆跑上塔顶平台，气喘吁吁。]

**学徒赫尔：** 大法师！暗潮军——暗潮军已经越过了外围警戒线！第三哨塔报告，先锋部队约八百人，以重装步兵为主体，后方有——

**蒼：** 八百四十二人。重装步兵四百六十，轻步兵二百一十，弓手一百二十，另有骑马斥候五十二人在侧翼游弋。后方跟随的辎重车辆中至少有两辆装载了反魔力装置——外壳覆盖了磐石矿石的涂层，用于吸收和消散低阶魔法。

[她连眼睛都没有从远视术投影上移开。]

**蒼：** 另外，他们的行军速度是每刻钟三百二十步，考虑到高原地形的坡度修正，预计在两个时辰后抵达七塔外围结界的有效范围。

[赫尔张着嘴，一时说不出话。]

**蒼：** 赫尔。

**学徒赫尔：** 是、是的，大法师！

**蒼：** 你跑上来报告这件事的时间，我原本打算用来读完《奥术微粒论》第四卷的最后三十七页。

[赫尔的脸涨得通红。]

**蒼：** 不过——没关系。那本书我已经读过十一遍了。

[她挥了挥手，远视术投影消散。然后她缓缓降落到平台地面上，法师袍的下摆拖过符文石板，发出细微的沙沙声。]

**蒼：** 来犯者有两种——无知者和绝望者。无知者不知道白塔的力量，所以敢来；绝望者知道，但已经没有别的选择。

[她走到平台边缘，俯瞰高原。七座塔的蓝光将她的影子拉得极长。]

**蒼：** 不论哪一种，结果都是一样的。

[她抬起右手。]

*——七塔共鸣。这是白塔同盟建立以来就存在的防御体系。七座塔互为节点，共享法力池中的魔力，形成覆盖整个高原的立体防御网。在这片土地上，每一个魔法的效率都被放大三倍——前提是，施术者是白塔的法师。*

**蒼：** 「七塔共鸣——启动。」

[她的声音不大，但通过魔力传导，同时响彻七座塔的每一个角落。]

[七座塔顶的蓝色光柱骤然增亮。塔与塔之间的弧光变成了粗壮的光束，在天空中编织出一张密不透风的网。地面上的符文石板全部亮起，整个高原如同一块巨大的发光电路板。]

[空气开始震颤。蒼的长发在魔力涌动中飞扬，袍角猎猎作响。她的双眼从深蓝变为纯白——那是七塔的魔力正通过她的身体进行汇聚和分配。]

**蒼：** （低语）……法力池状态正常。七塔节点同步率99.7%。结界强度——

[她停顿了一下。]

**蒼：** 足够了。

[她睁开眼睛，纯白的瞳孔直视远方的暗潮军队。]

**蒼：** 我给你们上一课。

[她张开双臂。主塔顶端凝聚出一个巨大的蓝色魔法阵，直径超过塔顶平台本身，悬浮在半空中缓缓旋转。]

**蒼：** 课程名称——「奥术理论·序章：为什么不应该挑战一个拥有无限法力的術師」。

[魔法阵释放出一道巨大的蓝色光束，精准地落在暗潮先锋部队前方五十步的地面上。爆炸掀起的土石飞到了十丈高空。那不是攻击——那是警告。]

**蒼：** 第一条——火力差距。你们所见的，只是七塔总输出的百分之一。

[第二道光束落下，在第一个弹坑旁边炸出更大的坑洞。]

**蒼：** 第二条——精度。我刚才瞄准的是地面上一块特定的石头。直径三寸。距离两千四百步。

[暗潮军队的推进明显减缓了。但没有停下。]

**蒼：** ……

[她微微眯起眼睛。]

**蒼：** 有趣。要么你们的指挥官很勇敢，要么——他比我想象的更了解白塔。

[她收回双臂，魔法阵消散。]

**蒼：** 那好。既然选择了参加考试——

[她从袍袖中取出一根水晶杖。杖身是透明的六棱柱，内部封存着流动的蓝色液体。杖顶镶嵌的宝石在七塔共鸣的魔力中疯狂闪烁。]

**蒼：** 就别指望我会手下留情。

*——贤者从不愤怒。愤怒是低效的情绪，消耗精力却不产生结果。贤者只会做出判断——而判断，往往比愤怒更加残酷。*

### 战斗阶段

[战斗开始。蒼悬浮在主塔上方，七塔共鸣持续生效。「七塔共鸣」发动——每回合对全体敌方单位造成INT×0.4魔法伤害，无法闪避。]

*——白塔同盟的战争方式与其他势力截然不同。没有冲锋，没有近身格斗，甚至没有战线。只有从天而降的魔法——精确、高效、毫无感情。就像一位外科医生在手术台上操作一样。*

[第1回合，暗潮重步兵举盾推进。蒼观察他们的阵型——标准的龟甲阵，金属盾牌顶部覆盖了磐石矿石涂层。]

**蒼：** 磐石矿石。能吸收低阶魔法的动能。设计合理——如果你们面对的是普通法师的话。

[她单手举杖，杖顶宝石射出一道细如发丝的蓝色光线。光线没有击中盾牌——它穿过盾与盾之间不到两指宽的缝隙，精准地击中了一名步兵的护膝关节。]

**蒼：** 磐石矿石的吸收效率与接触面积成正比。缝隙——是你们自己留下的。

[第2回合，暗潮弓手在后方展开，试图用箭矢射击塔顶的白塔法师。箭矢飞到半空中，撞上了透明的结界屏障，像是撞上了一面看不见的玻璃墙——箭头粉碎，箭杆折断，碎片纷纷扬扬地落下。]

**蒼：** （叹气）……这就是为什么我说来犯者要么无知，要么绝望。结界的魔力密度是每平方尺三千奥姆，普通箭矢的动能连它的百分之一都无法突破。

[第3回合，暗潮方面推出了反魔力装置。一辆覆盖着厚重磐石矿石装甲的战车缓缓前进，车上的锥形装置发出低沉的嗡鸣声——那是在制造局部的魔力真空区域。]

**蒼：** ……

[她的表情第一次产生了微妙的变化——不是担忧，而是学术层面的兴趣。]

**蒼：** 反魔力装置。利用磐石矿石的谐振特性制造局部法力枯竭区。有效半径——大约三十步。在这个范围内，低于四阶的魔法会被完全吸收。

[她微微前倾身体。]

**蒼：** 好问题。那么——五阶以上呢？

[她举起水晶杖，杖顶凝聚出一团紫蓝色的光球。那颗光球的魔力密度之高，使得周围的空气开始扭曲，像是高温下的热浪。]

**蒼：** 「奥术轰击——五阶·纯化」。

[光球射出。它穿过反魔力装置制造的枯竭区——设备疯狂运转，磐石矿石发出红热的光芒，试图吸收这道魔法。但光球的能量远超它的承载上限。三秒后，装置过载爆炸。战车被掀翻，零件散落一地。]

**蒼：** 答案是——装置过载。超过吸收阈值的魔法不但不会被消散，反而会在矿石内部引发连锁反应，导致输出能量大于输入能量。

[她推了推鼻梁上并不存在的眼镜——一个思考时的习惯动作。]

**蒼：** 你们带来的反魔力装置，在我面前反而成了额外的炸弹。

[受到重击时——]

**蒼：** ！……物理攻击？直接绕过结界的薄弱节点？

[她低头看向伤口处袍子上的裂痕，然后重新看向战场。]

**蒼：** ……记录：暗潮军对结界节点存在先期侦查。战术评估上调一个等级。

### 胜利结算

[战斗结束。暗潮先锋被击退至高原边缘。蒼缓缓落回塔顶平台。]

**蒼：** 赫尔，战损报告。

**学徒赫尔：** 是！我方——零伤亡。结界能量消耗约百分之七。敌方遗留残骸已由第四塔回收组处理。

**蒼：** 零伤亡。

[她重复了一遍，语气平淡得像在朗读一份实验报告。]

**蒼：** 预期之内。但他们的反魔力装置……

[她沉默了片刻。]

**蒼：** 记录：需要调查暗潮军的磐石矿石供应链。这种规模的反魔力装置不应该出现在一支先锋部队中。他们在做准备——认真的准备。

[她挥手关闭了塔顶的强化法阵，七塔共鸣降回待机状态。]

**蒼：** 有趣的对手。

[她拿起之前漂浮在身侧的古籍，翻到之前被打断的那一页。]

**蒼：** ……但也仅此而已。

*——蒼·阿尔卡迪亚。四十三岁。在白塔求学三十一年，从学徒到大法师。她背诵了白塔图书馆中超过七万册典籍中的六万三千册。她相信知识是世界的答案，魔法是知识的具现，而白塔——是知识的殿堂。*

*——她还不知道，殿堂的地基下埋着什么。*

## Event 02 — 前哨遭遇「贤者的试炼」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.31-35 |
| 情感基调 | 学者出题考验，三重试炼的冷酷美学 |
| 战斗条件 | 依次通过火之回廊、冰之迷宫、雷之阵列三道试炼（每道试炼为独立阶段） |
| 特殊机制 | 三阶段连续战斗，每阶段有独立环境效果：火之回廊（每回合全体受火焰伤害INT×0.2）、冰之迷宫（SPD降低50%且随机冻结1单位）、雷之阵列（移动时触发落雷INT×0.5） |
| 敌方配置 | 阶段一：火焰傀儡×6(Lv.31) + 炎术士×2(Lv.32)；阶段二：冰霜傀儡×6(Lv.31) + 冰术士×2(Lv.32)；阶段三：雷电傀儡×6(Lv.32) + 蒼(Lv.35, 远程支援，不可击败) |
| 奖励 | 600 EXP, 1400G, 三元素碎片×1, 解锁蒼档案第二页 |

### 场景描写

七塔外围的第一道防线——三色回廊。

这是白塔同盟在四百年间精心设计的入侵者筛选系统。三条并列的通道从高原边缘延伸到七塔核心区域，每一条都长达两里。通道的墙壁是半透明的魔力结晶，内部流淌着不同颜色的能量：左侧通道赤红如熔岩，中间通道冰蓝如寒冬，右侧通道金紫如雷霆。

这不是防御工事。这是考场。

蒼设计这套系统时年仅二十六岁，那时她刚刚成为白塔历史上最年轻的高阶法师。她的设计理念简洁到近乎残酷——入侵者必须依次通过三种极端环境，每一种都针对不同的弱点。火考验耐力，冰考验判断，雷考验速度。

能全部通过的人，证明他们至少值得蒼亲自面对。

通不过的人——通道会自动将他们传送回高原边缘。活着的。蒼不喜欢不必要的死亡，她认为那是"数据浪费"。

此刻，三色回廊的入口同时亮起。赤、蓝、紫三道光芒将高原边缘的岩石地面染成斑斓的色块。通道入口处的空气因为魔力密度差异而产生了可见的折射——像是三面扭曲的镜子，倒映着暗潮军队犹豫不前的身影。

蒼的声音通过魔力传导，在三条通道中同时回响。

### 剧情对话

[蒼的身影以全息投影的形式出现在三色回廊入口上方。她端坐在一把凭空凝聚的魔力椅子上，手中拿着一本翻开的笔记。]

**蒼：** 你们还在。

[她翻了一页笔记。]

**蒼：** 说实话，我以为昨天的演示已经足够清晰了。火力差距、精度、结界强度——每一项数据都指向同一个结论：你们应该撤退。

[她合上笔记，直视下方。]

**蒼：** 但你们没有。这意味着两种可能——第一，你们的指挥官无视数据；第二，你们有某种我尚未掌握的变量。

[停顿。]

**蒼：** 不论哪种，都值得我进一步收集样本。

[她站起身来。魔力椅子消散。她伸出一只手指向三色回廊。]

**蒼：** 所以——我准备了一场考试。

**蒼：** 这是白塔同盟的「三重考验」。火、冰、雷——三种最基础的元素魔法构成的筛选环境。四百年来，只有十七个人完整通过。其中十二个后来成为了白塔的法师，三个成为了受白塔承认的盟友，剩下两个……

[她微微停顿。]

**蒼：** ……成为了值得纪念的对手。

[暗潮指挥官在队伍前方高声下令。士兵们开始分成三队，准备同时进入三条通道。]

**蒼：** 哦？分兵三路同时突破？

[她歪了歪头，那个动作中有一丝——只有一丝——被取悦的意味。]

**蒼：** 有趣。大多数挑战者选择集中兵力逐个突破。分兵意味着你们对自己的综合素质有信心……或者你们知道三条通道之间存在时间联动机制。

[她的表情略微凝重了半秒。]

**蒼：** 看来你们确实做了功课。好——那就开始吧。

[她打了个响指。三色回廊的入口同时敞开。]

### 战斗阶段·第一阶段：火之回廊

[暗潮第一队进入赤红色的通道。瞬间，温度飙升。]

**蒼：** （投影声）火之回廊的规则很简单。前进或者倒下。傀儡不会追击撤退者——但通道的温度每过一刻钟就会上升一个等级。

**蒼：** （投影声）提示：火焰波的传导速度与地面材质有关。石板快于泥土，泥土快于沙砾。观察地面——那里有答案。

[暗潮第二队进入冰蓝色的通道。与火之回廊相反，这里的温度骤降到极寒。]

[通道内部并非直线，而是一座不断变化的冰晶迷宫。透过半透明的冰墙，可以隐约看到其他路径上的身影，但无法分辨是友军还是冰霜傀儡的倒影。]

**蒼：** （投影声）冰之迷宫的规则——找到出口。迷宫每一刻钟重组一次。你有三次机会。

[冰霜傀儡从冰墙中无声地剥离出来。它们的外表和冰墙完全一样——只有在移动的瞬间才能看出区别。]

[第1回合，暗潮士兵们在迷宫中艰难前行。地面是一层薄冰，每一步都可能滑倒。更糟糕的是，寒气正在侵入他们的四肢——SPD持续降低。]

**暗潮队长乙：** 该死的……手指快没知觉了……

[第2回合，一名暗潮士兵被随机冻结——一层薄冰瞬间覆盖了他的全身，将他定在原地。其他人必须花费行动力去破除冰封。]

**蒼：** （投影声）提示：冰的本质是水的静止状态。想让冰停止扩散——找到热源。你们自己的体温也是热源。

[暗潮士兵们开始聚拢在一起，用彼此的体温对抗寒气。一个聪明的军士点燃了随身携带的油布——火光在冰晶迷宫中折射出无数光点，同时让周围的冰壁微微融化，暴露出了隐藏的路径。]

**蒼：** （投影声）……合格。利用火光的折射来透视冰墙结构。这个方法……我设计迷宫时没有预料到。

**蒼：** 记录：冰之迷宫存在火光折射漏洞，需要后续修补。编号——缺陷报告#4,217。

**暗潮队长丙：** 不要踩金线！找空隙！

[但空隙极小，且不断变化——每次击败一个雷电傀儡，阵纹就会重新排列。]

[第2回合，暗潮方一名老兵注意到了规律——阵纹的变化遵循某种周期。每三次重组后会回到接近初始状态的分布。]

**暗潮老兵：** 每三轮一个循环！数着走！一、二、三——现在！左侧安全！

**蒼：** （投影声）看出周期了？用了两轮。比我预期的三轮要快。

**蒼：** 考试的规则也是会变的。适应——或者失败。

[暗潮士兵们在不断变化的雷阵中艰难推进。每前进一步都伴随着电闪雷鸣。但他们没有停下。]

### 三阶段汇合

[三支队伍几乎同时抵达了通道的终点——一片圆形的白石广场。广场中央，蒼的全息投影消失，取而代之的是她本人。]

[她站在广场中央，水晶杖拄在地面上。身后，七塔的蓝光映出她纤细而挺拔的剪影。]

**蒼：** ……

**蒼：** 在我的记录中，这个通过率排名第四。

[停顿。]

**蒼：** 前三名——分别是四百年前的建塔者、两百年前的远征军，以及……

**蒼：** 总之。你们通过了初试。

*——蒼看着那些在火中忍耐、在冰中互助、在雷中思考的士兵们。她在心中给出了一个评价。*

*——"不是无知者。"*

*——她不确定这让她感到安心还是警惕。*

**学徒赫尔：** 大法师，三色回廊能量消耗偏高。法力池补充需要——

**蒼：** 六个时辰。我知道。

**蒼：** 记录下来：暗潮军的适应能力超出预期模型。火之回廊的沙砾观察、冰之迷宫的火光折射、雷之阵列的周期识别——三项都在两轮内完成。这不是普通军队的素质。

[她合上笔记。]

**蒼：** 要么他们的训练体系非常出色——要么有人给过他们关于三色回廊的情报。

**学徒赫尔：** 大法师，您的意思是……有内鬼？

**蒼：** 我的意思是，任何可能性在被排除之前都值得考虑。

**蒼：** 加强法力池的输出频率。下一次，我不会再用考试的方式了。

*——"他们在火中互相扶持。在冰中分享体温。在雷中彼此提醒。这些人——是为了彼此而战。"*

*——她不知道为什么要记录这个。但她记录了。*

### 场景描写

深夜。主塔第七十二层——蒼的私人研究室。

这间房间占据了整个楼层，圆形的空间直径约四十步。墙壁上没有窗户——取而代之的是从地板延伸到天花板的书架，每一格都塞满了书籍、卷轴和各种魔法容器。书架之间的空隙里嵌着发光的符文灯，散发出柔和的蓝白色光芒，使整个房间看起来像一座微缩的星空图书馆。

房间中央是一张巨大的石桌，桌面上刻满了法阵。桌上散落着各种研究工具：水晶棱镜、符文笔、魔力测量仪、半打翻的墨水瓶，以及至少三十本翻开到不同页面的书籍。蒼的研究方式向来如此——同时阅读数十本书，在脑中编织它们之间的联系。

蒼坐在石桌前。她没有读书。

她在看自己的右手。

右手的手背上，从指根到手腕，有一条细如发丝的黑色纹路。那是三个时辰前——三色回廊之战的最后阶段——留下的痕迹。

### 剧情对话

[蒼独自坐在研究室中。房间里只有符文灯的光芒和她缓慢的呼吸声。她的右手平放在石桌上，手背上那条黑色纹路在蓝白光芒中格外醒目。]

**蒼：** ……

[她用左手的食指轻轻触碰那条纹路。指尖接触的瞬间，纹路微微颤动，像是一条极细的活物。]

**蒼：** 影系第六阶——「暗蚀」。

[她的声音很轻，像是在自言自语，又像是在做学术记录。]

**蒼：** 作用机理：将目标的魔力回路临时腐蚀，使其在三十秒内无法施术。对有生目标附带神经干扰效果——剧痛。

[她的手指停在纹路上。]

**蒼：** 这个魔法……我不应该会用。

[回忆闪回。三色回廊之战的最后阶段。蒼在远处支援雷之阵列时，一名暗潮士兵突然出现在她的侧翼——不知怎么绕过了外围结界。那一瞬间，她的身体比思维更快地做出了反应。]

*——她不记得自己是怎么施放那个魔法的。*

*——她甚至不记得自己学过那个魔法。*

*——但她的右手自然而然地画出了术式，嘴唇自然而然地念出了咒文，黑色的魔力从指尖涌出，精准地击中了那名士兵的胸口。男人无声地倒下，全身抽搐。三十秒后他恢复了意识——但那三十秒里，他经历了不亚于酷刑的痛苦。*

*——蒼事后将他传送出了高原。人没事。*

*——但蒼有事。*

**蒼：** 《白塔禁术目录·影系篇》。编号：Restricted-0042。

[她翻开。书页上的文字用红色墨水书写——那是白塔标记"禁止研习"的颜色。]

**蒼：** "影系魔法——利用负面情感（恐惧、痛苦、绝望）作为魔力催化剂的术式体系。所有影系魔法均被白塔同盟列为一级禁术。理由：魔力来源不可控，施术者有不可逆的精神污染风险。"

**蒼：** "影系第六阶·暗蚀——需要施术者对'他人痛苦'产生共鸣才能驱动。换言之，施术者必须在某种程度上……享受施加痛苦的过程。"

**蒼：** ……不对。

[她闭上眼睛。]

**蒼：** 我没有学过影系魔法。白塔禁术目录中的所有术式都存放在第零层的封印库中，只有历代大法师知道入口。我作为现任大法师——我知道入口。但我从未——

**蒼：** ……从未进去过。

**蒼：** 但我施放了它。不是通过思考，不是通过计算——是本能。就像……就像这个魔法一直在我的身体里，只是在等一个合适的时机出来。

[她低头看着自己的双手。]

**蒼：** 而且——

[她的声音变得更低，低到几乎是气音。]

**蒼：** 当那个士兵倒下的时候。当他因为痛苦而无声尖叫的时候。我感受到了……

**蒼：** ……兴奋。

[这个词像一块烧红的铁烙在她的舌尖上。]

**蒼：** 不是满足，不是快感——是一种更深层的、学术层面的兴奋。就像……发现了一个全新的研究领域。一个禁忌的、黑暗的、但极其迷人的研究领域。

[她猛地站起来。]

**蒼：** 不。

[她走到石桌前，拿起一支符文笔，在空白的笔记上快速写下一行字。]

**蒼：** "研究日志第12,847号。记录：本人于今日无意识施放影系第六阶禁术'暗蚀'。施术过程中未经任何学习或准备，疑似肌肉记忆触发。更严重的问题——施术过程中产生了正向情绪反馈。初步判断：精神污染已经发生。来源不明。"

[她自己也说不清为什么这个数字突然让她感到不安。]

**学徒赫尔：** 大法师？夜已深了。您还在研究吗？我给您带了热茶……

**蒼：** ……不用了，赫尔。回去休息。

**学徒赫尔：** 是……大法师，您的声音听起来有些不一样。您没事吧？

### 场景描写

黎明。高原上空的天色从灰白变为了——紫色。

那不是正常的天空。蒼在彻夜未眠的研究之后做出了一个决定：不再用考试的方式对待暗潮军。昨夜的发现让她不安，而不安让她想要通过行动来恢复秩序感。对蒼来说，最能恢复秩序的行动就是——展示绝对的控制力。

七塔共鸣被调至最高输出。法力池的能量通过地下管道源源不断地涌入七座塔的核心，再通过塔尖射向天空。七道能量流在高空汇聚，与大气中的水汽和尘埃发生反应，形成了一片巨大的紫色风暴云。

风暴云覆盖了整个高原。

云层内部不断闪烁着三种颜色的光芒——赤红、冰蓝、金紫。那是火、冰、雷三种元素在高浓度魔力催化下达到了临界状态。它们随时可以从云层中释放出来，落向地面。

### 场景描写

主塔地下。第负七层。

蒼从未来过这里。

作为大法师，她知道主塔地下有七层结构——这是公开的建筑资料。地下一至三层是储藏室，四至五层是实验室，第六层是封印库——存放禁术目录的地方。而第负七层——法力池核心——在所有官方文献中都只有一句描述："由第一代大法师建造的魔力枢纽，无需维护，自动运行。"

四十三年来，蒼接受了这个描述。她是一个学者，但她也是白塔体制的产物。有些事情，体制说"不需要知道"，她就不去追问。

直到今天。

赫尔在限制区的文献中找到了一份被多次涂改的建塔日志。日志的大部分内容已经无法辨认，但有一个词反复出现——"容器"。不是指装魔力的容器。字里行间的语境暗示着——那是一个活的容器。

蒼带着那份日志来到了第负七层的入口。

入口是一扇没有把手的石门，表面覆盖着密密麻麻的封印符文。封印的等级极高——需要大法师的魔力签名才能解除。蒼将手掌按在石门上，符文逐一熄灭。石门无声地向两侧滑开。

门后是一条向下的螺旋阶梯，幽暗而狭窄。阶梯的墙壁上没有符文灯——取而代之的是一种淡淡的蓝色荧光，从墙壁本身渗出。蒼伸手触摸墙壁：温热的，像体温。那不是石头，而是某种……活着的东西。

她一级一级地走下去。

### 剧情对话

**蒼：** ……三百七十二级。

[她在默数。学者的习惯。]

[阶梯终于到了尽头。面前是一个巨大的圆形空间——直径超过百步。]

[蒼停住了脚步。]

**蒼：** ……

[她的呼吸停滞了三秒。]

*——法力池核心。四百年来为七塔提供无穷魔力的源头。白塔同盟一切力量的根基。*

一具水晶棺。

棺体是完全透明的。棺内充满了蓝色的液体，液体中悬浮着一个——

一个女孩。

她很小。看起来不超过十岁。面容安详，仿佛只是在沉睡。银白色的头发在液体中飘散开来，像是水中的海藻。她穿着一件白色的长裙——那种款式至少在三百年前就已经绝迹了。

法力池的魔力来源。

不是天然的魔力结晶。

是一个孩子的生命力。

[蒼站在水晶棺前。她的双手在发抖——那不是害怕引起的颤抖，而是某种更深层的东西。]

**蒼：** ……不……

[她跪了下来。膝盖碰到水晶地面时发出清脆的声响，在空旷的空间里回荡了很久。]

**蒼：** 不，不对。这不可能。

**蒼：** "意识清醒但无法行动。"

### 场景描写

三天后。高原北侧丘陵——七塔外围第二道防线。

蒼选择在这里迎击暗潮军的第二波攻势。地形上，北侧丘陵是七塔防御体系中最薄弱的一环——这是她故意的。

她把战场选在了最容易被突破的位置。

如果有人仔细观察，会发现更多反常的细节：结界的覆盖面积比标准配置少了三分之一；法师的部署间距过大，存在明显的射界空白；奥术傀儡被设置在了次优位置——它们守护的不是关键通道，而是已经被废弃的辅助路径。

蒼站在丘陵顶端的一块平坦岩石上。她的法师袍在风中轻轻飘动，水晶杖拄在身侧。从远处看，她的姿态和三天前没有任何不同——挺拔、从容、居高临下。

但走近了就会发现：她的眼睛是涣散的。

蓝色的瞳孔中没有了以往那种精密计算的锐利光芒。取而代之的是一种……空洞。像是一台精密仪器的核心齿轮被抽走了，外壳还在，但内部已经停止运转。

她在想着地下七十四米深处的水晶棺。

### 剧情对话

[蒼站在丘陵顶端。暗潮军的先锋已经出现在视野中。身旁的白塔法师们在做最后的战前准备。]

**白塔法师甲：** 大法师，结界阵列已就位。但……您确定用这个配置吗？北侧的覆盖面积似乎——

**蒼：** 嗯？

[她从沉思中回过神来。目光对焦花了两秒——这对蒼来说是极其反常的。]

**蒼：** ……覆盖面积。对。够了。

**白塔法师甲：** 但大法师，标准配置应该是——

**蒼：** 我说够了。

[她的语气比平时更硬。不是因为自信，而是因为不想被追问。]

**白塔法师甲：** ……是。

[法师退下。蒼重新看向远方。暗潮军的旗帜在丘陵下方的平原上展开。]

**蒼：** ……

*——她在心里默算。法力池此刻的输出功率。七塔共鸣的同步率。每一个魔法的能量消耗。这些数字过去只是数字——冰冷的、客观的、不带感情的数据。*

*——"消耗魔力100单位"——等于那个女孩承受了100单位的痛苦。*

*——"七塔共鸣全功率运转"——等于那个女孩的生命力被以最大速率抽取。*

[学徒赫尔从阵地后方跑来。]

**学徒赫尔：** 大法师？

**蒼：** 今天的战斗——不使用七塔共鸣。

[赫尔的表情像是被人往脸上泼了一盆冷水。]

**学徒赫尔：** 什……什么？大法师，不用七塔共鸣的话，我们的魔法输出会下降——

**蒼：** 我知道会下降多少。百分之七十。

**学徒赫尔：** 那——

**蒼：** 用个人储备就够了。

[赫尔张了张嘴，最终没有反驳。他注意到蒼的表情——那不是商量，是命令。]

**学徒赫尔：** ……是，大法师。

[蒼转回前方。暗潮军开始冲锋了。]

**蒼：** ……来吧。

[她举起水晶杖。杖顶的宝石亮起——但光芒远不如之前耀眼。那是仅靠个人魔力储备驱动的光芒。]

[蒼回到主塔。赫尔在门口等她，脸上写满了焦虑。]

**学徒赫尔：** 大法师——北侧丘陵失守的消息已经传遍了整个白塔。其他塔主们……他们通过共鸣网络发来了质询。

**蒼：** ……

*——今夜，她要再去看看那个女孩。*

深夜。主塔地下第负七层。法力池核心。

法力池核心的空间比三天前更暗了。蒼注意到那些从女孩身上辐射出的蓝色光线变细了——法力池的输出正在下降。因为蒼关闭了七塔共鸣，需求端减少了，法力池的抽取速率也相应降低。

蒼坐在棺边，双手交叠放在膝盖上。她的目光穿过透明的棺壁，落在女孩的脸上。

### 剧情对话

[蒼坐在水晶棺旁。法力池核心的空间安静到能听见自己血液流动的声音。]

**蒼：** ……你好。

[沉默。当然没有回应。]

**蒼：** 我不知道你能不能听到。根据封印参数，"永恒沉眠术"维持对象的意识活动但阻断所有运动神经信号。理论上——你能听到。只是无法回应。

[她调整了一下坐姿。]

**蒼：** 我叫蒼。蒼·阿尔卡迪亚。现任——大法师。

[说到"大法师"三个字时，她的声音微不可察地卡顿了一下。]

**蒼：** 三天前我第一次来到这里。在那之前，我不知道你的存在。不——应该说，我知道"法力池"的存在，但不知道它的本质。四十三年来，我一直以为法力池是一个纯粹的魔力装置。就像……锅炉。只不过燃烧的是魔力而不是木柴。

[停顿。]

**蒼：** 我错了。

[她低下头，看着自己的双手。]

**蒼：** 日志上没有你的名字。第一代大法师们记录了你的年龄、你的魔力亲和度、你的身体参数——像是在记录一件实验器材的规格。但他们没有记录你叫什么。

**蒼：** 你有名字吗？

[沉默。]

**蒼：** ……当然有。每个人都有名字。只是没有人觉得那重要。

[她展开纸片。]

**蒼：** ……

**蒼：** 你一直在求救。四百年。每一秒。意识清醒地——在这个棺材里——对着无人的黑暗——说"救我"——

**蒼：** 而四百年来——没有一个人——来到这里——听到——

**蒼：** ……我听到了。

**蒼：** 我听到了。我来了。

[然后她站起来。她的表情——]

### 场景描写

七塔核心区——白塔同盟的心脏地带。

七座白塔围成的区域直径约三里，中心是主塔。塔与塔之间由半透明的结界墙连接——那些结界在正常状态下呈现出淡蓝色的微光，坚固到足以抵挡五阶以下的所有物理和魔法攻击。

但今天的结界看起来不太一样。

光芒是不均匀的。大部分区域仍然保持着标准的淡蓝色，但在某些位置——第二塔与第三塔之间、第五塔与第六塔之间、以及第七塔后方——光芒明显黯淡，甚至出现了肉眼可见的闪烁。

这些"故障点"不是自然衰减。结界的维护系统完全由大法师控制——蒼在昨夜的例行维护中，"忘记"了激活这三处节点的魔力供给。

她的记忆力可以精确到第12,847号研究日志中的每一个标点符号。她不可能"忘记"任何事。

蒼站在主塔的中层观测台上。这里可以俯瞰整个核心区。她手中的水晶杖暗淡无光——她已经连续数日没有从法力池中汲取魔力了。

她在等暗潮军发现那些缺口。

[蒼在观测台上。赫尔急匆匆地跑来。]

**学徒赫尔：** 大法师！结界诊断系统报告——第二-三、第五-六、第七后方三处节点出现魔力供给异常！结界强度下降至标准值的百分之三十！

**蒼：** ……哦。

[她的反应平淡到赫尔几乎以为她没听到。]

**学徒赫尔：** 大法师！这是严重故障！如果暗潮军发现了这些缺口——

**蒼：** 赫尔。

**学徒赫尔：** 是！

**蒼：** 法力池的输出最近不稳定。可能是管道老化导致的供给波动。我会安排检修。

**学徒赫尔：** 可是——暗潮军已经在核心区外围集结了！他们随时可能发起进攻！现在检修来不及——

**蒼：** 那就来不及。

[赫尔愣住了。]

**蒼：** 用现有的结界强度迎战。将法师和傀儡集中部署在三处故障点周围。如果敌军从那里突入——近距离拦截。

**学徒赫尔：** 大法师……

[赫尔的声音低了下来。他看着蒼的脸——那张脸上有一种他从未见过的表情。不是紧张，不是愤怒。是……平静。一种做出了某种重大决定后的、彻底的平静。]

**学徒赫尔：** ……您是故意的，对吗？

[长久的沉默。]

**蒼：** 赫尔。你是白塔最优秀的学徒。你的观察力——我一直很欣赏。

[赫尔的眼眶微红。]

**蒼：** 但现在不是问问题的时候。去执行部署。

**学徒赫尔：** ……是，大法师。

**学徒赫尔：** 大法师——不管您在做什么——我相信您有理由。

**蒼：** ……谢谢你，赫尔。

[暗潮军对七塔核心区发起进攻。斥候很快发现了三处结界故障点。]

**白塔法师甲：** （共鸣网络私频）大法师的攻击……怎么回事？她的命中率——

**白塔法师乙：** （共鸣网络私频）别问。照常战斗。

**蒼：** 这是命令。用个人储备维持现有防线。能守多久守多久。守不住——就撤。

**蒼：** ……够了。

**蒼：** 今天到此为止。

[她的声音通过魔力传导，在核心区内回荡。]

**蒼：** 赫尔。

**学徒赫尔：** 大法师。

第二塔主·炎——一个身材魁梧的中年男人，红色法袍，留着修剪整齐的胡须。火系专精。性格暴烈。

第四塔主·雷鸣——一个年轻得过分的男人，金色法袍，眼神锐利。雷系专精。野心家。

第五塔主·翠影——一位身形纤细的女性，绿色法袍，总是带着温和的微笑。风系专精。调解派。

第六塔主·暮光——一个看不出年龄的人，紫色法袍，面容被兜帽遮蔽。暗系专精。神秘主义者。

第七塔主·磐石——一个矮壮的老人，灰色法袍，双手粗糙如工匠。地系专精。务实派。

六道目光同时落在蒼身上。

### 剧情对话

[蒼走到中央石柱前。她没有激活自己的共鸣水晶——她人就在现场，不需要投影。她面对着六道全息影像，挺直了脊背。]

**炎：** 你来了。我以为你会躲在实验室里不敢出来。

[他的声音像燃烧的木柴，噼啪作响。]

**蒼：** 炎塔主。我从不躲避。我只是在优先处理更重要的事务。

**炎：** 更重要的事务？比白塔的防线被你亲手瓦解更重要的事务？！

[他的全息影像向前迈了一步，像是想冲过来。]

**炎：** 三处结界故障——六个人都查过了——不是设备问题！是你关闭了供能！蒼·阿尔卡迪亚，你到底在干什么？！

[蒼没有退后。]

**蒼：** 我在干一件早就该干的事。

[冰河的声音从侧面传来，冷如其名。]

**冰河：** 蒼。在你说出你的"理由"之前——你是否承认以下事实：你故意削弱了核心区结界，导致暗潮军突入七塔之间？

**蒼：** 承认。

[六位塔主的表情各异。炎是愤怒，冰河是冷静，雷鸣是若有所思，翠影是担忧，暮光看不见表情，磐石是失望。]

**冰河：** 理由。

**蒼：** 你们有多少人——知道法力池的真相？

[短暂的沉默。]

[蒼注意到了沉默的质地。不是困惑的沉默——是被揭穿的沉默。]

**蒼：** ……你们都知道。

**磐石：** （叹气）蒼——你去了第负七层。

**蒼：** 是的。

**磐石：** 我们知道迟早会有一任大法师走到那里。只是没想到——会是在这种时候。

**蒼：** "这种时候"？你的意思是——如果不是在战时，你们就可以继续保密下去？

**蒼：** 我很冷静。我已经花了一周时间冷静。冷静到我可以用完全理性的方式告诉你们——法力池里关着一个活人。一个八岁的女孩。她被清醒地折磨了四百年。而你们——六位塔主——全部知情。

[炎拍了一下石柱，全息影像因为魔力波动而闪了一下。]

**炎：** "折磨"？蒼，你以为我们没挣扎过吗？你以为这四百年来每一任塔主都心安理得吗？我们——每一个人——在第一次看到那个棺材时都和你一样——

**蒼：** 一样什么？一样震惊？一样愤怒？然后呢？

[她的声音突然升高了半个音阶。这在蒼身上极其罕见。]

**蒼：** 然后你们回到了自己的塔里。继续用她的痛苦施法。继续维护七塔共鸣。继续——假装一切都没问题。

[沉默。]

**雷鸣：** 蒼大法师。我理解你的愤怒。但请允许我指出一个事实——解除法力池意味着白塔同盟失去全部魔力支援。在暗潮军压境的当下，这等同于自杀。

**蒼：** 我知道。

**雷鸣：** 你"知道"——但你还是要这样做？

**蒼：** 是的。

**雷鸣：** ……以什么立场？

**蒼：** 以一个人的立场。

[雷鸣嗤笑了一声。]

**雷鸣：** "一个人的立场"。蒼大法师，你是白塔同盟的大法师。你的决定关系到七座塔——数千名法师和学徒——整个高原上所有人的安全。你没有资格用"一个人的立场"来做决定。

**蒼：** 那个女孩也是"一个人"。

[雷鸣沉默了。]

**翠影：** 蒼……我们不是不想救她。但——你想过后果吗？法力池崩溃后，我们用什么来抵抗暗潮军？用拳头吗？

**蒼：** 用我们自己的力量。

**翠影：** 我们自己的力量——没有法力池的白塔法师——连一支普通军队都打不过。蒼，你比谁都清楚这个数据。

**蒼：** 我清楚。所以我已经开始研究替代能源。风力和水力驱动的魔力转换装置——理论上可行——

**炎：** 理论上！理论上可行！蒼，暗潮军已经在七塔之间扎营了！你觉得他们会等你做完实验吗？！

[蒼没有回答。因为她知道答案是"不会"。]

**暮光：** ……

[暮光开口了。他的声音低沉、沙哑，像是从深井底部传来。]

**暮光：** 蒼大法师。我有一个问题。

**蒼：** 请说。

**暮光：** 如果——假设——你解除了法力池。那个女孩被释放了。然后暗潮军攻陷了白塔。数千名法师和学徒——死了。高原上的平民——死了。你能接受这个结果吗？

[蒼沉默了。]

[很长的沉默。]

**蒼：** ……不能。

**暮光：** 那么——

**蒼：** 但我同样不能接受用一个孩子的痛苦来换取我们的安全。这两个结果——我都不能接受。

[她抬起头。]

**蒼：** 所以我要找第三条路。

**冰河：** 不存在第三条路。

**蒼：** 不——是你们没有去找。因为法力池太方便了。有一个现成的答案摆在那里——只需要牺牲一个人——为什么还要费力去找第三条路？

[冰河没有反驳。]

**蒼：** 四百年。四百年的时间，足够研发出十种替代能源。但白塔没有做。因为没有人觉得有必要。

[她环顾六位塔主。]

**蒼：** 你们都是学者。你们知道真相。你们也知道这是错的。但你们选择了沉默——因为沉默是最省力的选项。

[磐石低下了头。翠影别过脸去。炎攥紧了拳头但没有再发言。冰河面无表情。雷鸣的眼神闪烁不定。暮光依旧隐在兜帽阴影中。]

**蒼：** 从今天起——我不会再使用法力池的魔力。我会在不依赖法力池的情况下守护白塔——或者失去白塔。不论哪种结果，那个女孩都会被释放。

[她走向议事厅的出口。]

**炎：** 蒼·阿尔卡迪亚！你——

**蒼：** 你们可以弹劾我。大法师的罢免需要六位塔主一致投票。我猜——你们不会一致。

[她没有回头。]

[她是对的。翠影不会投赞成票。磐石也不会。暮光——不确定。但只要有一票反对，弹劾就无法通过。]

**蒼：** 下次会议——我会提交替代能源的初步方案。如果你们有兴趣的话。

[她走出了议事厅。]

*——六位塔主的全息影像在她身后缓缓消散。议事厅陷入黑暗。*

*——蒼·阿尔卡迪亚，四十三年来第一次，站在了整个白塔同盟的对面。*

*——她不害怕。一个见过水晶棺内那双眼睛的人，不会再害怕任何活人的目光。*

*——但她累了。*

*——走出议事厅后，她在走廊的拐角处停了三秒钟。背靠着墙壁。闭上眼睛。深呼吸。*

*——然后她继续走了。*

*——因为地下七十四米处，有人在等她。*

## Event 10 — 据点攻略「奥术堡垒」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.37-41 |

### 场景描写

黎明。主塔。

七塔核心区已经完全被暗潮军控制。六座副塔在失去蒼的统一调度后各自为战——有的塔主选择了投降，有的选择了撤离，有的仍在抵抗但已无力回天。

只剩主塔。

主塔是七塔中最高的一座——一百层的白色尖塔，塔身覆盖着密密麻麻的符文。在法力池全功率运转的时代，这些符文会散发出耀眼的蓝光。但现在，大部分符文已经暗淡了。只有零星几处还在微弱地闪烁——那是蒼用个人魔力储备勉强维持的最后防御。

塔基被一层半球形的结界包裹。结界的颜色不再是以往的淡蓝色——而是带着灰色的浑浊。那是魔力不足的征兆。结界的强度——蒼心里有数——大约只有全盛时期的百分之十五。

足够挡住一次进攻。不够挡住第二次。

蒼站在塔顶。从这里看下去，暗潮军的营帐如同一片灰色的海洋，将主塔团团包围。清晨的阳光照在他们的旗帜上，猎猎作响。

她深吸了一口气。

今天结束之后，白塔同盟四百年的历史——就会画上句号。

### 剧情对话

[塔内。蒼从塔顶向下走到第七十二层——她的研究室。赫尔在那里等她。]

**学徒赫尔：** 大法师。准备好了。

[赫尔的眼睛红肿，但表情坚定。桌上放着蒼这几天赶制的东西——一个拳头大小的装置，由铜管、水晶片和微型风车组成。]

**蒼：** 测试结果？

**学徒赫尔：** 输出功率——大约相当于法力池的千分之一。不，万分之一。

[蒼看着那个粗糙的装置。]

**蒼：** ……万分之一。

**学徒赫尔：** 大法师，这不够用。连一个基础结界都——

**蒼：** 我知道。但它是干净的。

[她拿起装置，放在掌心。微型风车在她呼吸的气流中缓缓转动，水晶片发出极其微弱的蓝色光芒。]

**蒼：** 这个装置的能量来自风和水。不来自任何生命的痛苦。

[她将装置放入袍袖。]

**蒼：** 万分之一是起点。不是终点。

**学徒赫尔：** ……是。

**蒼：** 赫尔。接下来的战斗——我需要你做一件事。

**学徒赫尔：** 什么都行。

**蒼：** 当暗潮军攻入中层时——你带所有愿意跟你走的学徒离开。从主塔地下通道撤离。

**学徒赫尔：** ！大法师——

**蒼：** 不是请求。是命令。你和学徒们——是白塔的未来。如果白塔要重建——不是靠这些石头和符文——是靠你们脑子里的知识。

[赫尔的嘴唇在颤抖。]

**学徒赫尔：** 那您呢？

**蒼：** 我守塔顶。等暗潮军上来。然后——做我该做的事。

**学徒赫尔：** ……您不会死的，对吗？

[蒼看着赫尔。四十三年来，她教过无数学徒。但没有一个像赫尔这样——在知道了一切之后还选择留在她身边。]

**蒼：** 我尽量不死。

[赫尔行了一个白塔最高规格的礼——单膝跪地，右手覆心。]

**学徒赫尔：** 蒼·阿尔卡迪亚大法师。赫尔·维特鲁，您最愚笨但最忠诚的学徒。此去一别——

**蒼：** 不是一别。是暂别。去吧。

[赫尔站起来，擦了一下眼睛，转身跑向楼梯。]

[蒼独自站在研究室中。她最后环顾了一圈——那些书架、那些卷轴、那张刻满法阵的石桌。三十一年的记忆。]

**蒼：** ……再见了。

[她走上塔顶。]

### 战斗阶段·地面层

[暗潮军发起总攻。第一波冲击落在主塔结界上。]

*——结界在第一次冲击中碎裂了。不是缓慢的衰减——是像玻璃一样瞬间碎成千万片。蓝灰色的碎片在空中飘散，像下了一场水晶雪。*

[塔基暴露。六台奥术傀儡·重型激活——它们是蒼最后的自动防御。三米高的石制巨人从地面裂缝中站起，挡在主塔入口前。]

[第1回合，暗潮军与傀儡正面交锋。傀儡的每一拳都能砸碎盾牌，每一脚都能踏裂地面。但暗潮军已经有了丰富的对傀儡经验——三色回廊中学到的战术被完美复用。]

**暗潮队长：** 钩锁组！缠住四肢！近战组！关节部位集中攻击！

[第2-3回合，结界术士在傀儡身后释放延迟法阵——暗潮士兵踩上去会被减速和持续伤害。但暗潮军的工兵投掷沙袋覆盖法阵，强行创造出安全的通道。]

[傀儡逐一被击破。结界术士在失去掩护后很快被压制。]

### 战斗阶段·中层

[暗潮军攻入主塔内部，沿着螺旋阶梯向上推进。]

[中层——第四十至六十层——是白塔精英术士的最后阵地。他们在走廊和楼梯间设置了层层魔法陷阱。]

[第4回合，暗潮先锋踏入第四十层。走廊地面上的符文亮起——冰冻陷阱将三名士兵定在原地。后续的暗潮士兵用锤子砸碎冰面，救出同伴。]

**白塔精英术士甲：** 大法师已下令——阻敌但不杀敌。用控制型魔法！

[精英术士们执行命令。他们的魔法以减速、定身、眩晕为主——伤害被刻意降低。这让暗潮军困惑但也让他们的推进速度大幅放缓。]

[第5回合，暗潮军推进到第五十层。赫尔的身影出现在楼梯拐角——他手中拿着一根法杖，但没有举起。]

**学徒赫尔：** ……

[他看着涌上来的暗潮士兵。然后他深吸一口气。]

**学徒赫尔：** 白塔学徒——随我撤离！大法师命令！

[身后，十几名年轻的学徒从各个房间中涌出。赫尔带着他们转向地下通道入口——一个隐藏在第五十层墙壁中的暗门。]

[赫尔最后回头看了一眼。然后他推开暗门，消失在通道中。]

[暗潮军没有追击。他们的目标是塔顶。]

[第6-7回合，白塔精英术士们在赫尔撤离后放弃了抵抗。他们放下法杖，站到走廊两侧。]

**白塔精英术士甲：** 我们……不再战斗了。塔顶——大法师在那里。

[暗潮军穿过沉默的精英术士队列，继续向上。]

### 战斗阶段·塔顶

[第8回合。暗潮军抵达第一百层——塔顶平台。]

[蒼站在平台中央。身后是空旷的天空。风吹动她的长发和法师袍。她的水晶杖横在身前，杖顶的宝石散发着微弱但稳定的光芒——那是她个人魔力的最后储备。]

**蒼：** 你们到了。

[她环顾涌上平台的暗潮士兵。]

**蒼：** 比我预计的快了一个时辰。你们——每一次都比我预计的快。

[她举起水晶杖。]

**蒼：** 作为白塔大法师——我有义务进行最后的抵抗。这不是为了白塔。是为了——

[她停了一下。]

**蒼：** 是为了证明一件事。不依赖法力池——不依赖那个女孩的痛苦——我蒼·阿尔卡迪亚，到底还剩下多少力量。

[她释放魔法。纯粹的、来自自身储备的魔法。没有法力池的增幅，没有七塔共鸣的加成。只是一个四十三岁的术师三十一年积累的个人实力。]

*——她的魔法依然精准。依然优雅。但威力远不如之前。*

*——她曾经翻手间制造的奥术风暴，现在变成了一簇簇小型的魔力弹。她曾经覆盖整个高原的七塔共鸣，现在变成了一面勉强覆盖身周三步的微型结界。*

*——但她没有退后一步。*

[第9回合，暗潮军从多个方向逼近。蒼在平台上转着圈释放魔法——每一个方向、每一个角度，都计算到了极致。她的移动像是一支精密的舞蹈，每一步都有意义。]

**蒼：** 第四百七十二式——奥术散射。覆盖左翼30度扇形。第一百九十三式——冰锥连锁。中路纵深三排。第——

[她的呼吸越来越急促。个人魔力储备正在快速耗尽。]

[第10回合，蒼的结界碎裂。一名暗潮士兵的长矛刺穿了她最后的防护，在她的左臂上划出一道伤口。]

**蒼：** ……！

[她后退两步。看了一眼流血的左臂。然后她将水晶杖插在地面上。]

**蒼：** ……够了。

[她松开了杖。双手垂在身侧。]

**蒼：** 这就是答案。不依赖法力池的蒼·阿尔卡迪亚——能挡住十个回合。

[她闭上眼睛。]

**蒼：** 奥术堡垒——陷落了。

### 胜利结算

[暗潮军占领主塔。蒼站在塔顶平台上，没有逃跑，没有投降。她只是——站在那里。]

[暗潮指挥官登上塔顶。]

**暗潮指挥官：** 蒼·阿尔卡迪亚大法师。白塔同盟的据点——奥术堡垒——已被暗潮军攻陷。

**蒼：** 我知道。

**暗潮指挥官：** 您——

**蒼：** 在你处置我之前——我有一个请求。

[暗潮指挥官沉默，等待。]

**蒼：** 主塔地下有一个需要打开的东西。我需要你们的帮助。

*——蒼·阿尔卡迪亚。白塔同盟最后一任大法师。在她的据点陷落的那一刻，她想的不是自己的命运。*

*——她想的是地下七十四米处的一具水晶棺。*

*——终于可以打开了。*

## Event 11 — 角色事件·伍「打开棺盖」

| 项目 | 数据 |

| 类型 | 剧情 |
| 推荐等级 | — |

### 场景描写

主塔地下。第负七层。法力池核心。

蒼带着暗潮军的指挥官和数名士兵走下了螺旋阶梯。三百七十二级台阶。她这次没有数——她已经走过太多遍了。

指挥官在看到水晶棺的那一刻停住了脚步。

他是一个身经百战的军人。他见过尸横遍野的战场，见过被火焰吞噬的城市，见过绝望到疯狂的人。但他从未见过——一个被关在水晶棺中四百年的孩子。

士兵们也停住了。有人倒吸了一口凉气。有人低声咒骂。有人转过身去不忍再看。

蒼站在水晶棺旁。她的表情出奇地平静。

她等这一刻——等了太久了。

### 剧情对话

[法力池核心。蓝色的光芒将所有人的脸照得惨白。]

**暗潮指挥官：** ……这是什么？

**蒼：** 这是白塔同盟四百年力量的来源。

[她的手轻轻抚过棺壁。]

**蒼：** 一个八岁的女孩。被封印在水晶棺中。意识清醒。生命力被持续抽取，转化为魔力，供给七座塔。

[指挥官的拳头缓缓握紧。]

**暗潮指挥官：** ……你们——白塔——

**蒼：** 是的。我们。包括我。

[她没有回避。]

**蒼：** 我施放的每一个魔法——包括对你们施放的——都来自她的痛苦。奥术风暴、七塔共鸣、三色回廊——全部。

[暗潮指挥官深吸了一口气。他显然在极力压抑愤怒。]

**暗潮指挥官：** ……你带我们来这里——是要我们帮你打开它？

**蒼：** 是的。封印的暴力破解需要在物理层面击碎水晶棺。我的个人魔力已经耗尽。而且——

[她看向棺壁。]

**蒼：** 封印设计中有一个安全机制。如果是白塔法师试图暴力破解，封印会自动反噬施术者。第一代大法师们很有远见——他们预料到了未来某一代可能会良心发现。

[她苦笑。]

**蒼：** 所以——需要非白塔法师的力量。需要你们。

[暗潮指挥官看了蒼一眼。然后他转向身后的士兵。]

**暗潮指挥官：** 能砸开吗？

[一名暗潮战士走上前，用手指敲了敲棺壁。]

**暗潮战士：** 这水晶……硬度极高。普通武器不行。需要——

**蒼：** 需要反魔力手段。你们在第一天带来的磐石矿石——还有剩余吗？

**暗潮指挥官：** ……有。

**蒼：** 将磐石矿石研磨成粉末，覆盖在棺壁上。矿石会吸收封印的魔力，使水晶脆化。然后——用物理方式击碎。

[暗潮指挥官点了点头。他下达了命令。]

[十分钟后。士兵们将磐石矿石粉末均匀地撒在水晶棺表面。粉末接触棺壁的瞬间发出了刺耳的嘶嘶声——封印魔力与矿石的中和反应。蓝色的光芒在粉末覆盖处迅速黯淡。]

[水晶棺的表面开始出现细密的裂纹。像是冰面在阳光下融化时的龟裂。]

**蒼：** ……快了。

[她跪在棺旁，双手平放在棺壁上。她能感受到封印的力量正在迅速衰减。]

**蒼：** （低声）再忍一下。最后一次。

[裂纹越来越密。棺内的蓝色液体开始冒泡——那是液态魔力在失去封印约束后开始挥发。]

[蒼站起来。]

**蒼：** 退后。所有人退到阶梯口。

**暗潮指挥官：** 你呢？

**蒼：** 我留在这里。

**暗潮指挥官：** ——

**蒼：** 法力池崩溃时会释放大量原始魔力。对普通人致命。对术师——

[她看了一眼自己的双手。]

**蒼：** 对我来说，是我应得的。

[暗潮指挥官沉默了一瞬，然后带着士兵们退向阶梯口。]

[蒼独自站在水晶棺前。]

[裂纹扩展到了整个棺壁。棺内的液体沸腾了——蓝色的蒸汽从裂缝中喷出，充满了整个空间。]

**蒼：** ……

[她举起右拳。]

[然后她——砸了下去。]

*——水晶棺碎裂了。*

[四百年的封印在这一拳中终结。棺壁炸成了无数碎片，蓝色的液体泼洒在地面上，迅速蒸发为蒸汽。封印中积蓄的原始魔力以冲击波的形式向外扩散——击中了蒼的全身。]

[蒼被冲击波推后了三步。她的法师袍被撕裂了无数道口子。蓝色的魔力灼伤遍布她的手臂和脸颊。但她站住了。]

[棺中的液体流尽。残破的水晶碎片中——那个女孩静静地躺着。]

[封印解除了。]

[女孩的身体不再辐射蓝色光线。从每一个毛孔中抽取生命力的管道——断了。四百年来第一次——她的生命力是属于自己的。]

[然后——]

[地面开始震动。]

*——法力池崩溃了。*

[失去了能量核心的法力池结晶体开始连锁碎裂。从棺底开始，裂纹像闪电一样蔓延到地面、墙壁、天花板——整个负七层的结晶体在十秒内全部碎成了粉末。]

[蒼抱起了女孩。她很轻——像是抱着一团羽毛。四百年的生命力抽取让她的身体骨瘦如柴，但她还活着。她的胸口有极其微弱的起伏——呼吸。]

[震动沿着主塔的结构向上传导。蒼抱着女孩冲上螺旋阶梯。暗潮指挥官在阶梯口接住了她们。]

**蒼：** 出去——快——塔要——

[话音未落。]

[从地面传来一声深沉的、如同大地叹息般的轰鸣。]

[然后——七塔熄灭了。]

*——同时熄灭。所有符文、所有结界、所有共鸣连接——在同一瞬间全部归零。七道蓝色光柱消失。天空中的七芒星法阵碎裂。连接塔与塔的弧光断开。*

*——四百年来第一次，七座白塔变成了它们最本质的样子——七根没有魔法的石柱。*

[蒼抱着女孩站在主塔外面。她仰望着熄灭的塔顶。没有蓝光。没有符文。只有白色的石壁在晨光中泛着冷色。]

[怀中的女孩动了一下。]

[蒼低头。]

[女孩的眼睛——睁开了。]

[那是一双金色的眼睛。像琥珀。像阳光。像蜂蜜。像一切温暖的、干净的东西。]

[女孩看着蒼。四百年来第一次看到另一个人的脸。]

[然后——她笑了。]

[不是水晶棺中那种微弱的、无力的嘴角弧度。是一个真正的、完整的、从心底浮上来的微笑。]

[同时——她的身体开始变得透明。]

**蒼：** ！不——

[蒼收紧了双臂。但她的手臂穿过了女孩的身体——像是穿过了阳光。]

**蒼：** 不——等等——你——

*——四百年的生命力抽取已经超过了人体的极限。封印解除的瞬间，女孩的生命之火——其实早就该在几百年前熄灭了。维持她存活的，恰恰是封印本身。*

*——解除封印，等于解除了她最后的生命维持。*

*——蒼知道这个可能性。她查过所有文献。概率是——百分之八十七。*

*——但她还是做了。因为那百分之八十七的死亡，也是解脱。*

[女孩的身体化为了金色的光点。光点缓缓飘散，像是无数细小的萤火虫在晨风中飞舞。]

[在完全消散之前——女孩的嘴唇动了。]

[这一次不是"救我"。]

[蒼读出了那两个字——]

**蒼：** ……"谢谢"。

[金色的光点飘散在空气中。消失了。]

[蒼跪在地上。她的双臂保持着拥抱的姿势。但怀中已经空了。]

**蒼：** ……

[她跪了很久。]

[当她终于站起来的时候——暗潮的士兵们注意到了一个变化。]

[她的头发。]

[原本及腰的黑色长发——从发根到发梢——变成了银白色。像是被一夜之间抽走了所有颜色。]

*——法力池崩溃的魔力冲击，加上封印解除时的反噬——改变了蒼的身体。黑发变银——这是术师承受超负荷魔力冲击时的典型症状。不可逆。*

[蒼伸手拉过一缕银色的发丝，看了一眼。]

**蒼：** ……

[她松开手。]

**蒼：** 够了。

[两个字。声音很轻。但在寂静的高原上，每个人都听到了。]

**蒼：** 四百年。够了。

*——蒼·阿尔卡迪亚站在熄灭的白塔下。银发在晨风中飘动。她的脸上有泪痕——她不记得自己什么时候哭的。*

*——但她知道——那个女孩笑着离开了。*

*——那就够了。*

## Event 12 — 防守击退「赎罪之火」

| 项目 | 数据 |

### 场景描写

法力池崩溃后的第二天。高原。

七座白塔依然矗立着——它们本身是石头建造的，失去魔力不会倒塌。但它们不再发光。在清晨的阳光下，它们只是七根灰白色的石柱，投下长长的、没有任何魔力的影子。

但并非所有塔主都接受了这个结局。

炎——第二塔主——在法力池崩溃后暴怒了。他带着残余的塔兵冲向主塔，誓要"审判叛徒蒼·阿尔卡迪亚"。雷鸣——第四塔主——也加入了。他的动机更复杂——不是愤怒，而是野心。在蒼失去大法师权威之后，他想要填补权力真空。冰河和暮光虽然没有公开出手，但各自派出了精锐部队"维持秩序"——实际上是来观望并准备捡拾残局。

翠影和磐石没有参与。翠影在自己的塔中闭门不出。磐石带着他的工匠们开始研究——如何在没有法力池的情况下维持基础设施。

蒼站在主塔废墟前。她的银发在风中飘动。身上的法师袍已经换了一件——不再是白塔的深蓝色，而是一件素白的普通长袍。没有符文，没有魔力纹路。就是一件衣服。

她的手中没有水晶杖。杖在昨天的战斗中折断了。

但她的右手掌心——有一团微小的火焰在燃烧。

那是「赎罪之火」。

### 剧情对话

[蒼站在主塔前的空地上。暗潮军在她身后列阵——他们是她的盟友了，至少在这一刻。]

[远方，炎塔的残兵正在集结。红色的法袍在晨光中如同一片流动的熔岩。]

**暗潮指挥官：** 蒼大法师——不，蒼女士。他们来了。

**蒼：** 我看到了。

[她抬起右手。掌心的火焰微微跳动。]

**暗潮指挥官：** 你的魔力——法力池崩溃后——你还能战斗吗？

**蒼：** ……我的常规魔法输出下降了百分之九十七。没有法力池，白塔法师和普通人几乎没有区别。

**暗潮指挥官：** 那——

**蒼：** 但我找到了一个替代方案。

[她张开掌心。火焰升高了一寸。]

**蒼：** 我把它叫做「赎罪之火」。

**暗潮指挥官：** ……赎罪？

**蒼：** 原理很简单。法力池用的是别人的生命力作为魔力来源。而这个——

[火焰在她掌心旋转。]

**蒼：** 用的是我自己的。

[暗潮指挥官的眉头皱了起来。]

**暗潮指挥官：** 你的意思是——每次施法都在消耗你自己的生命？

**蒼：** 生命力、体力、精神力——所有我自身拥有的能量。每次施放，我会流失一部分……寿命。

[她语气平淡，像是在描述一个化学反应方程式。]

**蒼：** 但这是干净的。不伤害任何人——除了我自己。

**暗潮指挥官：** ……你——

**蒼：** 别用那种表情看我。这是我自己的选择。而且——

[她微微笑了。银发和微笑组合在一起，让她看起来像是另一个人——比以前柔和了很多。]

**蒼：** 四十三年来我用别人的痛苦施法。现在，用自己的疼一疼——很公平。

[远方，炎塔的残兵开始冲锋了。]

**蒼：** 来了。

[她向前走了一步。]

**蒼：** 让我——证明一件事。

### 战斗阶段·第一波

[炎塔残兵冲来。他们是火系法师，即使没有法力池也保持着相当的战斗力——炎塔主一直强调个人实力的培养。]

[第1回合，炎塔法师释放火球术。多枚火球从不同方向飞来。暗潮军举盾防御。]

[蒼走到阵线前方。她张开右手——「赎罪之火」激活。]

*——她的掌心射出一道白色的火柱。不是普通的红色或蓝色火焰——是纯白色的。白火的温度远超普通火焰，但它不烧焦泥土，不灼伤空气。它只在碰到目标时才释放热量。*

*——精准到不可思议。*

[白火扫过炎塔法师的阵列。他们的火球在白火面前如同蜡烛遇到了太阳——瞬间被吞噬。白火穿过火球，继续前进，在炎塔法师脚下的地面上炸开——冲击波将他们掀翻在地。]

**蒼：** ……！

[她的膝盖微微弯曲。脸上闪过一丝痛苦——赎罪之火的代价。她的皮肤在白火施放后微微泛灰，像是老化了几个月。]

**蒼：** （低声）……疼。但该疼的。

[第2回合，炎塔残兵重新集结。但他们的士气已经被那道白火严重打击。一些法师开始犹豫——他们认识蒼。蒼是他们的大法师。他们曾经敬仰她。]

**炎塔法师甲：** 大法师——为什么——

**蒼：** 不要叫我大法师。大法师的头衔——属于白塔。白塔已经不存在了。

[她再次举手。白火在掌心凝聚。]

**蒼：** 我只是蒼。一个需要赎罪的人。

[第二次赎罪之火。这一次她的目标更精准——白火分成六道细流，分别击中了炎塔法师手中的法杖。法杖被白火灼断，碎片四散。]

*——她没有伤害任何人。她只是——缴了他们的械。*

[炎塔残兵失去法杖后战斗力骤降。暗潮军趁机推进，将他们包围并缴械。]

### 战斗阶段·第二波

[第3回合，雷鸣塔主的亲卫队从东侧出现。他们装备精良——每人手持双短杖，身穿雷电附魔的战斗法袍。]

**雷鸣：** （远距离扩音）蒼·阿尔卡迪亚！你毁了白塔！你必须为此付出代价！

**蒼：** 雷鸣。你说得对。我确实毁了白塔。

[她的语气平静。]

**蒼：** 但白塔——本来就该毁。一座建在孩子的痛苦上的塔——不配存在。

**雷鸣：** 天真！你以为毁了法力池就能赎罪？你只是把所有人——数千名法师——推入了深渊！

**蒼：** 你关心的不是那数千名法师。你关心的是——没有法力池之后，你的力量也归零了。

[雷鸣的脸色铁青。他挥手——亲卫队发动进攻。]

[第4-5回合，雷鸣亲卫的战斗力远超炎塔残兵。他们配合默契，双杖术在近距离极其难缠——一杖攻击，一杖防御，交替施法。雷鸣傀儡在两翼掩护，释放范围雷电。]

[蒼再次使用赎罪之火。这一次，她的白火在空中分裂成数十道细丝，编织成一张网——落在雷鸣傀儡身上。傀儡的外壳被白火融穿，内部的驱动核心暴露。暗潮战士们趁机将长矛刺入核心。]

[蒼在施放后咳嗽了一声。手背上出现了一片老人斑——她的生命力在加速流逝。]

**暗潮指挥官：** 蒼女士！你——你的手——

**蒼：** ……没关系。继续。

[第6回合，雷鸣亲卫被击退。雷鸣本人没有亲自参战——他看到局势不利后选择了撤离。]

### 战斗阶段·第三波

[第7回合，最后一波。冰河精锐和暮光暗术士联合出击——这是六塔中最危险的组合。冰系的控制加上暗系的腐蚀，能让任何目标在被冻住的状态下被慢慢吞噬。]

[暮光的暗术士释放了暗影侵蚀——一团紫黑色的雾气向蒼蔓延。]

[蒼看着那团暗影。她认出了它的本质——影系魔法。和她之前无意识施放的「暗蚀」同源。]

**蒼：** ……影系。

[她的右手背上，曾经出现过的黑色纹路闪了一下——然后消失了。因为法力池已经不存在了。影系魔法失去了污染源。]

**蒼：** 再见了。

[她举起双手。赎罪之火——全力释放。]

*——白色的火焰从她的全身涌出。不仅是手掌——肩膀、背部、额头——白火像是从她体内燃烧出来的。她在火焰中仿佛一座燃烧的白色雕像。*

[白火向前推进，吞噬了暗影侵蚀。紫黑色的雾气在白火中蒸发殆尽——连影子都没留下。然后白火继续前进，在冰河精锐和暮光暗术士之间的地面上划出一条白色的火线——灼热到让地面玻璃化。]

**蒼：** 够了。

[她的声音从白火中传出。]

**蒼：** 回去。告诉冰河和暮光——白塔完了。不是因为暗潮军——是因为它本身就不应该以那种形式存在。

**暗潮指挥官：** 你的身体——那些斑点——

**蒼：** 今天大概——老了五六年吧。

### 场景描写

高原。清晨。

战斗结束后的第一个日出。

东方的天空从深紫渐变为浅橙，然后是金色。阳光越过高原边缘的山脊，照在七座白塔上——没有符文的白塔在朝阳中呈现出一种朴素的美。纯粹的石头的颜色。没有魔法附加的颜色。只是白。

高原上的空气清冽得几乎刺痛肺部。没有了高浓度魔力的弥漫，空气中那股臭氧和羊皮纸的味道消失了。取而代之的是泥土、晨露和远处山花的香气——高原本来的味道。

暗潮军的营帐在七塔之间错落分布。士兵们在做早饭，炊烟袅袅。有人在打水，有人在修补装备，有人裹着毯子在帐篷口打哈欠。这是一幅普通得不能再普通的军营晨景——没有魔法、没有战斗、没有紧张。

主塔的台阶上，蒼坐在那里。

她面朝东方，双手环抱膝盖。银色的长发披散在身后，在朝阳中泛着柔和的光。白色的长袍在晨风中微微飘动。她的脚边放着一杯已经凉了的茶——赫尔昨晚走地下通道回来后泡给她的。

她的表情——

很难用一个词概括。不是快乐。不是悲伤。不是释然。也许最接近的描述是——安静。一种深层的、从骨头里渗出来的安静。像是一台运转了四十三年的机器终于被允许停下来了。

[蒼坐在主塔台阶上。朝阳照在她脸上。]

**学徒赫尔：** ……您一夜没睡？

**蒼：** 嗯。

[简短的回答。以前的蒼会补充一个精确的数据——"准确地说，我在凌晨三时四十七分闭眼了十一分钟，但没有进入深度睡眠。"但今天，她只是说了"嗯"。]

**学徒赫尔：** 茶。热的。

[他把新茶放在蒼手边，拿走了凉掉的那杯。]

**蒼：** ……谢谢。

[她拿起茶杯，双手捧着。热度通过杯壁传到她的掌心。她低头看着茶面上倒映的自己——银发，眼窝略深，嘴角有一条以前没有的细纹。]

**蒼：** 赫尔。

**学徒赫尔：** 嗯？

**蒼：** 学徒们都安全吗？

**学徒赫尔：** 全部安全。十五个人，一个不少。他们现在在第三塔的地面层——磐石塔主收留了我们。

**蒼：** 磐石。

[她微微点头。]

**蒼：** 他是个好人。一直都是。

**学徒赫尔：** ……？

**蒼：** 没有了魔力弥漫的味道。高原本来的味道——泥土、露水、花香。四十三年来我一直闻着臭氧和羊皮纸。我以为那就是高原的味道。

**蒼：** 原来不是。

**蒼：** 别叫我大法师了。

**学徒赫尔：** ……蒼老师。

**蒼：** "蒼老师"。比"大法师"好听多了。

**暗潮指挥官：** 蒼女士。

**蒼：** 嗯。

**暗潮指挥官：** 不是作为俘虏。不是作为附庸。是——作为同伴。

**蒼：** ……你们不怕吗？我可是刚刚亲手毁了自己所属的势力。一个会背叛自己组织的人——

**暗潮指挥官：** 你没有背叛白塔。你背叛的是白塔的罪。

**蒼：** ——良心。

**蒼：** 我的良心醒得太晚了。晚了三十一年。如果早一天——那个女孩就少受一天的苦。

**暗潮指挥官：** ……

**蒼：** 所以，你问我会不会加入暗潮。

[她放下茶杯。缓缓站起来。银发在风中飘动。]

**蒼：** 我的回答是——是的。但有一个条件。

[她看向七座白塔。]

**蒼：** 这些塔——以后会成为真正的学院。不靠秘密运转。不靠牺牲运转。靠风。靠水。靠——

[她伸出手，让晨风吹过指间。]

### 场景描写

三个月后。高原。

七座白塔依然矗立——但它们变了。

最显眼的变化在塔顶。每座塔的顶端都安装了一组巨大的风车——不是魔法驱动的，而是纯机械的。铜制的叶片在高原永恒的强风中旋转，发出有节奏的嗡嗡声。风车的轴承连接着塔内的魔力转换装置——将风的动能转化为魔力。效率低得可怜——和法力池相比，大约只有千分之一。但它在运转。干净地运转。

塔身上的符文也变了。以前的符文是蓝色的，由法力池的魔力驱动。现在，一些符文被改造成了水力符文——利用高原上溪流的水压来维持基础功能。水流通过管道引入塔内，驱动微型水轮，水轮再带动符文阵列。效率同样很低。但——干净。

主塔前方的空地上多了一块东西。

一座小型的实验台。铜管、水晶片、微型风车、水轮模型、各种零件散落其上。旁边立着一块写满了公式和示意图的黑板——粉笔字迹潦草但精确，是蒼的笔迹。

蒼坐在实验台前。银色的长发扎成了一个简单的马尾——她说散着碍事。白色长袍换成了一件灰蓝色的工作袍——实用、耐脏、口袋很多。她面前摆着一个新的装置——比三个月前那个拳头大小的原型机大了三倍，结构也复杂得多。

她正在往装置里装最后一个零件。

### 剧情对话

[蒼在实验台前工作。赫尔在旁边帮忙递工具。]

**蒼：** 六号扳手。

**学徒赫尔：** 给。

[赫尔递过扳手。蒼接过来，拧紧了装置侧面的一颗螺栓。]

**蒼：** ……好。密封完成。水晶共振腔的对准度——

[她凑近装置，眯着眼看了看。]

**蒼：** 偏差0.03毫米。在可接受范围内。

**学徒赫尔：** 以前您会说0.03毫米"不可接受"的。

**蒼：** 以前我有法力池。0.01毫米的偏差都会被法力池的过剩能量补偿掉。现在——

[她直起腰。]

**蒼：** 现在每一份能量都是自己挣来的。浪费不起。所以——0.03毫米，够了。不完美。但够了。

[她从口袋里取出一个小型风扇——手摇式的。对准了装置的进风口，开始摇。]

[风进入装置。内部的微型风车转动。风车带动齿轮组。齿轮组驱动水晶片之间的共振。共振产生魔力——极其微弱的魔力。]

[装置顶端的一颗小水晶——亮了。]

[蓝色的光。微弱到几乎看不见。但——亮了。]

**学徒赫尔：** ！亮了！蒼老师！亮了！

**蒼：** ……

[她盯着那颗微弱发光的水晶。]

**蒼：** 输出功率——大约0.003奥姆。

**学徒赫尔：** 0.003……法力池的输出是——

**蒼：** 三千万奥姆。

[赫尔的兴奋微微冷却了。]

**学徒赫尔：** ……所以这个装置的输出是法力池的——

**蒼：** 一百亿分之一。

[长久的沉默。]

**蒼：** ……但它是干净的。

[她拿起装置，放在掌心。微弱的蓝光映在她银发上，泛出柔和的光泽。]

**蒼：** 赫尔。法力池的第一天也是从零开始的。四百年前，第一代大法师们面对的也是一个"不够用"的能源。他们选择了——走捷径。用一个孩子的生命力。

[她将装置放回实验台。]

**蒼：** 我们不走捷径。

**学徒赫尔：** ……是。

**蒼：** 0.003奥姆是今天。明天也许是0.004。后天0.005。一年后——也许能达到1奥姆。十年后——一百奥姆。一百年后——

[她停了一下。]

**蒼：** 我大概看不到一百年后了。赎罪之火的代价——我的寿命比正常术师短了很多。

[赫尔的表情暗了下来。]

**蒼：** 但你能看到。

[她拍了拍赫尔的肩膀。]

**蒼：** 你——以及那十五个学徒——你们会继续。你们的学生会继续。总有一天——这些风车和水轮会产生足够的魔力，让七座塔重新亮起来。

**学徒赫尔：** ……蒼老师。

**蒼：** 到那一天——塔的光不再是蓝色的。它会是——

[她想了想。]

**蒼：** 白色的。纯白色的。像阳光。像那个女孩消散时的光。

[脚步声。暗潮指挥官走了过来。他看到了实验台上发光的装置。]

**暗潮指挥官：** 成功了？

**蒼：** 如果你对"成功"的标准是"能亮"的话——是的。如果标准是"有用"——还差一百亿倍。

[暗潮指挥官笑了。]

**暗潮指挥官：** 一步一步来。

**蒼：** 嗯。一步一步。

[她从实验台上拿起了一样东西——一根新的法杖。不是以前的水晶杖。这根杖是铜和木混合制成的，杖身上缠绕着细铜管——那是水力回路。杖顶镶嵌的不是宝石，而是一片微型风车和一块普通的水晶。]

**蒼：** 我给它起了个名字。「风水之杖」。

**学徒赫尔：** ……很朴素的名字。

**蒼：** 以前的「水晶杖」太华丽了。华丽到让人忘记它的能量来自哪里。

[她握着风水之杖，走到主塔前方的空地上。]

[高原的风吹过。杖顶的微型风车旋转起来。水晶片发出微弱的蓝色光芒。]

[蒼举起法杖。从杖顶释放出一道——极其微弱的——光线。光线飘到半空中，化为几颗淡蓝色的光点，像萤火虫一样在风中飘散。]

[没有奥术风暴的震撼。没有七塔共鸣的壮丽。只是——几颗光点。]

**蒼：** ……

[她看着那些光点。]

**蒼：** 不完美。

[光点在风中飘散、消失。]

**蒼：** 但至少——

[她收起法杖。转身看向赫尔和暗潮指挥官。银发在风中飞舞。她的脸上有疲惫，有皱纹，有赎罪之火留下的苍老痕迹。]

[但她的眼睛是亮的。]

**蒼：** 至少它是干净的。

*——不依赖痛苦的火焰。*

[背景中，七座白塔顶端的风车在高原的风中缓缓旋转。嗡嗡的声音像是一首低沉的摇篮曲。]

[蒼回到实验台前。拿起粉笔，在黑板上写下新的公式。然后她又拿起工具，开始改进下一个原型机。]

[赫尔在旁边递工具。]

[暗潮指挥官在旁边看着。]

[高原的风不停。]

*——三个月前，蒼·阿尔卡迪亚是白塔同盟的大法师。她拥有无穷的魔力、七座塔的控制权、和整个高原的至高权威。*

*——现在，她是一个银发的术师。魔力储备只剩下巅峰时期的百分之三。法杖是铜管和木头做的。实验室是露天的。*

*——但她的研究桌上有一个东西在发光。*

*——一百亿分之一的光。*

*——干净的光。*

*——这就够了。*
