## Event 01 — 开战宣言「灰烬中的重逢」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.25-28 |
| 情感基调 | 战斗狂的纯粹宣战，野兽般的兴奋 |
| 战斗条件 | 击败焔先锋突击队（8回合内） |
| 特殊机制 | 焔「爆裂连锁」每2回合对随机3单位造成ATK×0.5火属性伤害；废墟地形：碎石掩体DEF+1，火焰区域每回合扣5%HP |
| 敌方配置 | 焔(Lv.28) + 白塔战斗法师×3(Lv.25) + 火焰傀儡×4(Lv.23) + 法师学徒×6(Lv.22) |
| 奖励 | 400 EXP, 1000G, 焦黑的短杖碎片×1, 解锁焔档案第一页 |

### 场景描写

白塔同盟领地南缘——曾经的第三分塔「炎柱塔」的废墟。

三个月前，这里还是一座高耸入云的白色尖塔，塔身上镌刻的火属性符文日夜不息地燃烧着橘红色的光芒，是方圆百里最醒目的地标。如今，塔身从中段断裂，上半截倾斜着砸入地面，留下一道触目惊心的弧形沟壑。碎裂的白色石块散落四周，每一块上的符文都还在微弱地闪烁——像垂死者指尖最后的抽搐。

空气中弥漫着焦糊与臭氧交织的气味。废墟中央有一个巨大的焦黑坑洞——那是炎柱塔核心魔力炉爆炸后留下的痕迹。坑洞边缘的石头被烧成了琉璃质地，在午后阳光下反射出诡异的橙色光斑。地面上到处是龟裂的烧痕，像是某头巨兽用爪子在大地上留下的抓痕。

一道人影站在废墟最高处——断塔残存的顶端平台上，不过三米见方的斜面。

火红的双马尾在热气流中猎猎飞舞，像两面战旗。红黑色的战斗法师外套大敞着前襟，露出黑色束胸和结实的腹肌。单肩甲上刻着白塔的纹章，但被她用烧痕涂成了一个歪歪扭扭的爆炸图案。短杖「业火」扛在肩上，杖尖无风自燃，火焰在午后的热浪中几乎透明。

焔·伊格尼斯。白塔同盟战斗法师。蒼·阿尔卡迪亚的弟子。自封「人形爆裂弹」。

她的嘴角咧到了耳根。

### 剧情对话

[焔站在断塔顶端，远远看到暗潮军团的先锋部队从南面平原上逼近。她的眼睛亮了——不是紧张，不是恐惧，是一只饿了三天的野兽看到猎物时的兴奋。]

**焔：** 来了来了来了来了！！终于来了！！等了三天了你们这帮蜗牛！！

[她从断塔顶端一跃而下。五米高度，不用缓冲，双脚砸在碎石地面上溅起一蓬灰尘。她的膝盖吸收了冲击力——战斗法师的体术不是说着玩的。]

**焔：** 嘿嘿嘿——好多人！！一、二、三——数不过来了！！但是不要紧！！数量不重要！！重要的是——

[她举起短杖「业火」，杖尖的火焰骤然膨胀，从透明变为耀眼的橘红色。热浪扭曲了周围的空气。]

**焔：** ——够不够炸的！！

[身后的白塔法师们从废墟各处现身。三名战斗法师排成三角阵型，火焰傀儡在他们身边缓缓运转，体内的核心发出低沉的轰鸣。]

**白塔法师甲：** 焔！不要冲动！蒼大法师的指令是固守废墟，等待——

**焔：** 啊？守？守什么守！！敌人来了就该冲上去打啊！！

**白塔法师甲：** 我们的任务是——

**焔：** 任务就是打赢！！打赢了什么都解决了！！蒼老师说过——最好的防御就是把敌人全部炸飞！！

**白塔法师乙：** ……蒼大法师从来没说过这种话。

**焔：** 嗯——好像是我自己说的！但意思差不多！！总之——

[她回头看了一眼身后的队伍。火焰傀儡低沉地嗡鸣着，法师学徒们紧张地握着法杖，有几个人的手在抖。焔看到了那些颤抖的手。]

[她收起了笑容。只是一瞬。]

**焔：** ……喂。你们几个。害怕吗？

[法师学徒们面面相觑。一个年轻的女学徒低下了头。]

**学徒紫苑：** ……焔前辈。炎柱塔已经毁了。我们连基地都没有了。真的……能赢吗？

[焔走到紫苑面前。她比紫苑高半个头，低头看着她，红橘色的眼睛里火光跳动。]

**焔：** 紫苑。你还记得入塔考核的时候吗？

**紫苑：** ……记得。

**焔：** 那时候你对着考核用的靶子哭了半天，说火球怎么都点不着。然后呢？

**紫苑：** ……然后焔前辈往靶子上扔了一个爆裂火球，把靶子连带半个考场都炸了，说「看，火就是这么简单」。

**焔：** 对啊！！火就是这么简单！！

[她用力拍了一下紫苑的肩膀——力气大得紫苑踉跄了一步。]

**焔：** 害怕的时候就放火！！难过的时候也放火！！什么都想不通的时候——轰一发爆裂火球，炸完就舒服了！！

[她转身面向来敌，短杖朝前一指。]

**焔：** 白塔的法师不需要城墙！不需要堡垒！也不需要什么鬼基地！我们的阵地——就在脚下！站着的地方就是阵地！能放火的地方就是战场！

[她深吸一口气。然后——吼了。]

**焔：** 所以别给我哭丧着脸——点火！！全员点火！！把这片废墟变成他们的坟场！！

[战斗法师们同时举杖。火焰从十余根法杖的杖尖同时喷涌而出，在废墟上空汇聚成一片炽热的红云。]

### 战斗阶段

[战斗开始。焔冲在最前方——她永远冲在最前方。短杖「业火」划出弧线，爆裂火球接连射出，在敌阵中炸开一朵朵橘红色的花。]

*——哈哈哈哈！！这种感觉！！活着就该是这种感觉！！*

**焔：** 来啊来啊来啊！！一个不够就两个！两个不够就十个！十个不够——那就全部一起上！！焔全部接下！！

[第3回合，「爆裂连锁」发动——焔将魔力注入地面的旧符文，引发连锁爆炸。三道火柱从敌阵中冲天而起。]

**焔：** 看到没！！这就是炎柱塔的火！就算塔碎了——火还在！！

[遭受重击时——]

[一记重锤命中焔的左肋。她向后滑出数米，单膝跪地，嘴角渗出血丝。]

**焔：** ……嘿。

[她用手背擦掉血迹，站起来。红橘色的眼睛里火焰不减反增。]

**焔：** 不错嘛！终于有点疼了！来来来——给我更疼的！！

[HP低于40%时——]

[焔的外套已经被烧得只剩半边，束胸上全是灼痕和血渍。她的呼吸急促但稳定——像持续燃烧的炉火。短杖「业火」的杖身出现了裂纹，火焰从裂缝中泄漏而出。]

**紫苑：** 焔前辈！你的杖——

**焔：** 没事！！业火这家伙皮实得很！跟我一样！就算碎了也能继续烧！！

[她将短杖横在身前，魔力灌注——裂纹中喷出的火焰反而形成了一道灼热的屏障。]

**焔：** 还没够呢！！焔的火——没那么容易灭的！！

### 战后

[战斗结束。暗潮先锋部队撤退，废墟中的火焰渐渐熄灭，只剩袅袅青烟。焔拄着短杖站在焦黑的战场中央，浑身是灰尘、血迹和烧痕，但嘴角的笑意比战前还要张扬。]

**焔：** 哈——哈——！！太爽了！！还有没有了！！

[白塔法师们瘫坐在废墟上，精疲力竭。只有焔还站着——她体力耗尽时的表现和别人完全不同，不是倒下，而是越来越亢奋。]

[但当她回头看向废墟深处时，笑容微微凝固了。]

[那里——断塔的残骸之间——有一面被烧得焦黑的旗帜，上面依稀可见白塔的纹章。旗帜下压着一具焦化的人形，已经无法辨认面容。]

**焔：** ……

[她走过去，蹲下身，把那面旗帜从碎石下抽出来。布料一碰就碎，灰烬从她指缝间簌簌落下。]

**焔：** ……这是谁？

**白塔法师甲：** ……应该是炎柱塔的守塔法师。暗潮攻来时第一个被——

**焔：** 我没问你。

[她把残存的旗帜碎片叠好，放在那具遗骸旁边。然后站起身，拍了拍膝盖上的灰。]

**焔：** ……走了。下一个战场在哪？

[她的声音还是那么大，那么冲。但紫苑注意到——焔握着短杖的手，指节发白。]

> **获得奖励：** 400 EXP / 1000G / 焦黑的短杖碎片×1 / 解锁焔档案第一页
>
> *「焦黑的短杖碎片」——从炎柱塔守塔法师的遗物中找到的碎片。木质已经碳化，但核心的魔力导管还在微弱地发热。握在手中时，能感受到一股温和而执着的余温——和焔那种狂暴的火焰截然不同。*

## Event 02 — 前哨遭遇「燃烧的废墟」

| 项目 | 数据 |
|------|------|
| 类型 | 战斗 |
| 推荐等级 | Lv.28-31 |
| 情感基调 | 焦躁中隐藏的不安，白塔的伤疤 |
| 战斗条件 | 突破焔火焰封锁线（击破5个火焰柱后进入核心区域） |
| 特殊机制 | 焔「业火全开」持续3回合，ATK+4但每回合损失5%HP；火焰柱地形：靠近火焰柱的单位每回合受10%HP伤害，焔在火焰区域SPD+2 |
| 敌方配置 | 焔(Lv.31) + 紫苑(Lv.27) + 白塔战斗法师×2(Lv.28) + 火焰傀儡×6(Lv.26) + 残存守卫×4(Lv.24) |
| 奖励 | 550 EXP, 1400G, 熔融的法师戒指×1, 解锁焔档案第二页 |

### 场景描写

白塔同盟领地，第五分塔「辉矢塔」遗址。

又一座塔倒了。

和炎柱塔不同，辉矢塔不是被外力摧毁的——它是从内部崩塌的。塔身上半段完好，下半段却像被掏空了内脏，白色的外壁向内凹陷，露出里面密密麻麻的魔力管道和回路。那些管道被野蛮地截断、抽取，断口处残留着烧灼的痕迹和凝固的魔力液——蓝色的，像冻结的血液。

焔站在辉矢塔前，难得沉默了五秒钟。

这不是暗潮的手笔。暗潮会摧毁、会碾压、会用蛮力推倒一切。但不会这样——这样精准地、手术般地剖开一座塔，取走它的核心。

这是懂魔法的人干的。

### 剧情对话

[焔蹲在辉矢塔的断裂截面前，红橘色的眼睛扫过那些被截断的魔力管道。她的表情很罕见——不是兴奋，不是狂热，而是一种皱着眉头的困惑。]

**焔：** ……这不对劲。

**紫苑：** 焔前辈？

**焔：** 这些管道。看到没？截面是平的——被魔法切割的。而且切割的角度——

[她伸手摸了一下断口，指尖触到凝固的蓝色魔力液，嘶了一声缩回来。]

**焔：** 冰属性的切割。精度很高。不是暗潮那帮只会用蛮力的家伙能做到的。

**紫苑：** 你的意思是……有法师参与了对辉矢塔的破坏？

**焔：** 不是「有法师参与」。是「只有法师能做到」。而且——

[她站起身，踢了一脚旁边的碎石。]

**焔：** 魔力核心被整个取走了。一点残渣都没留。你知道从一座分塔里完整取出魔力核心需要什么吗？

**紫苑：** ……至少需要三名高阶术师同时操作，并且——

**焔：** 并且需要知道白塔的核心封印解法。这个世界上知道这套解法的人——

[她竖起手指开始数。]

**焔：** 蒼老师。七座分塔的塔主。白塔议会的五名长老。再加上……

**紫苑：** ……再加上谁？

**焔：** 没了。就这些人。所以——

[她的眼中终于闪过一丝不同于战斗兴奋的光芒。那是警觉。]

**焔：** 是内鬼。白塔有内鬼。

[远处传来号角声。暗潮的第二波先锋出现在地平线上。焔的表情瞬间切回战斗模式——像翻开关一样快。]

**焔：** 先不想了！来活了！！

[她举起短杖，火焰从杖尖喷涌而出，在辉矢塔废墟四周竖起五根火焰柱，形成一道灼热的封锁线。]

**焔：** 紫苑！带人守内圈！我在外面——

**紫苑：** 你又要一个人冲出去？！

**焔：** 废话！！外面才爽啊！！里面缩着有什么意思！！

[她一跃翻过火焰柱，短杖指向来敌。]

**焔：** 来吧——！！今天的份——焔全包了！！

### 战斗阶段

[焔在火焰封锁线外单独作战，利用废墟地形和SPD优势进行高速游击。她的战斗方式极其耗蓝——但也极其华丽。]

**焔：** 爆裂火球！！爆裂火球！！再来一发爆裂火球！！不够？！那就业火全开！！

[「业火全开」发动——焔全身被火焰包裹，攻击力暴增但持续失血。她完全不在乎。]

**焔：** 这才对嘛！！燃烧自己才能释放最大火力！！这就是战斗法师——不——这就是焔的战斗方式！！

[第5回合，紫苑从内圈射出一道辅助法术，恢复焔的HP。]

**焔：** 哦！不错嘛紫苑！但是别奶我——我不需要治疗！！我需要更多敌人！！

**紫苑：** 你不需要但我需要你活着！闭嘴站好别动！

**焔：** 切——

### 战后

[战斗结束后，焔瘫坐在一堆碎石上。衣服烧了大半，手臂上全是自己「业火全开」造成的灼伤。但她笑着。]

[然后她的笑容僵住了。]

[在战场边缘的碎石堆里，她看到了一样东西——一枚熔融变形的法师戒指。那是白塔法师的身份标识，每个正式法师在入塔时都会获得一枚。]

[焔认出了戒指上残存的铭文编号。]

**焔：** ……这是六期的。比我早一届的师兄……

[她把戒指攥在手心里。金属的余温已经散尽了，冰凉的。]

**焔：** ……他怎么会在这里？辉矢塔不是他的驻地。他应该在——

[紫苑走过来，看到了那枚戒指，脸色发白。]

**焔：** 「熔炉」。好名字。

**紫苑：** ……焔前辈。这台魔力炉的结构——是改造过的。原本的燃料输入口被替换成了……生体接口。直接连接人类的魔力回路，将法师体内的魔力——连同生命力——一起抽取、燃烧。

**焔：** ……这是人。

**焔：** 蒼老师的地下书库里。我知道。

[焔的拳头还嵌在金属中。血从她的手指缝里一滴一滴地落。]

**焔：** 能接触到那些文献的人——只有蒼老师和议会长老。

**焔：** 莱纳斯……

**焔：** 那个老头……？

[她拔出拳头。金属外壳上留下一个拳头形状的凹陷，边缘被她的体温烧得发红。她低头看着自己血淋淋的手，然后——]

**焔：** 紫苑。

**紫苑：** ……在。

**焔：** 我不太会动脑子。你知道的。

**紫苑：** ……嗯。

**焔：** 但有些事——不需要动脑子也明白。

[她低下头。]

**焔：** 但我现在知道——火焰至少不应该是「这个」。

她知道那些光点是什么。

| 项目 | 数据 |
|------|------|

熔炉要塞内部。控制中枢被紫苑破解后，被囚禁的法师们获救了——但状态极差。大部分人的魔力被抽取了七成以上，虚弱到无法站立。几个人被抬到要塞外的空地上，裹着毯子躺在担架上。

焔在他们中间寻找一张面孔。

她没有找到炎司师兄。

从其他获救法师的口述中，焔得知——炎司是第一批被投入魔力炉的人。在焔到达之前……他已经被完全燃烧殆尽了。

焔在得知这个消息后，走到要塞外围一片荒芜的空地上。她在那里站了很久。

空地上什么都没有。没有草，没有树，没有花。只有被魔力污染的灰色泥土和碎石。

但在一块岩石的缝隙里——焔看到了一朵花。

一朵白色的小花。不知道怎么在这片死地里活下来的。纤细的茎从岩缝中探出来，花瓣微微颤抖着，像是在顽强地对抗周围的死寂。

[焔蹲在那朵花前面。她已经蹲了半个小时了。]

[紫苑远远地站着。她知道——现在不该打扰焔。]

[焔伸出手指。食指尖冒出一簇小小的火苗——条件反射。在焔的世界里，接触任何东西的第一反应都是「点燃」。]

[火苗靠近花瓣。]

[然后——焔熄灭了火焰。]

**焔：** ……

**焔：** 以前的我——会把你烧掉吧。不是故意的。就是——碰到什么都会烧。控制不住。

**焔：** 蒼老师以前在白塔的花园种了很多花。我经过的时候——总会不小心烧掉几朵。蒼老师从来不骂我。她只是叹气，然后重新种。

**焔：** 有一次我问蒼老师——「为什么你总是种花？反正我路过都会被烧掉的」。蒼老师说——

[她模仿蒼的语气——虽然模仿得很不像。]

**焔：** 「焔。花不需要永远不被烧。花只需要——在燃尽之前绽放过就够了。就像你的火焰。」

[她低下头。]

**焔：** 当时我完全不懂。我说「蒼老师你在说什么我听不懂」然后跑去打架了。

[她从口袋里掏出那个烧焦的布偶。小女孩送的「小火」。焦黑的面目、残破的小袍子、断了一半的小法杖。]

**焔：** 炎司师兄……你的火——已经熄灭了。

**焔：** 所以我——不会把你烧掉。

**焔：** 焔的火——从现在开始——不只是燃烧。也要——让不该被烧的东西，好好活下去。

[远处，一道深蓝色的魔力波动闪了一下。焔感觉到了——她对蒼老师的魔力特征比对自己的生日还熟悉。]

[她转身。在夕阳的逆光中——一个身影站在废墟的边缘。深蓝色的法师长袍，极长的黑发在风中飘动。]

**焔：** ——蒼老师？！

### 场景描写

大裂谷上空——焚星炉控制平台。

焔站在这里。在她身后是指挥官和全军。在她面前是焚星炉——那颗巨大的、吞噬了数百名法师生命的暗紫色球体。

球体表面的吞焰符文在脉动。每一次脉动，都能感受到一股令人窒息的力量——那是被压缩的、凝聚的、数百条生命的火焰。它们在球体内部尖叫、翻涌，却永远无法挣脱。

莱纳斯站在控制台后方。他看到焔时——微笑了。

那个温和的、慈祥的微笑。

### 剧情对话

**莱纳斯：** 焔同学。好久不见了。

[焔咬紧了牙。]

**莱纳斯：** 你长大了不少。蒼教出了一个好学生。虽然——我个人认为，你的天赋被浪费了。

**焔：** 浪费？

**莱纳斯：** 你的火焰——是白塔百年来最纯粹、最强大的。如果投入焚星炉——你一个人的魔力就能驱动这台装置运转三个月。多可惜啊——把这么完美的燃料浪费在打架上。

**焔：** ……炎司师兄。他临死前——你也是这么跟他说的吗？

[莱纳斯的微笑没有变。]

**莱纳斯：** 炎司同学很安静。他没有挣扎。一直到最后——他都没有出声。

**焔：** ……

**莱纳斯：** 不像你。你一定会尖叫。你一定会反抗到最后一刻。但结果是一样的——焚星炉会吞噬你的一切。你的火焰越强——它就越强。这就是火属性法师的宿命。

[焔深呼吸。一次。两次。]

**焔：** 你说得对。

[莱纳斯微微挑眉。]

**焔：** 我的火焰越强——它就越强。所以——

[她举起了新铸的短杖——用「业火」的核心碎片打造的、比原来更纤细但更精密的武器。杖尖没有燃起火焰。]

[而是亮起了一颗白色的星光。]

**焔：** ——我不会用火焰打你。

[莱纳斯的微笑终于出现了一丝裂纹。]

**焔：** 蒼老师教过我——火焰的本质不是燃烧。是光与热。我把火焰拿走——只剩下光。

**焔：** 你造了一个吃火的怪物。但它——吃不了光。

**莱纳斯：** ……有趣。但是——

[他转身按下控制台。焚星炉轰鸣着启动。暗紫色的球体膨胀，吞焰符文疯狂旋转——]

**莱纳斯：** 你试试看。

**【阶段一：外壳】**

[焔作为NPC加入战斗。她的攻击方式只有星火系——白色光弹。]

**焔：** 星火——穿甲射击——！

**焔：** 有效！！继续——集中火力——外壳裂缝处——！！

[莱纳斯发动「魔力收割」——全场魔力被抽取，造成大量伤害。焔咬牙承受。]

**焔：** 嗯——！这种吸魔力的感觉——恶心透了——！但是——

[她看了一眼身旁的指挥官。]

**焔：** 不只是我一个人在扛！继续！！

[外壳破碎。焚星炉的核心暴露——一颗脉动的暗红色球体，表面流动着数百条扭曲的光纹。那是被吸收的法师们的魔力——他们的生命以这种扭曲的形态被困在核心中。]

**焔：** ……

[她看着那些光纹。每一条——都是一个人。]

**焔：** 我来了。你们——很快就自由了。

**焔：** ——！

**焔：** ……你问我还剩什么。

**焔：** 因为那不是魔法——那是生命本身——！！

**焔：** ……这些火焰——是炎司师兄的。是所有被夺走生命的法师们的。

**焔：** 它们不应该被用来毁灭——它们应该——回家。

**焔：** ……回去了。他们——回去了。

[她的声音很轻。]

**紫苑：** ……焔前辈。

**焔：** 紫苑。

**紫苑：** ……你哭了。

[焔摸了摸自己的脸。确实——湿的。]

**焔：** ……没有。这是——汗。战斗法师出汗从眼睛出很正常。

**焔：** ……我准备好了。从现在开始——焔的火——跟你走。

一个月后。灯火镇。

镇中央的枯树——居然发芽了。枯死了多年的老树在焔用火焰净化了周围魔力污染后，重新长出了几片嫩绿色的叶子。焔本人对此的反应是「啊？树怎么着火了？哦是叶子——绿色的火焰？——不是火焰是叶子？——好无聊」。

此刻是黄昏。焔坐在那棵发芽的老树下。

她给新短杖取名叫「星焰」。

### 剧情对话

[焔坐在树下。手里拿着那个烧焦的布偶「小火」。]

[那个小女孩跑过来。]

**小女孩：** 焔姐姐！

**焔：** 哦！小鬼！

[小女孩一屁股坐在焔旁边。]

**小女孩：** 焔姐姐，法师哥哥……回不来了吗？

[焔沉默了一下。]

**焔：** ……嗯。回不来了。

[小女孩低下头。焔看着她——然后伸手揉了揉她的头发。力气控制得刚刚好——不疼。]

**焔：** 但是——法师哥哥的火焰——已经自由了。我亲眼看到的。它变成了金色的流星——飞到天上去了。

**小女孩：** 真的？

**焔：** 焔从来不说谎！……好吧偶尔说一两句。但这次是真的。

[小女孩抬头看着天空。虽然是黄昏，还看不到星星。]

**小女孩：** 焔姐姐。你还会走吗？

**焔：** 嗯。焔跟着指挥官走了。要去很远的地方打仗。

**小女孩：** ……会回来吗？

**焔：** 当然会！焔说回来就回来！到时候——给你看更厉害的火焰！不是爆炸那种——是、是漂亮的那种——蒼老师教我的——

**焔：** 嗯虽然我还没学会但是肯定会学会的！

**紫苑：** ……你自己哄。

**焔：** 我不会哄——！

**紫苑：** 学。

**焔：** ……小鬼。不哭了。姐姐的衣服会湿的。

**焔：** 你看好了——

**紫苑：** ……焔前辈。你什么时候学会的？

**焔：** 昨天！练了一整晚！烧掉了十七块木板才成功！

**紫苑：** ……果然。

**焔：** 好了！该走了！

**焔：** ——好了。

**焔：** 走吧！！下一个战场在哪！！焔准备好了！！

[她把新短杖「星焰」扛在肩上。杖尖的金色火苗在夕阳中跳动。]

**焔：** 这次不只是爆炸了哦！焔还会——嗯——保护人啦、精准射击啦、火焰开花啦——全都会！进步了很多吧！夸我！！

[她凑到指挥官面前，满脸期待。]

**焔：** 快夸！不夸的话——就打一架决定！！

[夕阳将她的身影拉得很长。红色的双马尾在风中飞舞，像两面永不降下的战旗。]

[远方的天空中，一颗星星在暮色中亮起——是今夜的第一颗星。]

**焔：** 看——星星！

[她指着天空。]

**焔：** 蒼老师说过——星星就是很远很远的火焰。远到——光要走很久很久才能到我们眼睛里。

[她放下手。]

**焔：** 所以——就算火焰熄灭了——光也还在路上。永远不会真正消失。

[她转身看着指挥官。红橘色的眼睛里——有了一颗小小的金色星光。]

**焔：** 焔的火——也是这样。就算有一天烧尽了——光也会继续走。走很远很远。一直走到——需要温暖的人那里。

[她深吸一口气。笑了。]

**焔：** ——好了！煽情结束！！走了走了走了——！！再不走就来不及在天黑前找到下一个战场了——！！冲冲冲——！！！

[她一马当先冲了出去。风卷起的尘土在她身后扬起一道金色的幕帘。]

[灯火镇的老树上，那朵金色的火焰花在夜风中摇曳。不灭。]
