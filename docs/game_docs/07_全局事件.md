# 07 全局事件设计

## 一、事件系统规则

- 每回合事件阶段(Phase 3)最多触发2个随机事件
- 每个事件有触发条件和2个选项
- 部分选项含赌博机制（成功/失败概率）
- 事件不重复触发（同一事件每局最多1次，除通用事件外）
