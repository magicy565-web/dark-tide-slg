### 八、招募费用公式
```
总费用 = 兵数 × 2金 × 兵种加价倍率
兵种加价: 足轻×1, 武士×1.5, 弓兵×1.2, 骑兵×2, 忍者×1.8, 祭司×1.5, 術師×2, 砲兵×2.5
```

### 九、模块接口
- 依赖: 无（纯数据模块）
- 被引用: 02_角色设定.md, 05_行动设定.md, 10_AI行为.md
- 数据导出: troop_data{}, passive_effects{}, recruit_cost_formula
- 实现文件: `project/systems/faction/faction_data.gd` (SHARED_UNIT_DEFS, UNIT_DEFS, LIGHT_UNIT_DEFS, ALLIANCE_UNIT_DEFS, ULTIMATE_UNITS)
- 实现说明: 兵种数据使用 `soldiers` 字段（兵数）, `spd`（速度）, `row`（前/后排）, `recruit_gold`（招募金币）, `special`（被动ID）统一格式
