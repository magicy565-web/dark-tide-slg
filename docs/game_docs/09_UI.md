# 09_UI设计文档 —— 《暗潮 Dark Tide》

### 一、UI总体设计
- 参考《战国兰斯07》UI风格
- 日式2D战棋界面，深色系底板+金边装饰
- 分辨率: 1280×720 (16:9)
- 字体: Noto Serif CJK（标题）+ Noto Sans CJK（正文）

### 二、主要界面

#### 2.1 标题画面
- 游戏Logo + 背景CG
- 按钮: 新游戏 / 继续游戏 / 设置 / 退出

#### 2.2 阵营选择
- 3个阵营卡片（兽人/海盗/暗精灵）
- 每张: 阵营图标 + 名称 + 核心机制简述 + 特色兵种预览
- 点击展开详细信息，确认后进入游戏

#### 2.3 战略地图（主界面）
- **地图区域**（占70%屏幕）: 节点连接图，颜色区分阵营
  - 己方: 明亮高光白边
  - 敌方: 各阵营色
  - 中立: 灰色
  - 可攻击: 红色脉冲
  - 迷雾: 灰色蒙版
- **顶部资源栏**: 回合数 | 金币 | 粮食 | 铁矿 | 奴隶 | 威望 | 秩序 | 威胁
- **右侧信息面板**: 选中节点详情（所有者/类型/地形/驻军/建筑/城防）
- **底部行动栏**: 攻击 | 建造 | 招募 | 移动 | 外交 | 英雄 | 结束回合
- **左下角**: 小地图/缩略图

#### 2.4 节点详情面板
- 节点名称 + 类型图标
- 所有者/阵营
- 地形/等级/城防(当前/最大)
- 驻军列表（兵种图标+兵数）
- 建筑信息（名称+等级）
- 战略资源产出
- 行动按钮（攻击/建造/升级/招募）

#### 2.5 战斗界面
- 战国兰斯式左右对阵
- 左方(玩家) vs 右方(敌方)
- 前排3 + 后排2 的槽位布局
- 每槽位: 指挥官半身立绘 + 兵种图标 + 兵数条 + HP条
- 行动顺序条(顶部): 显示SPD排序的行动队列
- 战斗日志(底部): 滚动文本显示攻击/伤害/技能
- 按钮: 自动 | 撤退 | 技能(选中单位时)
- 攻城阶段: 城墙图标+城防条

#### 2.6 事件弹窗
- 居中半透明暗色面板
- 标题 + 描述文本
- 2个选项按钮（显示效果预览）
- 如有赌博: 显示概率

#### 2.7 英雄管理面板
- 左侧: 英雄列表（立绘缩略+名字+好感度条）
- 右侧: 选中英雄详情
  - 全身立绘
  - 属性(ATK/DEF/INT/SPD)
  - 装备3个槽位
  - 技能列表
  - 好感度进度+解锁内容
  - 部署/解除/调教按钮
- 监禁区: 俘获英雄列表+招募/释放按钮

#### 2.8 建造/招募面板
- 建筑列表: 可建建筑+费用+效果预览
- 招募列表: 可招兵种+费用+属性预览
- 灰色不可用项显示缺少的资源

#### 2.9 存档/加载界面
- 5个手动存档槽 + 1个自动存档
- 每槽: 截图缩略+回合数+阵营+时间戳
- 保存/加载/删除按钮

#### 2.10 设置界面
- 音量(BGM/SE)
- 文字速度
- 战斗动画开关
- 自动存档开关

### 三、通知系统
- 屏幕顶部滑入/滑出的通知条
- 颜色区分: 绿色(成功) / 红色(失败) / 黄色(警告) / 蓝色(信息)
- 自动3秒消失

### 四、模块接口
- 依赖: 所有模块（UI是展示层）
- 被引用: 13_美术.md
- 数据导出: UI布局规范, 通知接口
