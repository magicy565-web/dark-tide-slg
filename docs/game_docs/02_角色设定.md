## 一、角色设计原则

| 原则 | 说明 |
|------|------|
| **命名规范** | 全部采用日式命名（凛、雪乃、红叶、冰华等） |
| **性别分布** | 中立方、人类方以女性角色为主，搭配少量男性兵种（NPC/小兵） |
| **画风参考** | 《战国兰斯07》系列，日式2D立绘风格（详见 → `13_美术.md`） |
| **角色总数** | 18名可收集英雄（12名敌方 + 6名中立） |

**每个角色标准配置:**

- 职业 / 兵种类型
- 性格简述与背景故事
- 偏好赐物（用于提升好感度）
- 主动技能 × 1（好感Lv7解锁第二主动技能）
- 被动技能 × 1（好感Lv3升级）
- 3段调教/好感事件链（好感Lv5专属事件触发）

## 二、指挥官属性系统

每名英雄拥有四项核心属性，范围均为 **1-9**:

| 属性 | 缩写 | 范围 | 说明 |
|------|------|------|------|
| 攻击力 | ATK | 1-9 | 伤害公式加成，直接影响技能伤害输出 |
| 防御力 | DEF | 1-9 | 伤害公式减免，降低受到的伤害 |
| 智力 | INT | 1-9 | 技能效果 × (1 + INT × 0.05)；影响法力恢复速度 |
| 速度 | SPD | 1-9 | 决定行动顺序，SPD高者先行动 |

> 属性详细计算公式参见 → `11_兵种数据包.md`

## 八、调教/好感度系统

### 8.1 好感度机制

| 项目 | 规则 |
|------|------|
| 好感度范围 | 0 - 10 |
| 提升途径 | 共同战斗 +1/次，事件正确选择 +2/次，赐物 +1~+3（依偏好） |
| Lv3 解锁 | 被动技能升级 |
| Lv5 解锁 | 专属事件（3段调教事件链中的关键节点） |
| Lv7 解锁 | 第二主动技能 |
| Lv10 解锁 | 专属结局路线 |

### 8.2 俘获与招募流程

```
战斗败北 → 按概率俘获 → 进入监禁状态
                              ↓
                    每回合腐化值 +1
                              ↓
                    腐化值 >= 5 → 可尝试招募（50%成功率）
                              ↓
                    招募成功 → 加入己方（好感度从0开始）
                    招募失败 → 继续监禁
```

| 监禁参数 | 数值 |
|----------|------|
| 初始监禁容量 | 3 人 |
| 最大监禁容量 | 5 人（需建筑升级） |
| 释放效果 | 威胁值 -10，威望 +5 |
| 腐化增长 | +1/回合 |
| 招募门槛 | 腐化 >= 5 |
| 招募成功率 | 50%（可通过设施提升） |

### 8.3 调教事件链

每名英雄配有 **3段调教事件**，随好感度推进解锁:

| 阶段 | 触发条件 | 内容方向 |
|------|----------|----------|
| 第一段 | 好感Lv2+ | 初步接触——了解角色性格与过去 |
| 第二段 | 好感Lv5+ | 深入互动——信念动摇或关系转变 |
| 第三段 | 好感Lv8+ | 最终抉择——决定角色结局走向 |
