## 一、世界观概述

《暗潮 Dark Tide》是一款架空中土魔幻世界观的**日式战棋策略游戏**。玩家扮演崛起的**黑暗领主**，统帅邪恶军团征服大陆。

- **核心体验**: 领地经营 + 回合制战棋 + 角色收集与养成
- **画风参考**: 《战国兰斯》系列（日式2D立绘 + SLG战棋）
- **世界基调**: 黑暗奇幻，道德灰色地带——邪恶阵营亦有苦衷，光明阵营亦有腐败

大陆上存在三大邪恶势力（兽人部落、暗夜海盗团、暗精灵议会）与三大光明势力（人类王国、高等精灵、魔法师族），以及散布各地的中立据点。玩家需逐步征服、收编或消灭这些势力，最终统一大陆。

## 二、邪恶势力背景

### 2.1 兽人部落 — 铁牙氏族

| 项目 | 内容 |
|------|------|
| 领地 | 大陆西部荒原 |
| 首领 | 铁牙·碎骨者 |
| 规模 | 十二部落联盟（大陆最大兽人势力） |

**社会结构**

- 兽人社会无文字，所有历史以**战歌口传**形式保留
- "强者为王"是唯一法则——首领通过决斗产生，败者服从或离开
- 铁牙·碎骨者以绝对武力统一十二部落，但联盟本质松散

**WAAAGH! 文化**

- 兽人拥有独特的**集体战斗狂热**——当足够多的兽人聚集在一起时，会进入被称为"WAAAGH!"的战斗亢奋状态
- 这种狂热是双刃剑：**不战则内斗**。长时间没有外敌时，部落之间会自相残杀
- 铁牙·碎骨者维持联盟的方式就是不断发动对外战争

**历史仇恨**

- 百年前的"**碎盾之战**"中，人类王国天城王朝联合高等精灵，将兽人从富饶的中原地带驱赶到西部荒原
- 这场战败是兽人世代仇恨的根源——每一首战歌都会提到"被夺走的土地"
- 铁牙·碎骨者的崛起口号便是"夺回碎盾之地"

**对玩家的战略价值**

- ✅ **最易驾驭**: 指向敌人即冲锋，无需复杂指令
- ⚠️ **最难控制**: 无战事时部落会内斗消耗，需持续提供战斗目标
- 📌 适合作为早期扩张的主力军，但后期需搭配其他势力维持稳定

### 2.2 暗夜海盗团 — 深渊之牙

| 项目 | 内容 |
|------|------|
| 领地 | 无固定领地（流动舰队） |
| 首领 | "血帆"雷牙 |
| 构成 | 叛逃水手、流亡贵族、各族亡命徒 |

**起源**

- 创始人"血帆"雷牙原为人类王国的伯爵，因宫廷斗争被**流放**
- 流放途中劫持囚船，纠集同船囚犯建立了第一支海盗舰队
- 数十年间吸纳各族亡命之徒，发展为大陆最大的海上势力

**经济体系**

- 纯粹的**掠夺经济**: 战利品统一分配，船长双份，水手单份
- 没有领土野心——他们不占领土地，只掠夺财富
- 与所有沿海势力既有仇恨也做生意：今天劫掠的港口，明天可能是走私贸易的合作伙伴

**对玩家的战略价值**

- ✅ **最务实的盟友**: 只要有利可图就会合作，不问道德立场
- ⚠️ **分赃不均则哗变**: 必须持续提供足够的财富分配，否则舰队会叛变
- 📌 适合用于沿海作战和经济掠夺，但不适合内陆长期驻扎

### 2.3 暗精灵议会 — 永夜议会

| 项目 | 内容 |
|------|------|
| 领地 | 地下暗城网络 |
| 统治者 | 七位"苦痛之主" |
| 起源 | 高等精灵分支（"永夜之裂"后分裂） |

**起源: 永夜之裂**

- 两千年前，高等精灵内部爆发了关于**暗影魔法**使用的根本性分歧
- 主张拥抱暗影力量的派系被放逐，他们自称"永夜之子"，后称暗精灵
- 这场分裂被称为"**永夜之裂**"，是大陆历史上最深远的种族内战之一

**社会结构**

- 七位"苦痛之主"各自统治一座地下暗城，通过**永夜议会**协调行动
- **奴隶经济**: 一切劳动依赖奴隶——采矿、建造、甚至作为"苦痛神殿"的祭品
- "苦痛神殿"是暗精灵汲取**暗影之力**的核心设施，需要持续的痛苦能量供给

**与高等精灵的血仇**

- 两千年来，暗精灵与高等精灵之间的战争从未真正停止
- 双方都视对方为"背叛者"，和解的可能性为零

**对玩家的战略价值**

- ✅ **完全驯服后最稳定**: 一旦收编成功，暗精灵的忠诚度极高（因为他们无处可去）
- ⚠️ **最难收编**: 骄傲、狡诈、阴谋不断——收编过程中随时可能背叛
- 📌 高风险高回报的盟友，适合中后期经营

## 三、光明势力背景

### 3.1 人类王国 — 天城王朝

| 项目 | 内容 |
|------|------|
| 领地 | 大陆中部（最大政治实体） |
| 统治者 | 女王千姬 |
| 历史 | 天城一世三百年前统一 |

**核心优势: 数量与工事**

- 天城王朝拥有大陆最密集的城镇网络和最庞大的人口
- 王都**天城**从未被攻破——三重城墙、地下水道防御体系、圣殿骑士团常驻
- 军事思想以**防守反击**为核心：用城墙消耗敌军，再以骑兵反击

**内部腐化**

- 三百年的和平导致贵族阶层严重腐化
- 地方领主各怀鬼胎，对王都命令阳奉阴违
- 面对外敌时反应迟缓——从动员到出兵往往需要数个回合

**以女性角色为主导**

- **女王千姬**: 年轻的女王，刚即位不久，尚在巩固权力
- **银甲公爵夫人·红叶**: 精明务实的领地贵族，掌控南部经济命脉
- **铁壁女伯爵·冰华**: 北方边境守护者，以铁壁防守著称
- **圣殿骑士团团长·凛**: 王国最后的防线，刚正不阿的女骑士

> 详细角色设定参见 → `02_角色设定.md`

**圣殿骑士团（女骑士团）**

- 王国最精锐的战斗力量，全员女性骑士
- 直属女王指挥，不受贵族节制
- 当威胁值达到一定程度时，圣殿骑士团将作为最终防线出战

### 3.2 高等精灵 — 银月议庭

| 项目 | 内容 |
|------|------|
| 领地 | 东部古老森林 |
| 统治者 | 银月议庭（长老合议制） |
| 自称 | "第一子嗣" |

**文明底蕴**

- 高等精灵是大陆最古老的种族，文明历史比人类古老十倍
- 自称"第一子嗣"——世界创生之初便已存在
- 对其他种族的态度普遍傲慢，尤其将人类视为"有用的短命者"

**灵脉网络**

- 所有精灵据点通过**灵脉网络**连接，可以共享魔法屏障
- 这意味着进攻精灵领地时，所有据点的防御是联动的——攻陷一个并不意味着其他据点变弱
- 切断灵脉连接是攻略精灵领地的关键

**军事特点**

- **远程战斗无与伦比**: 精灵弓手是大陆最强的远程单位
- 世界树中沉睡着**远古树人**——终极防御力量
- 近战能力相对薄弱，一旦被突破远程防线则脆弱

**与暗精灵的关系**

- 不共戴天——这是两千年的血仇
- 如果玩家同时收编了暗精灵和高等精灵，需要极高的外交手腕才能让两者共存

---

### 3.3 魔法师族 — 白塔同盟

| 项目 | 内容 |
|------|------|
| 领地 | 七座魔法高塔（分布各地） |
| 统治者 | 塔主合议制 |
| 创立 | 四百年前"大魔灾"之后 |

**创立背景: 大魔灾**

- 四百年前，一次跨维度召唤实验失控，释放了大量异界生物
- 这场被称为"**大魔灾**"的灾难造成了巨大破坏
- 幸存的魔法使用者（跨种族）联合成立了**白塔同盟**，誓言防止同类事件再次发生

**组织结构**

- 多种族魔法使用者的学术-军事同盟
- 七座魔法高塔通过**奥术共鸣网络**连接，共享全局法力池
- 法力池是白塔同盟的核心——所有高塔的魔法能量统一调配

**战略定位**

- 不追求领土扩张，核心目标是构建"**魔法防御圈**"
- 防御圈内的区域受白塔保护，攻击这些区域会遭到魔法反击
- 进攻型力量极为强大（大法师的毁灭魔法）

**致命弱点**

- **法力耗尽后几乎无战斗力**: 白塔的所有战斗能力依赖法力池
- 如果能切断奥术共鸣网络或持续消耗法力池，白塔将不堪一击
- 长时间的消耗战是白塔的噩梦
