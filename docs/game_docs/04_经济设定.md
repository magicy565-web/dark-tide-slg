## 六、兵种训练科技树（v0.8.4 重构）

> 科技研究需建造**学院**类建筑。三层结构: **基础训练**(属性) → **进阶训练**(被动) → **终极训练**(主动技能)。

### 研究建筑（3种×3级）

| 建筑 | Lv1费用 | Lv1效果 | Lv2效果 | Lv3效果 |
|------|---------|---------|---------|---------|
| 学院 | 200g+20i | 解锁研究, 速度+25% | 速度+50%, 可排队2项 | 速度+100%, 排队3项, 费用-10% |
| 兵法研究所 | 180g+25i | 前卫/射击/机动训练速度+15% | +30%, 战术模拟 | +50%, 精锐训练 |
| 魔导研究院 | 200g+15i+1晶 | 法术训练速度+15%, 术师ATK+2 | +30%, ATK+4, 魔晶+20% | +50%, ATK+6, 法力再生+2 |

### 研究机制

| 项目 | 规则 |
|------|------|
| 基础速度 | 1进度/回合 |
| 学院加速 | Lv1 +0.25 / Lv2 +0.50 / Lv3 +1.0 |
| 研究队列 | 默认1项, 学院Lv2→2项, Lv3→3项 |
| 取消退款 | 退还50%已投入资源 |
| 无学院惩罚 | 研究暂停(不丢失进度) |

### 训练科技树总览

```
每分支结构:
  T1 基础训练(属性提升) ──┬── T2 进阶训练A(兵种A被动) ── T3 终极训练A(兵种A主动)
                         └── T2 进阶训练B(兵种B被动) ── T3 终极训练B(兵种B主动)
```
