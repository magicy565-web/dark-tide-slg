# 10_AI行为文档 —— 《暗潮 Dark Tide》

### 一、AI总体设计
- 光明方3阵营(人类/精灵/法师)各自独立AI，但可协同
- AI行为由威胁值驱动，分4个等级
- AI遵循与玩家相同的回合结构（收入→维护→策略→战斗→结算）

### 二、威胁等级与行为

#### 等级1: 独立行动（威胁0-29）
- 防守自身据点，不主动进攻
- 每回合恢复城防
- 边境据点增援驻军(每回合+1兵，上限为驻军上限)
- 不与其他光明阵营协作

#### 等级2: 防御联盟（威胁30-59）
- 被攻击的阵营可召唤1个相邻光明阵营增援
- 边境据点城防额外+2/回合
- 联合巡逻事件概率增加（触发女骑士巡逻等）
- 开始对玩家据点进行侦查

#### 等级3: 军事联盟（威胁60-79）
- 2-3个光明阵营协同行动
- 主动派出远征军攻击玩家最弱边境据点
- 攻击逻辑: 选最强己方边境→找最弱相邻玩家节点→己方兵力>敌方则出击
- 联军混编部队出现（人类骑士+精灵游侠）
- 英雄（凛）率联军出击

#### 等级4: 殊死抵抗（威胁80-100）
- 全部光明阵营统一行动
- 所有单位ATK/DEF+10%全局buff
- 圣战号召事件触发（大规模反攻）
- 全力防守剩余据点，不计代价增援
- Boss级战斗难度提升

### 三、各阵营AI特性

#### 人类王国AI
- 防守优先: 优先加固城防，被攻击后请求增援
- 城防恢复+3额外(双倍)
- 仅威胁60+时主动出击
- 圣殿女卫不移动，固守王都

#### 高等精灵AI
- 远程骚扰: 法师持续消耗相邻玩家驻军(-2/回合)
- 屏障依赖: 据点间灵脉网络互相强化
- 不主动出击，用法术远程消耗
- 威胁60+时与人类联合远征

#### 法师公会AI
- 龟缩施法: 优先积攒法力
- 传送集中防守: 用传送法术把兵力集中到被攻击的据点
- 法力弹幕: 消耗法力对相邻玩家据点远程伤害(-4驻军)
- 防守极其顽强，但不主动进攻
- 威胁60+时用法力弹幕密集轰炸前线

### 四、AI决策流程（每回合）
```
1. 评估威胁等级
2. 收入/维护（同玩家规则）
3. 防御检查: 是否有据点被攻击/即将被攻击？
   → 是: 调集增援、加固城防
4. 攻击判断（仅等级3+）:
   a. 扫描所有边境己方据点
   b. 找相邻最弱玩家据点
   c. 己方兵力 > 敌方兵力 × 1.5 → 发动攻击
   d. 否则继续积蓄
5. 招募补充: 在内部据点补充驻军
6. 特殊行为: 根据阵营特性执行（法师积攒法力, 精灵远程消耗等）
```

### 五、难度调节
| 难度 | AI招募速度 | AI攻击阈值 | AI全局buff |
|------|-----------|-----------|-----------|
| 简单 | ×0.7 | 己方>敌×2.0 | 无 |
| 普通 | ×1.0 | 己方>敌×1.5 | 无 |
| 困难 | ×1.3 | 己方>敌×1.2 | ATK/DEF+2 |
| 噩梦 | ×1.5 | 己方>敌×1.0 | ATK/DEF+3, 产出+30% |
