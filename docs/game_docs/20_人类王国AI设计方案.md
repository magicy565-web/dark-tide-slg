# 人类王国 AI 设计方案 — 天城王朝

**版本：** v1.0（2026年4月）
**依赖：** `10_AI行为.md`、`01_背景故事.md`、`02_角色设定.md`、`07_全局事件.md`

---

## 一、设计定位

人类王国（天城王朝）是大陆上规模最大的政治实体，其 AI 行为应体现以下核心特质：

- **防守反击型**：不轻易主动出击，但被攻击后会以超出预期的力度反击
- **官僚迟缓**：贵族内部腐化导致动员缓慢，威胁等级低时反应迟钝
- **城防依赖**：三重城墙是王国的核心优势，AI 会优先维护城防而非扩张
- **英雄驱动**：凛（圣殿骑士团长）、冰华（铁壁女伯爵）、红叶（银甲公爵夫人）各自有独立的 AI 触发条件

## 二、行为树结构

```
HumanKingdomAI.tick()
├── Phase 0: 评估威胁等级 & 更新内部状态
│   ├── 更新 _mobilization_level（动员度 0-100）
│   └── 检查英雄出击条件
├── Phase 1: 防御检查（每回合必执行）
│   ├── 城墙修缮（受损城墙优先修复）
│   ├── 边境增援（内部守军向最弱边境输送）
│   └── 圣殿骑士团驻守王都（不移动）
├── Phase 2: 外交行动（威胁 30+ 时激活）
│   ├── 向精灵族发出结盟请求
│   ├── 向法师公会输送铁矿（军火交易）
│   └── 对玩家发出外交警告
├── Phase 3: 军事行动（威胁 60+ 时激活）
│   ├── 凛率圣殿骑士联军出击
│   ├── 冰华守卫北方边境（不离开银冠要塞区域）
│   └── 红叶提供骑兵支援（威胁 40+ 触发）
└── Phase 4: 殊死抵抗（威胁 80+ 时激活）
    ├── 圣战号召事件（全军 ATK/DEF+10%）
    ├── 女王千姬发表演讲（全军士气满）
    └── 所有贵族领主被强制动员
```

## 三、动员度系统（Mobilization）

人类王国的 AI 特色机制：**动员度（0-100）**。

| 动员度区间 | 状态 | 行为变化 |
|-----------|------|---------|
| 0-29 | 和平时期 | 只修城墙，不增兵，不外交 |
| 30-59 | 戒备状态 | 开始边境增援，发出外交警告 |
| 60-79 | 战争动员 | 凛出击，贵族领主开始响应 |
| 80-100 | 全面战争 | 圣战号召，女王亲自下令 |

动员度每回合根据威胁值自动调整：`mobilization = clamp(threat * 1.1 + turns_under_attack * 5, 0, 100)`

## 四、英雄行为规则

| 英雄 | 触发条件 | AI 行为 | 备注 |
|------|---------|---------|------|
| **凛** | 威胁 ≥ 60 | 率圣殿骑士联军主动出击玩家最弱边境据点 | 被击败50%俘获 |
| **冰华** | 银冠要塞相邻据点被攻击 | 守卫银冠要塞，不主动出击 | 攻陷后自动俘获 |
| **红叶** | 威胁 ≥ 40 且玩家威望 ≥ 30 | 主动接触玩家，提出合作条件（外交事件） | 可和平招募 |
| **雪乃** | 凛被俘后 | 追随凛，自动加入玩家 | 无战斗行为 |

## 五、特色事件链（人类王国专属）

### 事件链 A：贵族内乱（威胁 0-59 时随机触发）
```
回合 N:   [贵族密谋] — 某领主拒绝出兵 → 玩家可利用（煽动叛乱）或无视
回合 N+2: [女王镇压] — 千姬派凛镇压叛乱 → 凛暂时无法出击（1-2回合）
回合 N+4: [秩序恢复] — 王国内部稳定，动员度+10
```

### 事件链 B：圣殿骑士巡逻（威胁 30+ 每3回合触发）
```
触发: [女骑士巡逻] — 凛率小队巡逻边境
选项A: 迎战 → 战斗（敌8兵，凛参战）
选项B: 撤退 → 弃边境据点，威胁-2
```

### 事件链 C：红叶的合作提案（威胁 40+，威望 ≥ 30，一次性）
```
触发: [银甲公爵夫人来访] — 红叶主动接触
选项A: 接受合作 → 红叶加入（外交招募），王国动员度-15（内部分裂）
选项B: 拒绝 → 红叶返回，王国动员度+5（团结一致）
```

### 事件链 D：圣战号召（威胁 ≥ 80，一次性）
```
触发: [圣战号召] — 女王千姬发表演讲
效果: 全体人类单位 ATK/DEF+10%，持续5回合
选项A: 全力迎战 → 大型战斗（敌20兵，凛+冰华双英雄）
选项B: 战略撤退 → 弃2据点，威胁-10
```

### 事件链 E：冰华的最后防线（银冠要塞被围攻时触发）
```
触发: 银冠要塞相邻3格均被玩家占领
效果: 冰华宣布"死守令"，城防×2，驻军+5
选项: 强攻（正常战斗）或 围困（每回合-3驻军，3回合后自动陷落）
```

## 六、与其他光明阵营的协同

| 威胁等级 | 协同行为 |
|---------|---------|
| 威胁 30+ | 人类可请求精灵族增援（1次/3回合），精灵在相邻据点+3驻军 |
| 威胁 40+ | 人类向法师公会输送铁矿（AI军火交易系统） |
| 威胁 60+ | 人类+精灵联军出击（凛+精灵游侠混编） |
| 威胁 80+ | 三方全面协同，共享情报（玩家据点信息对所有光明方可见） |
