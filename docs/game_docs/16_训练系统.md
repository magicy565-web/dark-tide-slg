# 16. 训练系统

> 暗潮 Dark Tide — 训练与AI威胁阶梯设计文档

---

## 1 设计理念

### 1.1 核心原则

| # | 原则 | 说明 |
|---|------|------|
| 1 | **玩家专属** | 训练树仅服务于玩家所选的恶势力（兽人/海盗/暗精灵），不存在跨势力共享科技 |
| 2 | **AI无科技树** | 其余5个AI势力（含2个友方恶势力和3个光明势力）不使用训练系统，改由**威胁阶梯**驱动成长 |
| 3 | **势力差异化** | 三条训练树在可训练兵种、被动效果、终极技上完全不同，鼓励不同玩法风格 |
| 4 | **资源博弈** | 训练消耗金币与回合数，玩家需在"扩军"与"升级"之间做出取舍 |

### 1.2 兵种总表与势力分配

共8个基础兵种，各势力只能训练其中4个：

| 兵种 | 类型 | 兽人 | 海盗 | 暗精灵 |
|------|------|:----:|:----:|:------:|
| 足軽 (Ashigaru) | 前卫·轻步 | **可** | - | - |
| 武士 (Samurai) | 前卫·重步 | **可** | - | - |
| 弓兵 (Archer) | 后排·远程 | - | **可** | - |
| 砲兵 (Cannon) | 后排·攻城 | - | **可** | - |
| 騎兵 (Cavalry) | 机动·突击 | - | **可** | - |
| 忍者 (Ninja) | 机动·暗杀 | - | **可** | **可** |
| 祭司 (Priest) | 辅助·治疗 | **可** | - | **可** |
| 術師 (Sorcerer) | 后排·法术 | **可** | - | **可** |

> 兽人: 足軽/武士/祭司/術師 — 前排压制 + 后排辅助
> 海盗: 弓兵/砲兵/騎兵/忍者 — 火力输出 + 高机动
> 暗精灵: 忍者/祭司/術師 + 专属单位"暗影行者" — 魔法 + 隐匿

## 2 玩家势力训练树

训练树统一分为 **T1 基础 → T2 进阶 → T3 终极** 三层。每层需消耗金币并经过若干回合研究。

**设计主题：** 以暴力碾压为核心，强化前排输出与全军士气。

| T1 | O-1b | 重甲锻造 | 武士DEF+15% | 200 | 1 | — |
| T1 | O-1c | 战吼鼓舞 | 全军士气上限+10 | 250 | 1 | — |
| T1 | O-1d | 祭祀祈福 | 祭司治疗量+20% | 200 | 1 | — |
| T2 | O-2a | 狂暴突击 | 足軽/武士获得"狂暴"被动：HP<50%时ATK+25% | 400 | 2 | O-1a, O-1b |
| T2 | O-2b | 战场嚎叫 | 被动：每回合开始，敌方前排士气-5 | 350 | 2 | O-1c |

## 3 AI威胁阶梯系统

### 3.1 设计意图

AI势力不使用训练树，其战斗力通过**威胁值(Threat Value)**动态调节。威胁值反映玩家对该AI势力的压力程度，范围0-100。

### 3.2 威胁值计算

| 事件 | 威胁值变动 | 说明 |
|------|:----------:|------|
| 玩家占领AI城池 | +15 | 每座城池 |
| 玩家击败AI军团 | +10 | 每次战斗 |
| 玩家与该AI接壤 | +2/回合 | 持续施压 |
| AI夺回城池 | -10 | 反攻成功 |
| 回合自然衰减 | -1/回合 | 缓慢回落 |
| 玩家与AI签订协议 | -20 | 外交手段 |

### 3.3 威胁阶梯数据表

| 阶梯 | 威胁值区间 | 名称 | ATK加成 | DEF加成 | HP加成 | 特殊效果 |
|:----:|:----------:|------|:-------:|:-------:|:------:|----------|
| Tier 0 | 0 — 29 | 常态 | +0% | +0% | +0% | 无额外效果，AI使用基础数值 |
| Tier 1 | 30 — 59 | 防御态势 | +10% | +15% | +0% | 驻军每回合回复+1单位；防御建筑耐久+20% |
| Tier 2 | 60 — 79 | 军事动员 | +20% | +25% | +0% | 解锁T2被动技能；每3回合生成1支远征军 |
| Tier 3 | 80 — 100 | 绝境反击 | +35% | +40% | +20% | 解锁T3终极技；每5回合生成1个Boss级单位 |

### 3.4 AI Tier被动技能与终极技

各光明势力在达到对应Tier时自动解锁以下能力：

**光明势力 — Tier 2 被动**

| 势力 | 被动名称 | 效果 |
|------|----------|------|
| 人类联邦 | 坚守意志 | 防御战时DEF额外+10%，士气不降至0以下 |
| 精灵王庭 | 自然恩赐 | 每回合回复全军5%HP |
| 法师公会 | 魔力涌流 | 法术型单位ATK额外+15% |

**光明势力 — Tier 3 终极技**

| 势力 | 终极技名称 | 效果 |
|------|------------|------|
| 人类联邦 | 王之号令 | 全军ATK/DEF+20%(3回合)+召唤1支精锐近卫军 |
| 精灵王庭 | 古树觉醒 | 召唤Boss"远古树人"(HP300/ATK60/DEF50)+全军回复30%HP |
| 法师公会 | 禁咒·天罚 | 对敌方全体造成80点法术伤害+沉默2回合 |

**恶势力AI（非玩家方）— Tier 2/3**

| 势力 | Tier 2 被动 | Tier 3 终极技 |
|------|-------------|---------------|
| 兽人(AI) | 战意高涨：ATK+10%，士气不降 | WAAAGH!怒吼(同玩家版本) |
| 海盗(AI) | 劫掠之风：战后AI获得额外资源 | 黑旗恐惧(同玩家版本) |
| 暗精灵(AI) | 情报网络：AI获得全图视野 | 暗影支配(同玩家版本) |

> 注意：AI恶势力使用的T2/T3技能效果与玩家版本相同，但触发方式不同——AI版本在达到对应威胁阶梯时自动解锁，无需研究。

### 3.5 远征军与Boss生成规则

| 生成类型 | 触发条件 | 内容 | 行为 |
|----------|----------|------|------|
| 远征军 | Tier 2+，每3回合 | 3-5个中等单位组成的军团 | 主动进攻最近的玩家城池 |
| Boss单位 | Tier 3+，每5回合 | 1个Boss级单位(HP×3, ATK×2) | 随机进攻或驻守关键城池 |

## 4 数据接口

### 4.1 与其他模块关联

| 模块 | 接口方向 | 数据内容 |
|------|:--------:|----------|
| 战斗系统 | 训练→战斗 | 兵种加成(ATK/DEF/HP%)、被动技能列表、主动技能状态 |
| 战斗系统 | AI阶梯→战斗 | AI全局加成(ATK/DEF/HP%)、已解锁的被动/终极技 |
| 经济系统 | 经济→训练 | 可用金币余额，扣款请求 |
| 回合管理 | 回合→训练 | 回合推进事件，触发研究进度+1 |
| 回合管理 | 回合→AI阶梯 | 回合推进事件，触发威胁值衰减与远征军/Boss生成检查 |
| 外交系统 | 外交→AI阶梯 | 协议签订事件，触发威胁值变动 |
| 地图系统 | 地图→AI阶梯 | 城池占领/丢失事件，触发威胁值变动 |
| 势力选择 | 开局→训练 | 玩家所选势力ID，决定加载哪棵训练树 |

### 4.2 关键数据结构

```
TrainingTree {
  faction_id: enum(ORC, PIRATE, DARK_ELF)
  nodes: [
    {
      id: string          // e.g. "O-1a"
      tier: int           // 1, 2, 3
      name: string
      effect: Effect
      cost_gold: int
      cost_turns: int
      prerequisites: [string]
      status: enum(LOCKED, RESEARCHING, COMPLETED)
      progress: int       // 已研究回合数
    }
  ]
}

AIThreatState {
  faction_id: string
  threat_value: int       // 0-100
  current_tier: int       // 0-3
  unlocked_passives: [string]
  unlocked_ultimates: [string]
  expedition_cooldown: int
  boss_cooldown: int
}
```

## 5 实现要点

### 5.1 训练树

1. **开局初始化：** 根据玩家选择的势力，加载对应的训练树数据（仅1棵），其余2棵不实例化。
2. **研究推进：** 每回合结束时，遍历所有状态为 `RESEARCHING` 的节点，`progress += 1`；若 `progress >= cost_turns`，状态转为 `COMPLETED`，立即应用效果。
3. **同时研究限制：** 同一时间只允许研究 **1个** 节点，避免玩家过快成长。
4. **T3前置检查：** T3节点需要至少2个T2节点完成（具体前置见上方各势力表格）。
5. **效果叠加规则：** 同类百分比加成采用**加算**（如ATK+15%和ATK+25%合计ATK+40%），不做乘算。

### 5.2 AI威胁阶梯

1. **威胁值更新时机：** 每个触发事件发生时立即更新，不等回合结束。
2. **阶梯跳变：** 威胁值可能在单回合内跨越多个阶梯（如从25直接到65），此时应按最终值对应的阶梯生效。
3. **阶梯回落：** 威胁值下降时，对应阶梯的buff立即移除，但已生成的远征军/Boss不会消失。
4. **独立追踪：** 每个AI势力拥有独立的威胁值，玩家对兽人的威胁不影响精灵的威胁阶梯。
5. **Boss平衡：** Tier 3 Boss单位应有明确的弱点机制（如兽人Boss弱法术、精灵Boss弱火攻），避免无解局面。

### 5.3 UI呈现建议

| 界面 | 内容 |
|------|------|
| 训练树面板 | 树形结构展示已解锁/研究中/锁定节点，点击可查看详情并启动研究 |
| 势力情报面板 | 显示各AI势力当前威胁值(进度条)及当前阶梯名称，鼠标悬停显示加成详情 |
| 战斗准备界面 | 高亮显示己方已激活的训练加成，以及敌方AI的阶梯buff提示 |
