# 05_行动设定

> 《暗潮 Dark Tide》行动与战斗规则定义

---

## 一、回合结构（8阶段）

```
Phase 1: 收入 — 收集所有己方据点资源（应用阵营/秩序修正）
Phase 2: 维护 — 扣除粮食(1兵=0.1食)、维护费
Phase 3: 事件 — 触发剧情/随机事件、英雄互动
Phase 4: 策略 — 建造/招募/升级/部署/移动军队/外交
Phase 5: 战斗 — 解算所有战斗（攻方进入防守节点）
Phase 6: 结算 — 占领节点、俘获英雄、更新秩序/威胁
Phase 7: AI回合 — 敌方阵营执行同样阶段
Phase 8: 世界更新 — 衰减威胁、恢复城防、tick机制值
```

---

## 二、战斗系统（战国兰斯式）

### 布阵

```
前排: [槽位1] [槽位2] [槽位3]
后排: [槽位4] [槽位5]
每侧最多5个部队, 每个部队 = 1指挥官 + 兵种 + 兵数
```

### 伤害公式

```
base_damage = 兵数 × max(1, 攻击方ATK - 防御方DEF) / 10
final_damage = base_damage × 技能倍率 × 地形修正
soldiers_killed = floor(final_damage)  # 最少1
```

ATK = 指挥官ATK + 兵种基础ATK + 地形加成
DEF = 指挥官DEF + 兵种基础DEF + 地形加成

### 行动队列

- 按SPD降序排序，相同随机
- 每单位每回合行动1次（extra_action被动+1）
- 战斗持续最多12回合
- 12回合后: 防守方胜

### 地形修正

| 地形 | ATK修正 | DEF修正 | 特殊 |
|------|---------|---------|------|
| 平原 | ×1.0 | ×1.0 | 骑兵ATK+1 |
| 森林 | ×0.9 | ×1.1 | 弓兵ATK+1, 骑兵ATK-1 |
| 山地 | ×0.8 | ×1.2 | 骑兵不可进入(冷蜥除外) |
| 沼泽 | ×0.9 | ×0.9 | 全体SPD-2 |
| 城墙 | ×0.7 | ×1.5 | 需先破城防 |

### 攻城战

```
城防: 每据点固有(村5/镇10/要塞20)
攻城伤害: ATK × 0.5/次攻击, 砲兵×2
城防每回合恢复: +3(所有者回合)
城防<=0时才进入正式战斗
```

### 目标选择

- 前排近战→敌前排优先
- 后排远程→可攻任意行
- 忍者→可绕过前排攻后排
- 嘲讽(taunt)→强制攻击该单位
- 前排清空→后排可被近战攻击

|--------|------|
| regen_1 | 每回合恢复1兵 |
| charge_1_5 | 首次攻击×1.5 |
| escape_30 | 30%概率逃避致死 |
| preemptive | 先于行动队列攻击 |
| preemptive_1_3 | 先制攻击×1.3 |
| siege_x2 | 城防削减×2 |
| extra_action | 每回合行动+1 |
| assassinate_back | 可攻击后排 |
| ignore_terrain | 无视地形减益 |
| fort_def_3 | 防守己方据点DEF+3 |
| counter_1_2 | 被攻击时反击×1.2 |
| immobile | 不可在节点间移动 |
| taunt | 嘲讽（强制被攻击） |
| aoe_mana | AoE攻击，消耗5法力 |
| charge_mana_1 | 每回合+1法力 |
| aoe_1_5_cost5 | AoE×1.5，消耗5法力 |
| death_burst | 死亡时对敌全体ATK×2伤害 |

## 四、法力系统

- **法力池**: 每军队独立, 初始0, 上限=10+(最高INT单位的INT×2)
- **恢复**: +1/回合 + 学徒法师被动 + 月华被动
- **消耗**: 按技能标注
- **溢出**: 超过上限部分丢失

---

## 五、移动与补给

- **移动**: 每军队每回合1节点
- **强行军**: 2节点, 兵数-10%
- **侦查**: 2节点范围(忍者扩展至3)
- **补给线**: 至己方节点路径>3跳→-2兵/回合
